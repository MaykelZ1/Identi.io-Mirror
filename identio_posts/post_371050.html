MediaInfo V 0 7 73 PC Español Mega 






MediaInfo le proporciona un acceso rápido a los valores internos de audio y archivos de vídeo. Tanto si quieres echar un vistazo a la puesta a punto de los temas incluidos en un archivo o quieres conocer todos los detalles precisos acerca de los códecs utilizados, MediaInfo ofrecerá el punto de vista adecuado.
Se puede elegir entre una interfaz gráfica de usuario para seleccionar rápidamente la vista adecuada de los datos y una interfaz de línea de comandos para recuperar automáticamente la información y medios de proceso en sus propios scripts. A estos efectos, vistas MediaInfo se pueden personalizar de manera fácil.

Características MediaInfo:

Diferentes y personalizables puntos de vista sobre los medios de información (texto, Mesa, estructura de árbol, HTML, XML ...)
Exportar datos como texto, CSV, HTML, XML ...
Interfaz gráfica de usuario, Command Line Interface, DLL
Integración en MS Windows (arrastrar y soltar, menú contextual)
Localización

Información Categorías:

Conceptos básicos:. Título, Autor, director, álbum, título Nr, fecha, longitud ...
Vídeo: Codec, relación de aspecto, tasa de fotogramas, velocidad de bits ...
Audio: Codec, frecuencia de muestreo, canales, Idiomas, Bit Rate ...
Texto: Lenguaje de Subtítulos
Capítulos: Nº. de los capítulos, Contorno

Formatos y Contenedores compatibles:

Vídeo: MKV, OGM, AVI, DivX, WMV, QuickTime, Real, MPEG-1, MPEG-2, MPEG-4, DVD (VOB) ... (Codecs: DivX, XviD, msmpeg4, ASP, H.264, AVC ...)
Audio: OGG, MP3, WAV, RA, AC3, DTS, AAC, M4A, AU, AIFF ...
Untertitel: SRT, SSA, ASS, SAMI.

Título: MediaInfo 0.7.73
Nombre de fichero: MediaInfo_GUI_0.7.73_Windows.exe
Tamaño de fichero: 4.52MB 
Requisitos: Windows (Todas las versiones)
Idiomas: Múltiples idiomas
Licencia: Open Source
Fecha añadida: 11 de abril de 2015
