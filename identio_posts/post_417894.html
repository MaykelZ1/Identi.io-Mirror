Windows 7 no se actualiza? te muestro la solucion

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 9 años
Que tal queridos amigos, aquí les traigo una solución muy practica al problema que presenta Windows 7 cuando Windows update no hace nada respecto a su misión primaria actualizar el sistema operativo con actualizaciones de seguridad, controladores y otras. Un colega y amigo me pidio que le ayude con este inconveniente (se quedaba eternamente buscando y no bajaba ni instalaba nada). Lo comparto con ustedes...se que a mas de uno le va a servir.


Les dejo el archivo para reparar los problemas (probado funciona 100%) el mismo se llama reparar.bat, lo que deben hacer es clic derecho sobre el e instalarlo como administrador. Luego reinicia el equipo.




DESCARGA POR MEGA





Este es el contenido del archivo, pueden hacerlo ustedes mismos pegando el contenido en el bloc de notas y guardarlo como reparar.bat. Ejecutar como administrador, reiniciar el equipo y listo. Al hacer clic en windows update veran que actualiza sin problemas. Espero les sirva. Saludos cordiales amigos.








net stop bits 
 
net stop wuauserv 
 
net stop appidsvc 
 
net stop cryptsvc 
 
Ren %systemroot%SoftwareDistributionSoftwareDistribution.old6 
 
Ren %systemroot%system32catroot2 catroot2.old6 
 
Del "%ALLUSERSPROFILE%Application DataMicrosoftNetworkDownloaderqmgr*.dat" 
 
net start bits 
 
Net start wuauserv 
 
Net start appidsvc 
 
Net start cryptsvc 
 
wuauclt.exe /updatenow 
 
Ipconfig /flushdns 
 
Ipconfig /release 
 
Ipconfig /renew 
 
netshintipreset reset.log 
 
netshwinsockreset 
 
Netshintipv4 reset 
 
Netshintipv6 reset 
 
ping localhost -n 40 >nul 
 
exit
