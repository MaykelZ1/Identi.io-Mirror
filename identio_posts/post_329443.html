Crea Tu Juego Fps Creator [Tutorial]

  • Categoría: Software / Programas
  • Publicado hace más de 11 años







FPS Creator es un potente editor para crear juegos 3D, concretamente, para juegos de acción en primera persona. Además, todo se hace de forma muy visual, con lo que no tendrás que estudiar complejos manuales de programación.
FPS Creator pone a tu disposición desde habitaciones ya montadas, hasta todo tipo de texturas y pasillos para que puedas crear un completo nivel. Si quieres un mayor detalle, puedes poner muebles, ventanas o cuadros.




Windows XP / Windows Vista / Windows 7 / Windows 8

Pentium 4 2.0Ghz 

1Gb De Ram o Mas (Recomendado 2gb)

Placa Grafica Integrada 256Mb o Mas

5Gb Espacio En El Disco Duro
