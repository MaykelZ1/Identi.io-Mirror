Fundamentos de HTML

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 10 años
Conoce las bases de la web

Conocer HTML es imprescindible para ser programador: el formato final de la mayor parte de las aplicaciones que se crean, de los contenidos que se muestran o de las interfaces que se usan en nuestra sociedad es HTML. Y este curso está pensado para enseñarte todo lo que necesitas sobre HTML. Comenzando desde lo más básico aprenderás cómo se programa HTML, las etiquetas, las técnicas y trucos para programar contenidos HTML, enviar datos, trabajar con información, mostrar imágenes y contenidos multimedia y a sacar partido a HTML5 y los contenidos para dispositivos. Seguiremos una serie de buenas prácticas, estructuras de uso habitual y consejos de programadores expertos para al acabar poder crear contenidos HTML de calidad y de nivel profesional. 


HTML es hoy en día el lenguaje de programación más utilizado por ser el vehículo de transferencia de información en internet. Es por eso que independientemente del lenguaje de programación que conozcamos (Java, PHP, etc.), del medio para el que se vaya a mostrar contenidos (tablets, smart TVs, teléfonos,?), que seamos ingenieros, programadores o diseñadores, al final acabaremos necesitando de HTML para mostrar contenidos.
Y para aprender HTML no hay mejor manera que este curso, creado por un experto profesional con años de experiencia en desarrollo y programación de contenidos para internet y pensado para sirva de tanto de referencia HTML, como de guía de buenas prácticas y de repositorio de trucos para programar contenidos web.
A lo largo del curso se van a ir repasando las distintas etiquetas HTML agrupándolas por tipos de contenidos y usando ejemplos prácticos habituales, para que además de aprender, este curso sirva más adelante como guía de referencia en el momento de maquetar nuestros contenidos.
Vamos también a ver herramientas aconsejadas para la programación, tecnologías involucradas en la creación de HTML y recursos que te ayudarán en tu trabajo.
