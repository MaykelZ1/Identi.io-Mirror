Teclas FN en Laptop Samsung Windows 8.1 (Solución)

  • Categoría: Off-Topic
  • Publicado hace más de 11 años
Solución definitiva! , No te funcionan las teclas FN en Notebooks Samsung con Windows 8/8.1 ? 

Tuve este problema al igual que muchos, busque en TODAS partes y no encontraba ninguna solución verdadera, lo único que me funcionaba era ?resolver? el problema con el software de ?Support Center?, realmente no lo resolvía, puesto que cada vez que se reiniciaba el sistema tenía que volver a aplicarse para funcionar, investigué por mucho tiempo y vi muchos trucos que sugerían instalar software obsoleto como ?Easy Display Manager? , ?Easy Settings? configurándolos con compatibilidad a Windows 7, mas esa no era una verdadera solución, es decir no era una solución DE CALIDAD y mostraba unos molestos gráficos en la pantalla cada vez que se utilizaban las teclas FN, pensé, si el software ?Support Center? lo soluciona temporalmente es porque debe de reiniciar un servicio, proceso, etc, ahora solo tenía que descubrir que era lo que ese software hacía realmente, para yo resolverlo de manera permanente, así que decidí buscar alguna solución efectiva por mi propia cuenta. 

Gracias a Dios no soy una persona conformista, y gracias a eso, hoy estoy contento porque resolví mi problema, y estoy seguro que también se los solucionaré a MUCHOS de ustedes!


Descubrí que la causa del problema es que la tarea programada para que la aplicación ?Settings? inicie de forma automática cada vez que el sistema inicia queda bloqueada, lo que hace que el software no inicie. 



Ahora les daré la solución, aquí vamos.. 

Lo primero que necesitaremos es ya tener instalado el software ?Settings?, en caso de no tenerlo descarguen ?SW Update? de este link: 

http://orcaservice.samsungmobile.com/SWUpdate.aspx 

Y posteriormente instalen ?Settings?. 

Una vez lo tengan continuen con lo siguiente: 


1. Presionen la tecla Windows + S para iniciar una búsqueda, escriban ?regedit? y ábranlo 

(Por seguridad siempre se recomienda hacer una copia de seguridad del registro, ya que si modifican alguna otra cosa pueden dañar su sistema, para hacer una copia de seguridad, seleccionen Archivo > Exportar , seleccionen una ubicación para el respaldo y guardan). 

2. Naveguen a la siguiente ruta: 

HKEY_LOCAL_MACHINE > SOFTWARE > Microsoft > Windows NT > CurrentVersion > Tracing > SCM > Regular 

3. Agreguen/Modifiquen el siguiente valor 

Nombre = TracingDisabled 
Tipo = REG_DWORD 
Valor = 0 

4. Reinicien el sistema 

5. Listo! , así de facil ! 

El servicio de ?Settings? ya debería de cargar automáticamente, y tendrán funcionando sus teclas FN, espero les haya servido a muchos de ustedes, Saludos!


(Nota: Yo mismo eh publicado esta información en otro sitio, pero lo comparto aqui también para que esté al alcance de más personas y puedan así, resolver su problema sin mas batalla)
