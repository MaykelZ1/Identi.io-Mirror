Instalación de Photoshop CS5 en Ubuntu(similares).

  • Categoría: Hazlo tu mismo
  • Publicado hace más de 10 años

Bienvenidos a todos y todas.


Deja cargar el post

En este post mostraré como instalar Adobe Photoshop CS5.


PRIMER PASO

Instalaremos Wine.

Lo podemos instalar desde el Centro de Software de nuestro equipo de vía terminal.


Repocitorios

Cita: Mostrar
sudo add-apt-repository ppa:ubuntu-wine/ppa

Actualización


Cita: Mostrar
sudo apt-get update
Instalación de Wine

Cita: Mostrar
sudo apt-get install wine1.6 winetricks
o

Cita: Mostrar
sudo apt-get install wine
Y listo.



SEGUNDO PASO

Descargar & instalar 

Photoshop CS5.



Lo descargamos:

Clic acá.

contraseña:

 erckrivas



Una vez descargado podemos notar que es un archivo .exe ahora lo que debemos hacer es darle clic derecho sobre el y nos aparecerá una opción que seria ejecutar con Wine(cargador de archivos de Windows).


Ahí nos parecerá un archivo de extracción lo que debemos hacer es colocar la contraseña "erckrivas"(sin comillas) y empezara la extracción del programa.




Empezara a extraerse el archivo deseado.







Después nos mostrará un archivo que dice "Photoshop CSS5 (Full Español)" debemos abrirlo igual que al otro archivo.







Nos aparecerá una ventana con el asistente de instalación de Photoshop CS5. Debemos instalarlo como a cualquier otro programa.






Una vez hecho todo eso tendremos el programa deseado.






Pueden notar que la interfaz del programa ha cambiado un poco, pero las herramientas no.



DATO: Recomiendo no borrar los accesos del escritorio por que después les costara mucho más ejecutarlo.
