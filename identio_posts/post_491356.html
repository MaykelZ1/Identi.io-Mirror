Windows MiniOS v2017.11



¿Que es Windows MiniOS?


Windows MiniOs no es más que una de las tantas versiones Lites (pequeñas) de Windows que existen, esta versión es creada por Daniel Rodriguez Montano, un programador cubano que al necesitar un sistema operativa que se adaptara a lo que el necesitara se le ocurrio crear el MiniOS.

Existen 6 versiones de Windows MiniOs:

Windows XP (x86)
Windows Server SP2 + R2 (x86)
Windows 7 (x86 - x64)
Windows 10 (x86 - x64)
Windows Server 2016 (x64)




¿Es compatible con Software, como Office y otros mas?


Si Windows MiniOS, es un sistema que busca: Ser liviano, adaptarse a cualquier usuario, y ser compatible con todo el software con el que una version de Windows normal lo seria.

En esta edicion, el mayor nucleo o lo mas esperado es el nuevo Windows 10 MiniOS, Un cambio total al sistema que conocemos. Daniel ha hecho un trabajo increible. En cuanto al Windows XP y el 7 Han recibido Optimizacion y algunas mejoras menores como impresión, y varios problemas de compatiblidad han sido resueltos, además de que a partir de esta versión ya viene preactivado.


Requisitos Minimos


Windows XP y Server MiniOS

Son versiones lites, aun así, recomiendo exactamente los mismos requisitos que Microsoft para las versiones originales de los sistemas, aunque correrían bien en equipos de menos recursos.

*Procesador Pentium a 233 megahercios (MHz) o mayor velocidad (se recomienda 300 MHz)
*Al menos 64 megabytes (MB) de RAM (se recomienda 128 MB)
*Un mínimo de 1,5 gigabytes (GB) de espacio disponible en el disco duro
*Unidad de CD-ROM o DVD-ROM
*Un teclado y un mouse de Microsoft, o algún otro dispositivo señalador compatible
*Adaptador de vídeo y monitor con una resolución Super VGA (800 x 600) o mayor
*Tarjeta de sonido
*Altavoces o auriculares


Windows 7 MiniOS

*Procesador de 1 gigahercio (GHz) o más rápido de 32 bits (x86) o de 64 bits (x64)*
*512 MB de RAM (32 bits) o 1 GB de RAM (64 bits)
*4 GB de espacio disponible en el disco duro (32 bits) o 7 GB (64 bits)
*Tarjeta gráfica DirectX 9 con controlador WDDM 1.0 o superior


Windows 10 MiniOS

* Procesador de 1 gigahercio (GHz)* o más rápido compatible con PAE, NX y SSE2
*1 gigabyte (GB) (32 bits) o 2 GB (64 bits)
*Espacio en disco duro. 5 GB (32 bits) o 8 GB (64 bits)
*Tarjeta gráfica. Dispositivo gráfico Microsoft DirectX 9 con controlador WDDM

Preguntas Frecuentes:

– Que requisitos mínimos o versión debo instalar en mi equipo?

Son versiones lites, aun así, recomiendo exactamente los mismos requisitos que Microsoft para las versiones originales de los sistemas, aunque correrían bien en equipos de menos recursos.

– Es posible actualizar desde una versión ya instalada?

NO, no es posible actualizar sobre ninguna versión de Windows, aun ni siendo de MiniOS.

– Como se debería hacer una instalación correctamente?

Recomiendo una instalación limpia, eliminar la vieja partición(es) donde está instalado el sistema, para así no arrastrar antiguos archivos, cuando eliminemos la partición(es) creamos nuevamente he instalamos.

– Como se activa el sistema?
El sistema viene auto-activado desde la versión 2017.11 (esta versión), de tener algún problema con la activación, dentro del ISO hay una carpeta llamada “Activador” dentro tiene una herramienta para ello.

– Debo instalar actualizaciones?
NO, debido a que el sistema es un “Lite” no se deben instalar actualizaciones desde Windows Update, ya que al sistema carecer de componentes las actualizaciones podrían dejarlo inservible u hacerlo mucho más pesado, pero sin embargo si pueden instalar manualmente las actualizaciones que necesites.

– Cual es la diferencia de MiniOS7 boot7 y MiniOS7 boot10?
En MiniOS7 algunas notebooks y equipos no permiten que este sistema botee, pero sin embargo si a windows8 y windows10, por ello para aquellos que desean instalar MiniOS7 y su equipo no lo permite esta la opción del MiniOS7 boot10, el sistema ya instalado es exactamente igual a MinOS7 boot7, lo único que cambia es el instalador que usa el de Windows10.

– Es posible instalar MiniOS10 por UEFI?
SI, ya es posible pero no desde el TEU, para instalar por UEFI debes usar solo el x64 que se encuentra independiente, se separó del TEU con este propósito, pero el sistema es exactamente igual.

– Como me pongo al día con las novedades y lanzamientos del sistema?
