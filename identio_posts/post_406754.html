Windows MiniOS -2016

  • Categoría: Sistemas Operativos
  • Publicado hace más de 9 años






Información.
Windows Mini Os es un Windows rediseñado desde cero, el nucleo es Windows XP pero este sistema operativo es mucho mas eficiente, su consumo de recursos es extremadamente bajo, solo requiere de 64mb de ram y procesador de 0.5ghz para funcionar, el único y especial para revivir pc antiguas, su diseño es excelente y muchos mas beneficios que incluye el so. Pero no es solo para equipos de pocos recursos, puedes usarlo en un buen ordenador también, es cierto que es pequeño, pero  tiene la misma compatibilidad de hardware que el XP original.

Se eliminaron muchos componentes, servicios y aplicaciones, ya sea por estar obsoletos o por ser innecesarios para la mayoría de los usuarios, ayudando a si a mejorar el uso de espacio en disco, la velocidad, el consumo de RAM y recursos del sistema, se optimizaron los servicios, entre muchos más cambios. Este sistema tiene la misma compatibilidad de hardware que el XP SP3 original. 

Versión 2016

-Puede que mas software aparte del Rufus puedan pasarlo a USB sin problema alguno.
-Habilitada la búsqueda de impresoras automáticamente en una red.
-Casi todas las DLL modificadas no son reemplazadas como antes, sino que son nativas del sistema
-La partición del sistema es nombrada automáticamente SYSTEM.
*Mejoras de Estabilidad
*Nuevos Themes 
*En Español
*Incluye Drivers Sata
*Incluye Últimos Certificados
*Sistema operativo 32 Bits
*Peso de 281 mb
Servidor: Mega
*La instalación via usb ya esta soportada, pero en caso de que no les funcione pueden pasarlo a un cd.
*No es compatible en Vmware si desean virtualizarlo deben instalarlo en VirtualBox
