Revive  y Equipa tu Nokia Symbian(Nostalgicos)






Hola amigos de identi. Brevemente les comento que tengo un NOKIA N8 y que ya me presentaba muchos problemas, se trababa, no se veian las fotos, marcaba muchos errores y no me dejaba instalarle nada, y como soy un Nostalgico me resisto a dejar Symbian, y por eso me dia la tarea de revivirlo y me encontre con un programa que se llama Nokia Recovery Tool. Que les dejo en este post, ademas tambien les pongo un pack de aplicaciones necesarias para el hackeo a si como mi coleccion de aplicaciones para que dejen sus dispositivos Symbian como recien salidos de la fabrica pero con el software mas actual.

Bueno, basta de charlas y comensemos con el post.






Requisitos del sistema.
Microsoft Windows 7 o una versión posterior.
Cable USB para conectar el teléfono al ordenador.
Mínimo de 4 GB de espacio de almacenamiento libre.



La herramienta Software Recovery Tool es una herramienta original de microsoft que busca y recarga automaticamente el software mas adecuado para tu telefono puede ayudarle a restablecer y a recuperar el software del teléfono en casa en caso de que el teléfono no responda, se bloquee o tenga problemas relacionados con el software o con la actualización de software.

    Importante: Software Recovery Tool borrará todo el contenido almacenado en el teléfono, incluidas aplicaciones y juegos (junto con datos de aplicaciones y progreso de juegos), mensajes de texto, historiales de llamadas, música, fotos y todo lo demás. Si es posible, haga una copia de seguridad del teléfono (Ajustes > Copia de seguridad). Podrá restaurar la copia de seguridad cuando se haya completado la recuperación.

La herramienta Software Recovery Tool de Nokia se puede utilizar para reinstalar el software en muchos de los teléfonos Serie 30+/Serie 40/Asha/Nokia Belle y plataformas Nokia X.

Instrucciones.
*Instalar en pograma Nokia Recovery   este buscara los draivers necesarios para compatibilidad con los telefonos. ( En caso de no tener los drivers necesarios instalar el Nokia PC suite que tambien agrego a este post para que los instale)   

*Conectar el telefono con cable usb. (Es posible que no lo reconozca si no se utiliza el cable USB NOKIA, aunque a mi me funciono con un cable usb samsung)

*Seguir las instrucciones del programa y este buscara y descargara la version mas nueva disponible para su telefono symbian.

* Ya descargado el software comenzara a instalarlo en el telefono, hay que seguir las instrucciones del programa ( en este punto ya no hay marcha atras, debes dejarlo terminar para evitar cualquier daño)

*Y GUALA, tu Symbian a quedado como de fabrica, no importa de que pais seas, el programa buscara el software adecuado a tu telefono, region y compañia. ( ami me puso hasta los juegos que venian de fabrica en el telefono.)



Una vez que su telefono ha sido restaurado tenemos la opcion para descargar aplicaciones firmadas para poder instalarlas O hackearlo para que nos permita instalar cualquier aplicacion sin que salga el molesto letrero de Certificado caducado a si que que tambien les dejo el pack junto con un video paso a paso dentro del rar para que no haya fallas. ( Este proceso no lleva mas de 5 min).


Y ya que esta hackeado podemos instalar diversidad de aplicaciones y juegos sin ningun problema.

Esta es la lista de juegos, aplicaciones, y temas que tengo y quiero compartir para que symbian no muera.
