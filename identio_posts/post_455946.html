Daemon Tools Lite 10



DAEMON Tools Lite v10.5.1.230


Daemon Tools Lite 10 es un software muy conocido confiable y utilizado para crear y 
montar imágenes.

Con la última versión  el programa incluye no sólo las funciones básicas gratuitas para 
la emulación de dispositivos virtuales sino también una serie de herramientas avanzadas 
para trabajar con VHD (disco duro virtual) discos RAM destinos iSCSI memorias USB 
de arranque y un montón de otras características.

La software gratuito Daemon Tools Lite 10 proporciona las siguientes características:

* Posibilidad de montar todos los tipos populares de imágenes incluyendo
 VHD VMDK archivos y TrueCrypt;
* Creación de *.iso *.mdx and *.mds/*.mdf  .mds archivos de imagen;
* La compresión y la protección de sus discos virtuales personalizados;
* Emulación de hasta 4 dispositivos SCSI DT o de disco duro;
* El catálogo de nuevas imágenes se rellena automáticamente.

Tenga en cuenta que para trabajar con dispositivos SCSI y discos duros es necesario 
instalar el controlador SPTD. Asegúrese de que usted haya seleccionado esta opción 
durante la instalación o descargarlo desde el sitio oficial.
SPTD 1.89 está dedicado a Windows 2003 / Vista / 7/8 / 8.1.
SPTD 2.11 sólo se necesita para Windows 10.

Aunque Daemon Tools Lite es gratuito para uso no comercial será usted bienvenido a 
probar los beneficios de la licencia personal que incluye actualizaciones de por vida 
puede ser instalado en 3 PCs no contiene ofertas de ningún otro fabricante que se 
ofrecen durante el proceso de instalación y actualización además apoyo 24/7.

Además de la funcionalidad básica Daemon Tools Lite 10 proporciona una serie de 
herramientas avanzadas que se pueden comprar por separado a un precio razonable 
o conseguir el paquete completo con un descuento increíble. 


Características Básicas y Avanzadas

Daemon Tools Lite 10 incluye dos tipos de funciones: básico y avanzado. La funcionalidad 
básica es familiar a los usuarios de las versiones anteriores del programa y está 
disponible de forma gratuita. La funcionalidad avanzada se compone de varios grupos 
que se pueden comprar por separado.


Daemon Tools Lite funcionalidad básica

Utilizando la versión gratuita de Daemon Tools Lite 10 usted puede:

  * Montar distintos tipos de imagen;   
* Coge las imágenes de los discos físicos;   
* Emular hasta 4 DT disco duro o SCSI (con conductor) sptd dispositivos;   
* Comprimir y proteger con contraseña de imágenes personalizadas;   
* Escanear su PC para llenar el catálogo de archivo nuevo.


Las características adicionales de Daemon Tools Lite 10

Daemon Tools Lite 10 incluye un montón de características adicionales para los 
usuarios profesionales. Todos ellos están divididos en varios grupos que se pueden 
comprar por separado. Así que usted puede comprar solamente la característica 
que realmente necesita o no comprar nada si la funcionalidad básica es suficiente para ti. 
Además todas las funciones se pueden comprar junto con un descuento.


Los grupos de funciones disponibles en Daemon Tools Lite 10:

  * El editor de imágenes le permite convertir y crear imágenes de archivos y carpetas.
  * El grupo Virtual HDD incluye agregar un VHD añadir un disco RAM y añadir un 
archivo con características TrueCrypt. 
  * Disc Burn se puede utilizar para copiar borrar y grabar diferentes tipos de discos.
  * El grupo Bootable USB le permite escribir una imagen de arranque en una unidad flash y proteger los datos privados en una memoria USB.
  * Se necesita permiso de escritura de dispositivo virtual para emular el proceso de 
grabación de Daemon Tools Lite o cualquier otra aplicación.
  * Iniciador iSCSI le permite conectarse a destinos iSCSI remotos y trabajar 
con las imágenes compartidas VHD unidades ópticas y dispositivos USB físicos.
  * El monto avanzado le permite ajustar el punto de montaje conectar los dispositivos 
virtuales para las físicas y emular hasta 4 dispositivos IDE.
  * Dispositivos función ilimitadas le permite utilizar más de 4 dispositivos virtuales
     simultáneamente(256 DT y HDD 32 SCSI e IDE 4).Algunas de las funciones tienen 
un período de prueba por lo que son bienvenidos para evaluarlos antes de comprar.
Para iniciar la prueba visite la página de licencia.
Para averiguar las instrucciones detalladas vaya a la activación del artículo.
