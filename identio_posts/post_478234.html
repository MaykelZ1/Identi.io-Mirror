Microsoft Toolkit Collection Pack Julio 2017









Se trata de un conjunto de herramientas y funciones para administrar licencias, desplegar y activar Microsoft Office y Windows. Toda la salida de estas funciones se muestra en la consola de información. Todas las funciones se ejecutan en segundo plano y la interfaz gráfica de usuario se deshabilita para evitar que se ejecuten varias funciones, ya que podrían entrar en conflicto o provocar daños si se ejecutan simultáneamente. Las funciones de personalización del programa de instalación de Microsoft Office, la desinstalación de AutoKMS (si AutoKMS está instalado), el desinstalador de AutoRearm (si AutoRearm está instalado), Office Uninstaller y Product Key Checker funcionan incluso si Microsoft Office o Windows no están instalados o soportados. Para más información sobre las funciones individuales, lea más.
