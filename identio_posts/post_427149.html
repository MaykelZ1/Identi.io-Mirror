Camtasia Studio 8 Full MG






Hola comunidad Identi, Hoy les traigo una super herramienta  que utilizo demasiado. Nos ayudara a editar nuestros videos y lo mejor, grabar el escritorio de tu PC.















Camtasia es un capturador y editor vídeo que recoge como fuente lo que pasa en tu pantalla. Gracias a él podrás, entre otras muchas cosas, hacer screencasts y tutoriales como los que nosotros hacemos.
Requisitos 
Microsoft Windows XP SP3, Windows Vista, or Windows 7.
Microsoft DirectX 9 o mas grande.
Microsoft .NET 4.0 Client Profile (incluido, excepto con installer)
Dual-Core Procesador minimo, Quad-Core Procesador o mejor recomendado.
2GB RAM minimos, 4GB RAM o mas grande recomendado.
2GB de disco duro para la instalacion del programa.
Dimensiones de 1024??768 o mejor.
Dedicated Windows-compatible tarjeta de sonido.
