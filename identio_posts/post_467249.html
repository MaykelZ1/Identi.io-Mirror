AirStream


A veces necesitamos archivos de nuestros ordenadores personales y no podemos acceder a ellos hasta que no lo tengamos delante. En este caso se usa el streaming con un software muy completo y sencillo llamado, AirStream.

 
Lo que destaca de este streaming es la de acceder de forma local a los archivos con mucha sencillez, con ningún conocimiento necesario de informática podremos disponer de nuestros archivos locales en un momento, además su interfaz es muy intuitiva y todo es bastante natural.

En este caso es para la plataforma Windows y Android

características
Incluye programa para el servidor en Windows y el APK para Android
Windows xp, vista, 7, 8 y 10
Android 4.0 y Superior
conexion a intenet (no es necesario puedes usar un 
hostpot o anclaje de tu movil android)
