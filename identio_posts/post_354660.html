NETFLIX, llega a Linux vía Ubuntu y Chrome







Los usuarios de Linux (Ubuntu y derivados) ya no necesitan buscar soluciones mágicas para ver Netflix en sus equipos. Esto es así gracias a la conjunción de 

dos esfuerzos considerables:

Ubuntu desde su versión 14.04 ha agregado soporte para NSS (Network 

Security Service, un clon de las DRM's para navegadores) 

y Chrome desde su versión 37 ya soporta 

de manera nativa la tecnología NSS







Para poder hacer uso de esta gran combinación, se debe contar con los siguientes requisitos:

- Ubuntu 14.04 LTS / Linux Mint 17 con kernel 3.17
- Navegador Chrome versión para Linux.
- Una cuenta activa de Netflix.




Chrome 37:

Para instalar este navegador se debe hacer clic AQUI. Te llevará a la página de descarga de Chrome para Linux en las diferentes versiones que están soportadas por este navegador.

Una vez instalada la versión del navegador compatible con el Linux que usas, sólo debes iniciar tu cuenta de Netflix y.... Voilá! Podrás ver videos en Linux!
