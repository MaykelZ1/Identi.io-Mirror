[IDM] Internet Download Manager 6.21 Build 17 [MG]

  • Categoría: Software / Programas
  • Publicado hace más de 10 años




Internet Download Manager (IDM) es una herramienta para aumentar velocidades de descarga hasta en 5 veces, curriculum vitae y programar las descargas. Capacidad de recuperación de errores y reanudar Integral se reiniciará las descargas rotas o interrumpidas debido a conexiones perdidas, problemas de red, ordenador de maquinaria, o cortes de energía inesperados. Simple interfaz gráfica de usuario hace IDM fácil y fácil de use.Internet Download tiene un acelerador de la lógica de descarga inteligente que ofrece la segmentación dinámica de archivos y segura descarga de varias tecnologías para acelerar tus descargas Manager. A diferencia de otros gestores de descarga y aceleradores de Internet Download Manager segmentos de los archivos descargados dinámicamente durante el proceso de descarga y reutiliza conexiones disponibles sin necesidad de conectar y de acceso etapas para lograr el mejor rendimiento de aceleración adicional.


Internet Download Manager soporta servidores proxy, ftp y protocolos http, firewalls, redirecciones, cookies, autorización, audio MP3 y MPEG de procesamiento de contenido de vídeo. IDM se integra perfectamente en Microsoft Internet Explorer, Netscape, MSN Explorer, AOL, Opera, Mozilla, Mozilla Firefox, Mozilla Firebird, Avant Browser, MyIE2, y todos los demás navegadores populares para manejar automáticamente tus descargas. También puede arrastrar y soltar archivos, o el uso de Internet Download Manager de la línea de comandos. Internet Download Manager puede marcar su módem en el tiempo establecido, descargar los archivos que desea, luego cuelgue o incluso apagar el ordenador cuando está hecho. Otras características incluyen soporte multilingüe, vista previa de zip, descarga categorías, pro planificador, sonidos en diferentes eventos , HTTPS apoyo, la cola de procesador, ayuda y html tutorial, una mayor protección de virus en la terminación de descarga, descarga progresiva de las cuotas (útil para conexiones que utilizan algún tipo de política de acceso justa o FAP como Direcway, la PC directa, Hughes, etc), construido -in acelerador de descargas, y muchos otros.


Versión 6.21 añade compatibilidad de Windows 8.1, añade IDM panel de descarga web los jugadores que se pueden utilizar para descargar videos flash de sitios como MySpaceTV, y otros. También cuenta con Windows 7 y Vista apoyo completo, página de vídeos grabber, planificador reconstruido, y el soporte para el protocolo MMS. La nueva versión también añade una mejor integración de IE 11 e IE navegadores basados ??en el motor, descarga rediseñado y mejorado, la única de integración avanzada en todos los navegadores más recientes, barra de herramientas mejorada, y una gran cantidad de otras mejoras y nuevas características.
