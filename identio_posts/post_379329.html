Tutorial como borrar archivos que no deja windows


Aqui tenemos en este caso una carpeta que no se puede eliminar




Lo que hay que hacer es con el winrar es boton derecho sobre la carpeta (en este caso) y Añadir al archivo...




Una vez tomado este paso debemos dejar las opciones como estan: modo de actualizacion en añadir y reemplazar y opciones de compresion eliminar ficheros tras la compresion




Una vez echo esto la carpeta que no podiamos eliminar pasa a ser un archivo winrar




Ahora ya podemos borrar sin problemas el archivo rar




Como se ve la carpeta donde estaba el archivo imborrable quedo vacia




Ahora vamos a la papelera de reciclaje y borramos el archivo




Y con esto el el archivo que no podiamos eliminar fue eliminado completamente
