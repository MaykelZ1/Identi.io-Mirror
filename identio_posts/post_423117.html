Adobe Reader XI v11,0,16 [Español]

  • Categoría: Software / Programas
  • Publicado hace más de 9 años




Adobe Reader XI hace más que nunca. Así que usted también puede hacerlo.


El software Adobe Reader es el estándar de confianza gratuito que permite visualizar, imprimir, anotar y documentos PDF. Es el único visor de archivos PDF que puede abrir e interactuar con todos los tipos de contenido PDF, incluidos los formularios y multimedia.



*Características:
Ver y anotar los documentos PDF
Llenar, firmar y enviar formularios PDF
Documentos PDF Acceso fiable
Haga más con servicios de PDF
Gestione implementaciones de software y cumplimiento de las normas

*Requisitos del sistema de Adobe Reader XI:
Estos requisitos del sistema son para Adobe ® Reader ® XI, v11.0. Debido a los requisitos del sistema cambian con cada actualización de software, consulte las Notas de la versión de software para la información más reciente sobre los sistemas operativos y exploradores compatibles. Las especificaciones técnicas están también disponibles para la versión móvil de Adobe Reader .
- Procesador de 1,3 GHz o más rápido
- Microsoft ® Windows ® XP con Service Pack 3 de 32 bits o el Service Pack 2 de 64 bits; Windows Server ® 2003 R2 (32 bits y 64 bits); Windows Server 2008 o 2008 R2 (32 bits y 64 bits); Windows 7 (32 bits y 64 bits); Windows 8 o 8.1 (32 bits y 64 bits)
- 256 MB de RAM (se recomiendan 512 MB)
- 320 MB de espacio disponible en disco duro
- 1024x768 resolución de pantalla
- Internet Explorer 7, 8, 9, o 10 o 11; Firefox extendido de soporte técnico de lanzamiento; Chrome

Nota: Para las versiones de 64 bits de Windows Server 2003 R2 y Windows XP (con Service Pack 2), Microsoft Update KB930627 se requiere.
