El Complemento All in One Runtimes reúne todas runtimes importantes y actuales para Windows XP, Vista, 7, 8 y 10 en un paquete y ofrece una selección de lo que se desea instalar. Se incluyen todas las versiones de .NET Framework incluyen revisiones.
También se incluye en la descarga: complemento Java SE, DirectX 9.0c (x86, x64), archivos de tiempo de ejecución generales, Adobe Flash Player (IE plug-in), Adobe Shockwave Player (IE plug-in) y Silverlight.
Lo que se va a instalar, el usuario puede determinar antes de que sea instalado en un menú de selección en detalle. La rutina de instalación es el mismo para todos los sistemas operativos Windows de Microsoft..

- El paquete de tiempo de ejecución de AiO ofrece un instalador conveniente que proporciona la opción de seleccionar qué motores de ejecución para ser instalado.
- El instalador también detecta los tiempos de ejecución ya instalados para evitar la reinstalación.
- Las dependencias del sistema se comprueban automáticamente. Por ejemplo, el .NET Framework 4.6 se puede instalar en Windows 7 y Windows 8.1 – en Windows 10 no lo hará.

INCLUYE:

- .NET Framework 4.6 + Updates
- Java Runtime Environment 8
- DirectX 9.0c Extra files
- General runtime files
- Microsoft Visual C++ Runtimes (v2005 – v2015)
- Microsoft Visual J# 2.0 SE
- Microsoft Silverlight 5
- Adobe Flash Player (Opera, FireFox, Internet Explorer)
- Shockwave Player 12 (Internet Explorer Plugin)
