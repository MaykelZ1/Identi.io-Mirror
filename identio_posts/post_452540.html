5 herramientas para crear tu página web gratis

  • Categoría: Apuntes y Educación
  • Publicado hace más de 8 años
Cinco herramientas para crear tu página web gratis



Crear una página web era un proceso complejo hace unos cuantos años, pero cada vez hay más servicios web y herramientas que permiten simplificar el proceso, e incluso abren la posibilidad de crear tu propia web sin tener nociones de programación. Por ello, hemos recopilado los mejores programas y herramientas para crear tu página web gratis en con unos resultados visuales impresionantes.

De hecho, hay tantas herramientas para crear páginas web para usuarios sin nociones de programación que se hace difícil elegir entre ellas. Hay multitud de herramientas tanto online como offline. Los online tienen como ventaja que son más eficientes y rápidos, y puedes ver las modificaciones en tiempo real, mientras que uno offline permite modificar más aspectos, aunque no es tan directo.



Wix: crea tu página web guiada

Esta herramienta permite crear una página web viendo en tiempo real todos los cambios y modificaciones que vas realizando. La página cuenta con dos métodos para crear la web. El primero consiste en usar el Wix ADI (Artificial Design Intelligence), el cual te guía mediante un proceso de cuestiones básicas antes de crear una web de manera automática basada en esos parámetros. A pesar de ello, tiene un cierto grado de personalización, pero está más bien pensada para quienes no quieran complicarse la vida y quieran un diseño directo y rápido.



Si eres más atrevido, hay otro método que consiste en utilizar el Wix Editor, con cientos de plantillas para elegir las cuales se pueden modificar y personalizar a tu gusto, viendo en tiempo real todas las modificaciones, pudiendo cambiar la localización de los elementos. Cuenta con asistente SEO y está en disponible en español.

Prueba Wix aquí.


WordPress

Esta herramienta es una de las más populares para crear blogs, tiendas e infinitud de webs. Hay dos versiones de WordPress. La más compleja se descarga desde la página oficial de www.wordpress.org y necesita subirse a tu propio espacio web para ser instalada, y ofrece una gran variedad de opciones de personalización para usuarios avanzados.



La versión más sencilla permite crear tu propia página web online en www.wordpress.com, con un gran número de plantillas entre las que elegir. La principal desventaja de usar la versión web es que esta tiene menos plugins y plantillas que la versión de escritorio.

Descarga WordPress aquí.

Prueba WordPress online aquí.


CoffeeCup Free HTML Editor

Aunque no le llega a Adobe Dreamweaver, CoffeeCup es uno de los editores para profesionales gratuitos más completos que hay. Es necesario tener ciertos conocimientos de programación, pero es un programa realmente intuitivo ya que cuenta con un diseño de pantalla partida, donde tenemos a un lado el código y al otro el diseño, en el cuál se ven las modificaciones que vamos realizando en tiempo real.



Para iniciarse, el programa cuenta con una serie de plantillas y sugerencias de código. Si cuentas con algo de conocimiento de HTML y quieres avanzar en él, Coffee Cup es una gran opción. No obstante, el corrector de código y el optimizador sólo están disponibles en la versión de pago.

Descarga CoffeeCup Free HTML Editor aquí.


Joomla

Al igual que WordPress, Joomla ofrece la posibilidad de utilizar la propia web como host en los servidores de la compañía, o también permite descargar la herramienta para subirla e instalarla en tu propio sitio web.



La única pega es que es algo más complejo de utilizar que WordPress. Su principal baza es una gran cantidad de temas y extensiones para personalizar tu web.

Prueba o descarga Joomla aquí.


Jimdo y Yola

Jimdo permite crear una página web online en cuestión de minutos, haciendo el proceso lo más sencillo posible. El servicio online permite elegir entre cientos de temas, y permite un acceso sencillo para aquellos que no tienen conocimientos de programación web. La parte gratuita del servicio ofrece 500 MB de almacenamiento y 2 GB de tráfico, creando tu propia web con el subdominio .jimdo.






Muy similar a Jimbo tenemos Yola, que cuenta con 1 GB de almacenamiento y un límite de tráfico de 1 GB.

Prueba Jimbo aquí.

Prueba Yola aquí.
