Ardamax Keylogger v4.2 - DC 02-09-2014






Ardamax Keylogger es un registrador de pulsaciones de teclas que captura la actividad del usuario y lo guarda en un fichero de cifrado diario. El archivo de registro se pueden ver con el Visor de registro de gran alcance.

Utilice esta herramienta para averiguar lo que está sucediendo en su computadora mientras usted está ausente, mantener una copia de seguridad de sus datos mecanografiados automáticamente o utilizarla para supervisar a sus hijos. También se puede utilizar como un dispositivo de vigilancia para detectar el acceso no autorizado.

Los registros pueden ser enviados automáticamente a su dirección de correo electrónico, el acceso al keylogger está protegido con contraseña. Además, Ardamax Keylogger registra información acerca de las direcciones de Internet que el usuario ha visitado.


*Características Del Keylogger:
Registro de pulsaciones
- Registros escriben todas las pulsaciones de teclado, contraseñas y personajes ocultos.
Navegadores captura
- Graba un registro de todos los sitios web visitados por todos los navegadores.
Grabación de webcam
- Periódicamente hace imágenes de webcam y las almacena para iniciar la sesión.
La entrega de registro de correo electrónico
- Keylogger puede enviar registros grabados a través de entrega de correo electrónico a horas fijas - perfecta para el control remoto!
Entrega FTP
- Ardamax Keylogger puede cargar los registros grabados a través de la entrega de FTP.
Grabación con micrófono
- Grabar voz desde el micrófono con compresión en tiempo real.
Entrega en red
- Envía los registros grabados a través de a través de LAN.
Tala Portapapeles
- Captura todo el texto copiado al portapapeles de Windows.
Modo Invisible
- Hace que sea absolutamente invisible a cualquier persona. Ardamax Keylogger no es visible en la barra de tareas, la bandeja del sistema, Windows 2000/ XP/ 2003/ Vista/ Windows 7 Administrador de tareas, los espectadores de procesos (Process Explorer, WinTasks etc), Menú Inicio y la lista de inicio de Windows.
Vigilancia Visual
- Periódicamente hace capturas de pantalla y almacena las imágenes comprimidas para iniciar la sesión.
Chat en monitoreo
- Ardamax Keylogger se diseña para registrar y controlar ambos lados de una conversación en la siguientes charlas: AIM, Windows Live Messenger, ICQ, Skype, Yahoo Messenger Google Talk, Miranda, qip
Seguridad
- Le permite proteger la configuración del programa, modo oculto y archivo de registro.
Monitoreo de aplicaciones
- Keylogger registrará la aplicación que estaba en uso que recibió el golpe de teclado!
Tiempo / Fecha de seguimiento
- Se podrá localizar con exactitud el tiempo exacto de una ventana recibió un golpe de teclado!
Visor de registro de gran alcance
- Puede ver y guardar el registro como una página HTML o texto plano con keylogger Log Viewer.
Tamaño pequeño
- Ardamax Keylogger es varias veces más pequeño que otros programas con las mismas características. No tiene módulos y librerías adicionales, por lo que su tamaño es más pequeño y el rendimiento es mayor.
Unicode
- Ardamax Keylogger es totalmente compatible con caracteres Unicode que permite grabar las pulsaciones de teclado que incluyen caracteres del japonés, chino, árabe y muchos otros conjuntos de caracteres.

*Otras Características:
- Soporte con Windows 2000-2003/ XP/ Vista/ Windows 7/ Windows 8.x
- Monitores máquinas multi-usuario
- Inicio automático
- Interfaz amigable
- Fácil de instalar
