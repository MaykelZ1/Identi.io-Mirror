Magix Samplitude Pro X2 Suite


El hermano menor de Sequoia ha llegado, también es una excelente DAW (Digital Audio Workstation).

Samplitude fue creado en el año 1992 y desde entonces ha cosechado fama y éxitos en las altas esferas de las DAW.

Su motor ingeniería hibrida y natividad en PC, permiten trabajar de forma independiente sin importar la clase de tarjeta de sonido que tenga la PC.

Esta DAW esta a la altura de Studio One y Sonar.


Detalles de Magix Samplitude Pro X2 Suite

Sistemas Operativos:
Windows Vista  32 Bit y 64 Bit
Windows 7         32 Bit y 64 Bit
Windows 8         32 Bit y 64 Bit
Windows 8.1      32 Bit y 64 Bit

Idiomas:
Alemán
Ingles
Español
Francés
Italiano

Requerimientos de Hardware:
Procesador: 1,5 GHz o superior
Memoria RAM (Para sistemas de 32 Bit): 1 GB Ram
Memoria RAM (Para sistemas de 64 Bit): 2 GB Ram
Tarjeta Video: Independiente con una resolución minina de 1024 x 768
Tarjeta de Sonido: ASIO y/o WDM compatible
Espacio en Disco Duro: 10 GB
Lectora óptica: Lectora de DVD de doble capa para la instalación
Grabación : Requiere un grabador para CD/DVD ±R(W)
Exportación de MP3: Requiere Windows Media Player 10 o superior
Grabación MIDI: Requiere de interface MIDI

Formatos Soportados:
Exporta: MIDI-Standard-Format (MID), WAV, AAC, Broadcast WAV, MP3, CD-
             A, OGG-  Vorbis, AIFF, WMA, FLAC.

Importa:  MIDI-Standard-Format (MID, GM, GS, XG), WAV, AAC, Broadcast
