Antivirus en USB y CD para desinfectar tu PC

  • Categoría: Software / Programas
  • Publicado hace más de 7 años



Cuando la PC de Escritorio, o las Laptops tienen una invasión repentina de virus o de Malware que en muchos casos les impide arrancar , estos CDs/USB son muy útiles para recuperar la PC, ya que actúan antes de que cargue el sistema operativo, escaneandola en busca de virus
 
Aclaro que estos CDs/USB Bootables , son para Windows (Excepto Dr. Web que funciona para ambos) y todos están actualizados y descargados desde su página oficial respectiva.
 
Para crear un CD/DVD de autoarranque solo basta con tener una imagen en formato ISO y quemarla en una unidad grabadora.
 
Casi todos vienen con un programa especial para una vez teniendo la imagen, se pueda crear con ella, un pendrive auto arrancable. Se recomienda usar estos programas especiales para crear pendrives, utilizando una PC con un S.O. desde XP hasta 8.1 y ejecutarlos en modo administrador.
 
La memoria, deberá ser previamente formateada bajo FAT 16 o FAT 32
 
Previa modificación del BIOS, podremos bootear con el (o ellos), en la maquina afectada por los virus.
 
En síntesis, debido a  que los virus causan daño al sistema operativo, corrompiendo, robando información o impidiéndole arrancar, un  Antivirus de arranque, es un arma muy poderosa a nuestra disposición para remover los virus y tener nuestra PC de regreso y corriendo normalmente.



Antivirus Live CD




Antivirus Live CD es un tenedor 4MLinux incluyendo el escáner ClamAV.
Está diseñado para administradores de sistemas que necesitan un CD en vivo de peso ligero con un escáner antivirus.
Tanto Ethernet (incluyendo Wi-Fi) y dial-up (incluyendo módems USB rápidas) las conexiones a Internet son compatibles para activar las actualizaciones automáticas de la base de datos de firmas de virus.
Todas las particiones se montan automáticamente durante el arranque de forma que puedan ser escaneados por ClamAV (los sistemas de archivos soportados son: btrfs, ext2, ext3, ext4, FAT, HFS, HFS +, JFS, Minix, NTFS, ReiserFS, XFS y).
Imágenes Antivirus Live CD son totalmente compatibles con UNetbootin, que se puede utilizar para crear un programa fácil de usar Antivirus Live USB.

Los Antivirus de rescate más recomendados son:

 

AVG Rescue CD




Es un antivirus de autoarranque de “solo texto”, el cual puede buscar por programas potencialmente indeseados, escanear cookies, encontrar archivos con extensiones escondidas, e incluso escanear dentro de los archivos.
Antes de empezar a escanear, tienes la opción de escanear solamente una carpeta que escojas, o solamente el sector de arranque (boot), el registro o cualquier lugar que desees del disco duro.
Instrucciones:
Para obtener un CD auto arrancable, solo basta quemar el archivo ISO en un CD , esto es igual para todos los archivos ISO´s .
Para obtener una memoria autoarrancable, solamente extraiga los los archivos dentro del zip hacia una memoria USB, abriendo el ejecutable de la instalación del zip (setup.exe) y direccionando la instalación a la memoria en la cual deseemos instalar este antivirus.
Recuerde que debemos tener modificado el BIOS para arrancar desde el dispositivo de nuestra elección.
Archivos que incluye la descarga y peso:
Rescue CD (para creación de  CD ) Imagen en formato ISO cuyo  Peso es  169 Mb
 
Rescue CD (para creación de  USB ) Archivo Zip cuyo Peso es 199  Mb

Avira Rescue System






También funciona como un editor de registro, explorador de web y más, todos ellos bajo una interfaz gráfica estándar.
Tiene actualización automática de su base de datos, antes de desarrollar un escaneo, lo cual es muy útil, ya que no necesitas estar descargando este software cada vez que lo necesites.
Durante un escaneo, también los archivos comprimidos (en RAR, 7Zip, WinZip, etc.) son descomprimidos y escaneados para su mayor protección.
Mientras realiza el escaneo, se puede ver el número de virus encontrado en tiempo real y el número de archivos escaneados y el tiempo remanente para terminación del escaneo.
Este antivirus en particular escanea la computadora de manera total, no permitiendo, tal como hacen otros, el escaneo de ubicaciones específicas, tales como el registro o carpetas particulares.

Archivos que incluye la descarga y peso:
Imagen en formato ISO con un peso de 670 Mb
 
Manual del usuario en Inglés de cómo usarlo, como configurar el BIOS para el arranque, con peso de 611 Kb y en formato PDF

Bitdefender Rescue CD





Automáticamente busca actualizaciones cada vez que es iniciado.
Se pueden excluir ciertas extensiones de archivo de la búsqueda, permite la búsqueda o escaneo de carpetas individuales o si se desea un blanco especifico del directorio y no al disco entero.
Tiene una interfaz gráfica muy fácil de usar.
Su única desventaja es que toma un buen tiempo en iniciar el escaneo.  

Archivos que incluye la descarga y peso:
Imagen ISO con peso de 667 Mb
 
Stickifier, que es heramienta. gratuita para crear un USB de arranque con peso de 3.80Mb

Dr.WebLiveDisk





Funciona tanto para Windows como para Linux.
Puede escoger las acciones que tomara cuando encuentre archivos infectados, sospechosos o incurables, también permite escoger que acción deberá tomar si encuentra amenazas específicas como adware, dialers, jokes, hacktools y riskware.
Permite actualizar las definiciones de virus directamente desde el programa, lo cual significa que no será necesario bajar la última versión cada vez que se utilice.
Archivos que incluye la descarga y peso:
Archivo en formato ISO para quemar en CD/DVD cuyo peso es de 602 Mb.
 
Aplicación ejecutable para instalar en una memoria USB con un peso de 596 Mb

Eset SysRescue Live





ESET SysRescue Live es una herramienta de limpieza de malware que se ejecuta desde un CD, DVD o USB, independientemente del sistema operativo. Tiene acceso directo al disco y a los archivos del sistema, y por lo tanto es capaz de eliminar las amenazas más persistentes
Archivos que incluye la descarga y peso:
Archivo en formato ISO para quemar CD, con un peso de 324 Mb
Aplicación ejecutable para instalar en memoria USB cuyo peso es de 8.73 Mb
 
Guía del usuario en Inglés, en formato PDF con un peso de 4.61 Mb

Kaspersky Rescue Disk





También es conocido como Rescue Disk, escanea en busca de virus, troyanos, herramientas maliciosas, adware. Dialers y otros programas maliciosos.
Tiene Modo gráfico (recomendado) y el modo de solo texto.
Tiene una opción para buscar en archivos comprimidos, packs de instalación y en los objetos OLE, que es un estándar que permite la incrustación y vinculación de objetos (imágenes, clips de vídeo, sonido MIDI, animaciones, etc) dentro de ficheros (documentos, bases de datos, hojas de cálculo, etc). También hace posible la inclusión de controles ActiveX y la comunicación entre ellos)
Permite la opción de desinfectar la computadora cuando encuentre objetos maliciosos o darle la opción de tomar una acción determinada por el usuario.
Las actualizaciones de virus pueden ser descargadas dentro del programa y los resultados pueden ser exportados a un archivo de texto.  
Pasos para la creación de un CD autoarrancable, solo basta quemar el CD
Pasos para crear USB autoarrancable :
Paso 1– Para comenzar abra Kasperky Rescue USB y cargue la imagen ISO original de Kaspersky Rescue con el botón “Browse”:
Paso 2– Seleccione el Pendrive que va a usar y haga clic en “Start” para iniciar el proceso de creación.
Paso 3– Captura de Kaspersky Rescue CD iniciando desde una unidad Flash
