Hola Amigos Hoy les traigo un tutorial de como tener el Fifa 16 Pc Full desbloqueado con todas sus Funciones, Offline obviamente !!

Presten mucha atencion al tutorial es un poco larga pero muy facil, es solo seguir paso por paso !!



Lo Primero que debemos tener es  La version DEMO ( Link DEMO FIFA 16 )  y la version Desbloqueda de 13 Gb ( LINK ) que encontraremos en la Pagina de Skidrowgame.


Empezemos con el Tutorial, (Mas abajo les dejo el Video)


1- Copiamos todos los Archivos de Fifa 16 Desbloqueado de 13 Gb a la Carpeta de FIFA 16 DEMO,  Click en Omitir Archivos, Ojo no reemplazamos.
3-Borramos las Carpetas "Data y DLC" De La carpeta DEMO FIFA 16
4-Copiamos las carpetas "Data y DLC" de Fifa 16 desbloqueado de 13 gb a la carpeta de DEMO FIFA 16.
5- Borramos la carpeta  ''dlc_powdll'' que se encuentra dentro de la carpeta "DLC" del DEMOFIFA 16
6-Instalamos File Master 15 ( LINK )
7-Instalamos Creation Master 16 ( LINK )
8-Abrimos la Ubicacion donde instamos Creation Master, dentra de la carpeta "Templates" encontraremos la carpeta  "Data" que lo copiaremos dentro de DEMO FIFA 16
9- Abrimos el File Master, Buscamos la carpeta DEMO FIFA 16, Seleccionamos, esperamos que carguen los Archivo y damos en Regenerar.
10- Desactivamos el Firewall, para iniciar el juego.
