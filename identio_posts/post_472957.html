Windows 10 MiniOS|2017|v,05|UB|








Se eliminaron componentes, servicios y aplicaciones, por estar obsoletos o por ser innecesarios para la mayoría de los usuarios, ayudando a si a mejorar el uso de espacio en disco, la velocidad, el consumo de recursos del sistema, se optimizaron los servicios, entre otros cambios. Aparte de lo que fue eliminado siempre busqué que este sistema mantuviera la misma compatibilidad de hardware que el Windows10 original.
Cambios
 Se eliminaron componentes, servicios y aplicaciones, por estar obsoletos o por ser innecesarios para la mayoría de los usuarios, ayudando a si a mejorar el uso de espacio en disco, la velocidad, el consumo de recursos del sistema, se optimizaron los servicios, entre otros cambios. Aparte de lo que fue eliminado siempre busqué que este sistema mantuviera la misma compatibilidad de hardware que el Windows10 original.
La apariencia se mantiene la original del sistema.
NOTA: Para instalar desde USB recomiendo el Rufus, aunque pueden usar cualquier otro software que usaran para hacerlo.

NOTA: Si se va a instalar en una máquina virtual recomiendo usar VirtualBox, de usar VMWare no poner el ISO cuando creamos la máquina, lo hacemos después para que instale correctamente.
