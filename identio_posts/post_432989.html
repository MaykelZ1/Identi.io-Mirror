Qué hacer después de instalar Apricity OS

  • Categoría: Linux
  • Publicado hace más de 9 años


Ya que migré recientemente a Apricity, pues es necesario realizar algunas configuraciones básicas para lograr el máximo desempeño del SO o al menos en mi caso, lo hice para tener las mismas herramientas, software, configuración y estilos que en gnome con Fedora 24.

Actualizar el SO:
Cita: Mostrar
sudo pacman -Syu

Después de la actualización, reiniciar:
Cita: Mostrar
reboot

Instalar las librerías de compilación para desarrolladores:
Cita: Mostrar
sudo pacman -Sy base-devel
Seleccionar, todas las fuentes de instalación que presente el comando.



Instalar ZSH para motor de comandos:
Zsh, es un motor poderoso de interprete de comandos, que facilita el uso de los mismos.
Cita: Mostrar
sudo pacman -S zsh git gedit
git clone git://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh
cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc
cp ~/.zshrc ~/.zshrc.orig
sudo chsh -s /bin/zsh nombre_usuario
Cerrar sesión y volver a iniciar para que los cambios se apliquen.
Instalar Firefox Libre Office en Español, Nautilus, Evince, Brasero, Gnome Builder, Shotwell, Clipgrab, Geany, Gedit, Mplayer, Youtube-dl, Unetbootin, Pdfmod, Vlc, dconf-editor, gconf-editor, gparted, filezilla, audacity mediante el uso del comando Pacman:

Cita: Mostrar
sudo pacman -S firefox libreoffice-fresh-es nautilus evince brasero gnome-builder shotwell clipgrab geany gedit mplayer youtube-dl unetbootin pdfmod vlc dconf-editor gconf-editor gparted filezilla pinta audacity hunspell wget curl libdvdread libdvdnav lsdvd
Compresores de archivos:
Cita: Mostrar
sudo pacman -S xz bzip2 p7zip lbzip2 lrzip arj lzop cpio unrar

Mediante Yaourt sugiero: netbeans, yandex, sublime text 3, fritzing, ffmulticonverter, converseen, vokoscreen, dropbox, gnome-exe-thumbnailer, shotcut, dbeaver, android-studio:

NetBeans IDE es un entorno de desarrollo - una herramienta para que los programadores puedan escribir, compilar, depurar y ejecutar programas. Está escrito en Java - pero puede servir para cualquier otro lenguaje de programación. Existe además un número importante de módulos para extender el NetBeans IDE. NetBeans IDE es un producto libre y gratuito sin restricciones de uso.
Cita: Mostrar
yaourt -S netbeans


Yandex es un navegador ruso
Cita: Mostrar
yaourt -S yandex-browser-beta


Sublime  Text, es un editor de texto ligero para html, php, javascript, y muchos otros lenguajes.
Cita: Mostrar
yaourt -S sublime-text-dev


Fritzing, es un programa que permite crear, modificar y simular circuitos electrónicos.
Cita: Mostrar
yaourt -S fritzing

Programa de conversión de audio, video a otros formatos, permite convertir una lista de archivos.
Cita: Mostrar
yaourt -S ffmulticonverter


Software con una interfaz simple para screncast en linux, soporta salida de video MKV.
Cita: Mostrar
yaourt -S vokoscreen

Programa que permite sincronizar tu cuenta de drobox y la incluye en tu explorador de archivos, como nautilus.
Cita: Mostrar
yaourt -S dropbox

Librería que extrae una vista previa del los programas para winodows.
Cita: Mostrar
yaourt -S gnome-exe-thumbnailer


Editor de vídeo super poderoso para linux, es al estilo de sony vegas.
Cita: Mostrar
yaourt -S shotcut

Software con interfaz gráfica para realizar conexiones a gestores de bases de datos, como, mariadb, postgresql, sql, sqlite, postgresql, mysql, entre otros.
Cita: Mostrar
yaourt -S dbeaver

Software de desarrollo para aplicaciones móviles en android.
Cita: Mostrar
yaourt -S android-studio


Opcionales (o tal vez no):

Texlive y editor TexStudio, motor de librerías y paquetes para la creación de textos científicos:
Cita: Mostrar
sudo pacman -S python-pygments python-setuptools texstudio texlive-core texlive-bibtexextra texlive-fontsextra texlive-formatsextra texlive-games texlive-genericextra texlive-htmlxml texlive-humanities texlive-latexextra texlive-pictures texlive-plainextra texlive-pstricks texlive-publishers texlive-science

Utilerías adicionales
Las siguientes son programas adicionales, para recuperar archivos de usb formateada o hdd. Herramienta para visualizar los procesos que están corriendo actualmente en linux. Un software para resolver ecuaciones mátemáticas. Software para editar archivo binarios. Comando para visualizar las conexiones de red en el equipo.
Cita: Mostrar
sudo pacman -S foremost  htop maxima ghex iftop ogmrip xournal umbrello ninja-ide  wxmaxima gnuplot

VirtualBox:
Gestor de creación de máquinas virtuales multiplataforma.
Cita: Mostrar
sudo pacman -S virtualbox

Aquí les presenta dos fuentes de instalación, seleccionar la segunda. A continuación es necesario añadir nuestro usuario al grupo de virtualbox con el siguiente comando:
Cita: Mostrar
sudo usermod -G vboxusers -a tu-nombre-usuario

Para saber de más paquetes visitar la wiki de arch: https://wiki.archlinux.org/index.php/List_of_applications

Eso solamente al iniciar, poco a poco se agregan conforme surgen las necesidades.
