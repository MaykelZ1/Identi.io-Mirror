Anatomía y Función

¿Qué incluye?

Más de 500 preestablecidos vistas 3D en el cuerpo humano. Estos se pueden girar, agrandado, y disecados.
72 50-segundos animaciones y 77 ilustraciones que introducen funciones comunes
Pronunciaciones de audio y definiciones para miles de estructuras anatómicas
Múltiples funciones de búsqueda (navegar por el sistema, la función, relacionada puntos de vista o búsqueda) y la opción de crear una biblioteca de vistas favoritas
La capacidad de anotar y guardar imágenes
Licencia para dos ordenadores / un usuario


Requisitos del sistema

Se requiere el siguiente hardware para ejecutar Anatomía y Función para PC o Mac (Descargar).

Para PC:

2 GB de RAM
2 GHz procesador Pentium 4 o mejor
Windows XP / Vista / 7/8
Tarjeta de video dedicada con 64+ bordo on-MB RAM
1024x768 resolución de la pantalla, o mejor

Para usuarios de Mac:

2 GB de RAM
Procesador Intel
La última versión de OS X 10.6 / 10.7 / 10.8 / 10.9
Tarjeta de video dedicada con 64+ bordo on-MB RAM
1024x768 resolución de la pantalla, o mejor

Atlas de la Anatomía Humana

¿Qué incluye?

Sección Atlas 3D que es completamente interactivo e incluye más de 4.600 estructuras anatómicas
Cientos de imágenes en miniatura que se inician vistas predefinidas de las regiones y sistemas del cuerpo
Pronunciaciones, definiciones detalladas y términos latinos
Nuevas patologías comunes para cada sistema del cuerpo
La capacidad de crear, guardar y compartir puntos de vista personalizados en 3D
Cientos de pruebas
Dinámica capacidad de búsqueda, dibujo, notecard y captura de pantalla


Requisitos del sistema

Por escritorio de Windows o portátil:

2 GB de RAM
2 GHz procesador Pentium 4 o mejor
Windows XP / Vista / 7/8
Tarjeta de video dedicada con 64+ bordo on-MB RAM
1024 x 768 de resolución de pantalla, o mejor
Nota: Si usted tiene una computadora o dispositivo que ejecuta Windows con la interacción multi-touch (pantalla táctil), visite el Atlas de Anatomía Humana de la página del producto Windows Touch.
