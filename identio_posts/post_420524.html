UBUNTU 14.04.02 LTS Desktop Trusty Tahr (Español)


La versión Desktop de UBUNTU le permite probar Ubuntu sin cambiar el equipo en absoluto, y en su opción de instalar de forma permanente más tarde. Este tipo de imagen es lo que la mayoría de la gente va a querer usar. Necesitará al menos 384 MiB de RAM para instalar de esta imagen.

Hay dos imágenes disponibles, cada uno para un tipo diferente de equipo:

Para PCs con Procesador de 64 bits (AMD64)
Escoja esta opción para sacar el máximo provecho de los equipos basados en la arquitectura AMD64 o EM64T (por ejemplo, Athlon 64, Opteron, EM64T Xeon, Core 2). Si usted tiene un procesador no de 64 bits fabricado por AMD, o si necesita soporte completo para código de 32 bits, utiliza las imágenes para i386 en su lugar.

Para PCs con Procesador de 32 bits (INTEL/AMD)
Para casi todos los PC. Esto incluye la mayoría de las máquinas con Procesadores tipo Intel / AMD /  etc y casi todos los equipos que ejecutan Microsoft Windows, así como los sistemas de Apple Macintosh más recientes basados en procesadores Intel. Elíje esto si no estás del todo seguro.
