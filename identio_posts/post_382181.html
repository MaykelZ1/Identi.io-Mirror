Esta todo traducido pero en el menú de configurar las teclas se ve mal y no se distingue, así que será dificil reconfigurar el teclado, pero si q podremos usar un gamepad junto con xpader, os dejo la lista de controles para que reconfigureis con xpader o similar

CONTROLES DEL JUEGO:
Andar: Shift + Flecha Arriba
Paso Atrás: Flecha Abajo
Girar a La Derecha: Flecha Derecha
Girar a La Izquierda: Flecha Izquierda
Correr: Flecha Arriba
Esprintar: Pulsa - (Guión). Mientras Corres
Saltar: Alt
Paso Lateral Izquierdo: Shift + Flecha izquierda / Del. Puedes Utilizar Los Dos.
Paso Lateral Derecho: Shift + Flecha derecha / Av. Pag. Puedes Utilizar Los Dos.
Voltereta/Rodar: Fin
Desenfundar y Guardar Armas: Barra Espaciadora
Acción: Ctrl
Mirar: 0 en teclado numérico + Flechas
Agacharse: . (Punto)
Gatear: . Punto + Flechas
Salto En Parado: Alt + Flecha Arriba
Salto Corriendo: Flecha Arriba + Alt
Saltar y Agarrar: Alt + Ctrl
Salto Atrás: Alt + Flecha Abajo
Salto Lateral: Alt + Flecha Derecha/Izquierda
Descolgarse: Flecha Abajo + Ctrl
Franquear: Ctrl + Flecha Der/Izq
Atravesar Techos: Ctrl + Flechas de Dirección
Escalar: Ctrl + Flechas
Balancearse En Cuerdas: Aprieta - suéltala y aprieta rápidamente Alt, para saltar.
Mover Bloques: Ctrl + Flecha Arriba / Abajo
Hacer La Vertical: Ctrl + Shift + Flecha Arriba
Saltar De Cabeza al Agua: Shift + Alt + Flecha Arriba
Salto Mortal: Shift + Alt + Flecha Arriba y Abajo
Media Vuelta En El Aire: Alt + Arriba/Abajo + Fin
Saltar Hacia Atrás Desde Una Pared: Pulsa Alt sin soltar Ctrl
Media Vuelta y Agarrarse: (Alt + Fin) + Ctrl
Encender Bengala: , (coma) o 5 en el teclado numérico

PRISMÁTICOS:
Acercar: .
Alejar: -
Visión Nocturna: Ctrl
Soltar Binoculares: Barra Espaciadora

TRAJE DE GRAN PROFUNDIDAD:
Tirar Bengalas: Barra Espaciadora

TECLAS ESPECIALES:
P: Menú
1-8: Seleccionar armas disponibles
9: Botiquín pequeño
0: Botiquín grande
Alt - (bloque numérico): Reducir resolución
Alt + (bloque numérico): Aumentar resolución
F5: Grabar partida
F6: Cargar partida

Esta es la forma de hacer Capturas de pantalla durante el juego.
