Como hacer que W7 32bits reconosca mas de 4gb ram

  • Categoría: Software / Programas
  • Publicado hace más de 11 años
Bienvenidos a mi primer post


Importante: Este software ha sido utilizado por mi en un portatil con Windows 7 Ultimate. Tenga en cuenta que se hacen algunos cambios en el núcleo del sistema, que en algunos casos puede provocar consecuencias impredecibles para el sistema operativo. Varios programas antivirus identifican la aplicación como una amenaza para su equipo.

Toda la responsabilidad por las consecuencias de los riesgos de uso de este programa corren por cuenta del usuario.

Muy Importante: No instalar bajo ningún concepto en un SO diferente a Windows 7 y menos en Windows 7 de 64 bits.

Seguir las instrucciones:

1.- Inicie ReadyFor4GB.exe
2.- Haga clic en Check, a continuación, el botón Aplicar.
3.- Haga clic en Yes (Sí) para parchear ?ntkrnlpa.exe?
4.- Pulse el botón Aceptar para confirmar la ubicación para guardar el archivo ?ntkr128g.exe?
5.- Cierre la ventana ReadyFor4GB.
6.- Haga clic en el archivo ?AddBootMenu.cmd? y selecciona ?Ejecutar como administrador?
7.- Pulse el botón ?Y? para confirmar o presione ?Q? para salir y pulse Intro.
8.- Esperar hasta que el final de la secuencia de comandos y cerrar el programa.
9.- Haga clic en el archivo ?RemoveWatermarkX86.exe? y selecciona ?Ejecutar como administrador?
10.- Pulse el botón ?Y? para confirmar

Copia de seguridad de ?user32.dll.mui? se guardará como Windows System32 es-ES user32.dll.mui.backup.
String ?en-US? depende del idioma de su sistema.

11.- Espere a que el parche termine su secuencia y reinicie el equipo.
12.- El equipo se reinicia, se mostrará la entrada de inicio de Windows Boot Manager y cargaremos el Windows de 128 GB con ntkr128g.exe.


Si apareciera ? Version de evaluacion ? después del reinincio, entonces necesitas reconstruir la memoria caché de MUI. Para hacer esto:

a.- Haga clic en el archivo ?RemoveWatermarkX86.exe? y selecciona ?Ejecutar como administrador? .
b.- Pulse el botón ?I? o ?Y? para confirmar.
c.- Espere a la reestructuración de la caché y, a continuación, reinicie el equipo.
