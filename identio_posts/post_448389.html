Los mejores clientes para descargar torrent en 2017



A pesar del auge de los servicios de streaming, son muchos usuarios los que prefieren utilizar sistemas de descarga P2P para todo tipo de contenido gracias a que es una red abierta basada en compartir los archivos que vamos descargando. Para hacer esto hay una gran cantidad de programas, conocidos como clientes de torrent. Elegir uno u otro puede ayudarnos a maximizar el potencial de nuestra conexión. Por ello, vamos a recopilar cuáles son los mejores que podemos descargar en este 2017 para descargar torrent en el ordenador.


1.        uTorrent


El cliente más popular y utilizado a nivel mundial para descargar torrent. Conocido como uTorrent, o también como µTorrent, lleva desde 2005 facilitando descargas a sus usuarios, aunque no exento de polémica. Para financiarse utiliza anuncios y toolbar que hay que esquivar al instalarlo. A pesar de ello, los anuncios mientras se utiliza el programa no se pueden esquivar. El programa es de los más estables y completos que podemos encontrar, e incluye un monitor remoto para analizar cómo van nuestras descargas o añadir nuevos torrents desde nuestro móvil, incluso desde fuera de casa. uTorrent cuenta con clientes para Windows, Mac OS X, Linux y Android.


2.        BitTorrent


Aniversario BitTorrent

BitTorrent tiene su propio cliente con el mismo nombre. A pesar de soportar oficialmente uTorrent, también tienen este cliente, que básicamente es una versión de uTorrent con otro nombre y una interfaz ligeramente distinta. Como principales diferencias, tenemos que el cliente permite utilizar web-based seeding para cuando un torrent no tiene semillas y su creador lo ha subido a un HTTP. Los trackers privados suelen preferir también a clientes con este cliente antes que con uTorrent.


3.        qBittorrent * para mi uno de los mejores *


qBitorrent es probablemente el cliente de torrent que mejor combina el tener muchas funciones y capacidad de personalización con un aspecto sencillo y un peso muy ligero con bajos consumo de recursos del sistema, tanto de CPU como de RAM. Además, incluye un buscador, reproductor multimedia, cifrado, priorización de torrents frente a otros, así como de archivos dentro de los propios torrent, filtrado de IP y creación de torrent. Además, es software libre y está disponible para Windos, Linux y Mac OS X.

qBittorrent es lo más parecido que hay en la red a ?Torrent, sólo que sin anuncios ni malware que se descubrió que éste utilizaba para minar bitcoins en los ordenadores de los usuarios que lo descargaban. Por ello, con qBitorrent nos evitamos tener que recurrir a versiones antiguas de ?Torrent y podemos tener un cliente de torrent totalmente actualizado y con las últimas funciones.


4.        Deluge


Si algo caracteriza a Deluge es su gran capacidad de personalización gracias a que es compatible con extensiones, habiendo disponibles una gran cantidad de plugins en su página web oficial. Al igual que qBittorrent, su aspecto recuerda al de ?Torrent, pero con la ventaja de no tener software que incluya publicidad en nuestro ordenador. Entre sus funciones está una gran capacidad de programación de descargas o la posibilidad de mover descargas a un directorio determinado en función de su formato. Deluge cuenta con clientes para Windows, Mac OS X y Linux.


5.        Transmission


Este cliente es la opción más común si tienes Mac, y también está disponible para Linux. Su abanico de posibilidades permite tener un programa enfocado para usuarios avanzados, así como ser accesible para quienes quieran un simple cliente de torrent sin muchas complicaciones.


6.        Transmission-Qt Win


Transmission-Qt Win es un port de Transmission para Windows. Si estás acostumbrado a utilizar este cliente en otras plataformas, el port tiene las mismas funciones. El port no es del todo perfecto, y puede dar problemas al descargar archivos que alcancen el terabyte (no afectará a muchos), y puede ocasionar fugas de memoria si se deja mucho tiempo abierto.


7.        Vuze


Este cliente cuenta con versiones tanto gratuitas como de pago. Vuze, anteriormente conocido como Azureus, cuenta con dos versiones; una más completa llamada Vuze, y luego otra más ligera llamada Vuze Leap. La principal diferencia es que Vuze añade la posibilidad de utilizar plugins, conversión de video en el propio programa, y control remoto web. Vuze está disponible para Windows, Mac OS X y Linux.


8.        BitComet
BitComet mezcla la posibilidad de unificar todas tus descargas, tanto de torrent como de descarga directa, en un solo programa. Además, viene con VLC incluido, por lo que con una sola instalación puedes tener todas las ventajas que ofrecen varios programas de descarga. Bitcomet tiene clientes para descargar torrent tanto en Windows como en Mac OS X.


9.        BitLord
Este cliente para descargar torrent es el más liviano y simple disponible en la red. No incluye ningún software extra de publicidad. Al igual que BitComet, también incluye VLC para reproducir contenido multimedia, además de un buscador de archivos torrent. Si eres un usuario avanzado, quizá se te quede pequeño, pero para iniciarse es una gran opción.


10.      Tixati
¿Te molestan los fondos blancos y no has utilizado ninguno de los temas oscuros que recopilamos recientemente? Probablemente Tixati sea el programa ideal para ti. Además de tener una interfaz de fondo oscuro que recuerda en apariencia a lo que fue otrora WinMX, Tixati no incluye publicidad, además de ser especialmente ligero. Está disponible para Windows y Linux.
