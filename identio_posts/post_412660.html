Hola les traigo la mejor música electrónica, de varios géneros como lo son House, Techno, Dance, Electro y muchos otros más, Es una recopilación que he hecho de varios años. INFORMACIÓN DE LA MUSICA A DESCARGAR:



-Genero: varios

-Artistas: varios (por mencionar algunos David guetta,Dash berlin,Tiesto,Daft      Punk,Skazy,Alesso,Alex derron,Armin van buuren y muchos mas)

RECOMENDACIONES PARA UN MEJOR AUDIO: 
NOTA IMPORTANTE : recomiendo escoger alguna canción para escuchar de referencia antes de configurar el audio, escucharla y luego configurar el audio de Windows, luego de esto escuchar la misma canción y notaran una gran mejoría en el audio, Lo mismo aplica a la configuración de AIMP, si no quieren usar AIMP, más abajo explico cómo configurar el reproductor de Windows media para mejorar un poco el sonido. También aclaro que uso unos audífonos SENNHEISER, si ustedes usan audífonos de baja calidad también notaran que su sonido no es el mejor, pero con la configuración podrán mejorarlo un poco, y si quieren un audio excelente deben usar unos audífonos con cancelación de ruido activa.

 



1. Configurar el audio de windows




1.2 Configurar windows media player para mejorar el sonido (este paso se omite si no usan el wmp para escuchar musica y estan buscando un ecxelente reproductro de audio)




2. Instalar Aimp desde aqui Sitio oficial AIMP http://www.aimp.ru/index.php?do=download



3. Configurar AIMP, en preferencias tambien pueden configurarlo si tienen sistema de sonido mono, stereo, quadro ,5.1 ,7.1. tambien pueden elegir si usan algun preset de equalizador, si usan audifonos recomiendo usar el el que dice Headphones.
