Basta con arrastrar y colocar texto, gráficos y elementos multimedia en tus páginas web, sin necesidad de tener conocimientos sobre codificación. Utiliza el editor de línea de tiempo para crear y administrar sensacionales efectos visuales. Pronto crearás páginas web y sensacionales sitios web como un profesional. Site Safe
 Protege automáticamente tus recursos de sitios web y recupera con facilidad tu sitio desde ubicaciones de copia de seguridad locales, externas o en línea si algo se pierde.

NOVEDADES:
¡NOVEDAD! Fuentes web
 Las fuentes web te permiten aportar personalidad al contenido de tu sitio y, al mismo tiempo, aumentar su legibilidad, accesibilidad y dinamicidad. Corel Website Creator incluye una biblioteca de fuentes web de primera calidad diseñadas por profesionales.
 
¡NOVEDAD! Elementos HTML5
 HTML5 es una convergencia de varias tecnologías que sirven para añadir interactividad a los sitios web. Se trata de un enorme paso adelante respecto a HTML4, por lo que recomendamos que te asegures de utilizar HTML5 en tu sitio web y lo crees de forma profesional con Corel Website Creator.
 
¡NOVEDAD! Compatibilidad mejorada con CSS3
 CSS3 aporta numerosas funciones de estilos nuevas a las páginas web. Saca partido a la animación de los elementos de las páginas web y aplica distintos efectos, como esquinas redondeadas, sombras y muchos otros. Así proporcionarás a tu sitio web un aspecto elegante y cuidado.
 
¡NOVEDAD! Compatibilidad con Windows 8
 Sólo el software y los dispositivos compatibles con Windows 8 han superado las pruebas de compatibilidad y fiabilidad diseñadas por Microsoft en equipos que ejecutan el sistema operativo Windows 8. Los productos compatibles con Windows 8 se instalan sin problemas y se ejecutan a la perfección en Windows 8.
 
¡NOVEDAD! Interfaz de usuario optimizada
 Gracias a su nuevo espacio de trabajo moderno e intuitivo, serás más productivo y te sentirás como en casa. Corel Website Creator ofrece un nuevo y atractivo aspecto con herramientas actualizadas que te permiten mejorar el diseño y la creación de tus sitios web.
 
¡NOVEDAD! Carrusel de imágenes
 El nuevo carrusel de imágenes te permite visualizar cualquier colección de fotografías y constituye una magnífica alternativa a la galería de fotos. Tus fotos se muestran a tamaño completo y con la opción de animarlas automáticamente en la colección. No es necesaria ninguna programación; basta con que añadas las imágenes.
 
¡NOVEDAD! SiteStyles
 Los SiteStyles son colecciones de elementos gráficos y tipográficos que puedes aplicar a todas las páginas de tu sitio para crear un aspecto coherente. Puedes usar estilos predefinidos de Fusion o crear los tuyos propios personalizados y así obtener el aspecto perfecto que buscas para tu sitio.
 
¡NOVEDAD! Vista en línea
 El panel de control de Corel Website Creator también tiene un aspecto renovado. La vista en línea de Website Creator se ha adaptado al nuevo diseño de Windows 8 y te proporciona el mismo aspecto de la interfaz en mosaico.
 
¡NOVEDAD! Plantillas
 Corel Website Creator incluye nuevas plantillas. Las plantillas son páginas o sitios ya diseñados que sirven de punto de partida para la creación de tus propias páginas o sitios. Las plantillas suelen incluir una pancarta y botones de navegación, un SiteStyle asignado, texto y galerías de carrusel.
 
Site Safe
 La tecnología Site Safe protege tu contenido de diseño creando automáticamente una plantilla de copia de seguridad de los archivos de tu sitio web. Las plantillas de copia de seguridad pueden guardarse en una unidad externa o en una unidad de red, o publicarse en un directorio de tu servicio de alojamiento web.
 
Cloud Burst
 Recibe nuevos SiteStyles directamente en la vista en línea con Cloud Burst, el sistema de entrega en línea. Así podrás instalar esas plantillas, fotografías, imágenes y tutoriales para sitios web y utilizarlos directamente con un solo clic.
 
Sitios web interactivos
 Con su colección de widgets Asynchronous JavaScript and XML (AJAX), Website Creator te permite diseñar fácilmente sitios web con interactividad de usuario dinámica. Añade elementos de página personalizables al instante, como acordeones, paneles con fichas y paneles de conmutación, para mejorar la experiencia proporcionada a los usuarios del sitio web.
 
Desarrollo web integrado
 Crea sitios web de aspecto profesional gracias a la compatibilidad con una amplia gama de tecnologías de desarrollo web, como RSS, CSS, XHTML, PHP, ASP.NET, ColdFusion y JSP. Puedes sacar partido a un entorno de diseño interactivo que ofrece sencillos asistentes junto con inspectores de objetos HTML y CSS avanzados.
 
Diseño web con arrastrar y colocar
 Puedes situar imágenes, texto u otros elementos de página exactamente donde desees gracias a la función de diseño con arrastrar y colocar.
 
Editor de línea de tiempo
 El editor de línea de tiempo te permite dar vida al contenido estático del sitio web, como texto, fotografías, gráficos y vídeo. Puedes crear animaciones sofisticadas muy fácilmente en el familiar entorno de edición con tan sólo arrastrar y colocar.
 
Asistente para sitios
 Crea sitios web modernos con alto impacto visual en tres sencillos pasos. Website Creator incluye decenas de nuevos SiteStyles, plantillas con diseños de página modernos y plantillas para sitios web móviles optimizadas para su visualización en dispositivos móviles con iOS y Android.
 
La sencillez de arrastrar y colocar
 Arrastra y coloca cualquier texto, imagen o elemento de página exactamente en el lugar que deseas. Añade vídeos Flash, películas QuickTime y efectos JavaScript con unos pocos clics.
 
Editor de gráficos
 Recorta imágenes, corrígeles el color y cámbiales el tamaño con toda facilidad.
 
Administración de navegación sencilla
 Desplázate rápidamente por tu sitio sin salir de la vista de página gracias a la paleta de navegación del sitio. Cuando añades, eliminas o reorganizas páginas, todos los vínculos de navegación y de páginas se actualizan de manera automática.
 
Seguridad y mejoras integradas
 Disfruta de protección con contraseña, lenguajes PHP y ASP, formularios interactivos, libros de visitas, RSS y mucho más, sin necesidad de programación ni scripts personalizados. ¡El administrador de componentes te ayuda a localizar rápidamente todos los elementos que necesitas para la creación de sitios web interactivos!
 
Compatibilidad con AJAX
 Crea sitios web basados en AJAX con fichas, paneles de conmutación y asistentes para ofrecer a los usuarios una experiencia totalmente interactiva en el sitio web.
