Volver a usar msn

  • Categoría: Software / Programas
  • Publicado hace más de 8 años
Encontré esta información en la red y la quise compartir para recordar el viejo MSN

Entra en la pagina de Escargot

Escargot es un servidor, que simula los servidores de Microsoft y nos permite usar el MSN De la version 7.5 hacia abajo, y pronto se viene para Windows Live Messenger.

Entonces entras al sitio web de Escargot: https://escargot.log1p.xyz



Una vez alli dentro nos vamos a encontrar con una pagina web, que en el lado derecho no pone las versiones que se pueden usar, y en el izquierdo presionareos el boton de "Start Chatting", no se preocupen por el idioma que la aplicacion viene en español.



Una vez alli, hacemos un scroll hacia abajo para quedar en esta parte:



Alli se van a registrar y no se preocupen no hay correo de verificacion y lo puedes hacer tanto con GMAIL, YAHOO, OUTLOOK, HOTMAIL, o cualquier cliente de correo. Luego pones tu contraseña y es importante marcar el cuadro que esta alli, ¿Por que?: Ese cuadro permite la compatibilidad con todas las versiones antiguas de MSN (1, 2, 3 ,4). Y Luego le das a crear cuenta.

Luego de eso la pagina se reinicia y te notifica que tu cuenta se ha creado, Ok luego de eso vamos a bajar un poco hasta llegar a esta pantalla:



Una vez alli vamos a bajar para buscar la parte que pone "Spanish".



Una vez alli podemos ver que hay varias versiones de MSN, 5, 6 y 7 Vamos a clickear la que esta marcada (OJO: Bajar las versiones que pone "Pre-Patched", eso evita que tengas que parchear el MSN, de todas manera mas abajo explicare como parchearlo si te bajaste la version Sin Parche). Una vez hecho esto vamos a bajar hasta casi el final de la pagina y caer ene este sitio:



Este Paso es solo para los que bajaron la version "UnPatched" los demas pueden seguir al siguiente paso, Una vez estando alli, clickeamos a donde se esta marcando la flecha y nos bajara un ".DLL", No es ningun virus es el parche para el MSN.



Una vez terminado cerramos el navegador y vamos a donde tengamos nuestro instalador y para los que bajaron la "Unpatched", El instalador y el DLL.

A Partir de aqui hasta que termine la instalacion sigan los pasos de las imagenes.






Una vez terminada la instalacion, ¡LOS QUE BAJARON La version "Unpatched", deben copiar el .dll a la carpeta del MSN.(En mi caso se instalo en Program Files (x86) por que uso 64 Bits, sin embargo a ustedes dependiendo de que usen estara en Archivos de Programa.





Una vez hecho eso abrimos por Primera vez el MSN, Y Alli veremos la gloria...

Ok Nos logueamos con la cuenta que creamos antes y de manera magica, Tendremos de vuelta el MSN,




Si quieres charlar un rato, agregarme a tu lista de contactos puedes dejar tu cuenta acá abajo en los comentarios para que los demás te agreguen y tener así amigos con quien charlar .
