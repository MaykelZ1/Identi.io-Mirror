W7 Ultimate SP1 x64 Actualizado Septiembre 2017

  • Categoría: Sistemas Operativos
  • Publicado hace más de 8 años








Windows 7 SP1 (7601.23889) 64bit ultima versión final RTM, en este caso solo la edición Ultimate, actualizado completamente hasta el 17 septiembre, hecho completamente por mí a partir de la iso original de la MSDN.




Información:

- Esta ISO solo se basa en la edición Windows 7 SP1 Ultimate.
- Solo contiene el idioma español España (ES-es).
- Se agregaron todas las actualizaciones a la fecha que se pueden obtener vía Windows Update las cuales son: Actualización Acumulativa 2017-09, Actualizaciones de Seguridad, Actualizaciones de Corrección de errores (HotFix), Net Framework 4.7 (Este se instala automáticamente al actualizar Windows 7).
- Son 197 actualizaciones en total, no se agregan algunas porque ya están incluidas en las Actualizaciones Acumulativas.
- No se hizo ningún tipo de modificación.
- No contiene ningún tipo de software y/o programa.
- No viene activado (Recomiendo usar Windows Loader para activar, incluído en  la descarga)
