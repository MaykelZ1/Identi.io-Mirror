  • Categoría: Juegos de PC
  • Publicado hace más de 10 años
Bueno les dejo el juego de crysis 3, para todos aquellos que gustan de los mejores videojuegos de nuestros tiempos




No dejaré descripción ni nada de eso por que obviamente si estas aquí es por que ya sabes de que juego estamos hablando, lo que si dejaré son los requisitos mínimos del pc, así que si tienen un dinosaurio de computadora ya estuvo que no lo jugaran 


Requisitos mínimos para crysis 3


Requisitos mínimos:

? Windows Vista, Windows 7 o Windows 8

? Tarjeta DirectX 11 con 1Gb RAM

? Dual core CPU

? 2GB Memoria (3GB en Vista)

? (Nvidia/Intel): 

? Nvidia GTS 450

? Intel Core2 Duo 2.4 Ghz (E6600)

? AMD):

? AMD Radeon HD5770

? AMD Athlon64 X2 2.7 Ghz (5200+)


Requisitos recomendados:

? Windows Vista, Windows 7 o Windows 8

? Tarjeta DirectX 11 con 1Gb RAM

? Quad core GPU

? 4GB Memoria

? Nvidia/Intel):

? Nvidia GTX 560

? Intel Core i3-530

? AMD):

? AMD Radeon HD5870

? AMD Phenom II X2 565


Requisitos alto rendimiento:

? Windows Vista, Windows 7 o Windows 8

? Tarjetas gráficas recientes DirectX 11

? Quad core CPU reciente

? 8GB Memoria

? (Nvidia/Intel):

? NVidia GTX 680

? Intel Core i7-2600k

? AMD):

? AMD Radeon HD7970

? AMD Bulldozer FX4150
