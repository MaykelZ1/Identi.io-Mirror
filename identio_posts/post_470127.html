PS2|PES 2018 Top Game|ISO|Parche|UB|












El juego de Pro Evolution Soccer 2018 (TOP Games ) con el Campeonato Brasileiro 2017 - Series A y B, además de las ligas de Inglaterra, España, Holanda, entre otras. El parche fue lanzado el 27 de mayo de 2017 y contiene la base de datos (uniforme y escaladas) hasta esta fecha. El brasileño 2017 cuenta con los cuatro clubes que subieron a la primera división: Vasco, Bahía, Avaí y Atlético Goianiense. El PES 2018 by TOP Games se trata de un PATCH, hecho con base en el Pro Evolution Soccer 2014 (PES 2014).
