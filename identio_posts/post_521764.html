ParrotSec El OS Para Hackers



Parrot Security OS (o ParrotSec) es una distribución de Linux basada en Debian con un enfoque en la seguridad informática. Está diseñado para pruebas de penetración, evaluación y análisis de vulnerabilidades, análisis forense de computadoras , navegación web anónima, y practicar criptografía

El sistema incluye un arsenal completo de herramientas orientadas a la seguridad para cubrir muchas categorías del trabajo de un pentester.


Requisitos

CPU: x86 con minimo 700Mhz.
 
Arquitectura: i386, amd64 (x86-64bit), 486 (legacy x86), armel, and armhf.
 
RAM: minimo 256MB para i386 y 320MB para amd64. 512MB recomendado.
 
GPU: No graphic acceleration required.
 
HDD: ~16GB requerido para la instalación
 
BOOT: Legacy bios preferido

Herramientas y Utilidades
·              I2P: una network de peer to peer que permite una comunicación anónima

 
·              Tor Browser: navegador del sistema tor(super seguro)

 
·              TorChat

 
·              Wireshark: herramienta para monitorear el trafico de una red

 
·              XHydra: password cracker con interfaces gráfica

 
·              Zenmap: herramienta para escanear redes

 
·             Anon surf Anonymous mode start: asegura tu sistema, serrando todas la aplicación que puedan comprometer tu computadora, y te permite navegar redes como tor de forma más segura
