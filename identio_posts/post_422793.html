Windows 7 Ultimate Lite V3. 

Descripción:
Esta es una version modificada del Windows 7 original, donde se eliminaron componentes 
para dejar un sistema mucho mas optimizado y de menor peso 
para ser instalado en maquinas con tecnologías mas antiguas, 
netbooks, notebook y PCs de ultima generación.
La combinación perfecta de rendimiento y rapidez.

Cambios:
Drivers genericos WiFi y LAN incluidos
Soporte para Modems y Capturadoras de video
actualizaciones desde Windows Update,  impresora y escaner funcionando.

Se ha eliminado:
Drivers (excepto WiFi y LAN)

Media Center
Movie Maker
Restaurar sistema
Paquetes innecesarios que contiene Windows 7
Salvapantallas
Wallpapers
Entre otros...
