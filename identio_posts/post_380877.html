Guia Para Saber que Especificaciones tiene mi pc CPU,GPU RAM,OS X86, X64


Como bien saben no nacemos sabiendo y como muchos de nosotros por primera vez optenemos una Computadora PC ó Laptop y no sabemos que Especificaciones tiene.Pues bien aqui sabremos que es exactamente lo que tiene nuestro querido equipo de Computo, asi como:
- Procesador "CPU"
- Memoria "RAM"
- Sistema operativo "O.S."
- Tipo de Sistema ó Arquitectura x86 = 32bits y x64 = 64bits
- Tarjeta de Video + Vram "GPU"

Pasos seguir
1.- Anticlick (click derecho) en Este Equipo("Mi PC" en Windows XP)
2.- Propiedades


   Aqui se mostrara 
    - Sistema Operativo
    - Procesador "CPU"
    - Memoria "RAM"
    - Arquitectura -[/size][/color]Procesador x64 significa que soporta x64.


Para Mostrar Tarjeta de Video + VRAM- Método 1

  1.1 En cualquier parte vacia del Escritorio dar Anticlick y en Resolución de Pantalla.

  1.2 Configuración Avanzada
  1.3 Adaptador(pestaña)
      Aqui esta ,Tipo de Chip:en mi caso Geforce GT520
      Memoria Dedicada de Video VRAM:en mi caso 1024MB = 1GB

  Para Mostrar Tarjeta de Video - Método 2

2.1 Pulsar sin soltar Tecla Windows + R y Digitar DXDIAG y Aceptar.


2.2 Pantalla (pestaña)
Aqui esta Tipo de Chip: Geforce GT520 en mi caso.


Si volvemos a la pestaña Sistema también nos mostrara lo siguiente:
- Sistema Operativo y Arquitectura
- Procesador CPU
- Memoria RAM


Notas
Algunas Notas

-Sistemas  x86 ó 32bits 
Solo Soportan 3.2GB , 3.68GB hasta 4GB de RAM (depende del sistema operativo)

-Sistemas  x64 ó 64bits
 Solo Sopotan 4GB hasta 128GB e incluso 512GB de RAM (depende del sistema operativo)
Mas Información https://msdn.microsoft.com/en-us/library/windows/desktop/aa366778(v=vs.85).aspx

- Para saber Exactamente el modelo de nuestro hardware Aida64 nos lo muestra. http://www.aida64.com/downloads

Preguntas y Respuestas
Algunas Preguntas y Respuestas

¿Puedo ejecutar programas de 32bits en uno de 64bits?
  Si puedes,excepto algunos Controladores de Dispositivos(Driver Hardware) como Impresoras que solo estan disponibles en 32bits podrian no funcionar correntamente.

¿Puedo ejecutar programas de 64bits en uno de 32bits?
  No Funcionará,si el programa esta especificamente diseñado para 64bits.

¿Puedo instalar windows 64bits si tengo 3GB de ram?
 Es recomendable windows 32bits.

¿Puedo instalar windows 64bits si tengo actualmente 32bits?
 Si ,siempre en cuando tengas 4gb de Ram y  que soporte la CPU 64bits.
