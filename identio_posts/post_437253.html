(PC) Forza Horizon 3 (Full) (UB)




En Forza Horizon 3, que está a cargo del Festival Horizon. Personaliza todo, contratar y despedir a sus amigos, y explorar Australia en más de 350 de los mejores coches del mundo. Haga su horizonte la celebración definitiva de los coches, la música, y la libertad de la carretera. ¿Cómo se obtiene no es de usted.

NOTAS:

    -Este Método sólo funciona en la versión de Windows 10. 14936 de que pueda obtener el iso aquí.
    -La Herramienta que se utiliza para ejecutar el juego -ImmersiveHost- no está terminado.

INSTALACIÓN:

- Descarga los archivos del juego por debajo de los enlaces y el archivo de certificación.
- Abrir Powershell y el tipo de estos dos comandos, cambiando las rutas de acceso a los correctos:
Certutil -addStore TrustedPeople "c: \ PATH_TO_KEY \ MyKey.cer"
Add-AppxPackage "C: \ path_to_app \ forza_14.2.appx"
- Instalar Visual Studio 2015 Comunidad con los paquetes de C ++.
- Extraer y ejecutar el IH.7z runme.bat
- Copia este archivo en la carpeta Windows \ System32.
- Ahora Ejecutar el RUNME2.bat como Administartor
- Abrir la aplicación de Xbox Live, cree una cuenta ficticia, y configurarlo para acceder sólo para la aplicación XBL.

Nombre: Forza Horizon 3
Género: Carreras
Editorial: Microsoft
Desarrollador: Turn 10 Studios
Fecha de Publicación: 27 de septiembre el año 2016
Tamaño: 48.7 GB

REQUISITOS

OS : Windows 10 Anniversary Edition (64-bit, Version 14393.101)
CPU : Intel Core i7-3820 (3.6 GHz, Sandy Bridge E)
GPU : Nvidia GeForce GTX 970 or 1060 / AMD R9 290X or RX 480
RAM : 12 GB
Storage : 55 GB
DirectX Version : Version 12
