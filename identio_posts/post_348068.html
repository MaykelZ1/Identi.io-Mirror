Bueno amigos como pueden ver ya tenemos nuestro instalador solo nos falta crear el Autorun que es algo muy sencillo.


 
  

  
 





Fase 3: AutoPlay

 
Ahora Abrimos el AutoPlay Y creamos un nuevo Proyecto ya creado nuestro nuevo proyecto damos clic derecho y vamos a propiedades.




 
  

  
 




 Ahora vamos a ajuste personal y elegimos imagen.


  

  
 



 Ahora agregaremos botones a nuestra Imagen.


  

  
 



 
  

  
 



 Ahora solo queda asignarle a cada botón una tarea para eso damos clic derecho sobre el botón y le damos a propiedades.


  

  
 



 Ya estando en propiedades vamos a dar clic a acción rápida.


  

 

 



 Vamos a seleccionar la opción Run Program.
 
  



 



 Ahora solo buscamos la setup del repack que creamos.


  



 Si quieres que habrá un documento de texto solo elijes la opción open document, si quieres crear opción de salir elige exit/close.

 
 



 Ahora guardamos nuestro AutoRun donde guardamos nuestro Repack.


  

  
 



 Ahora que guardamos nuestro proyecto veremos esto.
  

  Tenemos que cortar del setup para abajo todos esos archivos en la carpeta AutoPlayDocs osea en la carpeta docs dentro del autoplay donde se encuentran los archivos. Les tiene que quedar algo así.



Ahora vamos a Probarlo
