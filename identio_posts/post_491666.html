Vegas Pro 15 (Multi-Idiomas)



VEGAS Pro le proporciona todas las herramientas necesarias para editar sus vídeos rápidamente. Codifique para entregas en 4K o publique en su canal de streaming favorito. O, cree un disco Blu-ray para entregar su producción en HD con sonido Dolby Digital. Sea cual sea su destino final, VEGAS Pro le llevará allí más rápido que nunca.
Trabaje más eficientemente que nunca con una interfaz personalizable y herramientas de edición precisas y rápidas. Acelere los tiempos de producción y entrega gracias a la edición y creación de discos mediante arrastrar y soltar. Mezcle, combine y edite los formatos más populares, incluyendo HEVC, ProRes, AVC y otros, directamente en la escala de tiempo y sin tener que esperar a una reconversión o transcodificación. La potente aceleración de hardware aprovecha las tecnologías NVIDIA e Intel QSV para acortar el proceso de renderizado. Diseñado y construido específicamente para ahorrarle tiempo y potenciar su creatividad.

Requisitos del sistema
 Sistema operativo: Microsoft® Windows 7 (64 bits), Windows 8 (64 bits) o Windows 10 (64 bits)
    Procesador: 2 GHz (se recomiendan varios núcleos o procesadores para HD o 3D estereoscópico; se recomiendan 8 núcleos para 4K)
    RAM: 4 GB de memoria RAM (se recomiendan 8 GB; 16 GB para 4K)
    Espacio en el disco duro: 1,5 GB de espacio en el disco duro para la instalación del programa; Disco de estado sólido (SSD) o multidisco de alta velocidad RAID para medios de 4K Tarjeta gráfica: GPU de NVIDIA®, AMD/ATI® o Intel® con al menos 512 MB de RAM (se recomienda 1 GB para 4K y procesamiento de vídeo acelerado por GPU)

Idiomas del programa:
English, Deutsch, Español, Français

GPU compatibles:

NVIDIA
Se recomienda 1 GB para formatos 4K, necesario para zoom inteligente, escala inteligente, desentrelazado adaptativo inteligente y para el procesamiento de vídeo acelerado en GPU.

Para el renderizado vía hardware (NVEnc):
 a partir de la serie GeForce 600 y siguientes (6xx, 7xx, 9xx, 10xx).
    Quadro Kxxx, Mxxx y Pxxx

AMD/ATI
Requiere una GPU con OpenCL habilitado y un controlador Catalyst 11.7 o posterior con una GPU Radeon HD 57xx o superior. Si usa una GPU FirePro, se requiere un controlador FirePro unificado 8.85 o posterior.

Intel
Es necesario un procesador Intel Skylake o superior para decodificación y codificación HEVC/AVC de 8 bits acelerada QSV, Intel KabyLake o superior para decodificación y codificación HEVC de 10 bits acelerada QSV
Formatos compatibles

Importación:
AAC, AA3, AVI, BMP, MPEG-1, MPEG-2, MPEG-4, WMV, HEVC, ProRes, AAC, AIFF, FLAC, LPEC, MP3, OGG, PCA, W64, WAV, WMA, BMP, JPEG, PNG, TIFF, WDP, WMPhoto, DDS, DPX, EX, AAF, Premiere/After Effects (*.prproj), Final Cut Pro 7/ DaVinci Resolve (*.xml), Final Cut Pro X (*.fcpxml), EDL Text File (*.txt), Broadcast Wave Format, DVD Camcorder Disc, subtitulado Closed Captioning

Exportación:
AAC, AA3, AVI, BMP, MPEG-1, MPEG-2, MPEG-4, WMV, HEVC, ProRes, AAC, AIFF, FLAC, LPEC, MP3, OGG, PCA, W64, WAV, WMA, BMP, JPEG, PNG, TIFF, WDP, WMPhoto, DDS, DPX, EXR, VEGAS Project Archive (*.veg), Pro Tools AAF File (*.aaf), Media Composer AAF File (*.aaf), Premiere/After Effects (*.prproj), Final Cut Pro 7/DaVinci Resolve (*.xml), Final Cut Pro X (*.fcpxml), EDL Text File (*.txt), disco XDCAM
