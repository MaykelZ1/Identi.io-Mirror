Jugar Online Cooperativo


Es Super Fácil Jugar Online Cooperativo, solo tiene que seguir estos pasos:


1. Tienen que descargar el programa que van a ocupar para jugar el juego. Este es el GameRanger y tiene que ser la ultima version.

2. Una ves descargado el programa se tienen que registrar o crear una cuenta.

3. Después que se hayan registrado el programa se les abrirá. Hay se van a la opción que dice ALL GAMES y seleccionan My GAMES. Esta opción buscara los juegos compatible con el programa que estan instalados en su PC (obviamente tiene que estar instalado el juego antes).

4. Para crear una partida o host. Se van a la opción FILE y HOST GAME..., esto creara una partida.
Hay mismo le aparecerá una pantalla con varias opciones y solo colocan los datos que quieran. (EJ: GAME ponen Borlerdans, en MAX PLAYERS cuantos de tus amigos van a jugar, en DESCRIPTION se pone el nombre del Host que no debe cambiar nunca y en PASSWORDS una contraseña si quieren).

5. Una ves creada el Host o la partida, solo esperan que tus amigos hagan los mismos pasos y se conecten al Host que acabas de crear.

6. Solo queda jugar, aquí el creador del Host le tiene que dar a la opción STAR y automáticamente se les iniciara el juego y no deben hacer nada, solo esperar.

7. disfrutar de jugar Online Cooperativo con tus amigos (Es recomendable jugar así, por que es mas entretenido el juego).

8. Esto es importante todos los que vallan a jugar la partida, tienen que tener la misma version que el que creo el Host o la partida.
