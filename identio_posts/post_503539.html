> > > > INFORMACION < < < <

--La Game Boy Advance posee un catálogo de unos 1000 juegos diferentes que salieron desde el 2001 al 2008, hubo juegos de todos los géneros para esta consola, hubo muchos juegos cancelados para el sistema siendo los más destacables Resident Evil 2 y Grand Theft Auto 3.

Uno de los juegos que no se eliminó, más conocido fue el Metal Slug Advance.

La consola también es compatible con todos los juegos de Game Boy y todos los de Game Boy Color lo que le da 730 juegos más en su catálogo teniendo un total de 1730 juegos diferentes. Y actualmente se esta incorporando la versión del emulador GBA con filtro HD y con versiones de 32, 64bits por primera vez.

Cuenta con un procesador ARM propio de 32bits a 16,7MHz, basado en la arquitectura RISC, con una potencia suficiente para permitir el desarrollo de juegos utilizando el lenguaje de programación C.

El microprocesador ARM es capaz de ejecutar tanto un juego de instrucciones con un tamaño de instrucción de 32bits, como un juego de instrucciones llamado "Thumb" de un tamaño de 16bit (pero igualmente, de 32bits).

El sistema, además, lleva un procesador Z80 a 8MHz para dar soporte al software de Game Boy clásica y color, pero sus dos procesadores no pueden estar funcionando a la vez debido a diferencias de voltaje y uso del bus de datos--

> > > > INSTRUCCIONES DE CONFIGURACION < < < <

1. Primero tenemos que descargar el enlace subido a MEGA o GOOGLE DRIVE.

2. Una ves descargado el archivo .RAR, le darán Click Derecho y descomprimir.

3. Cuando se haya descomprimido los archivos, les quedara un Block de texto y una carpeta llamada "Emulador de Game Boy Advance (GBA)".



4. Ahora le darán Doble Click a la carpeta y una ves dentro encontraran una gran variedad de archivos, el que vamos a utilizar es el archivo .exe llamado "mGBA" y simplemente lo ejecutan.



5. Una ves ejecutado, se les abrirá el tablero del emulador de Game Boy Advance (GBA) mGBA.



6. Simplemente Comenzamos con la configuración de este emulador, para eso lo primero nos dirigimos a ->Herramientas ->Ajustes...



7. Se nos abrirá una pestaña nueva, seleccionan la opción ->Audio/Vídeo y dejaran tal cual como les aparece en la imagen.



8. Luego seleccionan la opción ->Controladores, aquí podrán configurar el teclado o el joystick que tengan para emular los botones de la Consola portátil Game Boy Advance (GBA).





9. Con Por ultimo seleccionan la opción ->Shader, aquí cargaran los shader que mas le acomode para su equipo.



10. Para cargar los shader simplemente le dan Click al parámetro "Cargar Nuevo Shader".



11. Se nos abrirá una pestaña nueva, y como les mencione seleccionan los Shader que mas le acomode para su equipo.



12. Continuando con la configuración, nos dirigimos a la pestaña ->Audio/Video ->Objetivo de FPS y seleccionan 60fps.



13. Ahora Por ultimo vamos a ejecutar un juego para ver si el emulador quedo bien configurado y sin problemas, para esto se dirigen a la pestaña ->Archivo ->Cargar ROM...



14. Se nos abrirá una nueva pestaña y simplemente buscan la carpeta donde tienen sus juegos de Super Nintendo (SNES).
