Una vez ya estén en Windows 10 se dirigen a configuración y activación, les aparecerá algo como esto:

Ahora descargan desde el enlace que les dejare el Archivo KMSW10.rar y DESACTIVAN WINDOWS DEFENDER (MUY IMPORTANTE) para hacerlo van a la barra de búsqueda y ponen windows defender lo abren y van a donde dice configuración y desactivan la protección en tiempo real.(Tranquilos no hay ningún virus.) posteriormente lo descomprimen. Hecho esto ejecutan el KMSpico_setup.exe(click derecho sobre el icono y ejecutar como administrador) después siguen lo pasos de instalación.

Ya terminado de instalar buscan en la barra de búsqueda: kms y les saldrá el programa dan click derecho y ejecutar como administrador.

Presionan ese botón esperan un poco y les saldrá el logo de windows en verde y les dirá que ya acabo.
