Facebook tiene un juego de ajedrez oculto

  • Categoría: Ciencia y Tecnología
  • Publicado hace más de 9 años


mucho tiempo que las cosas se descubren y en Facebook y aunque hay muchos aficionados a jugar al ajedrez atrás, Facebook ha ocultado una opción para impugnar la persona que está conversando con una partida de ajedrez.

Este juego no se activa con un botón si un código " f@fbchess play ". El hecho de enviar este comando a un amigo, un tablero de ajedrez aparece y se puede empezar a jugar.Desafortunadamente, no podemos utilizar el ratón para mover las piezas, los movimientos serán controlados por comandos.

K para el rey.
Q para la reina.
B para obispo.
N para el caballo.
R para la torre.
P para el peón.
Así que si hacemos un movimiento que mustfbchess seguimos nuestro juego, por examplefbchess Pd3, esto servirá para mover el peón de cuadrar D3 y así terminar el juego.


Por desgracia, es un sistema mecánico, pero nos ayuda a comprender el idioma utilizado en este juego. Cuando es nuestro turno de recibir una notificación y puede jugar tanto en la web y aplicaciones móviles.
