Media Preview v.1.4.3.429






Si estás cansado del Explorador de Windows que no siempre muestra las miniaturas de todos los archivos multimedia, les traigo un programa que permite visualizar las miniaturas de archivos de video, audio y de imagen más comunes.

Este programa se llama Media Preview, en su versión 1.4.3.429.

Con esta herramienta prodrás personalizar las ventanas del explorador de Windows para darle una apariencia estética más bella.

Además de darle una apariencia personalizada te permite distinguir rápidamente de que archivo se trata.


En los links de descarga dejo además un tutorial para configurarlo.
