Ubuntu 15,04 Vivid Vervet & Derivados Final

  • Categoría: Linux
  • Publicado hace más de 10 años


Disponible Ubuntu 15.04 & Derivados
















Recibimos al cercopiteco verde o Chlorocebus pygerythrus como se merece, pues este primate africano tan vivaz representa hoy el espíritu tras Ubuntu 15.04 ?Vivid Vervet?, cuya versión final ya está disponible para su descarga.

Ubuntu 15.04 ?Vivid Vervet? se presenta con pocos cambios, estando Canonical más pendiente de Ubuntu Phone, pero acusados, y la gran novedad de este lanzamiento es sin duda la adopción de systemd como reemplazo de Upstart en las labores del arranque del sistema. Habemus systemd, sí.

En cuanto a generalidades, Ubuntu 15.04 incluye el kernel Linux 3.19.3, GTK 3.14, Firefox 36, LibreOffice 4.4 e, integrado en los repositorios oficiales, Ubuntu Make, un proyecto para facilitar a los desarrolladores el acceso a las herramientas de programación más demandadas entre las que se cuentan Android NDK, Android Studio, IDEA, Golang, Firefox developer edition, Dartlang, Stencyl y otras.


De las generalidades a las particularidades nos encontramos con:



Ubuntu:

Unity 7.3 reincorpora los menús de ventana integrados y habilita el Dash, el HUD y los diálogos de sesión para que se muestre sobre las ventanas a pantalla completa, además de pequeños retoques aquí y allá.
Web Oficial -->> www.ubuntu.com


Kubuntu:

Kubuntu 15.04 da el salto anunciado y se viste de Plasma 5.2, con las KDE Applications 14.12 como guarnición. Que dicho así, en un par de líneas, parece poca cosa, pero nada más lejos de la realidad: hay mucha tralla ahí y posiblemente éste sea el sabor más interesante de todos en esta versión.
Web Oficial -->> www.kubuntu.org


Xubuntu:

Xubuntu 15.04 no es menos y también actualiza su entorno de escritorio, en este caso, a Xfce 4.12. Y de nuevo pequeños retoques aquí y allá en el apartado visual que tienen un gestos con las aplicaciones Qt, que ahora se verán mejor integradas.
Web Oficial -->> www.xubuntu.org


Lubuntu:

Lubuntu 15.04, siendo como es el ultraligero de la familia, se conforma con correcciones y actualizaciones en LXDE que, como sabemos, se desarrolla a su propio ritmo.
Web Oficial -->> www.lubuntu.net


Ubuntu GNOME:
Ubuntu GNOME 15.04 sigue a rebufo de la edición principal y GNOME 3.14 es lo que trae, casi al completo. Los paquetes que faltan están en el GNOME3 Team PPA y advierten: para instalar paquetes, el Centro de Software de Ubuntu, no GNOME Software.
Web Oficial -->> www.ubuntugnome.org


Ubuntu MATE:

Ubuntu MATE 15.04 lanza su primera versión oficial dentro de la familia Ubuntu, incluye soporte para PowerPC y Raspberry Pi 2 y un nuevo tema visual, entre otros cambios.
Web Oficial -->> www.ubuntu-mate.org


En cuanto a otros sabores como Ubuntu Kylin, Ubuntu Studio, etc, también están saliendo, así como las ediciones profesionales para servidores y cloud o Ubuntu Core, que es donde si hay cambios notables de acuerdo a las notas de lanzamiento.
