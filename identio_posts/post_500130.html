> > > > INFORMACION < < < <

--La PlayStation fue lanzada a la venta el 3 de diciembre de 1994 en Japón, el 9 de septiembre de 1995 en EE. UU. y el 29 de septiembre de 1995 en Europa. Todo empezó con un contrato roto con Nintendo a finales de la década de 1980.

Nintendo acordó con Sony, a finales de los ochenta, desarrollar para su exitosa Super Nintendo un apéndice para incorporar juegos en CD, además del tradicional cartucho. El gigante de los videojuegos, sin embargo, rompió con la tecnológica japonesa, neófita entonces en la industria, porque consideraba que cedía demasiado en el control y beneficios derivados de la venta de juegos en CD.

ePSXe utiliza un sistema de plugins, inventado por PSEmu. Este sistema permite a usuarios y desarrolladores ajenos al emulador programar y utilizar sus propios plugins (añadidos requeridos para que ePSXe pueda funcionar). ePSXe permite plugins que emularán la GPU, (unidad que se encarga de generar los gráficos) SPU, (Para la música y los efectos de sonido) y el lector de CD-XA del sistema.

GPU: Los plugins que emulan este componente funcionan Direct3D, OpenGL, Glide, XVideo, MesaGL o renderización por Software. Destacan, entre otros, los plugins de Pete Bernert.

SPU: Para sistemas Windows, todos funcionan bajo DirectSound, y algunos más antiguos bajo MIDAS. En el caso de los sistemas que funcionan bajo Linux, encontramos plugins para ALSA y OSS.

CD-ROM: ePSXe incorpora sus propios plugins para ASPI en el caso de sistemas Windows 95/98/ME, e iocontrol para sistemas Windows NT/2000/XP/Vista/7. Entre las mejores alternativas encontramos, de nuevo, los plugins de Pete Bernert y el de SaPu, los cuales permiten hasta siete tipos diferentes de lectura--

> > > > INSTRUCCIONES DE INSTALACIÓN < < < <

1. Primero tenemos que descargar el enlace subido a MEGA o GOOGLE DRIVE.

2. Una ves descargado el archivo .RAR, le darán Click Derecho y descomprimir.

3. Cuando se haya descomprimido los archivos, les quedara un Block de texto y una carpeta llamada "Emulador de PlaySation (PSX)".



4. Ahora le daran Doble Click a la carpeta y una ves dentro encontraran una gran variedad de archivos, el que vamos a utilizar es el archivo .exe llamado "ePSXe" y simplemente lo ejecutan.



5. Una ves ejecutado, se les abrirá el tablero del emulador de PlayStation (PSX) ePSXe.



6. Simplemente Comenzamos con la configuración de este emulador, para eso lo primero nos dirigimos a ->Configurar ->Plugins ->Vídeo.



7. Se nos abrirá una pestaña nueva, aquí escogeremos el pluging que mas nos acomode a nuestro equipo. en mi caso es ->Pete's OpenGL2 Driver 2.9, y le dan a configurar.



8.una ves abierta la pestaña nueva, podrán configurar distintos parámetros de gráficos, resolucion, etc. si no saben para que sirve cada opción lo dejan tal cual como aparece en la imagen.



9. Continuamos con la configuración, para esto nos vamos a la pestaña ->Configurar ->Plugins ->Imput 1.



10. Se nos abrirá una pestaña nueva, aquí escogeremos el pluging por defecto para configurar el joystick y le dan a configurar.



11. una ves abierta la pestaña nueva, podran configurar el teclado o el joystick que tengan para emular los botones del control de PlayStation.



12. Ahora seguimos configurando, para esto nos vamos a la pestaña ->Opciones ->CPU Overcloking, y seleccionan el parametro "x1".



13. Para terminar vamos a ejecutar un juego para ver si el emulador quedo bien configurado y sin problemas, para esto se dirigen a la pestaña ->Archivo ->Ejecutar ISO.



14. Se nos abrirá una nueva pestaña y simplemente buscan la carpetas donde tienen sus juegos de PlayStation (PSX).
