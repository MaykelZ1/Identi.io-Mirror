*Windows Xp SP3 Pro Español,ISO 32 bits MEGA

* Actualizado a Noviembre del 2014 *





:: Descripción ::  


* Esta compilación está pensada para ofrecer una instalación desatendida de
*Windows XP SP3 edición Española lo más limpia y actualizada posible, incluyendo:
- Todas las actualizaciones hasta la fecha Noviembre del 2014 
- Pack de Drivers para Dispositivos de Almacenamiento (F6 text mode).
- *Packs de Drivers para Chipset, CPU, Gráficas, LAN y WLAN, Almacenamiento y Sonido.
- Internet Explorer 8 (sustituye a la versión 6).
- Windows Media Player 11 (sustituye a la versión 9).
- Microsoft .NET Framework 2.0 SP2 + 3.0 SP2 + 3.5 SP1 (como Componentes).
- Microsoft Visual C++ 2005 + 2008 + 2010.
- DirectX 9.0c (como Componente).
- Pack de Temas de escritorio para XP.
- Adobe Flash Player.
* Tras el primer reinicio, tardará unos minutos en descomprimir los packs
 de drivers (DriverPacks).
**La instalación parecerá detenerse sobre el minuto 23 debido a que es en
ese instante cuando preintegra los .NET Framework.
La ISO además incluye una carpeta de Archivos para que el usuario pueda, 
entre otras cosas, asignar/modificar su propia clave de Windows XP. 
Toda la información de la compilación en el archivo LEAME.HTML dentro de la ISO.
