Adobe Dreamweaver CC 2019 v19.0.1.11212 (x64) Multilingüe
Pre-Activado
Adobe Dreamweaver CC es una nueva versión de una aplicación diseñada para crear sitios web profesionales. El software de diseño web de Adobe Dreamweaver CC contiene una interfaz visual intuitiva que le permite crear y editar sitios web y aplicaciones móviles. Utilice "layouts de goma" (Fluid Grid Layouts) con soporte multiplataforma para crear páginas web. Antes de publicar, vea los diseños creados en el panel de vista multipantalla.

Sitios Web Adaptables
Realmente rápido. Dreamweaver CC le ofrece formas más rápidas y fáciles para diseñar, codificar y publicar sitios web y aplicaciones web que se ven increíbles en cualquier tamaño de pantalla. Cree y publique rápidamente páginas web casi en cualquier lugar con un software de diseño web que admita HTML, CSS, JavaScript y más.

Construye sitios para cualquier navegador o dispositivo
Crea rápidamente HTML, CSS, JavaScript y más.

Codificación rápida y flexible
Cree, codifique y administre sitios web dinámicos fácilmente con un motor de codificación inteligente y simplificado. Acceda a sugerencias de código para aprender y editar rápidamente HTML, CSS y otros estándares web. Y use ayudas visuales para reducir errores y acelerar el desarrollo del sitio.

Configuración para sitio en menos pasos
Ponga sus sitios en funcionamiento más rápido con plantillas de inicio que puede personalizar para crear correos electrónicos HTML, páginas Acerca de, blogs, páginas de comercio electrónico, boletines y portafólios. El color del código y las sugerencias visuales le ayudaran a leer el código más fácilmente para ediciones y actualizaciones rápidas.

Visualización dinámica en todos los dispositivos
Cree sitios web adaptables que se ajusten a cualquier tamaño de pantalla. Obtenga una vista previa de sus sitios y ediciones en tiempo real para asegurarse de que sus páginas se vean y funcionen como usted quiere antes de publicar.

Vea lo que puede hacer nuevo
Dreamweaver siempre está mejorando, con nuevas funciones que se desarrollan regularmente. Y con su membresía de Creative Cloud, obtiene las actualizaciones tan pronto como salen.

Soporte multi-monitor para Windows
Ahora puede ampliar su espacio de trabajo a múltiples monitores.

Integración de CEF
Dreamweaver ahora se integra con la última versión de Chromium Embedded Framework, para que pueda crear un sitio web HTML5 moderno y mostrar elementos, cuadrículas de CSS y mucho más.

Rediseñada con una moderna interfaz de usuario
Una interfaz optimizada y sin desorden le permite personalizar su área de trabajo para mostrar solo las herramientas que necesita para codificar.

Apoyo Git
La colaboración es fácil con el soporte de Git. Administre todo su código fuente dentro de Dreamweaver y realice todas las operaciones comunes directamente desde el panel de Git.

Novedades de Dreamweaver CC 2019


Actualizaciones de CEF
Dreamweaver ahora se integra con la última versión de Chromium Embedded Framework, por lo que los diseñadores y desarrolladores pueden crear un sitio web compatible con HTML5 y mostrar elementos de Flexbox, cuadrículas de CSS y mucho más.

Soporte ES6
La nueva compatibilidad con EcmaScript 6 incluye una lista de tipo rápido de clases, métodos, funciones de flecha y funciones de generador, así como la alineación del código ES6 para que pueda trabajar con las últimas actualizaciones de JavaScript.

Refactorización de Javascript
Use las funciones de renombrar y refactorizar para organizar de manera inteligente su código JavaScript.
