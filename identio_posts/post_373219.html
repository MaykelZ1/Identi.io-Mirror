MiPony (v2.2.4) Offline + Portable


Un gestor de descargas sencillo y potente





Mipony es un gestor de descargas especializado en descarga directa, es decir, que te ayudará a bajar contenidos de páginas como Rapidshare, Hotfile, Fileserve, Uploaded, Freakshare, Filefactory o Mediafire, entre otros muchos.

El proceso es muy sencillo, ya que puedes copiar uno o más enlaces a la vez. Mipony los analizará, y si funcionan, se iniciará la descarga automáticamente. Además, no tendrás que salir de Mipony para buscar enlaces, pues integra su propio navegador web. Mipony bajará los enlaces de uno en uno, consiguiendo velocidades altas y esperando por ti el tiempo entre cada descarga. Más tarde podrás comprobarlo con la pestaña Estadísticas, donde verás un gráfico con la cantidad descargada y el tiempo empleado. Otra función interesante de Mipony es la de unir varios ficheros comprimidos, los habituales en formato .001, y que sirven para separar contenido de gran tamaño en distintos archivos y facilitar así su descarga. Y para terminar, Mipony permite gestionar cuentas Premium de pago de estas páginas, evitando las limitaciones de las cuentas gratuitas.



Características

Con este gestor de descargas disfrutaras de grandes comodidades a la hora de llevar a cabo la descarga de archivos evitando así los inconvenientes de las páginas de hospedaje. Mipony se encarga de todo, accede a la web donde se encuentra el archivo, espera el tiempo necesario si se requiere e inicia la descarga de un archivo tras otro.

- Posibilidad de establecer prioridades para los ficheros en cola de descarga.
- Controla cuantos ficheros se pueden bajar simultáneamente de cada servidor.
- Muchos de los ficheros alojados en los servidores gratuitos se encuentran divididos en partes, Mipony tiene una opción para unir los ficheros descargados (HJSplit).
- Mipony puede captuar del portapapeles enlaces individuales o cualquier texto que contenga cualquier número de enlaces.
- Acualización automática de la base de datos con las páginas de alojamiento soportadas. Mipony se actualizará automáticamente para seguir funcionando con los sitios de descargas de f ficheros, además estamos añadiendo constantemente nuevos sitios de descargas soportados.
- Mipony puede analizar páginas completas en busca de enlaces, indicando si están aun disponibles en el servidor para descargar o si han sido borrados.
- Permite configurar Flashgot para enviar enlaces hacia Mipony.
- Plugin en formato barra para que desde Firefox y Explorer se puedan lanzar las descargas directamente en Mipony o abrir la página que se este viendo en el navegador de Mipony.


Descarga ininterrumpida

Una vez agregado el enlace de descarga a la lista de trabajo se descargaran los ficheros mientras seguimos navegando y, en el caso de cerrar el programa las descargas se retomaran al volverlo a iniciar.


Re intentos de descarga

Si por cualquier motivo un fichero da error descargando,Mipony se encargará de hacer los reintientos necesarios automáticamente hasta que el fichero se descargue.


Navegador

Lleva integrado un navegador desde el cual se detectan los enlaces de descarga de archivos.


Control remoto para Android

Se ha desarrollado una aplicación para Android para controlar Mipony en remoto con la que podrás manejar las descargas, introducir captchas, ser avisado de que hay captchas pendientes, añadir descargas, limpiar descargas completas y descargar archivos a tu móvil. Desde este enlace puedes descargar la aplicación.


Control remoto web

Mipony tiene un interfaz web para controlar de forma remota el programa. Desde de la interfaz web podrás ver el progreso de tus descargas y manipularlas y, además podrás introducir los captchas desde ella. El diseño del interfaz web esta adaptado tanto para tu ordenador como para móvil.


Plugins para Mipony para resolver captchas

- Imagetyperz.com
- Captcha Brotherhood
- Imagedecoders
- 9kw.eu (Recomendado)
