Requisitos Office 2013

-Procesador de 1 GHZ x86- o x64-bit como mínimo, si es mas rápido mucho mejor.
-1 GB de RAM  para 32 bit; 2 GB de  RAM  para 64 bit.
-3 GB de disco duro libres como mínimo.
-1024 x 576 px como resolución de pantalla, se recomienda 1366x786px.
-DirectX10 o una versión superior.
-Sistema operativo: Windows 7, Windows 8, Windows Server 2008 R2, o Windows Server 2012.
-3.5, 4.0, o 4.5 de .NET.
-Un navegador: Internet explorer, Chrome, Firefox, Safari, etc.
-Formato: ISO
-Peso: 770 MB(x32),890 MB(x64),48 MB(activador)


Como activar


-Ejecutar como administrador el activador.
-Elegir el logo de office.(Parte inferior derecha ,al lado del logo de windows)
-Ir a la pestaña Activation y dar en EZ-Activator.
-Esperar que el activador haga su trabajo y si no hay ningun problema tendra que salirle la siguiente imagen:

-Nos dirá que el producto se activo y que se pudo crear el archivo AutoKMS.exe ,responsable de que en cada inicio nos active el office por 180 dias.
-Talvez sea necesario desactivar el antivirus para que no haya problemas(Con Eset o MSE no hay problemas).
