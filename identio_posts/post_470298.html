Apariencia de MacOS en ubuntu

  • Categoría: Linux
  • Publicado hace más de 8 años



En esta ocasión les traigo un tutorial de como personalizar nuestro ubuntu y darle la apariencia de Mac OS.





Instalamos los paquetes que nos permitirán modificar todo lo necesario escribiendo en un terminal lo siguiente:

sudo apt-get install unity-tweak-tool
sudo apt-get install gnome-tweak-tool




Abrimos nuestra terminal y colocamos los siguientes comandos para obtener los temas de Macbuntu OS y themas.

sudo add-apt-repository ppa:noobslab/macbuntu
sudo apt-get update
sudo apt-get install macbuntu-os-icons-lts-v7
sudo apt-get install macbuntu-os-ithemes-lts-v7

Para hacer el camino de vuelta, en la terminal escribiremos:

cd /usr/share/icons/mac-cursors && sudo ./uninstall-mac-cursors.sh
sudo apt-get remove macbuntu-os-icons-lts-v7 macbuntu-os-ithemes-lts-v7





Una vez instalados, elegimos el tema, los iconos y el cursor desde Unity Tweak Tool

La alternativa al Launchpad se llama Slingscold y para instalarlo escribiremos lo siguiente.

sudo add-apt-repository ppa:noobslab/macbuntu
sudo apt-get update
sudo apt-get install slingscold




La alternativa a Spotlight se llama Albert y para instalarlo escribiremos lo siguiente.

sudo add-apt-repository ppa:noobslab/macbuntu
sudo apt-get update
sudo apt-get install albert





Para usar Albert tendremos que ejecutarlo, indicar qué atajo del teclado usaremos para invocarlo.

Instalamos Plank con el siguiente comando

sudo apt install plank

Instalamos los temas necesarios para que se parezca al Dock de Mac escribiendo lo siguiente.

sudo add-apt-repository ppa:noobslab/macbuntu
sudo apt-get update
sudo apt-get install macbuntu-os-plank-theme-lts-v7





Cambiamos el texto “Ubuntu Desktop” por Mac escribiendo lo siguiente.

cd && wget -O Mac.po http://drive.noobslab.com/data/Mac/change-name-on-panel/mac.po
cd /usr/share/locale/es/LC_MESSAGES; sudo msgfmt -o unity.mo ~/Mac.po;rm ~/Mac.po;cd

Comando para el camino de vuelta:

cd && wget -O Ubuntu.po http://drive.noobslab.com/data/Mac/change-name-on-panel/ubuntu.po
cd /usr/share/locale/es/LC_MESSAGES; sudo msgfmt -o unity.mo ~/Ubuntu.po;rm ~/Ubuntu.po;cd





Añadimos el logo de Apple en el Launcher escribiendo en una terminal lo siguiente:

wget -O launcher_bfb.png http://drive.noobslab.com/data/Mac/launcher-logo/apple/launcher_bfb.png
sudo mv launcher_bfb.png /usr/share/unity/icons/

Para volver al estado anterior:

wget -O launcher_bfb.png http://drive.noobslab.com/data/Mac/launcher-logo/ubuntu/launcher_bfb.png
sudo mv launcher_bfb.png /usr/share/unity/icons/





Instalamos los iconos Monochrome para LibreOffice. Para ello, abrimos una terminal y escribimos:

sudo apt-get install libreoffice-style-sifr

Después de la instalación, en LibreOffice vamos a Herramientas/Opciones/LibreOffice/Vista y seleccionamos “Sifr” que está en el apartado “tamaño del icono y estilo”.





Instalamos las fuentes de Mac abriendo una terminal y escribiendo lo siguiente:

wget -O mac-fonts.zip http://drive.noobslab.com/data/Mac/macfonts.zip
sudo unzip mac-fonts.zip -d /usr/share/fonts; rm mac-fonts.zip
sudo fc-cache -f -v
