los 21 errores que causan el problema de la pantalla azul y cómo solucionarlos
No hay duda que
el problema de la pantalla azul
es uno de
los problemas más frecuentes y graves
, Y que ya está causando problemas a muchos de nosotros, por esto he preparado este articulo donde intento darles soluciónes para cada uno de
los errores que causan este problema
Los problemas de la pantalla azul que hacen que el equipo deja de funcionar y esto es debido a un fallo técnico en el equipo, el cual debemos reparar, aquí les dejo 21 errores causantes de la aparición de la pantalla azul en el equipo y com solucionarlo:

1 ( Stop 0x0000000A IRQL_NOT_LESS_OR_EQUAL )

Este mensaje indica que uno de los drivers no es compatible con el sistema operativo, o que hay un virus entro a los archivos del sistema, para resolver el problema asegúrese de que su ordenador no contiene ningun virus y eliminar cualquier controlador recientemente instalado y instalar un controlador compatible con Windows XP.

2 ( Stop 0x0000001E KMODE_EXCEPTION_NOT_HANDLED )

Parece que hay un acceso anónimo a la memoria de una manera no es válido, y este mensaje aparece a menudo al instalar un controlador no compatible con Windows XP y para resolver este problema se debe desinstalar el driver que fue instalado recientemente.

3 ( Stop 0x00000024 NTFS_FILE_SYSTEM )

Este mensaje indica que hay un problema en el sistema de archivos NTFS y hay un mensaje similar termina con 23 y suele aparecer en sistema de archivos FAT32, y para resolver este problema deben abrir este comando "chkdsk.exe" y asegúrese de que la conexión de los cables entre los discos esta en buen estado.

4 ( Stop 0x0000002E DATA_BUS_ERROR )

Hay varias razones para la aparición de este mensaje, y el más común es un problema en la memoria del sistema o la memoria RAM, otras de las razones seria el disco duro y tiene que ser examinado y la última razón seria un fallo en la placa madre


5 ( Stop 0x0000003F NO_MORE_SYSTEM_PTES )

Este mensaje no es muy frecuente y es probablemente causado por el mal uso de la recuperación de datos en el programa de copia de seguridad "Backup", intente utilizar la característica Restaurar sistema para resolver el problema.


6 ( Stop 0x00000050 PAGE_FAULT_IN_NONPAGED_AREA )

Este mensaje indica que un Driver o servicios a convocado los datos de la memoria, pero no los encontró y lo que causa este problema puede ser un driver o un software no compatible con Windows XP, como algún tipo de software anti-virus, en este caso se debe desinstalar el driver o programa que aparecera en el mensaje de error.

7 ( Stop 0x00000077 KERNEL_STACK_INPAGE_ERROR )

Parece ser que el sistema no ha podido llegar a los datos necesarios de la memoria virtual y la causa de este problema se debe a varias razones, entre ellas problemas en la memoria que puedes ser un fallo en el disco duro, o incluso un virus que pudo haber infectado al disco duro del equipo.

8 ( Stop 0x00000079 MISMATCHED_HAL


Este mensaje indica que hay una falta de compatibilidad entre los archivos del sistema de Windows XP y la capa hardware abstraction layer>HAL de Windows este problema se produce a menudo en los equipos que funcionan con la especificación ACPI cuando la configuración de ACPI en la BIOS se cambia inesperadamente, y para resolver el problema tienen que re-instalar el Windows después de confirmar que en la configuración de la BIOS es todo bien.

9 ( Stop 0x0000007A KERNEL_DATA_INPAGE_ERROR )


Este mensaje es similar al Stop 0x00000077Bad blocks
y el problema es causado por una mala instalación o configuracion en los Drivers SCSI, o tener partes dañadas
en el disco duro, como puede ser tambien un conflicto entre la configuración de la BIOS y la controladora SCSI.

10 ( Stop 0x0000007B INACCESSIBLE_BOOT_DEVICE )

Este mensaje indica que Windows XP no reconoce las particiones del disco duro al arrancar el equipo, y para resolver este problema arranca la computadora desde el CD-ROM con un CD Windows XP, entrar en la consola de recuperación "
Recovery Console"

y escribir este comando Bootcfg.

11 ( Stop 0x0000007F UNEXPECTED_KERNEL_MODE_TRAP )

Este mensaje indica que hay un problema en el hardware y a menudo es causada por problemas en las memorias RAM o el sobrecalentamiento del procesador, que de su parte puede ser causado por un mal funcionamiento en el ventilador, o un intento de aumentacion de velocidad del procesador a un limite insoportable.

12 ( Stop 0x0000009F DRIVER_POWER_STATE_FAILURE )

Este mensaje indica un problema en el proceso de Desconeccion del equipo, y puede ser causado por un Driver o software no compatible con Windows,en la mayoria de los casos el mensaje viene acompañado por el nombre del Driver dañado, y hay que eliminarlo iniciando el sistema en modo seguro (Safe Mode).

13 ( Stop 0x000000C2 BAD_POOL_CALLER )

Parece que hay un intento fallido por un driver al intentar llegar a ciertas partes de la memoria, y hay que eliminar del último controlador instalado.

14 ( Stop 0x000000D1 DRIVER_IRQL_NOT_LESS_OR_EQUAL )

Este mensaje de error es uno de los mensajes comunes despues de la instalación de Windows XP o actualizar una versión anterior de Windows XP y que es causada por la incompatiblidad de un controlador o un programa con la version Windows, y para superar este problema hay que tratar de conseguir el Driver apropiado antes de la instalación de Windows.

15 ( Stop 0x000000D8 DRIVER_USED_EXCESSIVE_PTES )

Un Driver dañado causa este problema, y la solución es similar al del problema anterior y se consiste en deshacerse del Driver que causa el problema.

16 ( Stop 0x000000EA THREAD_STUCK_IN_DEVICE_DRIVER )

Este mensaje aparece cuando se instala una nueva tarjeta de video o se utilizar un conductor incompatible con Windows XP, para resolver el problema de su tarjeta existente, y hay que cambiar la tarjeta o eliminar este driver inapropiado.

17 ( Stop 0x000000ED UNMOUNTABLE_BOOT_VOLUME )

Windows XP no logró alcanzar el disco que contiene los archivos de arranque si ves este mensaje después de tratar de actualizar el sistema a Windows XP, asegúrese de que dispone de los factores desencadenantes (Drivers) apropiados para el controlador de disco "disk controller", en la mayoria de los casos este mensaje desaparece automáticamente sin ninguna intervención manual despues de reiniciar el dispositivo.

18 ( Stop 0x000000F2 HARDWARE_INTERRUPT_STORM )

Parece ser un fallo de hardware en el uso de IRQ Debido a que el Controlador de este dispositivo no es compatible con Windows XP, Lea los detalles del error para ver nombre del driver que causa el problema y después de la entrada en modo de seguro, hay que ir al Administrador de dispositivos y Retirar todos los dispositivos vinculados a la misma línea IRQ y luego añadirlos de Nuevo.

19 ( Stop 0xC000021A STATUS_SYSTEM_PROCESS_TERMINAT ED )

Un problema creado por un software en el archivo del sistema Crss.exe, y para resolver el problema Quite el programa responsable que puede identificarlo a partir de la información adjunta al mensaje de error.

20 ( Stop 0xC0000221 STATUS_IMAGE_CHECKSUM_MISMATCH )

Este mensaje indica que hay daños en uno de los archivos del sistema y el nombre del archivo se indica en el mensaje, y para resolver el problema al arrancar la computadora presione F8 y elegir Última opcion "
Last Known Good Configuration"

o arrancar con el CD de Windows y entrar en la consola de recuperación y recupera los archivos necesarios del CD-ROM de Windows XP

21 ( STOP 0xC0000218 UNKNOWN_HARD_ERROR )

Hay problemas en disco duro como "Bad Sectors" o debido a un fuerte choque, hay que verificar el disco duro para asegurarse de que está libre de los malos sectores "Bad setors" utilizando un programa para analizar su disco, en este caso les recomiendo el mejor programa para analizar el o duro y aislar los "Bad setors
