Transmission, para descargas por P2P (torrent)

  • Categoría: Software / Programas
  • Publicado hace más de 9 años
Que tal amig@s:

Hoy comparto Transmission que es uno de los mejores clientes de descargas torrent de toda la red. Este cliente se caracteriza por ser de código abierto y totalmente gratuito.

Este año recién de forma oficial fue lanzado para Windows (ya que tenía una versión no oficial hace ya un tiempo) y me ha parecido muy estable, sencillo y practico a la hora de realizar este tipo de descargas.

Para los que trabajan con OS X y en Linux ya lo deben conocer muy bien.




Les dejo las dos versiones para Windows, espero lo pruebe y den sus cometarios.
