Crear USB booteable Windows 8.1

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 10 años
Veremos como se crea una unidad flash usb booteable para instalar windows 8, 8.1, 7 .


En este caso necesitamos unas herramientas como por ejemplo, Windows 7 USB-DVD- tool, tambien podemos necesitar Ultraiso si es para windows 7 seguro lo necesitaran.
La unidad USB que van a necesitar tiene que tener espacio para almacenar una iso de un sistema operativo asi que  de 8 GB o mas estara bien.

Son unos pasos muy simples y rapidos, sin mas queda el video cualquier duda comenten
