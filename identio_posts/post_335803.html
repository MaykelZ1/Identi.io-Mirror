Instrucciones Para Correr El Juego. Solo Para Usuarios Con Sistema Operativo de 32 Bits. "Las Personas Con 64 Bits Omitir Este Paso Ya Que No Es Necesario."

Cita: Mostrar
En Windows Vista/7/8 ( 32 bits)

1 ) Abra un símbolo del sistema "CMD" con privilegios de administrador


- Abra el menú Inicio.
- Escriba cmd en el campo de búsqueda.
- Haga clic derecho sobre el icono del símbolo del sistema y seleccione "Ejecutar como administrador"

2 ) Deje un espacio y Escriba  " bcdedit /set IncreaseUserVa 3072" (sin las comillas)
3 ) Después de pulsar ENTER , debería decir algo así como " La operación se realizó con éxito"


4 ) Reinicie su ordenador.
5 ) Ejecute "TheForest.exe"


6) Después de jugar asegurese de eliminar el comando, vaya a inicio, Escriba cmd.exe, Ejecutar
como administrador Escriba: "BCDEdit/deletevalue IncreaseUserVa"


Nota: Asegúrese de eliminar el comando, ya que puede disminuir el rendimiento del sistema después de jugar el juego.








ModAPI V0.1.8a
(Editor de Partidas)

Lo Que Puedes Hacer:

- Crear Backups de tus partidas actuales o restaurarlas."compatible con backup de las
versiones anteriores V0.04, V0.05, V0.06 , V0.06b."

- Modifica tu inventario, agrega o remover items.
- Modifica tiempo, salud, resistencia, hambre, blindaje.
- Editor de planos, con un simple click puedes finalizar cualquier construccion.
- Elimina objetos y contrucciones de tu partida.

- Nota: Se recomienda hacer una copia de seguridad de tu partida, lo puedes hacer con el App o puedes acceder directamente al directorio donde se aloja el archivo.

Ruta: C:/Users/User-Name/AppData/LocalLow/SKS/TheForest
Nombre del archivo "PlayerPrefsFile___RESUME__"

"La carpeta AppData es una carpeta oculta del sistema, ir a opciones de carpeta y
mostrar  archivos, carpetas y unidades ocultos"
