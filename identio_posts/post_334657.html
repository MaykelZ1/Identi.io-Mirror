Hoy les traigo tres libros en
 formato pdf.

los tres son para aprender la gramatica en ingles. 

aqui el nombre de los tres libros, completitos. listo para hacer los ejercicios.

el primero se llama gramar; GRAMMAR IN USE, BASIC 

este libro lo puden usar aquellas personas que comiensan con el idioma ingles y quieren tener una buena gramatica.

el segundo se llama; GRAMMAR IN USE INTERMEDIATE. para este ya se deben de tener conocimientos previos para resolver los ejercicios gramaticales que trae. 

el tercer libro: GRAMMAR IN USE ADVANCED. este ya es para personas con un nivel arriba de intermedio para que puedan resolver los ejercicios


los libros bienen con ejercicios. pero ilustrados. son super buenos y entendibles.
