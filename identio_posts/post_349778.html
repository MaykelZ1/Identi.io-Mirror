Como usar Whatsapp en PC [con Pidgin] Linux

  • Categoría: Linux
  • Publicado hace más de 10 años







Hoy voy a enseñarles como hacer andar el Whasapp desde su computadora con el programa Pidgin y lo vamos a explicar para un sistema Gnu/Linux, Fedora 20 especificamente.


ACLARACIONES:
1. Esto es para las personas que no estan ocupando el whatsapp en el celular ya que no he confirmado que se pueda usar la misma cuenta desde ambas plataformas (PC y celular) a la vez.

Para que sirve??
Si  solo tienes plata para comprarte un :

 pero queres ocupar de todos modos whatsapp esta es tu opción


Empecemos
Primero intalar el Pidgin
#yum install pidgin




Usaremos ademas 2 programas:
1. Whatsapp purple: https://github.com/davidgfnet/whatsapp-purple
2. Yowsapp: https://github.com/tgalal/yowsup

El primero es un plugin para que pidgin maneje el protocolo whatsapp, el segundo es un programa de consola que nos permitira registrar nuestro numero de telefono y obtener la clave para usar el whatsapp.

Tenemos que instalar todo lo que necesitamos para compilar los programas, primero
#yum install git libpurple libpurple-devel python-dateutil python-argparse python libxml2 make gcc-c++

[editado: si estas en Debian o derivados debes instalar:
 aptitude install python2.7 python-dateutil python-argparse pidgin libglib2.0.0 libglib2.0-dev libpurple-dev git make g++
 ]


Descargamos los programas
$git clone https://github.com/davidgfnet/whatsapp-purple.git whatsapp
$git clone https://github.com/tgalal/yowsup.git yowsup

Nos movemos al directorio de creado de whatsapp donde hemos guardado el programa whatsapp purple y :
$cd whatsapp
ejecutamos make ARCH=i686 o make ARCH=x86_64 segun tu sistema sea de 32 o 64 bits respectivamente
$make

esto debe crear un archivo llamado libwhatsapp.so en la misma carpeta
[nota: Si sucede algun error durante la compilacion es porque falta algun programa, informarme el output del error por si me he olvidado de algo]

Ese archivo libwhatsapp.so lo movemos al directorio /usr/lib/pidgin/
#cp ./libwhatsapp.so /usr/lib/pidgin/

luego nos movemos al directorio yowsup y creamos un archivo de configuracion para el registro de nuestro numero
cd yowsup
le damos permisos de ejecucion al programa
#chmod +x yowsup-cli
creamos un archivo de configuracion llamado wsconfig:
$cat << fin >> wsconfig
cc=54
phone=54xxxxxxxxxx
id=
password=
fin

donde cc es el codigo del pais y phone es tu numero de telefono completo incluyendo codigo pais. para saber cual es el codigo de tu pais o prefijo telefonico internacional ver:
http://es.wikipedia.org/wiki/Anexo:Prefijos_telef%C3%B3nicos_mundiales
por ejemplo el codigo de EEUU y Canada es 1, para Mexico es 52, Argentina 54, Chile 56 etc

luego ejecutamos
#./yowsup-cli registration -c ./wsconfig -r sms
status: sent
retry_after: xxxx
length: 6
method: sms

Nos enviaran a nuestro telefono un codigo de seis digitos en el formato xxx-xxx que usaremos para terminar el registro.

luego ejecutamos
#./yowsup-cli registration -c ./wsconfig --register xxx-xxx
donde xxx-xxx es el codigo de seis digitos que te han enviado a tu celular, te enviaran el password (pw) el password luce asi :
status: ok
kind: free
pw: SJKxG56HHHGiyku6Y/Bi9BJ/P0V=
price: $x.xx
price_expiration: xxxxxxxxxxxxx
currency: USD
cost: x.xx
expiration: xxxxxxxxxxxx
login: 54xxxxxxxxxx
type: new

asegurate de copiar el password y guardarlo bien.

Con el password solo resta abrir el pidgin y configurar la cuenta, elegimos protocolo whasapp,

nuestro numero de telefono completo en formato internacional es el usuario y copiamos y pegamos el password que recibimos.

Si todo ha ido bien estaremos logeados en whasapp desde pidgin, para agregar contactos debe agregarse el numero de telefono completo en formato internacional incluyendo el codigo pais.

[editado 2
para los emoticons descargar desde https://github.com/stv0g/unicode-emoji y copiar las carpetas en ~/.purple/smileys
luego ir a Pidgin>Tools>Preferencias>Temas>Temas de emoticons  y elegir un tema ]

Saludos.
