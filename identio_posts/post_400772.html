Start8 Menu Inicio Clasico Para Windows 8

  • Categoría: Reciclaje
  • Publicado hace más de 9 años

Start8 devuelve el menú Inicio familiarizados con el último sistema operativo de Microsoft Windows 8. Start8 arrancará directamente en el escritorio de Windows 8, dando a los usuarios un punto de partida amistosa. Los usuarios pueden buscar, pin y lanzar aplicaciones de interfaz de usuario clásicas y modernas en el menú Inicio con Start8.
Organizar:

 Devolver el botón Inicio de Windows 8

 De Windows menú Inicio 7 de estilo con Windows 8 mejoras

 Búsqueda de Windows 8 de estilo (UI) Moderno aplicaciones
 Pin de escritorio y Metro Apps para el menú Inicio
 Apoyo Jump List
 Búsqueda unificada de aplicaciones, configuraciones y archivos
 Arrancar directamente al escritorio de Windows 8
 Con un solo clic el acceso a cerrar, dispositivos, música, documentos y vídeos
Personaliza:

 Opcionalmente desactivar el escritorio de Windows 8 "puntos calientes"
 Soporta WindowFX 5.1 Start animaciones de menú
 Grupo de Apoyo a las políticas a través de la plantilla .admx incluido
