Game is NOT saving progress with the current crack. However, you can force it to start from any of the 36 available checkpoints.

1. Make backup of the folder “…\Wargame\Checkpoints\”.
2. Example: To start game from checkpoint 17, rename file “…\Wargame\Checkpoints\chapter17.sav” to “chapter0.sav”.
3. Start New Game in main menu in usual way.
4. Quit to desktop and repeat steps 2-3 for any other checkpoint. No other solution for now
