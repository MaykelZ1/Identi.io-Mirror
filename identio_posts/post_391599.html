L2 Net/ L2 Walker

  • Categoría: Software / Programas
  • Publicado hace más de 10 años


Buenas gente, hoy les traigo el dichoso programa "L2 Net" o "L2 Walker"


Desde ya antes de cualquier cosa, vale aclarar que esto esta TOTALMENTE prohibido en cualquier servidor, así que les recomiendo que hagan un PJ secundario y lo usen para botear. 

Esta guía sirve para cualquier Crónica del l2 y también el programa!


Antes de empezara hablar de configurar el NET, primero hay que cambiar la configuración del INI del system del servidor en el que queremos BOTEAR.

Importante: tienen que instalar en FRAMEWORK antes de poder usar el NET


Primero se descargan los programas que vamos a usar
Tutorial para descargar con Jdownloder


Tutorial para descargar con Jdownloder


Para ello primero tienen que entrar al servidor para averiguar el IP, sin esto no van a poder logear con el NET. 
Entramos al server, una vez adentro vamos a INICIO y donde dice "buscar programas y archivos" escribimos "CMD" le damos a enter y les va a abrir el DOS.


Una vez abierto el DOS escriben el comando NETSAT -N les va a salir una serie de IP'S ustedes buscan el que aparece con 7777 o 2106, por lo general aparacen con ese final de IP

Una vez que les sale el IP del server la vamos a guardar en el "LOGINLIST" que se encuentra en la carpeta del NET. Es un TXT asi que lo pueden abrir con el bloc notas, borran todo lo que hay y escriben de la siguiente manera:



Le ponen el nombre del server, yo le puse "L2" porque soy vago
Ahora pasamos a configurar el "INI" abrimos el programa "FILE EDIT, INI" que mas arriba les deje el enlace. Vamos a "Open and decrypt"




Buscamos el archivo que dice "L2" lo abrimos y va a salir algo así


Donde dice serverAdrr=........... puede que llegase a tener el IP del sever, si la tiene se ahorran todo el paso anterior. Bien la borran y escriben la siguiente IP 127.0.0.1. Una vez hecho esto, van a "Save and encrypt" y le dan "OK" al cartel que salga.

Listo, ya tenemos el INI configurado para entrar con el BOT  al l2 (cuanto laburo, igual se hace 1 sola vez)
La carpeta del net la pueden poner donde se les antoje, yo por comodidad la pongo dentro de la carpeta del L2. 
Abrimos el "L2Net" que se encuentra en nuestra carpeta. Hay 2 maneras de abrir el bot, iniciando el cliente y otra manera con sin iniciarlo, ami me gusta iniciarlo así.



Luego de que le damos click en IG  les va a quedar algo así



Bien, una vez hecho esto pueden abrir el L2 desde nuestro system donde cambiamos el INI, loguean (OJO que si ponen mal el usuario o contraseña tiene que cerrar y volver a abrir el net) lo mas bien y debería quedarles así



Si no les queda así, en algo la cagaron o puede que el server tenga protección, en caso de que el server tenga protección de GAME GUARD o algo por el estilo, en la pantallita de abajo les va a tirar algo como game guard crash o algo así. 
Si les quedo así, felicidades pueden empezar botear!

Ahora vamos a la parte de programar el BOT, nada complicado el NET es bastante completo por lo que nos permite hacer varias cosas, pueden abrir 20000 bots, desde la misma carpeta haciendo los mismos pasos abriendo desde el mismo system no hay necesidad de otro cliente.

Si quieren pueden cambiarle el idioma a español, pero me da pachorra tener que subir otro screen con eso, así que se los escribo. Van a la parte de HELP que se encuentra arriba de todo LENGUAGE y buscan el español

A programar el BOT para que haga el trabajo sucio, farmear. Clikean donde dice BOT OPTIONS, deberia quedarles así




Como pueden ver tenemos varias secciones, Party, Autofighter, etc. vamos a la parte de Autofighter


Si son arqueros y magos tienen que activar las casillas de 

-Active target y Move before target (este es para cuando el mob esta atrás de algun arbol o lo que sea y el bot lo va a buscar) les recomiendo cambiar el MOVE RANGE a 700 o 500 eso jueguen ustedes y vean.

Cada vez que quieran aplicar algún cambio al bot pongan apply una vez hecho eso si quieren sacar el cartel del bot option le dan a CLOSE

La parte de combate/items/buffs se lo dejo que averigüen ustedes porque ya me estoy cansando



Ahora para armar un cuadrado de farm y targeteo vamos a BOUNDING POLYGON



Bien ahora primero que nada tienen que llegar con el PJ al lugar de farmeo, si quieren usar esta opción del bot, les recomiendo en el sector de Autofight que desactiven la opción MOVE BEFORE ATTACK, ya que esto les permitira quedarse en el centro del cuadrado que ustedes armen. En la parte de Z RANGE es el rango que le dan al targeteo, y en donde dice radius side y other es para armar el cuadrado de farmeo, cuando el mob entra en el cuadrado que armaron el bot automaticamente lo targetea. Cuando terminan de cambiarle los rangos y todo eso dan click en ADD CURRENT LOCATION y GENERATE BOX. Al costado les va a aparecer una serie de coordenadas, si quieren limpiar esas coordenadas le dan click derecho y ponen clear all o como este en su idioma.



Para el resto de las opciones investiguen ustedes, si no les saco toda la diversión



Bueno eso fue todo, si necesitan algo o tienen alguna duda, dejen comentarios. Saludos y buen boteo!!!


Recuerden comentar es agradecer!
