Generador de Post para IDENTI.


El programa esta escrito en Python 2.7.
Si quieren ver el código fuente se bajan el archivo que es para LINUX que esta crudo. Solo pesa 7kb. Lo deje así apropósito, el que quiera modificar o hacer lo que le parezca tiene permiso.




El programa genera post de este estilo:
Los campos del programa que no se llenan, luego no aparecen con etiquetas erroneas. Así que simplemente pueden llenar solo los datos que necesitan y ya.

EJEMPLOS: 
http://www.identi.li/index.php?topic=346528
http://www.identi.li/index.php?topic=346101
http://www.identi.li/index.php?topic=346055




Gracias Char4n por las ideas, imágenes, y el feedback.


Versión 1.0
Cambio de nombre a uno mas identificable: Post-Identi
Se agrego la opción para elegir:
Tipo de letra
Color
Tamaño
También se puede agregar password(opcional)
El cual se agregara junto con la barra correspondiente.
