¿Qué es el paquete de idioma?

En la terminología de Microsoft, un paquete de interfaz de idiomas (LIP) es una piel para la localización de software en idiomas como el lituano, serbio, hindi, marathi, kannada, tamil y tailandés. Basado en la tecnología MUI, un LIP también requiere el software para tener un lenguaje base instalada y proporciona a los usuarios una experiencia de aproximadamente 80 por ciento de usuario localizada mediante la traducción de un conjunto reducido de elementos de la interfaz de usuario. A diferencia de paquetes MUI que están disponibles sólo para clientes de licencias por volumen de Microsoft y para selecto SKUs de Windows Vista, un paquete de interfaz de idiomas está disponible libremente y se puede instalar en una copia con licencia de Microsoft Windows u Office y un lenguaje de base fija.

Normalmente, un paquete de interfaz de idiomas está diseñado para los mercados regionales que no tienen paquetes MUI completos o versiones totalmente localizadas de un producto. Es una solución localizada intermedia que permite a los usuarios de computadoras para adaptar su software para mostrar muchas características comúnmente usadas en su lengua materna. Cada nuevo paquete de interfaz de idiomas se construye utilizando el glosario creado por el Proyecto de la Comunidad Glosario en cooperación con el gobierno local, el sector académico y expertos lingüísticos locales.
