Malwarebytes 3.0.4 Premium



Malwarebytes 3.0
Cuatro capas de tecnología para destruir el malware. Protección más inteligente. Protección especializada frente al ransomware. Es la seguridad que usted necesita.

Le protege de las amenazas avanzadas.
Detecta y elimina el malware en tiempo real con tecnología avanzada anti-malware, anti-spyware y anti-rootkit. Busca las amenazas más recientes y peligrosas automáticamente para que usted esté protegido sin tener que hacer nada. 

Impide que se usen sus propios programas en su contra.
Envuelve el navegador y los programas de software en cuatro capas de defensas capaces de detener los ataques que aprovechan las vulnerabilidades de dichos programas para infectar los ordenadores. 

Examina de forma más rápida e inteligente.
El modo de análisis superrápido se centra únicamente en las amenazas que se encuentran activas. Análisis más rápidos sin renunciar a los resultados. Ejecute un análisis en segundo plano mientras arranca su juego favorito. Cuando esté listo para jugar, el análisis ya habrá finalizado. 

Protege sus archivos frente a bloqueos y secuestros.
Detiene el ransomware conocido y desconocido gracias a la tecnología de última generación exclusiva que protege de manera proactiva sus archivos. Se trata de una defensa potente y completa que bloquea el ransomware, no de una simple herramienta de descifrado. ¡Desde hoy mismo puede asegurarse de que nunca protagonizará una noticia sobre un ataque de ransomware! 

Le protege frente a los sitios web falsos o infectados.
Detecta e impide el contacto con los sitios web falsos y los enlaces maliciosos. Recibe una protección proactiva frente a las descargas de malware, los intentos de toma de control y los anuncios infectados. ¿Le preocupa acabar en sitios web peligrosos? Ya no tendrá que preocuparse más. 




3.0 / 8 de diciembre de 2016
Capacidad de rendimiento/protección
    Se ha añadido protección anti-exploit y anti-ransomware sin firma (solo Premium)
    Velocidades de análisis hasta 4 veces más rápidas, incluidos los análisis superrápidos más veloces
    Se han eliminado los reinicios no esenciales después de ciertos eventos de eliminación de malware
    Motor heurístico avanzado (Shuriken) ahora habilitado por defecto
    Autoprotección ahora habilitada por defecto (solo Premium)

Usabilidad
    Interfaz de usuario rediseñada para mejorar la experiencia del usuario
    Ahora configurable para integrarse con Windows Action Center/Windows Security Center (solo Premium)
    Las verificaciones de las actualizaciones se realizan de forma automática para que no sea necesario programarlas
    Soporte mejorado para navegación por teclado y lectores de pantalla

Estabilidad/problemas resueltos
    Se ha añadido la posibilidad de bloquear la pantalla Mi cuenta a través de la política de acceso de usuarios
    Se ha añadido la posibilidad de recopilar registros mejorados cuando lo indique el soporte
    Se ha resuelto el problema que ocasionó que el programador no se ajustara al horario de verano
    Se ha resuelto el problema que ocasionó que el programador se ajustara por defecto a la detección de amenazas a cada hora en lugar de diariamente
    Se ha resuelto el problema que ocasionó que Chameleon informara de mensajes de error en varias ocasiones a Windows Event Viewer
    Se ha cambiado la configuración 'Recuperar si se pierde por' en el programador a 'Desactivado' por defecto
    Se impide el análisis de rootkit en análisis personalizados si no se selecciona la unidad completa del sistema
    Actualizado a la última versión de 7-Zip DLL
    Abordados otros defectos varios

Requisitos de software
Windows 10 (32/64-bit)
Windows 8.1 (32/64-bit)
Windows 8 (32/64-bit)
Windows 7 (32/64-bit)
Windows Vista (Service Pack 1 o posterior, 32/64-bit)*
Windows XP (Service Pack 3 o posterior, 32-bit only)*
Conexión activa a Internet para actualizaciones de base de datos, productos y registro

Requisitos de hardware
CPU de 800 MHz o superior con tecnología SSE2
2048 MB (sistema operativo de 64 bits), 1024 MB (sistema operativo de 32 bits, excepto 512 MB para Windows XP)
250 MB de espacio disponible en el disco duro
Resolución de pantalla mínima de 1024 x 768

*El componente de protección anti-ransomware solo está disponible para Windows 7 o superior.
