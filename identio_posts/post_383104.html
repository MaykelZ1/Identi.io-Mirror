Windows 10 ISO Creation Tool

Liberado el 29 de julio de 2015 por Microsoft Corporation













Windows 10 ISO Creation Tool es una herramienta que te permite instalar o reinstalar el nuevo Windows 10 Final. Crea tus propios medios de instalación (limpios con la herramienta oficial de Microsoft) con una unidad flash USB o un DVD.

Puedes escoger la creación de un medio de instalación de Windows 10 Home, Home N, Home Single Language, Pro y Pro N (32 y 64 bit).

Windows 10 ISO Creation Tool puede ser ejecutada para descargar los medios de instalación de Windows 10 desde Windows 7, Windows 8.1 y en Windows 10.


Antes de empezar


Asegúrate de que tienes:

- Conexión a Internet.

- Espacio de almacenamiento de datos suficiente en el equipo, USB o unidad externa para la descarga.

- Un DVD o una unidad USB vacíos (y una grabadora de DVDs) con al menos 4 GB de espacio si quieres crear medios Te recomendamos que uses una unidad USB o un DVD vacíos ya que se eliminará todo el contenido.

- Consulta los requisitos del sistema.

- Si vas a instalar el sistema operativo por primera vez, necesitarás la clave de producto de Windows (xxxxx-xxxxx-xxxxx-xxxxx-xxxxx). Para obtener más información acerca de las claves de producto y cuándo se necesitan, consulta la página Preguntas más frecuentes.

- Para las ediciones Enterprise, visita el Centro de servicio de licencias por volumen.

Usa la herramienta de creación de medios para descargar Windows. Esta herramienta proporciona la mejor experiencia de descarga para los clientes que tienen instalado 7, 8.1 y 10.


La herramienta incluye:

- Formatos de archivo optimizados para la velocidad de descarga

- Opciones de creación de medios integradas para unidades USB y DVDs

- Conversión opcional al formato de archivo ISO
