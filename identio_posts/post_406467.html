Jdownloader 2 Instalacion Limpia sin Fallas, Pasa!

  • Categoría: Software / Programas
  • Publicado hace más de 9 años

Bienvenidos a un nuevo post de...







Hola gente de identi hoy les traigo el gestor de descargas jdownloader 2, como sabran este programa es uno de los mejores que pueden encontrar para descargar archivos sin ningun problema. Sin embargo los instaladores de este programa traen barras o porquerias que no queremos y en muchos casos el instalador falla asi que les traigo un metodo para instalarlo y evitar todo ese embrollo





Pasos a seguir para su Instalación

Lo primero que tenemos que hacer es tener instalado java, es fundamental tener la ultima versión para que el programa funcione sin problemas

https://www.java.com/es/download/

el segundo paso sera descargar el archivo que nos permitirá instalar jdownloader sin problemas y funcional

http://installer.jdownloader.org/JDownloader.jar

Una vez instalado java y bajado este archivo mueven el mismo a una carpeta puede ser cualquiera pero si o si deben meterlo en una carpeta
Hecho eso ejecutan como administrador el archivo .jar que bajaron y dejan que descargue los ficheros y actualize puede tardar un rato no se preocupen.
Al finalizar le dan click al jdownloader.jar y tendran su programa funcionando sin problemas. Recuerden que cada vez que quieran abrir el Jdownloader deben ejecutar siempre este fichero .jar, para mas comodidad recomiendo que hagan un acceso directo y listo
