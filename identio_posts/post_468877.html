OneClikFirewall (Bloquea el Internet a ejecutables

  • Categoría: Software / Programas
  • Publicado hace más de 8 años




Compañeros de Identi,hoy quiero compartir con todos ustedes este sencillo progama para bloquear los ejecutables (que nos interesen) con el Firewall,quien no ha descargado algún programa donde nos indican bloquearlo para prohibir la salida a internet,con este sencillo soft lo tenemos bien a mano,para hacerlo mucho mas sencillo.







Muchas veces necesitamos controlar el acceso a internet de aplicaciones y se nos solicita en algunos manuales (la mayoría de cracks, patchs y similares… que bloqueemos tal o cual aplicación en el Firewall de Windows.

Los que habitualmente hacemos este tipo de actividades, no nos resulta muy complicado, pero es posible que para algunos usuarios noveles o poco habituados a meterse en las entrañas de las utilidades de Windows se les complique acceder y configurar reglas en el Firewall.

Para todos esos usuarios, esta pequeña, útil y gratuita herramienta les puede ayudar mucho en esa tarea.

La aplicación no tiene una interfaz propiamente dicha, ni es un firewall alternativo. Añade dos opciones a los menús contextuales cuando hacemos clic derecho encima de un archivo ejecutable, uno para bloquear el acceso a Internet de esa aplicación, y otra para darle permiso.






Cuando bloqueamos el acceso a un ejecutable, la aplicación lo que hace es crear automáticamente una regla de denegación de acceso en el Firewall de Windows, lo mismo que haríamos nosotros manualmente, pero sin la necesidad de navegar hasta los ajustes del firewall, seleccionar la aplicación y configurar la regla.

Esta herramienta hace todo el trabajo manual de forma transparente y en mis pruebas se ha comportado muy estable y confiable, realizando todo el trabajo de forma precisa.

Otro recurso más para tener a mano y que se nota en falta cuando no está ahí.
