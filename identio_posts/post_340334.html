Drive SnapShot









Drive Snapshot crea una imagen exacta del disco de tu sistema en un archivo. No se deje engañar por el peso de este software, ya que es uno de los mas livianos pero el mas potente y confiable a la hora de hacer sus respaldos del sistema.


Con Drive Snapshot podrás crear una imagen incluyendo el sistema operativo, programas instalados, los datos y todos los atributos de seguridad, mientras que Windows está funcionando y seguir trabajando. 


Gracias a este software el ideal en caso de algún problema causado por un virus, troyano o fallo del disco duro, podrás realizar una restauración completa y exacta de tu sistema en sólo unos minutos. 


El archivo de imagen creado por Drive Snapshot puede ser montado como un disco virtual, de modo que usted puede restaurar o usar fácilmente solos algunos archivos o directorios, utilizando el Explorador de Windows u otro programa.
