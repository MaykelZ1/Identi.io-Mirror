KEKA

Keka es un compresor de archivos gratuito para macOS
the main compression core is p7zip (7-zip port)
Puede comprimir en estos formatos:
7z, Zip, Tar, Gzip, Bzip2, DMG, ISO
Puede extraer todos estos formatos:
RAR, 7z, Lzma, xz, Zip, Tar, Gzip, Bzip2, ISO, EXE, CAB, PAX


Aunque existen multitud de programas para realizar este mismo cometido, el programa gratuito Keka destaca principalmente por su sencillez de uso. Tras arrancarlo y configurar el programa, solo será necesario arrastrar el fichero o ficheros que desees comprimir hasta el icono del Dock para que el programa realice el resto.Podremos elegir entre cinco diferentes tipos de compresión, 7Z, GZIP, TAR, ZIP o BZIP2. El programa nos permite incluso escoger el método de compresión para que esta se realice más rápida o por contra tarde más consiguiendo mayor compresión. Nos permitirá incluso dividir el archivo comprimido en partes. También podremos proteger el archivo con contraseña para evitar que sea descomprimido por quien no deba.
El programa posee una versión para añadir a las librerías del sistema propiciando que nos sea posible realizar la compresión desde el menú contextual, de esta manera nos resultará más rápido utilizarlo puesto que podremos utilizar desde el menú que aparece pulsando con el botón derecho cualquier fichero.
