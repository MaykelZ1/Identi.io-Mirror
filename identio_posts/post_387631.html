Instalar Windows Desde Pendrive

  • Categoría: Software / Programas
  • Publicado hace más de 10 años

Hola amigos de identi hoy les enseñare como instalar cualquier Window en "PENDRIVER", lo vamos a hacer utilizando símbolo del sistema de window por lo cual no utilizaremos sin ningún tipo de programa sigan los pasos que mostraran a continuación.
al final del post les dejare un link con las instrucciones en un archivo de texto para que no quede duda alguna

29-08-2015 

Aviso deben tener conectado el pendriver a la hora de hacer esto, y siempre ejecutar el símbolo del sistema como administrador.

1- Ejecutar "SÍMBOLO DEL SISTEMA" como ADMINISTRADOR y teclear lo siguiente.

diskpart



list disk


select disk
aquí tienen que colocar select disk y el numero de su unidad USB, ejemplo: select disk 6


clean


create partition primary


select partition 1


active


format fs=NTFS
aqui deben esperar que la carga llegue a 100%



assign


exit
exit


2- copiar los archivos del window al pendriver y esperan a que se pase por completo



3- expulsar el pendriver y conectarlo nuevamente, y les quedara así

¡¡¡ATENCIÓN!!!

Para formatear su pc solo tienen que reiniciar su pc y pulsar varias veces la tecla F10 y hay eligen el USB y dan enter para iniciar desde el USB,así hago yo a la hora de formatear mi pc mediante un USB.

Eso fue todo por hoy amigos de identi espero que les guste cualquier duda comenten, si les gusta unos punticos nunca están de mas. Como Instalar Cualquier Windows en Pendriver
