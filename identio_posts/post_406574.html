Acronis True Image 2016

Proteja su vida digital con la solución líder de copia de seguridad

Características esenciales:

Copias de seguridad de imagen de disco completa

Realice copias de seguridad de todo el contenido de su equipo, incluyendo su sistema operativo, aplicaciones y datos, a un disco duro externo o NAS.

Universal Restore

Restaure los datos de sus copias de seguridad, aplicaciones y el resto de datos en su equipo existente o en un hardware completamente nuevo, o restaure datos específicos según sea necesario.

Optimizado para Windows 10 y las versiones más recientes de Mac OS X

Certificado para Windows XP, 7, 8 y 10





Requisitos del sistema:

Windows

Windows XP SP3
Windows 7 SP1 (todas las ediciones)
Windows 8 (todas las ediciones)
Windows 8.1 (todas las ediciones)
Windows Home Server 2011
Windows 10

Sistemas de archivos compatibles

NTFS
FAT32
Ext2/Ext3/Ext4 *
ReiserFS *
Espacio de intercambio Linux *

* Los sistemas de archivos Ext2/Ext3/Ext4, ReiserFS y espacio de intercambio SWAP solo pueden utilizarse en operaciones de copia de seguridad o recuperación de discos o particiones
