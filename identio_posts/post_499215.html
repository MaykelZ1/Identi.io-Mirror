> > > > INFORMACION < < < <

--La Game Boy Advance posee un catálogo de unos 1000 juegos diferentes que salieron desde el 2001 al 2008, hubo juegos de todos los géneros para esta consola, hubo muchos juegos cancelados para el sistema siendo los más destacables Resident Evil 2 y Grand Theft Auto 3.

Uno de los juegos que no se eliminó, más conocido fue el Metal Slug Advance.

La consola también es compatible con todos los juegos de Game Boy y todos los de Game Boy Color lo que le da 730 juegos más en su catálogo teniendo un total de 1730 juegos diferentes. Y actualmente se esta incorporando la versión del emulador GBA con filtro HD y con versiones de 32, 64bits por primera vez.

Cuenta con un procesador ARM propio de 32bits a 16,7MHz, basado en la arquitectura RISC, con una potencia suficiente para permitir el desarrollo de juegos utilizando el lenguaje de programación C.

El microprocesador ARM es capaz de ejecutar tanto un juego de instrucciones con un tamaño de instrucción de 32bits, como un juego de instrucciones llamado "Thumb" de un tamaño de 16bit (pero igualmente, de 32bits).

El sistema, además, lleva un procesador Z80 a 8MHz para dar soporte al software de Game Boy clásica y color, pero sus dos procesadores no pueden estar funcionando a la vez debido a diferencias de voltaje y uso del bus de datos--

> > > > INSTRUCCIONES DE INSTALACIÓN < < < <

1. Descargar el link de descargas.

2. Una ves descargado el archivo .RAR, le daran Click Derecho y descomprimir.

3. Cuando hayan descomprimido los archivos, les quedara un archivo de texto y una carpeta llamada "Emulador de Game Boy Advance".



4. Ahora le dan Doble Click a la carpeta y una ves dentro encontraran una gran variedad de archivos, el que vamos a utilizar es el archivo .exe llamado "mGBA" y simplemente lo ejecutan.



5. Una ves ejecutado, les abrirá el tablero del emulador de Game Boy Advance mGBA.



6. Simplemente Comenzamos con la configuración de este emulador, para eso lo primero nos dirigimos a ->Herramientas ->Ajustes...



7. Se nos abrirá una pestaña nueva, aquí aplicaremos la mayor parte de las configuración.



8. Primero en la pestaña ->Audio/Video, verán la opción de sistema de Audio y colocan SDL.

9. Luego en la opción de Sistema de Vídeo colocan OpenGL.

10. Y en la opción de Objetivo de FPS, colocaran los 60fps.



11. Segundo en la pestaña ->Controladores, aquí se configuran el Joystick o Teclado para emular las teclas del Game Boy Advance.



12. Tercero en la pestaña ->Shader, aquí colocaran los shader HD, para que juegos se ejecuten con una mejor calidad de imagen.



13. Para esto en la opción ->Cargar nuevo Shader, seleccionaran el Shader HD que mas les guste.







14. Ahora continuamos con la configuración y nos vamos a la pestaña -> Audio/Vídeo ->Bloquear proporción de aspecto desmarcan esta opción para que los juegos se ejecuten en pantalla completa.



15. Por ultimo para ejecutar un juego se dirigen a la pestaña ->Archivo ->Cargar ROM y simplemente buscan la carpetas donde tienen sus juegos de Game Boy/Game Boy Color/Game Boy Advance.
