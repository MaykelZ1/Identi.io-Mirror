[Gestor de Descarga] Download Ninja [MEGA]

  • Publicado hace más de 10 años




Free Internet Download Manager

Download Ninja



Download Ninja es un  Internet Download Manager y acelerador que aumenta no sólo dramáticamente la velocidad de descarga, sino también le permite reanudar y reiniciar descargas interrumpidas. Download Ninja acelera tus descargas dividiendo cada archivo en trozos lógicos y la descarga de cada fragmento mediante una conexión diferente a la del servidor, el resultado de esto es mucho más fiable y permite descargas rápidas.



Reanuda Descargas

Reanuda las descargas después de dejar, sin tener que reiniciar desde el principio.



Download Accelerator

Aumentar la velocidad de descarga de hasta 10 veces mediante el uso de múltiples conexiones simultáneas



Escritura secuencial de archivos

A diferencia de otros gestores de descarga, descargar archivos de Download Ninja de principio a fin para que pueda empezar a obtener una vista previa de archivos de audio y vídeo antes de la descarga se ha completado.



Audio / Video Downloader

Descargar archivos de vídeo y audio de los sitios web de medios de comunicación populares, como YouTube, Vimeo y SoundCloud (más de 200 páginas web en total)



Conversión de Medios

Convertir descargas multimedia de un formato a otro o extraer audio del vídeo automáticamente cuando finalice la descarga.



Descargar Colas

Descarga colas le dan un control fino a sus descargas de inicio a fin, así como la programación y el limite de velocidad.



Programar las descargas

Programar una cola de descarga para iniciar o detener en una fecha y hora determinada o configurar las descargas para iniciar / detener al mismo tiempo todos los días









Limitador de velocidad

Limitar la velocidad de descarga de una sola descarga, toda una cola o todas Download Ninja descargas para que pueda seguir usando el Internet, mientras que las descargas están en curso



Control de Conexiones

Controlar el número de descargas y conexiones por descarga simultáneas



Triture los archivos descargados

Una vez que haya terminado con los archivos descargados Download Ninja puede destruir tanto los datos como los metadatos que aseguran que nunca se pueden recuperar, incluso con recuperación de especialistas y herramientas forenses



Monitoreo Portapapeles

Download Ninja supervisa el portapapeles para añadir una descarga todo lo que tienes que hacer es clic derecho y copiar el enlace



Arrastrar y gota de la ayuda

Arrastra nuevos enlaces directamente sobre Download Ninja para agregar la nueva descarga



Login / Password Manager

Con seguridad un seguimiento de sus diferentes nombres de usuarios web de modo que se ajustan automáticamente utilizando nuestro cifrada gestor de inicio



Soporte Proxy

Set Download Ninja permitirá descargar archivos utilizando la configuración de proxy de Internet Explorer o especificar su propia configuración del proxy desde el programa



Chrome / Firefox Plugin

Descargar archivos y vista y descarga de audio y archivos de vídeo sin salir de su navegador



Descarga Grupos

Definir grupos de descarga para que las nuevas descargas se descargan automáticamente a una carpeta específica según el tipo de archivo o sitio web de origen.



Actualizar Dirección

Actualización expiró descarga URL después de la descarga ha comenzado sin volver a comenzar desde el principio



Virus Scanner

Escanear automáticamente todos los archivos de descargas de virus y malware usando tu antivirus favorito



Multi-Idioma

Traducido en 6 idiomas con más que se añade todo el tiempo. Vea nuestra página de traducciones de los créditos e información sobre cómo crear una nueva traducción



Temas y Editor de temas

Utilice el editor de temas para crear skins personalizados para Download Ninja. Echa un vistazo a el editor de temas guiar más información



Descargar Resumen

Mantenga un ojo en las descargas que funcionan con la ventana de resumen de descarga flotante
