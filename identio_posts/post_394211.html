Programar el apagado automàtico

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 10 años
Cuando estás descargando un archivo de Identi, por ejemplo, es posible que tengas que salir de casa y dejar el ordenador encendido. Ahora bien, también puedes hacer exactamente lo mismo y, teniendo en cuenta el tiempo que necesitarás mantenerlo así, programar el equipo para que se apague de forma automática después de un determinado tiempo, el que tú quieras. Te explico cómo programar el apagado automático en Windows.

Si utilizas Windows XP,Windows 8 o 8.1 podrás programar el apagado automático de tu ordenador accediendo al menú de Inicio y pulsando sobre Ejecutar. Una vez aquí, es tan sencillo como escribir el comando ?shutdown -s -t? seguido de los segundos que quieres que tu ordenador se mantenga encendido. Una vez ejecutemos el comando pulsando Intro con el teclado, o bien sobre Aceptar con el ratón, comenzará la cuenta atrás hasta el apagado, que se hará automáticamente.



Ahora bien, si tienes Windows 7, tendrás que acceder al menú de Inicio, y escribir ?shutdown -s -t? seguido del tiempo, en segundos, que quieras mantener encendido tu ordenador. Si queremos mantenerlo encendido durante una hora, por ejemplo, utilizaremos el comando shutdown -s -t 3600. Una vez hayamos hecho esto, se desplegará una ventana emergente avisando de que el ordenador se apagará de forma automática en el tiempo programado.



EL TIEMPO SE LO PROGRAMA EN SEGUNDOS EN ESTA FORMA

shutdown -s -t 600 -f   10 minutos (60seg x 10mnts) 

shutdown -s -t 900 -f : 15 minutos 

shutdown -s -t 1200 -f : 20 minutos 

shutdown -s -t 1800 -f : 30 minutos 

shutdown -s -t 2700 -f :45 minutos 

shutdown -s -t 3600 -f : 1 hora 




Si te arrepientes ¿cómo lo cancelo?

En todas las versiones podemos cancelar el apagado automático, ya sea porque finalmente no deseamos utilizar esta función, o bien porque queremos cambiar el tiempo de la cuenta atrás. Para ello, es tan sencillo como avanzar por los mismos pasos anteriormente descritos correspondientes a la versión de nuestro sistema operativo y, como comando, utilizaremos?shutdown -a?

Simple
copien y pegen esto en un block de notas y guardenlo con la extension .bat y listo. mas simple  

@echo off
color 1f
cls
:menu
cls
echo,
echo 1.Asignar programado de apagado automatico
echo,
echo 2.Cancelar programado
echo,
echo 3.Salir

set /p opt=
if %opt%==1 goto continuar
if %opt%==2 goto cancel
if %opt%==3 goto s

:continuar
cls
echo Introduce el valor en horas[enteras] para apagado automatico
set /p n1=
echo Introduce el valor en minutos[enteros]
set /p n2=
set /a res=(%n1%*3600)+(%n2%*60)
shutdown -s -t %res%
echo Su pc sera apagada en %n1% horas con %n2% minutos
echo Pulse una tecla para volver al menu...
pause>nul
goto :menu

:cancel
cls
shutdown -a
echo Cancelado
echo Pulse una tecla para volver al menu...
pause>nul
goto :menu

:s
cls
exit
