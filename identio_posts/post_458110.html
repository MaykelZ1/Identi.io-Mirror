Soporte para plugins de terceros

Soporte de archivos extensa



Tanto si va a importar desde la cámara, escáner, la web u otro editor de fotos, PaintShop Pro es compatible con los formatos raster y vectoriales archivo más populares, incluyendo archivos .psd de Photoshop de Adobe. Salida de sus archivos a una amplia selección de formatos de imagen para compartir fácilmente.


Laboratorio retro

Miniaturas hacen que sea fácil para seleccionar el efecto que desea.

Obtener el efecto de los costosos lentes de cambio de inclinación sin el costo! Enfoque diferencial que permite a los efectos de cambio de inclinación imitan en sus fotos. También puede usarlo para jugar con la profundidad de campo para mantener partes de la imagen en foco y el resto borrosa.


herramientas web



Usar una variedad de herramientas de la web versátiles para guardar y optimizar las imágenes para su uso en línea. Trabaja con mapeo de imágenes para enlazar HTML con sus gráficos, y utilizar rebanar y rollovers para crear diseños que funcionan a la perfección con las páginas web.


restauración de fotos increíble

Utilice la herramienta Corrección de decoloración para restaurar el color y el contraste de las fotos desvaídas y el Scratch Remover para reparar grietas antiestéticas en jirones fotos para que se vean nuevos otra vez.


herramientas de texto integrales



Añadir texto a cualquier proyecto de impresión, web o foto con herramientas completas tipografía esta foto de software de edición. Ajustar tipos de letra, tamaño, estilo, alineación, lo que lleva / ajuste entre caracteres y aplicar efectos de obtener el texto creativo que desee.
