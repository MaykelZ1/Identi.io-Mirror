Rom CyanogenMod 12 punto uno









Hola amigos les traigo una rom para nuestro Samsung Galaxy Trend Plus S7580 ya que nuestro dispositivo solo tiene android 4.2.2 y esta hermosa rom es la  CyanogenMod 12.1 con android lollipop 5.1
En la carpeta que descargaran encontraran dos archivos, uno es la rom y el otro es el gapps
primero deberan instalar el archivo rom y luego el gapps.

CÓMO INSTALAR:
1. Recuperación de Flash a través de Odin
2. Vaya al modo de recuperación
3. Limpie : datos , caché y caché de Dalvik
4. Instale ROM
5. Limpie caché Dalvik caché +
6. Instale GAPPS
7. Disfrute !!

HOW TO INSTALL:
1. Flash recovery via Odin
2. Go to recovery mode
3. WIPE: Data, Cache and Dalvik cache
4. Install ROM
5. Wipe Cache + Dalvik cache
6. Install Gapps
7. Enjoy!!
