Regular brillo de laptop con HUAYRA

  • Categoría: Linux
  • Publicado hace más de 10 años
Hola a todo mundo, hoy traigo esta pequeña publicación ya que tuve un problema con el cual batalle bastante y era que las tecla de función servían todas a excepción de las de bajar o subir brillo.

Navegue por toda la red y no encontré nada hasta que di con la solución.

Les relataré todos los procesos que seguí siendo que ninguno funciono y al final dejaré el bueno

Todas las webs decían mira prueba esto

Cita: Mostrar
1) Editar el archivo /etc/default/grub:

sudo nano /etc/default/grub

2) Buscar la linea con el siguiente contenido:

GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"
y anexarle las siguientes opciones del kernel "acpi_backlight=vendor acpi_osi=Linux".

Así tiene que quedar la línea:

GRUB_CMDLINE_LINUX_DEFAULT="quiet splash acpi_backlight=vendor acpi_osi=Linux"

Para finalizar la edición del archivo solamente basta con grabar (Ctrl+x, presionar ?Y? para confirmar grabación).

3) Actualizar el grub con el comando update-grub:

sudo update-grub

Reinician el equipo, y si nada falló, deberían poder utilizar las teclas de función de brillo.

Cita: Mostrar
1. Abrimos nuestro editor de texto y ponemos lo siguiente :

#! /bin/bash
brillo=$(cat /sys/class/backlight/cmpc_bl/brightness)
brillo=$(expr $brillo + 1)
echo $brillo > /sys/class/backlight/cmpc_bl/brightness


y lo guardamos como "SubirBrillo.sh"

2. Abrimos nuevamente nuestro editor de texto y ponemos lo siguiente :

#! /bin/bash
brillo=$(cat /sys/class/backlight/cmpc_bl/brightness)
brillo=$(expr $brillo - 1)
echo $brillo > /sys/class/backlight/cmpc_bl/brightness


y lo guardamos como "BajarBrillo.sh"

3. una vez que tenemos los dos archivos le damos permisos para ejecutarlo

chmod +x BajarBrillo.sh SubirBrillo.sh


4. Ahora vamos a darle permisos al archivo brightness para que los scripts puedan modificar su valor.

Para ello abrimos el archivo /etc/rc.local con permisos de root o sudo

nano /etc/rc.local
Un vez abierto, añadimos la siguiente linea justo antes de la linea exit0:

chmod 777 /sys/class/backlight/cmpc_bl/brightness
Y guardamos los cambios.

PD: Tanto como para subir o bajar el brillo hay que ejecutar los archivos BajarBrillo.sh SubirBrillo.sh hasta llegar a el nivel de brillo de nuestra preferencia ¡¡

Cita: Mostrar
Primero vamos actualizar el sistema ... ¿Como?
abrimos la terminal yendo a Menu -> Accesorios -> Terminal
y escribimos lo siguiente :
Código: 
sudo apt-get update

poner la contraseña de sudo (por defecto "alumno"), luego
Código: 
sudo apt-get upgrade
en caso que nos pregunte le damos que si, y por ultimo ingresen
Código: 
sudo apt-get dist-upgrade
despues de hacer todo esto nos dirijimos a Menu -> Sistema -> Administracion -> Gestor de paquetes Synaptic y buscamos los paquetes :
intel-classmate-drivers y intel-classmate-function-keys (en caso de que no aparezcan reiniciar el sistema y volver a buscarlo en Synaptic)
marcamos para instalar los dos paquetes , instalamos y reiniciamos . Con eso ya tendrian en funcionamiento las teclas fn y el brillo, al igual para usar proyector o pantallas externa con fn + f6 , al igual que activar/desactivar wifi , bueno creo que eso es todo , cualquier duda pregunten !!

Todos publicaban lo mismo, siempre respondían lo mismo y yo pensé que ya me tendría que aguantar a tener mi laptop o al máximo del brillo o como iniciara, pero después de mucho tiempo encontré la solución


¿SIRVE? SI, este método fue el único que me funciono perfectamente y se los dejo a continuación
Cita: Mostrar
Crean un archivo en este directorio (si no existe xorg.conf.d, tambien lo crean):  /usr/share/X11/xorg.conf.d/20-intel.conf
con el siguiente contenido:
Código:
Section "Device"
        Identifier "card0"
        Driver "intel"
        Option "Backlight" "intel_backlight"
EndSection

y ahora les dejo una pequeña muestra de que funciona
