ESET Mobile Security & Antivirus v4,0,8,0 +keys

  • Categoría: Moviles / Celulares
  • Publicado hace más de 7 años

ESET Mobile Security es la protección de dispositivos contra virus y troyanos "móviles". Y esta no es una de las funciones principales. A diferencia del módulo "BlockVor", la configuración y la lógica del módulo "Antivirus" son análogas a la lógica de la protección antivirus clásica, bien conocida por todos los productos antivirus para computadoras personales.

El módulo "Antivirus" ESET Mobile Security brinda protección del dispositivo móvil mediante un monitor antivirus (en esta aplicación se llama protección en tiempo real) y analiza el dispositivo o la carpeta seleccionada a pedido. El escaneo programado, así como el escaneo bajo demanda en segundo plano, no son compatibles.

El propietario del dispositivo puede cambiar la configuración del monitor antivirus y escanear a pedido.

Módulo antivirus ESET Mobile Security usa una única base de amenazas para todas las plataformas móviles. La actualización de las bases de datos de firmas en el sitio web de ESET ocurre regularmente al menos una vez al día. No hay necesidad de preocuparse por los graves gastos de tráfico para la actualización, ya que el tamaño del archivo de actualización es bastante pequeño (no más de 1000 bytes). El único parámetro que se puede configurar en el sistema de actualización de la base de datos antivirus es la frecuencia de actualización. Y, por supuesto, si lo desea, puede iniciar la actualización no solo en un horario, sino también manualmente.

Módulo "Protección contra el spam"

Una posibilidad interesante de ESET Mobile Security para la plataforma Android es filtrar no solo los mensajes SMS y MMS entrantes, sino también las llamadas entrantes y salientes. El bloqueo de mensajes y llamadas entrantes significa que el módulo Anti-Spam mueve mensajes e información sobre llamadas al registro de spam. Ninguna notificación de recepción de mensajes o llamadas consideradas como spam no distraerá al propietario del teléfono.

Las reglas para el filtrado de spam se presentan en ESET Mobile Security con una lista de contactos, para cada una de las llamadas y mensajes que se procesan. La lista de contactos se puede ingresar manualmente, o puede seleccionar desde la libreta de direcciones de su teléfono inteligente.

Verificación general del estado del dispositivo y administrador de tareas

Como se dijo al principio del artículo, ESET Mobile Security, además de las funciones anteriores, incluye un administrador de tareas y un módulo para analizar varios parámetros críticos del teléfono inteligente. Esta funcionalidad está disponible a través del elemento "Configuración del sistema" en la pantalla principal de la aplicación.

Se inicia una verificación general del estado del teléfono inteligente solo a pedido. En función de los resultados de la comprobación del dispositivo, se proporciona al usuario información completa sobre el estado del sistema, incluidos datos sobre aplicaciones en funcionamiento, carga de la batería, Bluetooth, memoria libre y dispositivos conectados.

El administrador de tareas le permite ver los procesos, servicios y tareas que se ejecutan en el dispositivo, así como también aprender sobre la memoria que ocupan. ESET Mobile Security le permite detener un proceso, servicio o tarea que el sistema no inicia. Dichos elementos están marcados en la lista con una cruz roja.
