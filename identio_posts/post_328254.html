Microsoft Service Pack 2 para Windows 7


* Acumula un total de 90 actualizaciones *
*Como saben la gente de Microsoft nunca hablo sobre lanzar o no , un Service Pack 2 para Windows 7.  Igualmente desde hace tiempo varios rumores afirmaron con bastante seguridad que dicho upgrade nunca llegaría a Windows 7.  la actualización de este Service Pack 2 si acumula un total de 90 actualizaciones publicadas tras el lanzamiento de SP1 en febrero de 2011.
 
*Este paquete mejora tanto la fiabilidad como el rendimiento general del equipo, encontrando novedades como DFSN (Distributed File System Namespaces), mejoras en la redirección de ficheros, WMI (Windows Management Instrumentation), cliente SMB y directivas de grupo.
*La actualización está disponible para Windows 7 y Windows Server 2008 y puede ser descargada desde aquí:
