  • Publicado hace más de 9 años
hola a todos hoy les traigo la bios de esta canaima oficial desde la pagina de intel sin logos del gobierno tambien sirve para tecnicos por si una cnaima se le da単o la bios

MG101A4-MG101A7 (actualizado 08/abr/2024)


Link


como flashear la Bios


1.Metodo: Shell EFI ( el mas recomendado)

 Necesitamos Un Pendrive con todos los archivos de la bios
 Carga modo Windows 8
 Presionar F11 vamos a "Shell EFI"  o sino tienes SO te mandara directo a la consola Efi Shell
 Escribimos F.nsh
 esperar una vez finalizado se apaga se cargan todos los valores de fabrica no se necesita entrar a la bios para establecerla


2.Metodo: Windows ya sea con Modo Heredado o UEFI(no es muy recomendable aunque yo flashee por este metodo salio todo bien)
 
 Ejecutar FWIN32.bat solo para Windows 32bits
 Ejecutar FWIN64.bat tanto para Heredado como UEFI  Windows 64bits
 Se abre una ventana de cmd para verificar si la bios es compatible
 Se apaga el equipo automaticamente
 Se vuelve a encender nos aparecera una pantalla negra de Phoenix flash o algo asi se comezara a llenar la barrita ***********_______****_** sonaran pitidos
 Se apaga el equipo lo volvemos a encender entre 3-5 segundos no se necesita carga los valores de fabrica
 Listo
 

Metodo 3: Crisis Recovery(este metodo sirve si la bios esta muerta)

 1. Copiar BPxxxx.cap al directorio del pendrive cambiar el nombre a BIOS.cap.
 2. Presionamos "Fn + R" con el equipo apagado
 3. Conectamos el cargador el LED sin soltar las teclas parpadeara el LED
 4. Presionamos el boton de encendido
 5. Esperar que termine
