(1) YouTube HTML5 everywhere MOD



YouTube HTML5 everywhere MOD, es un script creado para añadir soporte html5 en algunos sitios web que por dejados nos obligan a usar Flash para poder reproducir vídeos en este caso de youtube, soporta completamente identi.li, Se a modificado para activar el botón FullScrean en el reproductor html5, para probarlo o desinstalan flash o lo deshabilitan desde la configuración del navegador.
