Process Hacker 2.38 rev: 343 -Controla al detalle tu Sistema-

Process Hacker 2.38 revision 343 (26 febrero 2016) es una aplicación multi-proposito de supervision del sistema (cpu, memoria, red, disco, etc) que te será de gran utilidad para monitorear los procesos que se están llevando acabo en tu ordenador. 
Con él podrás ver "absolutamente todos los procesos", incluidos los ocultos producidos por posibles malwares. Podríamos decir que es una versión avanzada del clásico Administrador de Tareas nativo.
.
Process Hacker nos brinda información avanzada acerca de cada proceso y te permite realizar búsquedas en distintos campos, permitiendo ver y manipular hilos, fichas, DLLs, módulos, regiones de memoria, etc, etc. Podrás relacionar en un click cualquier módulo de memoria activo con la región de memoria en la que se encuentra, accediendo a datos como la dirección, estado o tipo de protección, o también puedes abrir la carpeta contenedora y copiar cualquiera de los datos que te proporciona Process Hacker.
.
 CARACTERÍSTICAS: 

? Vista personalizable con resaltado que muestran los procesos que se ejecutan en el equipo.
? Estadísticas detalladas con gráficas del sistema.
? Funciones avanzadas que no se encuentra en otros programas, como el desprendimiento de depuradores, identificadores GDI, inyectar y cargar archivos DLL, y mucho más.
? Proceso de finalización de gran alcance que trabaja a nivel del núcleo de Windows que normalmente evita o pasa el software de seguridad y/o rootkits.
? Ver y editar procesos/servicios de control, incluidos los que no se muestran en la consola Servicios.
? Ver y cerrar las conexiones de red.
? Inicia casi al instante, a diferencia de otros programas.
? Implementa todas las funcionalidades ofrecidas por el Administrador de Tareas, además de funciones más avanzadas.
? Permite ver todo lo que esta pasando en un hilo, incluso creando, editando y controlando servicios.
? Cuenta con capacidades avanzadas de análisis de cadenas, así como la expresión regular de filtrado.
? Seguimiento detallado en modo kernel, soporte a WOW64 y .NET.
? Muestra máscaras de acceso simbólico (por ejemplo, lectura, escritura), en lugar de números (por ejemplo, sólo 0x12019f).
? Muestra los nombres de los objetos del gestor de transacciones y objetos de registro ETW.
? Muestra información detallada token, así como permitir los privilegios se activación y desactivación.
? Obtiene información en tiempo real sobre el acceso al disco.
? Controla qué programas tienen conexiones de red activas, y las puede cerrar en caso de que sea necesario.
.

Cambios Importantes v: 2.38 

* Added labels to indicate the maximum data point in each I/O graph.
* Graph grids now scale correctly when resized.
* Improved high DPI scaling.
* Added exploit mitigation policy information to process properties (Windows 8 and above).
* Added File modified time and File size columns for processes and modules.
* Added Key modified time column for services.
* Clicking a tray icon now shows the pop-up UI (useful for touch-enabled devices).
* The NetAdapters plugin has been renamed to HardwareDevices.
* Updated UserNotes plugin.
.

Other changes:

* Added "Start when I log on" option.
* Added "Not responding" text to tray icon rich pop-up for programs that are hung.
* Added right-click menu and double-click action for environment variables.
* Added dialog box to show long command line strings.
* Added Time stamp column for processes.
* Added -sysinfo command line parameter for opening System Information at startup.
* Added 32x32 icons for high DPI displays.
* Digital signature verification is now performed with very low I/O priority.
* Improved performance when handling a large number of threads, modules or handles.
* The pop-up UI no longer displays when double-clicking the tray icon.
* Fixed ASLR state being shown as N/A in process properties.
* Fixed multi monitor window placement bug.
* Fixed handle enumeration bug affecting processes with PID >= 65536.
* Fixed Interrupts being missing from the max CPU usage history.
* Updated ToolStatus plugin.
.
