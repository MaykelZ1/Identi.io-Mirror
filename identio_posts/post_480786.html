  • Categoría: Software / Programas
  • Publicado hace más de 8 años
VirtualBox 5.1.26 ESP-x32-x64


Instala y corre varios sistemas Operativos desde 1 solo escritorio


Programa muy util para correr windows XP u otros sistemas desde Windows 10




Ejemplo de escritorio listo para correr varios sistemas pre-instalados



Puedes correr todos los windows que desees e incluso distros de Linux!


Este programita es una joya ya que nos permite ejecutar cualquier sistema trayendo consigo mismo retrocompatibilidad para juegos y programas que pueden no correr en tu sistema actual, olvidate de problemas y ejecuta Oracle Virtual Box!


Además es PORTABLE para llevarlo en un pendrive y no dejar rastros y ficheros por todo tu disco duro manteniendo un optimo estado.






espero les sea de utilidad!



Los requisitos dependen del sistema o sistemas que vayas a correr a la vez

yo les recomendaria un cpu de doble nucleo con almenos 2 gigas de RAM minimo.
