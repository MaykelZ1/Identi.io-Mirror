Adobe Shockwave Player 12.1.7.157
Publicación: 12 de Febrero de 2015



Adobe (Macromedia) Shockwave Player es un Plug-In de navegador, este le permite al usuario disfrutar de animaciones interactivas y contenido web como juegos, presentaciones, tutoriales de software y muchas otras cosas. Potenciado por Adobe Director, Shockwave Player, que es una plataforma de autoría multimedia poderosa parte del paquete de software Adobe Systems, logra aprovechar al máximo las herramientas y servicios avanzados que pueden ser utilizados para construir contenido multimedia que no se encuentra disponible en Flash.

Java JRE 8 UPDATE 31 (32 BITS)
Publicación: 21 de Enero de 2015




El Java Runtime Enviroment (JRE) contiene todo lo que tu computadora necesita para ejecutar aplicaciones Java o cargar aplicaciones Java en tu navegador. La tecnología Java es un entorno de programación orientado a los objetos, independiente de plataformas y con múltiples hilos. Es la base para los servicios web y en red, aplicaciones, escritorios independientes de plataforma, robótica y otros dispositivos integrados. Esto permite que incontables desarrolladores de todo el mundo creen contenido web que puede ser fácilmente reproducido en todos los dispositivos modernos conectados a Internet, sin la necesidad de optimizar las aplicaciones para cada navegador, sistema operativo o plataforma individual.
