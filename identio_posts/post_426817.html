Crear Pendrive booteable Windows & Linux (USB)

  • Categoría: Hazlo tu mismo
  • Publicado hace más de 9 años
Holis! Otra vez les traigo un poco de información, he estado navegando y veo aparecen muchas maneras de hacer un pendrive booteable pero no funcionan, por lo menos las que yo probé no funcionaron, aquí les voy a dejar una manera que es 100 por 100 funcional. 

Con el programa Universal USB Installer descargable desde aquí: 
http://www.pendrivelinux.com/universal-usb-installer-easy-as-1-2-3/ 

El programa también sirve para instalar muchos otros sistemas operativos de linux y Windows y hasta para hacer el pendrive booteable del Hiren's Boot, el procedimiento es el mismo con la imagen ISO respectiva. 

Bueno empecemos... (Formatear pendrive antes de empezar con el programa se puede formatear pero es recomendable que también lo formateen antes) 

Una vez descargado el Universal USB Installer buscamos donde quedo descargado y lo ejecutamos 























Espero les sirva y como siempre espero comentarios, criticas, preguntas y puntos para esta novata n.n
