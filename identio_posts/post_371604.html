R-Undelete 4 9 Build 156783 Portable

  • Categoría: Software / Programas
  • Publicado hace más de 10 años


Utilizando la misma tecnología única de recuperación de datos IntelligentScan como el peso pesado de R-Studio, R-Undelete es una potente solución de bajo costo y fácil de usar pero de restauración de archivos para sistemas de archivos FAT y NTFS. R-Undelete recupera archivos en los discos locales reconocidas por el software. Además, la recuperación de archivos en bruto (escanear para tipos de archivo conocidos) se puede utilizar para sistemas de archivos fuertemente dañados o desconocidos. Para comodidad del usuario R-Undelete puede ejecutarse desde discos y carpetas menú contextual. Los archivos gráficos (ver la lista), vídeo y archivos de audio (todos los archivos son compatibles si un códec adecuado está instalado en el sistema) se pueden previsualizar antes de que se recuperan.

En comparación con las versiones anteriores, las versiones 4.x R-Undelete se complementan con una mejor interfaz a modo de asistente orientadas a los novatos de recuperación de datos y una opción para recuperar archivos de particiones dañadas y eliminados que traen R-Undelete al R- alta calificación Estudio planta a excepción de algunas de las características especiales de recuperación de datos como RAID, edición hexadecimal y soporte de red. Archivos y carpetas encontrados ahora pueden ser fácilmente ordenados por la estructura original, la extensión de archivo o tiempo de archivo de creación / modificación. Agregado S.M.A.R.T. (Self-Monitoring, Análisis y Tecnología de Información) de atributos de la vigilancia para unidades de disco duro permite a los usuarios examinar la salud del disco y predecir sus posibles fallos. Nuevos ReFS del sistema de archivos de Microsoft (Sistema de archivos Resilient) está soportado junto con Windows 2012 OS Server.

A partir de la versión 4.5, R-Undelete crea una imagen y al mismo tiempo analiza los datos que se copian. El programa crea dos archivos almacenados en el mismo lugar: un archivo de imagen y un archivo con la información de exploración. El visor de archivos mejorada compatible con los formatos de archivo adicionales, incluyendo Adobe Acrobat PDF, documentos de Microsoft Office doc, xls, ppt (Office 97-2003), docx, xlsx, pptx. Una nueva, más rápido, y el algoritmo más sólido para tipos de archivo conocidos escaneo y sistema de archivos FAT de procesamiento se ha implementado. El algoritmo se adapta para el procesamiento multi-núcleo y evita que el programa de congelación y estrellarse en una situación crítica.

Qué Hay De Nuevo:

Nuevas características:
Exploración + disco acelerado para los dispositivos de lectura rápida.
+ Búsqueda de versiones de archivos borrados. R-Undelete puede buscar las versiones de archivos eliminados utilizando sus tamaños, nombres y extensiones y tipos de archivos reconocidos como los parámetros de búsqueda.

Mejoras:
* Más fiable y significativamente más rápido reconocimiento de particiones NTFS y FAT.
* Archivos encontrado utilizando tanto buscar tipos de archivos conocidos y presentar análisis de registros ahora se muestran sólo en la estructura de archivos / carpetas.
* Mejora de la selección de datos en el editor hexadecimal. Ahora bien, el principio y el final de la zona de ser seleccionado se puede ajustar por la Selección desde y Seleccionar a los comandos en el menú contextual.
* El lector hexadecimal muestra ahora una posición de búsqueda junto con la barra de progreso.

Una plataforma compatible con Intel con Windows 2000 / XP / 2003 / Vista / 2008/2012 Windows 7/8 / 8.1.
Por lo menos 256 MB de RAM, un ratón, y suficiente espacio en disco para los archivos recuperados, archivos de imágenes, etc.
Se requieren los privilegios administrativos para instalar y ejecutar R-Undelete bajo Win 2000 / XP / 2003 / Vista / 2008/2012 Windows 7/8 / 8.1.
