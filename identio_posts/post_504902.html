SoftMaker Office 2018

Cree impresionantes documentos, hojas de cálculo y presentaciones sin esfuerzo.

Deje que sus documentos hablen por sí mismos: gracias al uso intuitivo, la enorme variedad de funciones y el sobresaliente rendimiento de SoftMaker Office 2018 podrá obtener rápidamente resultados de primera.

Intercambio directo de documentos con Microsoft Office

Olvídese del engorroso paso de importar y exportar: SoftMaker Office 2018 utiliza los formatos DOCX, XLSX y PPTX de Microsoft como sus formatos de archivo predeterminados. Esto le permite editar sus documentos en Microsoft Office directamente sin convertirlos.

Escriba. Calcule. Presente.

Escriba con TextMaker 2018

La moderna aplicación de procesamiento de textos TextMaker 2018 le permite crear documentos de cualquier tamaño en un santiamén, tanto si se trate de un folleto o catálogo como de una carta de negocios o un completo artículo científico. Gracias a su uso nativo del formato de archivo DOCX, sus documentos siempre serán compatibles con Microsoft Word.


Calcule con PlanMaker 2018

El potente programa de hojas de cálculo PlanMaker 2018 hace que crear hojas de cálculo, cálculos y gráficos de cualquier complejidad sea tarea fácil, tanto si se trata de un simple horario como de una completa planificación financiera de su empresa. Como utiliza de forma nativa el formato XLSX, puede editar archivos de Microsoft Excel de forma fiel al original.


Presente con Presentations 2018

Cree presentaciones impresionantes con Presentations 2018 Gracias a sus imponentes animaciones, sus transiciones de diapositivas y su sofisticado concepto de página maestra estará perfectamente preparado para cualquier presentación. Como Presentations usa el formato PPTX de forma nativa, el intercambio de documentos con PowerPoint se realiza sin fallos.






Requisitos del sistema


SoftMaker Office para Windows se puede utilizar en los siguientes sistemas operativos:

Microsoft Windows 10, 8, 7 (64 Bits)
Microsoft Windows Server 2008 R2 o superior
Es compatible con servidores de aplicaciones como Citrix XenApp y Microsoft Terminal Server.
