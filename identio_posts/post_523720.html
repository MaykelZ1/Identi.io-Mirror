Emulador FB Alpha +10,920 Roms Multi-consola [GDRIVE] y [MEGA]

FBA es un multi-emulador, es decir, es capaz de emular diferentes placas arcade y algunas consolas (algo así como Mame pero no engloba tantos sistemas). Esta versión partió del extinto Final Burn que desarrolló Dave y que posteriormente daría lugar a este fork que estamos comentando con el apodo de ‘Alpha’ y que esta siendo mantenido en la actualidad por los desarrolladores BarryHarris, Jan_Klaassen, KEV, LoopMaster, Mike Haggar, Hyper Yagami, iq_132,Ayeye y dink.

Sirve para emular todo a continuación:
Capcom CPS-1
Capcom CPS-2
Capcom CPS-3
Cave
Data East DEC-0, DEC-8 and DECO IC16 based games
Galaxian based hardware
Irem M62, M63, M72, M90, M92 and M107 hardware
Kaneko 16
Konami
Neo-Geo
NMK16
Pacman based hardware
PGM
Psikyo 68EC020 and SH-2 based hardware
Sega System 1, System 16 (and similar), System 18, X-Board and Y-Board
Super Kaneko Nova System
Toaplan 1
Toaplan 2
Taito F2, X, Z and others

FB Alpha también tiene drivers para las siguientes consolas:
Sega Megadrive/Genesis
ColecoVision,
Sega SG-1000,
Sega Master System/Game Gear
PC-Engine/TurboGrafx 16/SuperGrafx.


Tutorial Configurar Mando:
1) Abrir FB ALPHA
2) Cargar el juego
2) Ir a la pestaña INPUT > MAP GAME INPUT (F5) > SELECCIONAR PLAYER 1 > JOYSTICK O TECLADO > PONER NORMAL Y USE PRESET
