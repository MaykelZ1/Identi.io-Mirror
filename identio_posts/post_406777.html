Windows 10 LTSB x64 Dic2015 esp + OEM-DWS-AOMEI


Amigos de identi.li
En esta oportunidad les comparto Windows 10 Enterprise LTSB x64
en Español actualizado a Diciembre 2015
Las gracias a Generation2 @KAT & @TPB por este gran release
Asimismo incluyo algunas herramientas de ajuste final para Windows 10




1. Requisitos mínimos de hardware

Procesador: 1 GHz o más rápido o SoC
RAM: 2 GB para 64 bits
Espacio en disco duro:  20 GB para 64 bits
Tarjeta gráfica:  DirectX 9 o posterior con un controlador WDDM 1.0
Pantalla: 800 x 600


2. Branches

Se supone que LTSB es el branch que viene con menos apps Metro (más liviana en algunos aspectos), además que la actualización no es mandatoria como en CB y CBB.
Mejor les dejo la sgte tabla (tomada de es.wikipedia) donde se ven las diferencias entre los diferentes branches de Windows 10: CB, CBB, LTSB





3. Instalación

El iso (links de descarga al final) Windows 10 Enterprise LTSB x64 viene con 3 opciones:
STD: standard, sin activar, sin OEM
OEM: le pone a Windows el wallpaper y los logos de la marca de la computadora, sin activar
KMS: instalación con Windows 10 LTSB activado con KMS



En mi caso elegí la opción KMS; apenas finalizada la instalación le apliqué OEM Info Updater y DWS, y limpieza con Ccleaner, para recién activar WiFi





4. OEM Info Updater 7  web

Se ve cool una computadora que tiene los logos y el wallpaper de la marca. Por ello les dejaré este software portable para que personalicen su instalación de Windows 10.
Screenshot luego de aplicar OEM Info Updater.
 




5. DWS Final (Destroy Windows 10 Spying)  web

Este pequeño software de código abierto permite alivianar Windows 10, ya que le quita/elimina aplicaciones Metro que consumen recursos y de los cuales pueden prescindirse.
Además bloquea actualizaciones automáticas, así permite al usuario actualizar su Sistema Operativo cuando lo estime conveniente.
Otra función de DWS es bloquear la telemetría y seguimiento, mediante los cuales, y utilizando recursos de hardware y ancho de banda de internet, Windows 10 envía info de lo que realizamos en nuestra computadora a Microsoft y sus partners.
Aquí unos screenshots de las funciones y utilidades de DWS aplicadas al máximo para Windows 10.









6. AOMEI Backupper Standard  web

Con sólo 36MB este software (gratuito, no requiere serial) puede realizar backups del sistema operativo Windows 7, 8.1, 10 incluyendo los aplicativos instalados.
Como usuario de Windows 7 no estaba convencido de probar Windows 10. Este software me permitió hacer backup de Windows 7 y tranquilo probar Windows 10 antes de decidirme a usarlo.





Las herramientas 4, 5, 6 me ayudaron a dejar Windows 7 Home Premium, y darle una oportunidad a Windows 10. Lo tengo instalado en mi netbook 11.6" que tiene un hardware limitado con procesador Celeron N2830 de doble núcleo, 2GB de RAM. Siento la netbook más liviana con Windows 10 que con Windows 7, obtuve algo así como un Windows 10 LTSB Lite, además que se ve más cool una netbook actualizada con Windows 10, MS-Office 2016 + Project 2016, AutoCAD 2015.
