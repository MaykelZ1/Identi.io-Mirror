AIDA64 Engineer v5,70,3800

  • Categoría: Software / Programas
  • Publicado hace más de 9 años




AIDA64 recopila y muestra una amplia gama de detalles para su hardware y software. Para averiguar lo que está dentro de una PC, ¿qué tipo de hardware que se está tratando, sólo tiene que echar un vistazo a la página Resumen. Si usted necesita más detalles para un componente de hardware en particular, siempre estás a sólo unos clics de distancia de una página específica de información de hardware. La página de resumen también le ofrece enlaces a sitios web de los fabricantes de hardware, y señala de actualización de controladores y BIOS de descarga instalaciones.


AIDA64 Engineer tiene un motor de detección de hardware sin igual en su clase. Proporciona información detallada sobre el software instalado y ofrece funciones de diagnóstico y apoyo para el overclocking. Como está monitoreando sensores en tiempo real, que pueda reunir las lecturas de voltaje exacto, la temperatura y la velocidad del ventilador, mientras que sus funciones de diagnóstico ayudan a detectar y prevenir los problemas de hardware. También ofrece un par de puntos de referencia para medir el rendimiento de los componentes individuales de hardware o el sistema entero. Es compatible con todas las versiones de Windows de 32 bits y 64 bits, incluyendo Windows 8.1 y Windows Server 2012 R2.



*Características:
Diagnóstico
AIDA64 ofrece más de 50 páginas de información sobre la configuración de hardware y más de 50 páginas de programas instalados, licencias de software, aplicaciones de seguridad y configuración de Windows.
Las pruebas de Stress
AIDA64 Prueba de estabilidad del sistema utiliza un módulo de prueba de esfuerzo multi-hilo de 64 bits para conducir el equipo a sus límites absolutos. Disco duro, SSD y OpenCL GPGPU pruebas de estrés adaptador de vídeo también está disponible.
Precisión
AIDA64 tiene las capacidades de detección de hardware más precisos en su clase, para proporcionar información detallada sobre el funcionamiento interno de ordenador sin necesidad de abrirlo. El módulo de detección de hardware se ve reforzada por una base de datos de hardware exhaustiva celebración de más de 176.000 entradas.
Auditoría de software
AIDA64 ofrece más de 50 páginas de información sobre los programas instalados, licencias de software, aplicaciones de seguridad y configuración de Windows. Lista de los procesos iniciados, servicios, archivos DLL, programas de inicio y las páginas web visitadas también está disponible.
Soporte para pantalla externa
AIDA64 puede mostrar los datos vitales del sistema en una variedad de dispositivos de LCD / VFD conectados al ordenador. La última versión AIDA64 soporta más de 50 pantallas LCD y VFD, e introduce el monitoreo remoto inalámbrico de valores de los sensores en los teléfonos inteligentes y las tabletas.
Mediciones
AIDA64 es compatible con más de 250 diferentes dispositivos sensores para medir temperaturas, voltajes, velocidades del ventilador y el consumo de energía. Los valores medidos se pueden visualizar en los iconos de la bandeja del sistema, el panel OSD, gadgets de escritorio, Logitech G15/G19 Gaming Keyboard LCD, y Razer Switchblade LCD.

*Requisitos del sistema:
Sistema operativo: Windows 95/98/Me, Windows NT4/2000, Windows XP, Windows PE, Windows Server 2003, Windows Vista, Windows Server 2008, Windows 7, Windows Server 2008 R2, Windows 8, Windows Server 2012, Windows 8.1, Windows Server 2012 R2, Windows 10, Windows Server 2016
Procesador: Intel Pentium o posterior
Espacio en disco necesario para instalar: 30 MB

*Cambios:
AVX and FMA accelerated FP32 and FP64 ray tracing benchmarks
Vulkan API information for graphics accelerators
Support for Microsoft Windows 10 Redstone RS1 Insider Preview
Improved support for AMD Zen Summit Ridge CPUs
Improved support for Intel Kaby Lake CPUs
Sensor support for Corsair AXi power supply units
Corsair Link sensor support
Support for Corsair Strafe, Logitech G13, Logitech G19, Logitech G19s RGB LED keyboards
Support for Corsair, Logitech, Razer RGB LED mice
Improved support for Koolance and T-Balancer sensor devices
Support for multiple pages in Logitech Arx and RemoteSensor
ACPI 6.1 support
Kingston SSDNow UV300, Samsung CM871, Samsung PM871 SSD support
GPU details for AMD Radeon R5 340X and Radeon R7 350X
GPU details for NVIDIA GeForce 920MX, GeForce 930MX and GeForce 940MX
