Extensiones para Sublime Text 3 [Linux]

  • Categoría: Linux
  • Publicado hace más de 8 años
 
 
SublimeText
Sublime Text es un excepcional editor de textos que aporta muchas características útiles a la hora de programar o editar código. El editor está cargado de funcionalidades útiles y cómodas desde el punto de la usabilidad y eficiencia, utilizando el método geek y convirtiendo nuestro trabajo de edición de texto en una experiencia cada vez más sencilla y agradable, a medida que vamos aprendiendo a utilizar todas sus funcionalidades.

Sin duda alguna, es uno de los editores de código más utilizados por los desarrolladores y programadores. Algunos por sus múltiples características, usabilidad, estética.

Una de las razones por las que me gusta, es por la cantidad de plugins adicionales que se le pueden agregar, a pesar de las funcionalidades que ya tiene, que lo hacen sumamente útil.


1.- Package Control:
Por defecto, el Package Control viene desactivado en Sublime Text 2. Para activarlo, vamos al menú View/Show Console y pegamos el siguiente texto (ST3): Esto es para sublime text 2, para sublimetext 3 en su build 3126 solo basta presionar la teclas CTRL+SHIFT+P

Cita: Mostrar
import urllib.request,os,hashlib; h = '7183a2d3e96f11eeadd761d777e62404' + 'e330c659d4bb41d3bdf022e94cab3cd0'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://sublime.wbond.net/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by)

Ahora, cada vez que queramos buscar un paquete en el repositorio para instalarlo, pulsamos CTRL+SHIFT+P y escribimos Install Package. Entonces podremos buscar entre la gran cantidad de paquetes que existen para Sublime Text e instalarlos con un simple click.
 
  

 
 
De la misma forma, pulsando  CTRL+SHIFT+P y escribiendo Remove Package podemos desinstalar paquetes que no queramos utilizar más.

Si lo deseas, también puedes acceder a Package Control Repository para buscar paquetes vía web, categorizados en secciones como nuevos, tendencias, populares, autores u otros.


- IMPORTANTE: Asegúrate de instalar plugins compatibles con la versión de Sublime Text que estés utilizando. Si estás utilizando ST3, es posible que no puedas utilizar paquetes diseñados para otra versión.

Este plugin es necesario para instalar paquetes adicionales, configurar demás funciones etc.


2.- BracketHighlighter
Este plugin está disponible en BracketHighlighter, y su principal función es resaltar con colores las aperturas y cierres de {}, [], "", <>, (), y de más etiquetas de html, php, y muchos lenguajes de programación.




Esto es realmente útil cuando se tiene muchas líneas de código y no se puede diferenciar dónde comienza una sección y termina otra.

La instalación de paquetes adicionales mediante Package Control, es simple, primero presionar CTRL+SHIFT+P luego, escribir Install Package, luego abrirá una ventana en la que escribimos el nombre del paquete a instalar, solo clic en instalar y listo.


3.- CSS Format
Este plugin es mi favorito, lo que hace es identar/desglosar código css, por ejemplo, cuando se descarga un archivos css de algún sitio web, usualmente viene una sola línea todo el css, pues con esta extensión se evitar desglosarlo manualmente. El link oficial es CSS-Format
  
 

 
 
4.- Emmet
Es un plugin que autocompleta fragmentos de código, a parte de los que ya incluye sublimetext, de una manera más cómoda y más fácil. Su link oficial es Emmet
  
 


5.- DocBlockr
Esta extensión permite agregar comentarios en las funciones, o código en el que se ejecute. Siguiendo la estructura correspondiente a documentación de java. Su link oficial es: DocBlockr
 
 

 
 
6.- ColorPicker
ColorPicker es un menú auxiliar para seleccionar un color desde una paleta de colores, esta aparece al presionar CTRL+SHIFT+C  Su link oficial es: ColorPicker
 
 


 
 
Y esas son mis extensiones favoritas para sublime text 3, en su build 3126. Estos paquetes también se pueden descargar por separado y agregarlos en la ruta ~/.config/sublime-text-3/Installed Packages al menos eso es para Linux, más específicamente Fedora.
