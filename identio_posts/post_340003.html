(Truco) Agua Flotante  (revelado)

----------------------------------------------------------

Jugar con agua es muy divertido, pero toma algunas precauciones cuando hagas este truco. Coloca el vaso sobre una vasija grande o una bandeja. Utiliza un vaso de agua de cristal grueso que no se rompa fácilmente o mejor uno irrompible. Al hacer este juego, vuelca el vaso lo más rápidamente que puedas. Necesitarás practicarlo primero. Si quieres saber como hacer este truco de magia mira el siguiente artículo de unComo de cómo hacer el truco de magia el agua flotante
----------------------------------------------------



----------------------------------------------------

Primero de todo coloca un vaso vacío con el borde hacia abajo sobre una lámina de plástico duro. Marca el borde con un bolígrafo (a). corta con unas tijeras siguiendo la linea (b) para obtener un círculo.
----------------------------------------------------


----------------------------------------------------

Después para hacer el juego, esconde el circulo de plástico bajo el trozo de papel duro. Llena el vaso. Sujeta el círculo bajo el papel y colócalo sobre el vaso.
----------------------------------------------------

----------------------------------------------------

A continuación comprueba si el círculo está cubriendo exactamente el borde. Presiona el papel sobre el vaso con una mano y vuélvelo con la otra.
----------------------------------------------------


----------------------------------------------------

Para finalizar separa el papel muy despacio y con cuidado. El círculo de plástico se pegará al vaso, sosteniendo el agua. A corta distancia, parecerá un vaso mágico.

----------------------------------------------------
