Hola Amigos Hoy les vengo Hablando de El nuevo Sistema Operativo de Microsoft!!
 
Windows 10 Será el Nuevo S.O a lanzar por la compañia de Microsoft el cual se espera para este año existen muchos rumores sobre este sistema sin embargo la compañia a lanzado la primera version de lo que podremos esperar de este S.O Hoy les traigo algunas capturas de las novedades de este S.O


El botón de inicio a regresado junto con el menú de inicio y se ah acoplado las aplicaciones en el de una mejor manera y mas optimizada


la compatibilidad con los programas se mantiene me tome la libertad de instalar photoshop y para ser un S.O de prueba lo corre bastante bien.




la implementancion de poder agregar mas de un escritorio (Característica de Distribuciones Linux) y el efecto de las ventanas el cual recuerda al ambiente de Gnome 



Hasta hace poco tiempo se tenia problemas para instalar los Office 2013 en el S.O sin embargo este error ya fué solucionado y ya puede ser instalado en la version 64 bits


Windows Defender Seguirá siendo el Antivirus Por Defecto del Sistema De Microsoft 


El CMD trae algunas mejores bastante utiles y muy atractivas una de ellas es poder modificar la transparencia de la ventana



Tendremos una nueva carpeta llamada Windows System la cual tendrá múltiples herramientas útiles.

Con la llegada de windows 10 se rumora que este podría ser gratis cambiando de esta manera el modelo de negocio de la compañía de Microsoft por ahora solo habrá que esperar hasta que la compañía lance su nuevo sistema operativo el cual parece ser muy prometedor espero les haya gustado el post y cualquier información adicional que tu sepas de este sistema y quieras hacer tu aporte puede dejarlo en los coments nos vemos en la próxima Identiqueros!!!
