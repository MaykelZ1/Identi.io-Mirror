letras en varios sitios web

TAudio Converter
Convertidor de Audio




TAudioConverter apoya varios codificadores. Puede convertir a MP3, AAC, AAC HE, AC3, Musepack, Ogg Vorbis, Opus, WMA, DTS, ALAC, FLAC, Monkey Audio, TAK, TTA, WavPack, AIFF y PCM.


TAudioConverter es multiproceso. Esto significa que se ejecutará varios codificadores en paralelo para acortar el tiempo de codificación. El número de procesos paralelos depende de los núcleos de la CPU que tenga.


También puede extraer pistas de audio de archivos de vídeo. Esto se puede hacer con o sin que codifica la secuencia de audio. Incluso si el archivo de vídeo tiene más de un flujo de audio, puede seleccionar el que usted desea.

Extractor de CD tiene soporte freedb.


Fusión de audio. Combinar varios archivos de audio en una sola.


Se puede copiar las etiquetas y carátulas de la fuente.

TAudioConverter puede aplicar ciertos efectos a audio. Se puede cambiar el nivel de volumen de audio y velocidad de reproducción.


Y mucho más ...

TAudioConverter es software libre de código abierto.

BlueGriffon es una aplicación intuitiva que proporciona a los autores web (principiantes o avanzados) con una interfaz de usuario sencilla que permite crear sitios web atractivos sin necesidad de amplios conocimientos técnicos sobre estándares web.


Debido a que Gecko vive dentro BlueGriffon, el documento se edita se verá exactamente igual en Firefox. Los usuarios avanzados pueden utilizar siempre la vista de origen para codificar su página.
