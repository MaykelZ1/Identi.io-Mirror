1. Si tienes instalado algún componente de Microsoft Visual C+, procura tener una sola versión de cada año ejemplo:

Microsoft Visual C++ 2010  x86 Redistributable - 10.0.40000
Microsoft Visual C++ 2010  x64 Redistributable - 10.0.40000
Microsoft Visual C++ 2010  x64 Redistributable - 10.0.40100

Desinstalamos la versión mas antigua y dejamos asi:

Microsoft Visual C++ 2010  x86 Redistributable - 10.0.40000

- Clic derecho en el rar#1
