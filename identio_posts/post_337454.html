Proteja a sus Hijos Bloqueando Páginas Porno
Un software de filtro de red para proteger a los niños frente al contenido web inadecuado
No puedes permitir que tus hijos lean cualquier cosa. Pero, cuando navegan por Internet, donde no hay filtros, ellos pueden ver cualquier cosa si no estás allí para instalar Anti-Porn como software de filtrado.
Anti-Porn es un software de filtro de red completo, para proteger a los niños contra el contenido web pornográfico. Anti-Porn filtrará los sitios web para adultos, que muestran imágenes que no son adecuadas para los niños y también los chats por Internet, cuando se utiliza lenguaje ofensivo.
Anti-Porn puede filtrar automáticamente decenas de miles de sitios web pornográficos y también puede bloquear cualquier sitio web que desees prohibir.
Además, Anti-Porn puede filtrar las informaciones de Internet mediante el bloqueo de algunas palabras ofensivas - se pueden incluir palabras especiales y los niños no podrán ver ningún tipo de material de Internet, que contenga esas palabras.
Anti-Porn puede suministrar la hora estándar del mundo y puedes configurar el programa para que permita a tus hijos usar el Internet en determinados momentos de ciertos días. Anti-Porn guarda un registro completo de todos los sitios web visitados, filtrados o no, y mantiene la pista de todo el material surgido en el ordenador, incluyendo archivos, imágenes, música, etc.
Puedes tener el inventario del historial del ordenador de tu hijo. Anti-Porn ha sido un filtro neto para tus hijos.
Por otro lado, los adultos que conocen la contraseña pueden navegar por Internet ilimitadamente. Filtra los sitios web con contenido para adultos - Anti-Porn ha bloqueado miles de sitios web con contenido pornográfico y palabras depredadoras, y también puedes predefinir las cosas que no quieres que tus niños vean, por ejemplo, sitios web y palabras ofensivas.
Limita el acceso a Internet por día y hora - puedes asignar fácilmente el tiempo de acceso a Internet a ciertas horas, incluso en ciertos días.
Limita el chat de Internet - Anti-Porn puede limitar los populares softwares de chat por Internet y también el chat de páginas web. Mantiene un registro completo del historial del ordenador. Registra los datos de tráfico para obtener una descripción general del uso de Internet.
Estas son algunas de las características:
Bloquea los sitios web con contenido pornográfico:
Anti-Porn bloquea docenas de miles de sitios web con contenido pornográfico. Además, puedes elegir los sitios web que deseas ser prohibidos para tus niños.
Limita el chat por Internet:
Anti-Porn puede limitar el uso de softwares populares de chat y el acceso a las principales páginas web de charla.
Limita el acceso a Internet por día y hora:
Puedes asignar fácilmente el tiempo de acceso a Internet a ciertas horas de ciertos días. Una característica especial de Anti-Porn es que el tiempo del PC será calibrado como tiempo de Greenwich.
Mantén un registro completo del historial del ordenador:
Anti-porn mantiene un registro detallado de todos los sitios web visitados, bloqueados o no.
Registra los datos de tráfico para una visión general del uso de Internet:
Puedes realizar un inventario del ordenador de tus hijos y encontrar todas las fotografías, películas, archivos de música y páginas web.

Anti-Porn ha recomendado varios sitios web que son aptos para los niños. También puedes agregar cualquier otro.
