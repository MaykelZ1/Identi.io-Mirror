Repositorios para Debian Stretch

  • Categoría: Linux
  • Publicado hace más de 7 años

 
Repositorio Backports
 
Este repositorio tiene paquetes actualizados de muchas aplicaciones y librerías, para tener versiones actualizadas de los que cuenta los repositorios oficiales de Debian GNU/Linux. Los repositorios Backports se subdividen en main, contrib y non-free. Este repositorio cuente con software no libre.
 
Para a agregar este repositorio, tecleamos en una terminal.

echo deb http://ftp.debian.org/debian stretch-backports main contrib non-free | sudo tee /etc/apt/sources.list.d/stretch-backports.list

Para eliminar el repositorio.

sudo rm  /etc/apt/sources.list.d/stretch-backports.list


Repositorio de Deb-Multimedia
 
Este repositorio cuenta con aplicaciones y librerías multimedia, podemos utilizar este repositorio para actualizar aplicaciones existentes en los repositorios oficiales o para instalar aplicaciones que no lo están. Este repositorio cuente con software no libre.
 
Para a agregar este repositorio, tecleamos en una terminal.

echo deb http://www.deb-multimedia.org tretch main non-free | sudo tee /etc/apt/sources.list.d/deb-multimedia.list

Agregamos la llave de seguridad del repositorio.

sudo apt-key adv --keyserver keys.gnupg.net --recv-key 65558117


Para eliminar el repositorio.

sudo rm  /etc/apt/sources.list.d/deb-multimedia.list


Repositorio Google Chrome
 
Este es el repositorio que contiene los paquetes del navegador web Google Chrome.
 
Para a agregar este repositorio, tecleamos en una terminal.

echo deb http://dl.google.com/linux/chrome/deb/ stable main | sudo tee /etc/apt/sources.list.d/google-chrome.list

Agregamos la llave de seguridad del repositorio.

wget -q -O - https://dl-ssl.google.com/linux/linux_signing_key.pub | sudo apt-key add -

Para eliminar el repositorio.

sudo rm  /etc/apt/sources.list.d/google-chrome.list

Repositorio WineHQ
 
Este es el repositorio que contiene los paquetes de Wine. Esta herramienta nos permite instalar Software de MS Windows en GNU/Linux.
 
Para a agregar este repositorio, tecleamos en una terminal.

echo deb https://dl.winehq.org/wine-builds/debian/ stretch main | sudo tee /etc/apt/sources.list.d/winehq.list

Agregamos la llave de seguridad del repositorio.

sudo apt-key adv --keyserver keys.gnupg.net --recv-key 5FCBF54A


Para eliminar el repositorio.

sudo rm  /etc/apt/sources.list.d/winehq.list



Repositorio PlayOnLinux
 
Este es el repositorio que contiene los paquetes de PlayOnLinux. Esta herramienta nos facilita la instalación Software de MS Windows, a través de Wine, en GNU/Linux.

sudo apt-key adv --keyserver keys.gnupg.net --recv-key 88E3D6C3

Para eliminar el repositorio.

sudo rm  /etc/apt/sources.list.d/playonlinux.list



Repositorio Virtualbox
 
Este es el repositorio que contiene los paquetes de Virtualbox. Este software multiplataforma nos permite la creación de Maquinas Virtuales.
 
Para a agregar este repositorio, tecleamos en una terminal.


echo deb http://download.virtualbox.org/virtualbox/debian stretch contrib non-free | sudo tee /etc/apt/sources.list.d/virtualbox.list
Agregamos la llave de seguridad del repositorio

sudo apt-key adv --keyserver keys.gnupg.net --recv-key 2980AECF

Para eliminar el repositorio.

sudo rm  /etc/apt/sources.list.d/virtualbox.lis

 
Repositorio Soundnode
 
Este es el repositorio que contiene los paquetes de Soundnode (cliente de SoundCloud). Necesita el paquete apt-transport-https para el funcionamiento de este repositorio.
 
Para a agregar este repositorio, tecleamos en una terminal.

echo deb https://packagecloud.io/JonasGroeger/soundnode/debian/ stretch main | sudo tee /etc/apt/sources.list.d/JonasGroeger_soundnode.list

Agregamos la llave de seguridad del repositorio.

curl -L https://packagecloud.io/JonasGroeger/soundnode/gpgkey | sudo apt-key add -

Para eliminar el repositorio.

sudo rm  /etc/apt/sources.list.d/JonasGroeger_soundnode.list

echo deb http://dl.google.com/linux/webdesigner/deb/ stable main | sudo tee /etc/apt/sources.list.d/google-webdesigner.list

wget -q -O - https://dl-ssl.google.com/linux/linux_signing_key.pub | sudo apt-key add -

Para eliminar el repositorio.

Repositorio Skype
 
Este es el repositorio que contiene los paquetes de Skype. Este es un software propietario distribuido por Microsoft tras haber comprado la compañía homónima y que permite comunicaciones de texto, voz y vídeo sobre Internet (VoIP).
 
NOTA: Necesita el paquete apt-transport-https para el funcionamiento de este repositorio.
 
Para a agregar este repositorio, tecleamos en una terminal.

echo deb https://repo.skype.com/deb stable main | sudo tee /etc/apt/sources.list.d/skype.list

Agregamos la llave de seguridad del repositorio.

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 1F3045A5DF7587C3

Para eliminar el repositorio.

sudo rm  /etc/apt/sources.list.d/skype.list
