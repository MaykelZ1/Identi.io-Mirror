Windows Repair Toolbox (Reparar windows XP a W10)

Windows Repair Toolbox

Windows, por defecto, no proporciona herramientas nativas que nos permitan conocer de forma rápida y sencilla información detallada sobre nuestro software o hardware o sobre el estado de los mismos. Por ello, siempre que tengamos que realizar ciertas labores de mantenimiento o reparación debemos optar por usar herramientas de terceros. Mientras que el número de aplicaciones de terceros desarrolladas para estos fines es muy elevado, existe una solución desde donde podremos acceder rápidamente a cualquiera de ellas, según nuestras necesidades: Windows Repair Toolbox.

Windows Repair Toolbox es una herramienta formada por un gran número de herramientas y aplicaciones de terceros con las que poder monitorizar el estado de nuestro ordenador, controlar prácticamente cualquier aspecto del mismo y ayudarnos a solucionar posibles problemas existentes en él.

Este kit de herramientas es compatible con todas las versiones de Windows a partir de XP y el único requisito para funcionar es que tengamos .Net Framework instalado en el sistema. Esta aplicación se distribuye de forma portable y ocupa tan solo 2 megabytes, sin embargo, por defecto no incluye ninguna de las herramientas de terceros de las que hace uso, aunque estas se descargan automáticamente cuando las lanzamos por primera vez. Si no tenemos conexión a Internet, este kit de herramientas hace uso de varias aplicaciones nativas de Windows (como chkdsk, sfc, regedit o msconfig), las cuales no son necesarias descargar.

Cómo funciona Windows Repair Toolbox

Lo primero que debemos hacer es descargar este kit de herramientas de forma totalmente gratuita desde su página web principal. Como hemos dicho, el kit de herramientas ocupa tan solo 2 MB, y es portable, por lo que no necesita instalación.

Una vez descargado lo ejecutamos y veremos la ventana principal del programa. Todas las herramientas incluidas en este programa se dividen en varias categorías y pestañas:

Tools: Una serie de herramientas de análisis, monitorización y recuperación del sistema operativo.

Windows Repair Toolbox - Tools
Malware Removal: Herramientas para la eliminación de todo tipo de software malicioso. Este apartado nos va a permitir también actualizar nuestro Windows y otras aplicaciones críticas, como Flash, Java o Adobe Reader.

Windows Repair Toolbox - Malware
Final Tests:  Una serie de tests que nos permiten comprobar el estado de determinados ajustes o la asociación de Windows para ciertos formatos de archivos. Este apartado nos permite también, por ejemplo, comprobar el funcionamiento de nuestra Webcam y nuestro micrófono.

Windows Repair Toolbox - Final Tests
Notes: Un bloc de notas con formato, muy útil, por ejemplo, para copiar, pegar y guardar rápidamente los logs de los diferentes programas.

Windows Repair Toolbox - Notes
Settings: Ajustes de la herramienta, por ejemplo, para buscar actualizaciones o eliminar los programas descargados.

Windows Repair Toolbox - Settings
Como podemos ver, la mayor parte de las herramientas se encuentran dentro de la pestaña ?Tools?. En esta pestaña podremos ver también, en la parte inferior, cierta información sobre nuestro equipo, como el sistema operativo, el procesador, la memoria la temperatura y si tenemos conexión a Internet.

Si pulsamos sobre una de las herramientas de Tools, por ejemplo, CPU-Z, automáticamente comenzará la descarga de la versión más reciente de la aplicación y, una vez finaliza, se ejecutará la aplicación en cuestión.

Windows Repair Toolbox - Cpu-Z
Sin duda, Windows Repair Toolbox es la mejor navaja suiza para todos aquellos que necesiten tener a mano las diferentes herramientas de mantenimiento, monitorización y control del sistema operativo. Gracias a su naturaleza portable podemos llevarla siempre con nosotros en una memoria USB y, además, si previamente hemos descargado las correspondientes aplicaciones y copiamos la carpeta ?Windows-Repair_Toolbox? que nos crea en raíz de C: podremos hacer uso de las aplicaciones siempre que las necesitemos, en cualquier ordenador, incluso sin conexión a Internet.

Para un usuario sin conocimientos puede resultar un poco caótica la interfaz pero con un mínimo de conocimientos sobre Windows y sus herramientas vamos a poder localizar rápidamente la que necesitamos en un momento en concreto. En la pestaña de herramientas tenemos varias utilidades para diagnosticar el hardware, realizar copia de seguridad y recuperar archivos, acceso a utilidades nativas de Windows, ?reparadores? del equipo, utilidades para desinstalar programas o una colección variada de otras utilidades.

Pero no es la única pestaña interesante a la que tenemos acceso. La pestaña Malware Removal nos da acceso a varias utilidades para limpiar el ordenador de virus y otras amenazas que comprometen su funcionamiento. En esta pestaña también tenemos algunas herramientas de optimización y la posibilidad de actualizar directamente Windows, Java, Adobe Flash o Adobe Reader.

Por último, la pestaña Final Tests nos permite realizar una serie de comprobaciones sobe algunos componentes, configuraciones del sistema o asociaciones de extensiones con determinados programas. Como vemos, no es la herramienta más intuitiva y sencilla del mercado pero sí nos da acceso a decenas de utilidades para resolver casi cualquier problema con Windows.




Requisitos:
Net Framework 4
Conexion a Internet
Compatible con Windows XP hasta Windows 10

Peso: 2mb

Servidor: Mega

Portable
