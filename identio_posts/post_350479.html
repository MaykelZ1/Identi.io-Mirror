Microsoft Desktop Optimization Pack 

2014 R2 Español 






¡Virtualiza, maneja y restaura sistemas operativos!







El Microsoft Desktop Optimization Pack (MDOP) es un conjunto de tecnologías disponibles como una suscripción para clientes de Software Assurance. Las tecnologías de virtualización de MDOP ayudan a personalizar la experiencia del usuario, simplificar el despliegue de aplicaciones, y mejorar la compatibilidad de las aplicaciones con el sistema operativo de Windows (UE-V/App-V/MED-V). Además, MDOP ayuda a administrar y proteger su dispositivo, lo que permite el seguimiento y la implementación de las funciones de Windows clave (MBAM / AGPM). El uso de MDOP cambia reparación escritorio de reactivo a proactivo, ahorrando tiempo y eliminando los retos asociados con la solución de problemas y la reparación de los fallos del sistema (DART).
