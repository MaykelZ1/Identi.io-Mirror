Al hacer clic en un enlace de descarga en un navegador, IDM se hará cargo de la descarga y acelerarlo. IDM apoya HTTP, FTP, HTTPS y protocolos MMS.
• Aceleración de la velocidad de descarga.
Internet Download Manager 6.28 Build 8 puede acelerar las descargas por hasta 5 veces debido a su tecnología inteligente de segmentación dinámica de archivos. A diferencia de otros gestores de descarga y aceleradores de Internet Download Manager 6.28 Build 8 segmentos de los archivos descargados dinámicamente durante el proceso de descarga y reutiliza conexiones disponibles sin etapas adicionales de conexión y acceso para alcanzar el mejor rendimiento de aceleración.
• Reanudar la descarga.
Internet Download Manager 6.28 Build 8 se reanudará sin terminar la descarga desde el lugar donde lo habían dejado.

programa de instalación rápida y sencilla hará los ajustes necesarios para usted, y compruebe su conexión al final para garantizar sin problemas la instalación de Internet Download Manager 6.28 Build 8

Usted puede simplemente arrastrar y soltar los enlaces a IDM, y arrastrar y soltar los archivos descargados de Internet Download Manager 6.28 Build 8 .
• Antivirus de control automático.
Antivirus control hace que su descarga libre de virus y troyanos.
• Navegador de integración avanzada.
Cuando está activada, la función se puede utilizar para la captura de cualquier descarga desde cualquier aplicación. Ninguno de los gestores de descarga tienen esta característica.
• Construido en el planificador.
Internet Download Manager 6.28 Build 8 puede conectarse a Internet a una hora determinada, descargar los archivos que desee, desconectar o apagar el ordenador cuando está hecho.
• IDM sitio web incluye araña y grabber.
IDM descarga todos los archivos necesarios que se especifican con los filtros de sitios web, por ejemplo, todas las imágenes de un sitio web, o subconjuntos de sitios web, o páginas web completas para la navegación fuera de línea. Es posible programar múltiples grabber proyectos a ejecutar una vez en un tiempo especificado, detenerlos en un momento determinado, o ejecutar periódicamente para sincronizar los cambios.
• IDM apoya muchos tipos de servidores proxy.
Por ejemplo, IDM trabaja con Microsoft ISA, FTP y servidores proxy.
• IDM apoya principales protocolos de autenticación:
Básica, negociar, NTLM, y Keberos. Así IDM puede tener acceso a Internet y muchos servidores proxy usando nombre de usuario y contraseña.
• Descargar toda la característica.
IDM puede agregar todas las descargas relacionadas con la página actual. Es fácil de descargar varios archivos con esta característica.
• Interfaz personalizable.
Usted puede elegir el orden, y qué botones y columnas aparecen en la ventana principal de IDM.
• Descargar las categorías.
Internet Download Manager 6.28 Build 8 puede utilizarse para organizar las descargas automáticamente utilizando las categorías definidas de descarga.
• Función de actualización rápida.
Rápida actualización puede comprobar si hay nuevas versiones de IDM y actualización IDM una vez por semana.
• Descargar límites.
Descarga progresiva de las cuotas función. La función es útil para conexiones que utilizan algún tipo de política de acceso equitativo (o FAP) como Direcway, la PC directa, Hughes, etc
