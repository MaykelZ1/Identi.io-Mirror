DriverPack Solution 14.14 Edicion full 2015

  • Categoría: Software / Programas
  • Publicado hace más de 10 años


El programa más popular para instalar automáticamente, actualizar y buscar los controladores. 
Es un gestor versátil para instalar controladores para todas las versiones de Windows. A diferencia de los 7 actualizaciones de controladores incorporados en Windows, este programa se puede utilizar incluso sin Internet y encontrar / instalar controladores para dispositivos no sólo VENDER populares (como es el caso de Windows Update).

 

Características principales:
- Funciona con cualquier paquetes de controladores (archivos 7z, zip), para cualquier sistema operativo desde Windows XP
- Trabajar como un poco 32 y una de 64 bits
- Capacidad de grupo y la costumbre instalación de controladores
- Capacidad para manejar archivos en lotes terminar Bashrat'a
- Configuración de la aplicación transparente (todos los ajustes se cambian a través de los ajustes). Todos los ajustes se almacenan en un archivo ini
- Crear rápidamente una base de datos de los conductores (la capacidad de crear todas las bases de datos con un solo clic)
- Interfaz amigable y sensible
- Ver una lista de dispositivos para el PC local
- Ver conductores oxidasa para el paquete de controladores
- Capacidad para trabajar con cualquier operador
- Posibilidad de añadir y lanzamiento de la interfaz de herramientas adicionales (para agregar, utilice el menú Opciones - herramientas adicionales). Es decir, en cierta medida se puede utilizar como un autorun para sus colecciones
- Compruebe si hay actualizaciones del programa a través de Internet
- Posibilidad de instalación silenciosa
- La interfaz multilingüe
