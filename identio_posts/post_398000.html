Software Textil - Gerber Accumark

  • Categoría: Software / Programas
  • Publicado hace más de 9 años



El programa AccuMark es el estándar para el diseño de patrón, graduación y trazo/marcadas de corte en las industrias de ropa, interiores automóviles, de transporte, muebles e industrias de confección. Es utilizado por miles de compañías en todo el mundo en más de 20 idiomas.
 
 Contiene el juego más amplio de herramientas de graduación y trazo de costura en la industria, junto con muchas otras características para satisfacer las necesidades rápidamente cambiantes en los usuarios de hoy:
 
 - Controles más potentes de posición de muestras
 - Creación avanzada de tolerancia de costura
 - Creación automática de más de 20 diferentes tipos de costura
 - Procesamiento acelerado de los grupos de moldes
 - Compatibilidad de la base de datos Microsoft® SQL para usuarios de empresas grandes








** Instalación **
1. Abrir el archivo setup.exe.
2. Verificar si tenes instalado Microsoft .NET framework 1.1 y 2.0, sino bajarlo e instalarlo.
Si ya esta instalado verificar que el mismo este activo (en Panel de control >> Herramientas Administrativas >> Servicios), sino esta activo, ACTIVARLO.
3. Comenzar la instalacion de "AccuMark TM" e instalar todo.


 
 
INSTALANDO EMULADOR Y LICENCIA
 

1. Copiar a870.sentinel y SentinelFilter.sys (cd del progama>> LEGEND >> emulator) a la carpeta windowssystem32.
 2. Abrir Emulator Utility (cd del progama>> LEGEND >> emulator) y apretar el boton install, luego cerrar el programa. No se preocupen si ven que parece que no hizo nada, despues de apretar install cierrenlo nomas.
 3. Abran wechoid.exe y lo mantienen abierto, (cd del progama>> LEGEND) el selector tiene que ser 1e o 1c.
 4. Copien LicGen a cualquier carpeta de su disco duro. (cd del progama>> LEGEND) a cualquier carpeta.
 5. Vayan a Inicio >> Ejecutar y pongan cmd.exe, luego desde la consola vayan al directorio donde copiaron el LicGen.
 6. Aqui escriban para abrir el LicGen lo siguiente:
 LICGEN XX ???? WORKING.LIC
 Donde XX es lo del selector sin el 0x (1e o 1c tiene que ser) y ???? es el codigo que aparece al lado del selector en el wechoid.exe (tambien sin el 0x)
 7. Copiar el archivo working.lic (que se creo en la carpeta donde copiaron el LicGen) a la carpeta C:Archivos de programaCommon FilesGerber Technology, luego abran
 installLic.exe (deberia estar en el escritoria sino busquenlo con el buscador de archivos de windows) o en (cd del programa >> Utility >> Tools) y apliquen el working.lic.
 
 
Cita: Mostrar
NOTA
El Programa puede producir un mensaje de error. Hay dos tipos:
 
1) ***ERROR - Licence file not locked to this key - ***ERROR - Original Licence restored
 Si te aparecio este mensaje de error, mira abajo RESOLUCION DE PROBLEMAS.
 
2)***ERROR - Licence file not locked to this key
 Si te aparecio este mensaje de error, el programa acepto la liciencia, intenta abrir el Gerber LaunchPAD, si el color cambio a marron la liciencia ya esta funcionando.

 

RESOLUCION DE PROBLEMAS


 

 1. Verifica que estes usando el selector y codigo correcto, debes usar el que aparece en wechoid.exe sin el 0x.
 2. Verifica que tengas los drivers correctos instaldos para el sentynel system. Deberias tener: Sentilel System Driver 5.42.0 (32bit).
 4. Intante repetir el proceso. Primero debes desinstalador el emulador, despues desinstalar el programa. Elimina tambien los archivos de C:Archivos de programaCommon FilesGerber Technology.
 

 


Si la siguiente pantalla, se mantiene con este color, el software no se ha activado


Una vez aplicado la medicina, debería quedar del siguiente color, con el software activado



Cita: Mostrar
NOTA:
Si bien se puede instalar en windows 7 x64 recomiendo instalarlo en windows 7 x86 y no van a tener problemas para instalar la medicina. En Windows XP profesional funciona de forma excelente.

Este software es un tanto viejito, pero es el que mejor funciona, las versiones sucesivas le han incorporado algunas mejoras y compatibilidad con versiones del Sistema Operativo actuales, pero basicamente el diseño de patrones no ha cambiado de forma sustancial.
