Conectar al router con el pin wps en windows

  • Categoría: Software / Programas
  • Publicado hace más de 10 años
Ayer les presente dumpper, un programa para windows capaz de darnos todos los posibles pines que puede tener nuestro router por defecto. Este dato es muy interesante, ya que generalmente mucha gente cambia la contraseña wpa/wpa2 de su router, pero a nadie se le ocurre modificar el pin wps.

Hace aproximadamente un año comence a probar unos cuantos métodos para conectar a una red wifi con el pin wps, ya que reaver en ocasiones, solo me daba este dato, hoy les traigo un método mas efectivo y sencillo, para poder hacerlo.

Se trata de hacerlo, mediante software, este programa se llama Jumpstart.



JUMPSTART
Antes de comenzar a utilizar el programa les dire que he leído que recomiendan realizar un punto de restauración del sistema, a mi no me ha dado ningún problema las veces que lo he usado.
Primero vamos a descargarnos el programa, y también vamos a necesitar el programa wincap
Una vez abrimos el programa nos va a salir esta pantalla, en la que vamos a seleccionar la segunda opción, la que pone Configure a wireless network:



Manual Jumpstart
Una vez pulsamos sobre siguiente y nos pedirá que pongamos el wps pin de la red a la que queremos conectarnos.




WPS PIN JUMPSTART
Una vez que pongamos el pin vemos como trata de conectarse a una red probando entre todas las disponibles. Si queremos seleccionar nosotros la red que queramos, debemos desmarcar la casillaAutomatically select the network.




CONEXIÓN JUMPSTART
Cuando empiece con este proceso, nos va a desconectar de la red en la que estemos y nos va a dar lacontraseña para conectarnos al router.




JUMPSTART
Pulsamos sobre siguiente y ya estamos conectados a nuestra red.
Espero que les sea util tanto como a mi y no se olviden de instalar Winpcap antes del Jumpstart.
Aqui les dejo el link de Jumpstart & Winpcap, los 2 programitas en el mismo Rar:
