Algunas aplicaciones para linux que debes probar

  • Categoría: Linux
  • Publicado hace más de 10 años


 
 
Si una queja es bastante común en quienes prueban por primera vez una distro Linux, es aquella que dice que no hay aplicaciones, o que no puedes usar todo lo que usas en Windows, esos programas a los que te habías acostumbrado y que te parecen geniales. Esto es verdad hasta cierto punto, y una completa tontería hasta otro. En Linux tenemos muchísimo software de excelente calidad, solo hay que saber buscar y estar dispuesto a probar cosas nuevas, en muchos casos te van a llegar a gustar más que lo que usabas antes. En otros no, y el mundo no acabará por eso.
 
Ya sea que estés estrenándote en los territorios del pingüino, o quieras leer sobre aplicaciones nuevas e interesantes que quizás no conocías. Este post es para ti.
 TLP

 
Dependiendo del equipo que tengas en tus manos, cuando cambias de sistema operativo en rendimiento de la batería de tu portátil puede variar, si te pasa que luego de mudarte a Linux se te acaba la pila mucho más rápido que antes, entonces deberías probar la herramienta de gestión avanzada de energía TLP.
 
TLP optimiza la gestión de energía de tu equipo y no tienes que entender detalles técnicos ya que viene preconfigurado para sacarle el mayor jugo a tu batería.
 
Para instalarlo en Ubuntu 12.04 y versiones posteriores, así como derivados (Linux MInt, elementary OS, etc.) solo necesitas abrir una terminal y escribir lo siguiente:
 
sudo add-apt-repository ppa:linrunner/tlp

sudo apt-get update

sudo apt-get install tlp tlp-rdw

 
Para iniciar el servicio sin necesidad de reiniciar, puedes escribir en la terminal:
 
sudo tlp start

 
Si usas una thinkpad deberás intalar algunos paquetes adicionales:
 
sudo apt-get install tp-smapi-dkms acpi-call-tools

 Easystroke

 
Otra herramienta ideal para quienes tienen un pc portátil. Easystroke es una aplicación de reconocimiento de gestos y pulsaciones perfecta para ejecutar acciones predeterminadas elegidas por el usuario. Puedes usar el touchpad de tu portátil, o simplemente el ratón si usas una PC de escritorio.
 
La interfaz del programa es muy sencilla y fácil de usar, y puedes "grabar" cuantos gestos quieras. Para instalarlo en Ubuntu y derivadas, abre una terminal y escribe:
 
sudo apt-add-repository ppa:easystroke/ppa

sudo apt-get update

sudo apt-get install easystroke

 Caffeine
Es una de las cosas más simples que vas a conseguir en esta lista, pero que a muchos les va a resultar completamente imprescindible. Lo único que hace Caffeine es asegurarse de que tu ordenador no se vaya a dormir o entre en modo hibernación si pedirte permiso, como cuando ves un vídeo en YouTube o una película en Netflix.
 
Solo necesitas instalar un indicador que aparecerá en tu panel de ahora en adelante con forma de taza de café. Al activarlo se evitará que la pantalla se apague jamás.
 
sudo add-apt-repository ppa:caffeine-developers/ppa

sudo apt-get update

sudo apt-get install caffeine

 
En Ubuntu 14.04 y versiones posteriores, también debes instalar estas dependencias adicionales:
 
sudo apt-get install libappindicator3-1 gir1.2-appindicator3-0.1

 MenuLibre

 
Una de mis aplicaciones favoritas el editor de accesos directos y lanzadores de aplicaciones MenuLibre. Es una simple pero muy útil herramienta que te permite editar cualquier parametro de los botones para lanzar tus programas en Linux, desde el icono hasta la descripción y el comando. Muy bueno si te gusta personalizar tu dock o tu launcher.
 
Para instalarlo solo debes añadir el PPA:
 
sudo add-apt-repository ppa:menulibre-dev/devel

sudo apt-get update

sudo apt-get install menulibre

 Geary

 
Es en mi muy subjetiva opinión uno de los mejores clientes de correo para Linux. Geary es simple y minimalista pero al a vez muy poderoso, y lo vengo usando desde hace ya unos 4 o 5 años sin ver necesidad de cambiar. Además es bonito.
 
Para instalar la última versión en Ubunty derivadas, abre una terminal y escribe:
 
sudo add-apt-repository ppa:yorba/ppa

sudo apt-get update

sudo apt-get install geary

 WPS Office

 
Los chinos, expertos en crear imitaciones de todo, tienen su propio clon de Microsoft Office y se llama Kingsoft WPS Office. Es un suite de ofimática que se parece igualito al office que a todos nos gusta de Windows, pero este es gratis, aunque no de código abierto. Su ventaja es que dificilmente te va a romper la compatibilidad de documentos como suele pasar en OpenOffice o LibreOffice, que como todos sabemos se quedan cortos en ciertos aspectos, y matan a de dolor de cabeza a quienes necesitan sí o sí la mejor compatibilidad con los documentos hechos en MS Office.
 
La única pega es que WPS Office está dispomible solo en inglés por ahora, de hecho, las versiones más recientes aún son alfas, eso sí, funcionan muy bien. Para instalar solo necesitas descargar los paquetes .deb o .rpm desde su web oficial.
 Minitube

 
Minitube es un genial reproductor de vídeo en streaming que te deja mirar el contenido de todo tipo de sitios como YouTube, Vimeo, Vevo, etc. Todo en una aplicación nativa, minimalista y fácil de usar.
 
Puedes descargar el paquete deb de 32 o 64 bits para Debian, Ubuntu y derivados desde si web oficial.
 Nuvola Player

 
Una aplicación poco conocida que resulta verdaderamente genial si quieres reunir todos los servicios de música en streamign posible en un solo lugar. Nuvola Player soporta Google Play Music, Amazon Cloud Player, Deezer, 8tracks, Pandora Radio, Rdio, y Hype Machine. Si quieres instalar la versión estable puedes usar el siguiente repositorio:
 
sudo add-apt-repository ppa:nuvola-player-builders/stable

sudo apt-get update

sudo apt-get install nuvolaplayer

 
Si quieres probar Nuvola 3 en beta, puedes seguir las instrucciones oficiales.
