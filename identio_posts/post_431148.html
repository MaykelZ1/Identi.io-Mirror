Como crear un pendrive maestro con multiples O S

  • Categoría: Software / Programas
  • Publicado hace más de 9 años



Cómo crear un pendrive maestro con múltiples instaladores de Sistemas Operativos.







Si eres aficionado a probar sistemas operativos o un profesional que necesita llevar varios en el bolsillo para instalar en múltiples equipos, hoy vamos a revisar la creación de un pendrive USB maestro con múltiples instaladores.
Si te gusta el tema, es probable que tengas una colección de imágenes ISO de sistemas almacenadas en varios sitios o quemadas en discos ópticos. Una buena manera de recopilarlas y tenerlas todas al alcance en incluirlas en un solo pendrive.
Su creación es sencilla mediante la herramienta gratuita WinSetupFromUSB, que nos permite añadir instaladores hasta el límite de capacidad del pendrive. Y no solo varias versiones de sistemas Windows o Linux, de 32 o 64 bits, sino que podremos añadir otras aplicaciones como medios de arranque contra virus o discos de rescate generales que nos permitan identificar y solucionar errores de sistema.

Tutorial de uso

Descarga WinSetupFromUSB y descomprime. Ejecuta la versión de 32 o 64 bits, según la versión de Windows que estés utilizando.
La herramienta buscará el pendrive a utilizar que tengas conectado y mostrará una interfaz simple pero potente que ya muestra sus posibilidades, especialmente si seleccionas sus opciones avanzadas, por ejemplo para personalizar las carpetas que alojarán los instaladores y los nombres del menú de arranque:



El siguiente paso es el más importante porque la herramienta debe formatear la unidad y dejarla preparada para arrancar. Marca el formato automático “FBinst”. También tenemos que marcar el sistema de archivos dependiendo si tu equipo tiene BIOS o UEFI. Si no estás seguro marca “FAT32” para mayor compatibilidad.


Para la prueba vamos a añadir una ISO de Windows 10 Pro 64 bits por lo que seleccionamos esa opción y asignamos la ruta donde tenemos la imagen:


La herramienta advertirá que todos los datos del pendrive serán borrados. Si los tienes a salvo, continúa.
Introduce un nombre para la carpeta donde se creará el instalador y otro para el nombre que quieras que aparezca en el menú de arranque.




WinSetupFromUSB formateará, preparará el gestor de arranque y realizará el quemado de la imagen al pendrive. Sé paciente, tardará tiempo, especialmente si no estás utilizando una unidad USB rápida, 3.0 o superior.



Todo listo. Ya tenemos el primer instalador incluido. Pero lo que queremos es añadir otros más. Para ello lo haríamos de la misma forma que el primero con las siguientes salvedades:
Cuando quieras incluir ISOs adicionales debes asegurarte de tener desmarcado el formato automático “FBinst” porque solo se usa la primera vez para preparar el sistema de arranque. Por defecto está desactivado.
Para cada sistema que agregues, debes repetir el proceso que vimos al principio de opciones avanzadas, para poder personalizar el nombre de cada uno en el menú de arranque.
No necesitas añadir las ISO de una vez. Puedes cerrar la aplicación y agregar el resto cuando necesites.
Puedes incluir los sistemas que quieras mientras tengas espacio en el pendrive.
No puedes incluir ISOs preparadas que mezclen varias imágenes de 32 y 64 bits como los AIO. Debes incluir uno a uno por separado.
No solo sistemas. Puedes añadir la mayoría de medios de arranque contra virus y otras herramientas, por ejemplo UltimateBootCD.  En esos casos usa la opción ISO/Otras ISO compatibles con Grubdos. Puedes revisar una lista de compatibilidad en la página de soporte.
Una vez tengas tus ISOs quemadas en el pendrive solo te falta arrancar con él en el equipo que desees y verás un menú de arranque con los sistemas o aplicaciones que hayas preparado:



Si no conocías este WinSetupFromUSB, pruébalo porque es tremendamente útil, fácil de usar y gratuito, permitiendo tener en un solo pendrive (o disco duro externo USB) todos los instaladores que necesitemos o en el caso de sistemas Linux o aplicaciones LiveCD, ejecutarlos directamente desde la unidad externa.

Nota. en la descarga encontraran este mismo tutorial en Word para utilizarlo en posteriores ocasiones sin tener que volver a este post.
