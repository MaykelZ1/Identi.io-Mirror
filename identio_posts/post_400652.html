¿Que es el Smart Install Maker?:



Esmart Install Maker, es un poderoso creador de instaladores en el cual el usurio puede crear un instalador de manera 100% personal.Con este programa,se puede modificar el icono,las letras,poner una imagen de fondo,etc.Un programam demasiado util a la hora de hacer nuestros propios archivos a la hora de compartirlos.


 ¿Por qué usar Smart Install Maker?:


Porque reduce el tiempo de desarollo del instalador
No hace falta ser un genio para crear un paquete de instalación. Simplemente haz click, escribe el contenido y añade los archivos siguiendo las diferentes pestañas de la interfaz. Después pulsa en 'Compilar' para crear el instalador y listo. Cualquier parte del proceso de instalación puede ser personalizado, incluyendo la secuencia de ventanas, transferencia de archivos, reinicio y desinstalación.


Por el pequeño tamaño de los instaladores creados con él
Los paquetes de instalación creados con Smart Install Maker son los más pequeños. Su pequeño tamaño se consigue gracias a la compresión Cab, la cual puede ser personalizada para ajustarla así a tus necesidades (rápida, normal, máxima, ultra). Los archivos pequeños ofrecen a los usuarios finales una mejor experiencia en la instalación del producto, puesto que los instaladores son descargados más rápido.


Porque podrás personlizar las ventanas de diálogo de la instalación
Smart Install Maker te permite controlar la apariencia del instalador al permitirte personalizar el color de fondo al poner degradadoso imagenes, añadir una imagen de encabezado, un logo, iconos, etc. También puedes personalizar las ventanas de diálogo a ser mostradas durante la instalación, como los acuerdos de licencia, los archivos léeme, etc.


Porque posee características avanzadas para usuarios avanzados
Smart Install Maker dispone de capacidades avanzadas tales como modificaciones del registro y archivos .INI, ejecución de comandos personalizados, soporte de variables para el instalador, verificación de contraseña, desinstalación personalizada y soporte para desinstalaciones con feedback (envío de comentarios).

Porque permite comprobar los requisitos del sistema
Smart Install Maker te permite comprobar las espeficiaciones del sistema para ver si se adecuan a los requisitos necesarios para el programa a instalar. Estas comprobaciones incluyen la comprobación del sistema operativo, derechos de administrador y versión de .NET Framework. Si el sistema no cumple los requisitos, el instalador puede alertar al usuario o abortar la instalación.

Por sus instalaciones silenciosas
Smart Install Maker te permite crear instalaciones silenciosas (desatendidas) que no muestran ninguna ventana de diálogo ni ningúnmensaje. La instalación es directa y automática.

Por su soporte multilenguaje
El programa te ofrece todo lo que necesitas para dar soporte a tus usuarios de todo el mundo. Los instaladores creados con Smart Intall Maker identifican el idioma del sistema operativo y ajustan las ventanas de diálogo y los mensajes para mostrarlos en dicho idioma. Los idiomas actualmente soportados incluyen Árabe, Búlgaro, Checo, Holandés, Inglés, Estonio, Italiano, Coreano, Polaco, Ruso, Español y Ucraniano.
