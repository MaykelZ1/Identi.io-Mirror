Liberar IP, renovar IP y refrescar DNS

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 10 años
Esta guía te ayudará a liberar, renovar y refrescar la información de tus DNS y dirección IP. Estos pasos se usan normalmente para intentar arreglar problemas de conexión

Windows
Windows 8

    En el escritorio pulsa la tecla de Windows + X.
    Selecciona Símbolo de sistema (Ejecutar como administrador).
    Escribe ipconfig /flushdns y pulsa Intro.

Windows Vista/ Windows 7

Tanto en Windows Vista como en Windows 7 necesitarás ejecutar la Ventana de Símbolo del Sistema en modo administrador para poder realizar estas funciones. Para hacerlo sigue estos pasos:

    Haz clic en Inicio.
    Selecciona Programas.
    Selecciona Accesorios y busca el Símbolo de Sistema.
    Haz clic derecho sobre el Símbolo de Sistema y clic izquierdo sobre Ejecutar como Adiministrador.
    Sigue con las instrucciones de Windows XP (a partir del paso 2) más abajo.

Windows XP

    Haz clic en Inicio y a continuación en Ejecutar. Escribe cmd en el cuadro de diálogo y pulsa Aceptar. Debería abrirse la ventana de Símbolo del Sistema.
    Escribe ipconfig /release y pulsa Intro.
    Espera unos segundos hasta que aparezca un mensaje que confirme que la dirección IP ha sido liberada.
    Escribe ipconfig /renew y pulsa Intro.
    Espera unos segundos hasta que aparezca un mensaje que confirme que la dirección IP ha sido renovada.
    Escribe ipconfig /flushdns y pulsa Intro.
    Cierra la ventana de Símbolo del Sistema e intenta conectar.
