Tutorial Virtualbox (3ra parte)

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 9 años
Continuamos con el tutorial de Virtualbox amigos, si recien llegan les recomiendo empezar por el principio, donde explico de que se trata Virtualbox y alli tienen los links de descarga para Windows y Linux. En el tutorial anterior les mostre como preparar su maquina virtual para instalar un sistema operativo que queramos probar. Hagamos un rapido raconto de lo visto en tutoriales anteriores para refrescar la memoria. Vimos que virtualbox es un programa que virtualiza un sistema operativo dentro de otro. Para ello tenemos un sistema operativo "anfitrion" (el que usamos todos los dias) en el cual virtualizaremos un sistema operativo "huesped" (el sistema operativo que vamos a probar. Para ello en el tutorial uno los proveo del material necesario para nuestro proposito (virtualbox, el extension pack y el guest addition).

Las ventajas de tener un sistema operativo virtualizado son muchas. Primero que nada no debemos hacer particiones ni cosas raras. La ventaja principal, sobre todo para usuarios noveles que quieren explorar otros sistemas operativos es innegable pueden tener tantas maquinas virtuales con distintos sistemas operativos y probarlos cuando ustedes quieran, pueden usar en todo momento su sistema anfitrion y el huesped al mismo tiempo. Otra ventaja es no tener que gastar en dvd ya que con la imagen iso alcanza, igual se puede con dvd.

Este proceso es ideal para usuarios de Windows que quieren pasarse a Linux y no se animan. Amigos, aca pueden hacer lo que les plazca, pueden meter la gamba todo lo que quieran, practicar, meter mano y aprender. En ultima instancia si se mandaron un golazo con el sistema huesped, nunca afectara al sistema anfitrion. Es cuestion de borrar la maquina virtual y empezar de nuevo jaja. Aclarado esto vamos al grano.



En el tutorial anterior les mostre como preparar la maquina virtual para la posterior instalacion del sistema operativo huesped. En esta oportunidad les mostrare como instalar una distribucion de Linux (mi preferida jaja) Opensuse Leap 42.1. Ustedes pueden hacerlo con otras. Los pasos que muestro en las capturas son iguales para todos los sistemas operativos y distribuciones. Volveremos brevemente sobre nuestros pasos y haremos lo del tutorial anterior, es decir preparar nuestra maquina, luego si les muestro todo el proceso de instalacion de Opensuse.
