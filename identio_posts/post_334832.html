Windows Fundamentals for Legacy PCs




Windows Fundamentals for Legacy PCs


     Windows Fundamentals for Legacy PC (WinFLP) es un sistema operativo de Microsoft orientado a los clientes ligeros (también conocidos como thin-clients) u obsoletos. Se lanzó el 8 de julio de 2006.

     La intención de Microsoft es la de proveer servicios básicos al hardware obsoleto, mediante un sistema operativo orientado a clientes ligeros, mientras mantiene gran parte de la tecnología usada en Windows XP SP2 (como es el Firewall, las actualizaciones automáticas, las políticas de grupo y otros servicios). Los usuarios podrán hacer uso de las aplicaciones empresariales, dejando que estas sean gestionadas en un servidor usando Escritorio remoto. Aunque esta versión de Windows no ofrezca el marcado telefónico o las conexiones VPN, sí permitirá el arranque remoto y el uso de terminales sin disco duro.


Características:


     Windows Fundamentals for Legacy PC está diseñado para sistemas antiguos (como primeras versiones del Pentium). Está disponible para clientes empresariales que quieren actualizarse a Windows XP por razones de seguridad, pero no pueden comprar hardware nuevo. Las aplicaciones están orientadas a la ejecución en Escritorio Remoto. Esta es una versión con menos complementos de Windows XP, modificada para funcionar con PCs de bajos recursos como Pentium II con 64 de RAM y 16 de V-RAM.


Requisitos mínimos:


64MB RAM (128MB Recomendado)

Procesador Pentium 233 MHz (300 MHz Recomendado)

610MB HDD (1,2GB Recomendado)

800x600 de Resolución Mínima

Tarjeta de red

Instrucciones De Instalación 


1. Primer Paso: Para empezar la instalación de esta versión es algo singular, ya que en vez de iniciar en modo texto, lo que hace, es iniciar en modo gráfico, como las actuales versiones de Windows, por lo que al iniciar, comenzara con forma de asistente que nos guiara por el resto de la instalación, en este caso para continuar pulsen en "Next".



2. Segundo Paso: En este paso se nos presentara las condiciones de licencia para el usuario final, por lo que se tiene, que leer, pero al estar solamente en un idioma, simplemente dan clic, en el botón de radio que nos desbloquee en botón "Next", lo normal es que sea el que no esta seleccionado en un principio, por lo que al final presionaremos "Next" para continuar.



3. Tercer Paso: En este es donde se selecciona el modo de instalación de asistente, en donde se puede optar por una instalación interactiva, es decir, donde tu controles manualmente las configuraciones, y la desatendida, en donde usando un archivo de formato especial puedas instalar la versión de Windows empleando las configuraciones de ese archivo, volviéndola automática, en este ejemplo solo dejaremos esto como esta, sin tocar ningún control, solo basta con darle al botón "Next" para proseguir con la instalación.



4. Paso Cuatro: Este es el paso mas importante, ya que de el depende que continúes la instalación, solo basta con ingresar el numero de serie del Windows, una vez escrito el código correctamente, se nos permitirá continuar con la instalación de lo contrario, se nos marcara el error, y se tiene que ingresar correctamente.



5. Quinto Paso: Bueno, este paso es el importante ya que con este menú, pueden configurar las opciones de región e idioma de su teclado, también el uso horario y la moneda local, solo basta con seleccionar en los espacios correspondientes, las configuraciones que van con la descripción, cabe mencionar que aunque pongas el idioma en Español, el sistema operativo seguirá estando en Ingles, ya que esto solo es para escribir en Español.



6. Sexto Paso: Bueno, este es el mas importante, y es lo mas destacado de esta versión de Windows XP, ya que nos permite elegir entre los diferentes niveles de personalización de componentes, es decir, cuales componentes quieres que se instalen junto con el sistema operativo.



Esta nos ofrece cuatro niveles de personalización de componentes:

Minimum Configuration: En esta opción solo se instalara el Escritorio de Windows y la aplicación de Escritorio Remoto.

Typical Configuration: Es parecida a la opción anterior, con el Escritorio de Windows y Escritorio Remoto, salvo que incluye el Internet Explorer 6 para la instalación.

Full Configuration: Instala todas las opciones de componentes, esta difiere a las demás opciones, porque instala el componentes de DirectX 8.0c, Juegos, entre otros extras.

Custom Configuration: Esta es el nivel que mas nos interesa y es el que seleccionaremos, para nuestra instalación, porque nos da un control completo sobre los componentes que queremos que se instalen, difiriendo de las demás opciones, donde solo nos colocan uno o dos componentes.

Para continuar escogeremos esta ultima.
PD: Yo le saco la opción del messenger

7. Séptimo Paso: En este paso, seleccionaremos los cuadros de texto de los componentes que queramos, el hecho de que componente queramos fuera, no afectara en nada, salvo la ausencia de este, ya que solo son la aplicaciones que se retiraran, pero estas pueden ser "trasplantados" de un Windows a este, en caso de que se necesite.



8. Octavo Paso: En este pasaremos a acondicionar el disco duro para la instalación con la creación de una nuevo partición, para los mas novatos, solo basta con seleccionar en el cuadro blanco la parte denominada "Partition 1", y después dar clic en el botón "Create", en donde se nos aparecerá un recuadro con los datos para la partición solo damos clic en "OK" y listo.



9. Noveno Paso: En este se definirán las configuraciones de la partición en el que se instalara el sistema operativo, tales como el formato que llevara, el tamaño unitario, la etiqueta del volumen de partición y la letra que llevara este disco en la carpeta Mi PC. Podemos pasar esta opción por alto ya que la instalación automáticamente, predetermina estos valores, por lo que se le dará clic al botón "Next".
PD: Yo en la etiqueta la renombre con "Windows" para hacerlo mas parecido a sistemas mas nuevos



10. Décimo Paso: Aquí es donde ingresaremos nuestro nombre y la organización a la que pertenecemos, no necesariamente, tiene que ser valores constantes, pueden ser personalizados de acuerdo al modo que uno quiera, la condición es que se deben llenar estos dos pasos para proseguir con la instalación.



11. Onceavo Paso: En este apartado es el que muchos por lo regular se confunden, el caso es que, al ser un sistema destinado a las empresas, este pide sin excepción una contraseña, para poder ingresar al sistema y en este caso especial, para continuar con la instalación de Windows. Aquí solo puedes escoger una contraseña de acuerdo a las especificaciones que de da el asistente, que son una letra mayúscula, dos letras minúsculas, y cuatro números, eso si esto puede ser en el orden que uno quiera, una vez que se ingrese la contraseña el asistente la evaluara y se podrá continuar con la instalación, de lo contrario habrá que pensar en una contraseña que siga estos parámetros y a su vez pueda ser fácil de recordar.



12. Doceavo Paso: Los pasos que le siguen después del once, tiene que ver con la configuración en red y la creación de dirección IP, pero en el caso de un usuario domestico estas no harán falta, por lo que se le da "Next" hasta llegar a la parte final, en esta se nos indicara las opciones que hemos seleccionado en los pasos anteriores, solo para confirmar de que estemos seguros para continuar con la instalación.




Una vez completado los pasos anteriores se cerrara el asistente de instalación y se procederá de manera automática con las configuraciones que hemos ingresado, por lo que se puede tomar un descanso y esperar a que termine la instalación.



En el momento en que reinicie el programa de instalación, este dará los ajustes finales, para el primer inicio del sistema operativo, una vez acabado eso les aparacera la pantalla de inicio donde podrán ingresar con su contraseña, empleando la cuenta "Administrator".
PD: Deben escribir en user "Administrator" o si no no les funcionara, luego cambiaremos esa opción con los  archivos que les dejare para que descarguen



Bueno, para concluir esta de mas decir que este método es diferente al tradicional, todo esto lo hace mas simple y un poco menos intimidante, ya que antes, se tenia que pasar primero por el modo texto, antes de entrar al gráfico, este radica en que desde el inicio todo es por medio de un asistente que recoge las configuraciones hechas y que al final realiza todas las acciones que mandamos sin tener que realizar algún ajuste particular.
