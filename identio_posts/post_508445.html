Windows 10 no permite elegir versión [Solución]

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 7 años
¿A quien no le a pasado que cuando hacen una instalación limpia de Windows 10 se dan cuenta que el instalador no les da la opción de elegir que versión quieren y muchas veces les predetermina la versión Home de Windows 10?

La solución es añadir un archivo llamado "ei.cfg" a la ISO o USB creado con la herramienta de creación de 
medios para que permita elegir al inicio de la instalación que versión se quiere instalar.

El archivo en concreto se llama "ei.cfg" y hay que copiarlo dentro de la carpeta "Source" que 
hay en la iso o USB creados con la herramienta de creación de medios.

Este archivo se puede crear fácilmente. Simplemente hay que abrir un bloc de notas y copiar lo siguiente:

[Channel]
Retail

Una vez copiado eso hay que guardar el archivo con el nombre "ei.cfg" fijándose que quede
con extensión CFG y no así "ei.cfg.txt"

Una vez se tiene creado este archivo "ei.cfg" se copia en la carpeta source y cuando 
se inicie la instalación limpia de Windows 10 te preguntará que versión quieres instalar.

@LASZChile2  la explicación del archivo ei.cfg
{Edition ID} Especifica la versión de Windows por ejemplo "Enterprise"
{Channel Type} tiene que ser "OEM" o "Retail"
{Volume License} tiene que ser 1 si se trata de una licencia por volumen, o 0 si no se trata de una licencia por volumen
Un ejemplo del contenido como debe quedar:
[EditionID]
Enterprise
[Channel]
Retail
[VL]
0

Así es como normalmente yo lo ocupo para poder instalar W10 en las máquinas que ya lo traen instalado de fábrica con la verisón Home, ya que como el serial viene en la Bios de nada te sirve hacer el ei.cfg sin especificar la edición porque nunca va aparecer que verisón instalar, para cualquier otra máquina basta con hacerlo con Channel y VL te dejará instalar cualquiera de las versiones que tenga el disco y el archivo es totalmente válido desde Windows 7 hasta el 10.
