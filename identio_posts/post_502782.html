K-Lite Codec Pack : el paquete gratuito más popular de códecs, filtros DirectShow y herramientas asociadas a ellos. Se distingue por la ausencia de conflictos entre códecs, facilidad de instalación y uso. Le permite ver y escuchar todos los populares y una gran cantidad de formatos raros de video y audio.

Principales ventajas del K-Lite Codec Pack:
[ul][list][li]Compatibilidad interna de componentes. Todos los especialistas prueban la compatibilidad de la distribución K-Lite. Con la instalación independiente de códecs, es fácil llegar a la situación de "Cólera infierno", ya que muchos filtros no se llevan bien entre sí. Para resolver conflictos, debe buscar la causa del "mal" y usar herramientas especiales para eliminar un componente.[/li][li]Buena compatibilidad con Windows. Además de la ausencia de conflictos antes mencionada, el paquete puede eliminarse completamente del sistema sin ningún rastro. Durante la instalación, K-Lite comprueba la presencia de otros códecs en el sistema y sugiere su uso durante la instalación, entre otras opciones. También durante la fase de instalación, busca codecs defectuosos y sugiere corregirlos.[/li][li]Configuración flexible. Hay 5 variantes del paquete con diferentes paquetes disponibles para descargar, también se ofrecen 3 modos de instalación y perfiles. Para cada formato multimedia, puede seleccionar el decodificador apropiado entre las opciones disponibles, de forma predeterminada, K-Lite elige el más óptimo. Para los usuarios de Windows 7 y 8, hay una utilidad especial llamada Preferred Filter Tweaker para Windows 7 y 8 para configurar los códecs preferidos evitando el SO.[/li][li]Actualizaciones regulares El usuario no necesita monitorear la actualización de cada códec de audio o video u otros componentes: K-Lite siempre contiene las versiones actuales.[/li][/list][/ul]

Comparar las versiones del paquete:
[ul][list][li]Basic contiene todos los códecs básicos para reproducir recursos multimedia en una PC. Este es el paquete más fácil y simple, con el que podrá ver / escuchar música y películas utilizando cualquier reproductor conveniente para usted. Aquí hay un DirectShow de audio / vídeo filtros de decodificación (LAV audio y vídeo), y los filtros DirectShow son la fuente de los subtítulos (LAV Splitter y VSFilter), la utilidad Codec Tweak Tool, así como la expansión de Icaros ThumbnailProvider y Icaros PropertyHandler.[/li][li]Standard también incluye reproductor MediaClema Classic HomeCinema, descodificador de DVD MPEG-2 (abre más posibilidades para la reproducción de DVD), MediaInfo Lite, que muestra información detallada sobre el archivo multimedia.[/li][li]Completo en comparación con la versión estándar, incluye madVR, un procesador de alta calidad, una utilidad para diagnosticar GraphStudioNext y filtros DirectShow adicionales.[/li][li]Mega contiene codecs ACM y VFW, filtros DirectShow, utilidades adicionales. Cuando instale, se le ofrecerá un reproductor de video para elegir: Media Player Classic HomeCinema o Media Player Classic Regular.[/li][/list][/ul]
