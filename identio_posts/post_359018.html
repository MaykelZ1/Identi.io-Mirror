DOLPHIN 4.0-X86


     Dolphin es un emulador de Gamecube y Wii. La mayoría de los juegos funcionan perfectamente o con errores menores. Los juegos son jugables en calidad de alta definición, con 1080p y más. Esta es una característica notable de las consolas actuales para GameCube y Wii no son capaces de hacer. Como proyecto de código abierto cualquiera puede cometer mejoras. El código se encuentra alojado en code.google.com. Dolphin es desarrollado activamente y se añaden casi cada día nuevas características y errores corregidos también. Dolphin es un proyecto multiplataforma y se admiten muchos sistemas operativos. Dolphin se ejecuta en Windows 7 (x86 y x64), Mac OSX y Linux.



REQUISITOS DEL SISTEMA
Windows XP o superior, o Linux o MacOSX Intel.

GPU con Pixel Shader 2.0 o superior. Algunos chips gráficos integrados funcionan pero depende del modelo (y sólo con DirectX 9).

Necesitará las bibliotecas se indican a continuación para ejecutar Dolphin:

MS Visual C++ Runtime 2008 SP1 (fixes the "side by side" / "application config" & model errors)
Microsoft Visual C++ 2008 SP1 Redistributable Package (x86)
Microsoft Visual C++ 2008 SP1 Redistributable Package (x64)
DirectX March 2009 Runtime (fixes the plugin loading errors)
DirectX End-User Runtime Web Installer

Video Card Requirements
You must have a video card that supports OpenGL 2.x fully or Direct3D 9.0c. ASEGURATE DE TENER LOS DRIVERS DE TU PC ACTUALIZADOS.
