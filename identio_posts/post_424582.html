8 Trucos para reparar Outlook

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 9 años




Microsoft Outlook es uno de los programas que más se utiliza en las oficinas y empresas para gestionar el correo, el calendario, los contactos y las tareas. Se ha convertido en el programa estándar para manejar de forma centralizada diferentes cuentas de correos empresariales y de casi todos los proveedores de correo como Gmail, Hotmail, Yahoo, AOL, entre otros. Pero no todo es color de rosas, en ocasiones se tienen algunos problemas con Outlook.


SOLUCIONAR PROBLEMAS COMUNES EN OUTLOOK

Al igual que con otros programas que son ampliamente utilizados día tras día, las personas tienden a tener un sinnúmero de problemas con Outlook que pueden convertirse en un verdadero dolor de cabeza hasta para los usuarios avanzados. Es por ello que hoy en CaminoGeek te mostraré algunos trucos o consejos útiles y muy sencillos para reparar el Outlook o solucionar los diferentes problemas que puede presentar este popular cliente de correo electrónico.


1. ANALIZAR Y REPARAR ARCHIVOS DE DATOS


Cuando las cuentas están configuradas como POP o IMAP en Outlook, los elementos de correo se almacenan en un archivo de tipo PST. A medida que la cantidad de correos crece, este archivo se vuelve más grande o pesado y puede comenzar a corromperse. Si el archivo se daña el Outlook dejará de procesar el flujo de correos nuevos, se frisará e inclusive podría no abrir en absoluto. Con el tiempo, y mientras más uso se haga del correo, los archivos PST desarrollan errores casi indefectiblemente.

Si Outlook empieza a dar problemas relacionados con algún archivo PST, lo mejor que se puede hacer es analizar el archivo en cuestión para detectar errores y repararlo. Outllok integra una herramienta nativa llamada Scan PST para ayudarnos con esto. Scanpst.exerealiza un escaneo completo del archivo PST, detectando los problemas o errores que tiene y finalmente reparándolo. Antes de iniciar el proceso te pregunta si quieres hacer un respaldo o backup del archivo, lo cual es muy recomendable para evitar sorpresas ante cualquier eventualidad. El proceso requiere de paciencia, ya que dependiendo del tamaño del PST, se puede tomar varias horas. Pero esto valdrá la pena. Para utilizar la herramienta debes seguir los siguientes pasos:


1º Ejecutar Scan PST. El ejecutable scanpst.exe está en la carpeta de instalación de Outlook, cuya ubicación puede variar dependiendo de la versión del sistema. Una forma sencilla de encontrarla es entrando al disco C y escribir scanpst.exe en el recuadro de búsquedas.





2º Seleccionar archivo PST. Una vez la herramienta esté abierta, clic al botón Examinar para buscar el archivo que queremos reparar. Cuando se haya hecho esto solo se debe hacer clic en el botón Iniciar y el proceso comenzará.




2. ARCHIVAR ELEMENTOS

Como he mencionado más arriba, si el PST crece demasiado empezará a causar muchos problemas con Outlook. Se corre el riesgo de que el archivo se corrompa y no se tenga acceso al correo. Por lo tanto, es mejor evitar que este archivo crezca más de lo recomendado y ocasione problemas mayores. Esto lo podemos lograr archivando los elementos de correo antiguos para que el tamaño del PST se mantenga balanceado.

Para archivar correos antiguos debes dar clic en Archivo (File), después dar clic enHerramientas de limpieza (Cleanup Tools) y seleccionar la opción Archivar (Archive).




Se abrirá una pequeña ventana que pedirá seleccionar las carpetas de donde se desea archivar elementos de correo. También se debe seleccionar desde qué fecha hacia atrás serán archivados los elementos, es decir, podemos seleccionar que se archiven todos los correos que tengan más de un mes de antigüedad, por ejemplo. Además se puede elegir en cuál archivo de datos serán almacenados los elementos que sean archivados.



En caso de que el archivo de datos haya alcanzado un tamaño excesivo y estés teniendo problemas con tu correo, sería aconsejable que realices el archivo de correo por años. De esta forma te asegurarás de archivar una gran cantidad de datos, disminuyendo significativamente el tamaño del PST. Y por supuesto, esto aliviará los problemas relacionados con un archivo muy grande. Para mejores resultados, después de hacer ésto, trata de realizar un escaneo y reparación del archivo con la herramienta Scan PST.



3. RENOMBRAR EL ARCHIVO OST

Cuando las cuentas no están configuradas como POP o IMAP, es porque están como cuentas de Exchange, lo que sirve para que los usuarios tengan todas las características del servidor de correos disponibles. En estos casos los correos se almacenan en un archivo llamado OST. Si este se daña el Outlook comenzará a presentar problemas y el correo no funcionará como es debido. A veces, simplemente renombrando el OST se resolverían muchos problemas. Para hacerlo, simplemente cierra el Outlook y abre la carpeta en que están sus archivos de datos; asegúrate de que puedes ver las extensiones y cambia la extensión .ost del archivo en cuestión por otra como .abc o .old. Cuando vuelvas a abrir el Outlook, al no encontrar el archivo .ost actual, éste creará un archivo nuevo y descargará del servidor todos tus elementos de correo. Ahora tu Outlook debería funcionar como nuevo.


4. ELIMINAR PERFIL Y CREAR UNO NUEVO

Si todo lo que has intentado de nada ha servido, entonces puedes recurrir a una opción un poco más drástica: eliminar el perfil de Outlook y crearlo de nuevo. Antes debes tomar en cuenta que si el correo está configurado como POP, tendrás que exportar los elementos de correo (bandeja de entrada, calendario, ect.) en un archivo de datos, para luego reimportar dichos elementos al nuevo perfil. Por otro lado, si la cuenta está configurada como IMAP o cuenta de Exchange, no necesitas guardar ningún archivo ya que los correos se cargarán automáticamente del servidor.

Para eliminar el perfil debes entrar al panel de control y abrir la herramienta Correo.


Después clic al botón Mostrar perfiles. En la ventana que aparece seleccionas el perfil y le das clic al botón Quitar. Para crear un nuevo perfil, clic al botón Agregar y sigues el asistente.


5. DESHABILITAR COMPLEMENTOS

Los complementos de Outlook agregan funcionalidades o características adicionales al programa. Un ejemplo de esto lo es el complemento del antivirus para el correo, el cual analiza el correo entrante y saliente para protegerte contra el malware. El problema con esto es que mientras más complementos estén conectado al programa, más lento se pondrá éste. Para comprobarlo trata de iniciar Outlook en modo seguro y verifica la diferencia en rapidez y agilidad del programa. Si llegas a la conclusión de que éste es el problema, entonces ve a la ficha Archivo ? Opciones ? Complementos y desactiva los que te resulten sospechosos y vuelve a iniciar el Outlook en forma normal. Debes repetir el procedimiento hasta encontrar el responsable del problema.


6. DESHABILITAR ANTIVIRUS TEMPORALMENTE

Muchos antivirus incluyen una herramienta dedicada para analizar el correo entrante y saliente a través de Outlook. A veces este proceso de escaneo puede hacer que Outlook funcione más lento y hasta que falle. Para verificar si tu problema se debe a esta razón, deshabilita el componente del antivirus para el correo de forma temporal y abre el Outlook. Si el problema se resuelve entonces se trataba de ésto, tal vez necesites actualizar el programa antiviruspara solucionar el inconveniente. Solo ten en cuenta que si dejas el componente inhabilitado Outlook estaría siendo vulnerable.

7.  INICIAR OUTLOOK EN MODO SEGURO

Abrir Outlook en modo seguro no siempre será suficiente para resolver el problema, pero nos ayudará a saber cuál es el origen del mismo. Al iniciar en modo seguro éste se ejecutará con las características mínimas, por lo que varias de sus funciones no estarán disponibles. El modo seguro sólo se ejecutará con los elementos  básicos que el programa necesita para funcionar. Para iniciar Outlook en modo seguro debes utilizar la herramienta Ejecutar, escribes el comando outlook.exe /safe y presionas la tecla Enter o clic a Aceptar.


8. RESETEAR PANEL DE NAVEGACIÓN
Muchos usuarios personalizan el Outlook agregando o quitando elementos del panel de navegación. Generalmente estos cambios no afectan el funcionamiento del Outlook, pero hay casos en que pueden provocar que el programa deje de funcionar. Para revertir estos cambios Outlook tiene una herramienta llamada resetnavpane, que resetea toda la configuración de dicho panel a su forma original. Para resetear todos los cambios tendrás que abrir la herramienta Ejecutar, escribir el comando Outlook.exe /resetnavpane y presionar Enter o clic en botón Aceptar.


Conclusión

Espero que con esta guía puedas resolver el inconveniente cuando tu correo Outlook no abre. La mayoría de este tutorial ha sido realizada utilizando la versión 2010 del programa, pero también puede utilizarse para otras versiones como Microsoft Outlook 2013 ó 2007. Las opciones varían, pero no tan drásticamente.
