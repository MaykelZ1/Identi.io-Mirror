TeraByte Image for Windows 2 95

  • Categoría: Software / Programas
  • Publicado hace más de 10 años


Image for Windows es una copia de seguridad de imagen de unidad económica y confiable y restaurar solución que proporciona una manera fácil y conveniente de copia de seguridad completa de todos los datos de su disco duro, los programas y los sistemas operativos. Las copias de seguridad se pueden guardar directamente a USB externo y unidades FireWire ?, a las unidades internas o de la red, e incluso directamente en un CD, DVD o BD. La imagen para el paquete de Windows incluye un asistente fácil de usar MakeDisk para crear un disco de arranque de recuperación. Para restaurar los datos, los programas y los sistemas operativos de nuevo a la forma en que estaban cuando se creó la copia de seguridad, simplemente arrancar el disco de recuperación y restauración de la partición (s) o unidad (s) que necesita para recuperarse. Es así de fácil.

Image for Windows utiliza la tecnología proporcionada por Volume Shadow Copy Service (VSS) o PHYLock ?, un componente complemento incluido en la imagen para el paquete de Windows. VSS y PHYLock ? le permite continuar utilizando su computadora mientras que la copia de seguridad está bloqueado a un punto en el tiempo. Esto elimina las inconsistencias experimentadas típicamente mientras que sostiene una partición que está en uso.

Image for Windows se ejecuta bajo el sistema operativo Windows, sin embargo, puede copia de seguridad o restaurar cualquier partición, incluyendo las de DOS o Linux, y particiones incluso cifrado por software de terceros.

Los archivos de imagen de duro de un terabyte producidos por Image for Windows son compatibles con todos los terabyte ilimitadas productos de imagen de disco del mismo número de versión principal (es decir, todos los productos 1.x son compatibles con otros productos 1.x). Esto le da la flexibilidad para restaurar imágenes de disco usando su producto de elección.

Imagen de Características de Windows:
Fiabilidad total - Ofrece la opción de verificar todos y cada byte que realiza copias de seguridad.
Conveniencia - Se ejecuta directamente desde Windows y cuando se utiliza con el PHYLock libre ? add-on, se puede crear una verdadera imagen de la partición de Windows sí mismo, sin necesidad de reiniciar.
Seguridad - Las copias de seguridad se pueden fijar con protección de contraseña simple, o el cifrado completo de 256 bits AES.
Data Backup Cambiado - opción de copia de seguridad diferencial que permite hacer copias de seguridad sólo los datos que han cambiado desde que se creó la última copia de seguridad completa.
Simple, interfaz Driven-Wizard - Fácil de usar la primera vez, y cada vez.
Compresión máxima - Por defecto, la compresión se aplica a las imágenes de disco que cree.
Velocidad máxima - Si la velocidad de copia de seguridad es de suma importancia, se puede omitir la compresión del todo.
Soporte unidad óptica - Usted puede guardar directamente en un CD, DVD, DVD-DL, BD.
Los discos de rescate de inicio - Cada vez que se guarda directamente en un CD, DVD, DVD-DL, BD, se convertirá automáticamente en un disco de restauración de arranque.
Compatibilidad con discos grandes - las unidades de apoyo compatibles GPT mayores de 2TiB.
Restaurar archivos o carpetas individuales - Uso de la TBIView libre y TBIMount complementos, se puede ver el contenido de los archivos de imagen de unidad, y opcionalmente restaurar archivos individuales o carpetas dentro de ellas.
Compatibilidad - reconoce automáticamente y realiza copias de seguridad área utilizada de FAT16, FAT32, NTFS, Ext2, Ext3, Ext4, ReiserFS y XFS particiones. Particiones con otros tipos de sistemas de archivos se pueden copiar en su totalidad. Soporta todas las particiones estándar en un MBR, EMBR o GPT
Windows Servers - Soporta tanto Workstation y Server versiones de Windows.
Rentabilidad - Incluye la imagen para DOS, Imagen para Linux, y TBOSDT sin cargo adicional, por lo que es una solución de imagen de disco de forma única asequible y completa.
Flexibilidad - Restaurar particiones, independientemente del sistema de archivos en la partición siendo restaurados. Esto le da el poder de restaurar particiones NTFS desde Windows 98 o incluso DOS (como sólo un par de ejemplos).
Amplia compatibilidad con dispositivos - Compatible con USB 2 y IEEE 1394 unidades externas. Mejor aún, la imagen para DOS admite estas unidades directamente, sin necesidad de instalación del controlador complicado. (Image for Windows simplemente utiliza los controladores ya están disponibles en Windows.)
Potencia - Uso de la imagen para DOS o Imagen para Linux (cada uno de los cuales se incluye con la compra de Image for Windows) versiones que ofrece la posibilidad de recuperar sus datos, incluso si Windows no se iniciará.
Red de Apoyo - Guarde copias de seguridad a cualquier unidad de acceso al propio Windows.
Soporte RAID - Funciona sin problemas con hardware de a bordo y basado en firmware RAID.
Command Line Support -, soporte de línea de comandos completa Potente está integrada, lo que le permite automatizar la copia de seguridad o procedimientos de restauración.
Programación - Aprovecha el planificador incorporado en Windows a través de un simple click de un botón Programar copia de seguridad.
Imagen de archivo de validación - Le permite comprobar a fondo la integridad de los archivos de imagen de disco, ya sea como parte del proceso de copia de seguridad, o en cualquier momento posterior. Productos de imagen TeraByte son reconocidos por su proceso de validación completa y exacta.
Restaurada expansión de la partición - Al restaurar una imagen, puede ampliar la partición restaurada a ocupar cualquier espacio libre que quedaría terminado.
Tamaños de archivos seleccionables - Si lo desea, puede especificar un tamaño máximo para los archivos de imagen de disco que se crean. Esto le permite crear archivos de imágenes que caben fácilmente en el medio extraíble de su elección.
Restauración Independiente de Hardware - Restauración de Windows a un hardware completamente diferente y luego utilizar la herramienta TeraByte OSD Suite para instalar los controladores necesarios para apoyar el nuevo hardware.

OS: Windows NT4, 2000, XP, Vista, 7, 8 y Windows Server y SBS versiones (NT4, 2000, 2003, 2008, 2008R2, 2011, etc ..)
Idioma: multilingüe
