Conoce los fundamentos de CSS para diseño web



34 videos HD | MP4 | 2:38 Hrs. | 383 MB | Archivos Base | Español | V2B | Ene. 27, 2015
Formadores: Marlon Ceballos, James Williamson

CSS es el lenguaje de presentación para la web. Mientras que con HTML se define la estructura del contenido de un sitio web, con CSS se define y aplica la capa de presentación que corresponde a características como colores, fuentes tipográficas, tamaños, etc. En este curso aprenderás los conceptos fundamentales de CSS combinando la explicación de los conceptos con demostraciones totalmente prácticas.





En el capítulo inicial de este curso aprenderemos conceptos fundamentales que son necesarios para entender cómo funciona el lenguaje y cómo lo implementaremos en los archivos HTML. Posteriormente conoceremos las especificaciones de este lenguaje que son necesarias como punto o documentos de referencia, seguiremos conociendo y usando en la práctica los aspectos más comunes del lenguaje.

Finalizaremos conociendo un listado de recursos que te serán de gran ayuda en determinados escenarios del diseño web.

El objetivo del curso es que aprendas a usar adecuadamente el lenguaje CSS desde la correcta escritura de la sintaxis hasta el uso lógico de las reglas y aplicación de las mismas.
