Locale Emulator - Descarga, instalacion y uso

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 10 años

¡Bienvenidos!

Este es un rápido tutorial de como descargar, instalar y configurar "Locale Emulator" el cual es una herramienta para emular la configuración regional de una PC japonesa y así poder ejecutar aplicaciones y juegos en el idioma Nippon (ademas de muchos otros).
Para este tutorial usare la versión 1.4.2.pre2.3

AppLocale es básicamente el programa "ideal" para correr aplicaciones en otros idiomas, sin embargo debido a que ha cesado su desarrollo tiene problemas de compatibilidad con algunos juegos o aplicaciones "recientes" haciendo que generen errores o no se muestren correctamente.

El ejemplo que pondré en este caso es la VN (visual novel) "Saya no Uta" la cual si es ejecutada con AppLocale no mostrara algunos sprites (imágenes de personajes), la única solución antes era aplicar manualmente la configuración regional a la PC y reiniciarla para que el juego funcionara bien, sin embargo hacer esto cada que se quiere jugar es molesto ademas de que si se deja la configuración regional y se sigue usando la computadora para otras cosas se notaran algunos errores con la tipografía de ciertas paginas web. Para evitar esto usaremos este programa.

Para una explicación mas detallada de lo que es AppLocale, su uso e instalación ademas de una explicacion mas a fondo de porque se tienen que usar estos programas para ejecutar las aplicaciones y juegos en japones les recomiendo visitar otro de mis tutoriales:

Instalar y ejecutar novelas visuales en japones

Descarga:
Locale.Emulator.1.4.1.2.pre2.3.zip

La ultima versión la pueden encontrar aquí:
https://github.com/xupefei/Locale-Emulator/releases

Una vez que hayan descargado el zip, lo descomprimiremos en la raíz de nuestro disco duro, debería de quedar algo así:



Ejecutaremos el archivo llamado LEInstaller.exe y daremos clic en Install



La instalación durara un par de segundos (literal) y nos avisara cuando termine, ahora solo resta abrir el archivo LEGUI.exe el cual automáticamente creara los accesos para ejecutar los archivos exe en japones. Sin mover nada solo cerramos la pequeña ventana.

Y todo debería estar listo, ahora vamos a la carpeta donde tenemos instalado el juego, le damos clic derecho al ejecutable y seleccionamos como se muestra en la imagen:


(si no aparecen las opciones de "Run in Japanese" asegúrense de haber ejecutado LEGUI.exe)

Creé este pequeño tutorial debido a que estoy subiendo Saya no Uta y para su correcto funcionamiento es necesario utilizar este programa

Suerte!
