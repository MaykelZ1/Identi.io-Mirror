DATOS TÉCNICOS

EASEUS Data Recovery Wizard Professional v61.0+Serial
Inglés, Alemán | Medicina Incl. |9.3 MB | Windows 8-  8.1/7/Vista/XP/2000 y Windows Server 2012/2008/2003 | NFO

DESCRIPCIÓN

Recupera todo tipo de archivos de particiones perdidas o formateadas
EASEUS Data Recovery Wizard restaura ficheros eliminados incluso de discos formateados. También puede recuperar particiones que han sido borradas manteniendo la estructura de archivos.
La pantalla inicial del programa nos da tres opciones: Recuperar archivos eliminados, restaurar todos los ficheros de una partición o restaurar la partición misma.

En el primer caso podemos incluso buscar el archivo que nos interesa recuperar de entre todos los borrados.
Se trata de una herramienta muy interesante para enmendarle la plana al típico virus que se dedica a borrar documentos o corregir un error muy común como es borrar una carpeta sin haber comprobado qué había dentro.
