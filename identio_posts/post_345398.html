Cómo defiendo mi PC Windows del nefasto Junkware

  • Categoría: Noticias
  • Publicado hace más de 10 años









Junkware es el software adicional, basura y/o malicioso que camuflado en algunas aplicaciones y herramientas Windows se instalan con ellas a poco que no andemos con sumo cuidado, incluyendo barras de herramientas adicionales, modificando los motores de búsqueda, metiendo complementos activos, cambiando las páginas de inicio de navegadores o lo que es peor, instalando software espía o redirigiendo a páginas maliciosas donde es habitual infectarse con todo tipo de malware.


 
Además de malware, este Junkware sobrecarga nuestro hardware ocupando recursos innecesariamente y aunque no todo el software instalable es necesariamente basura, como mínimo es un auténtico coñazo de algo que ni quieres ni necesitas. Si lo sumamos a la gran cantidad de software basura (crapware, bloatware?) que viene instalado en los flamantes equipos nuevos con Windows y que penaliza su rendimiento antes de sacarlo de la caja, entenderemos una situación preocupante en los PC Windows y casos como el ERE de Softonic, más que por los cambios en el algoritmo mundial por convertirse en líder de distribución de software basura.
 
No creas que son únicamente desarrolladores menores los que persiguen al usuario con Junkware porque ahí tenemos a gigantes como Oracle metiendo con calzador en Java basura como Ask, o Adobe, software innecesario como el McAfee Scan Plus en la instalación de Flash Player.


 
Es entendible la necesidad de monetizar el desarrollo del software pero es preferible cobrar una cantidad de dinero (tipo app móvil por 0,99 o 1,99 dólares) o añadir publicidad no invasiva ni espía, antes que causar al usuario -como mínimo- una gran molestia con este nefasto Junkware que nos inunda y al que hoy intentamos poner coto con algunas propuestas para frenarlo:
 Precaución y sentido común
La mayoría de este tipo de software como el mencionado de Oracle o Adobe puede impedirse desmarcando la casilla de instalación, por lo que es imprescindible tomarse su tiempo para leer a fondo todas las pantallas por las que va pasando la instalación. No siempre está tan accesible como debería y se emplea ?letra pequeña?.


 

 


Para intentar colarla, te puedes encontrar que hay que desmarcar una casilla para no instalar el Junkware y en la siguiente pantalla precisamente hay que hacer todo lo contrario por lo que hay tener que cuidado. Otra de las precauciones es realizar las descargas de la aplicación desde los sitios oficiales, nunca de los sitios publicitarios que verás en los primeros puestos en los motores de búsquedas.
 Unchecky
No importa lo cuidadosos que seamos, en alguna ocasión tendremos que lidiar con junkware. En ocasiones los usuarios avanzados van demasiado rápidos y el usuario de a pie no siempre comprende lo que está instalado. Para estos últimos, aplicaciones como Unchecky hacen el trabajo por su cuenta, en segundo plano, avisando y desactivando el software basura que pretende instalarse.
 AdwCleaner ? Junkware Removal Tool
Para eliminar este tipo de software basura no basta una desinstalación de una aplicación como haríamos con una estándar de Windows. Aplicaciones como AdwCleaner o Junkware Removal Tool ayudan en la tarea aunque no siempre funcionan completamente ni con todo el junkware ya que a menudo hay que revisar a mano la configuración del navegador, motores de búsqueda, accesos directos, ajustes, tareas camufladas y hasta el registro del sistema.


 





 Software antivirus
De igual manera que las anteriores, las soluciones de seguridad habituales no detectan ni eliminan todas estas aplicaciones pero sí muchas de ellas especialmente con las funciones de ?detección de software no deseado?. La conveniencia de contar con un buen antivirus en Windows está fuera de debate para un usuario estándar.
 Listas blancas de aplicaciones
Seguramente es la manera definitiva para asegurarse que este tipo de software no infecta nuestros PCs y consiste en aprobar expresamente la lista de aplicaciones que pueden ejecutarse en una cuenta determinada. Esta característica está disponible en las versiones Enterprise de Windows con AppLocker pero puede conseguirse algo similar en todas las versiones de Windows 7 y Windows 8 con el Family Safety de Microsoft, una función que va más allá del control parental y permite restringir accesos en cuentas específicas.
