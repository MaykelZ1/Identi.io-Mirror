2- En caso que se les presente el error 0xc0150002 en Duke Nukem Forever al intentar ejecutarlo:


Bajar Microsoft Visual C++ 2008 SP 1 Redistributable Package y / o Microsoft NET Framework 4 dependiendo de cuál te falte o ambos si es necesario (Las soluciones propuestas fueron probadas en windows 7 32 bits, no en windows 8, 8.1 o 10) ADVERTENCIA! -> si tenés Windows XP hay que instalar Visual C++ 2005 en lugar del 2008.

Acá el link con los paquetes mencionados todos juntos (Visual C++ 2008 SP 1 (pesa 3.9 mb) Net Framework (pesa 48.1 mb) Visual C++ 2005 (pesa 2.0 mb)) en una misma carpeta llamada -> Contenedor de posibles soluciones
