soft camaras de seguridad (grabacion y monitoreo)

  • Categoría: Software / Programas
  • Publicado hace más de 10 años
como anda la gente de identi. en esta oportunidad comparto un soft que sirve tanto para grabación como para monitoreo de cámaras de seguridad las misma nos permite grabar, monitorear tanto cámaras IP como cámaras web de tal forma podemos usar como cámara de seguridad nuestra cámara web. es un soft multimarca así que contamos con muchas opciones para poder monitorear,
el programa en cuestion es 
asi mismo podemos usar el equipo de grabacion (nuestra pc) como un servidor y poder ver desde el exterior nuestras camaras con proteccion (usuario y contraseña)
 

H264 WebCam

codificar video :
H264, H263, H261, XVID, MPEG4, MPEG2, MPEG1, FLV, WMV3, MSMPEG4 etc.
(codificacion de tasas de bits puede ser puestas arbitrariamente por el usuario)
codificar audio :
AAC, MP3, WMA etc.
(codificar tasa de bits a 32k,24k,16k, or8k,definidas por el usuario.)
grabar formato archivo :
MP4, MOV, AVI, FLV, SWF, ASF, WMV etc. soporta grabacion manual,programacion de grabacion y grabacion de movimiento.
servidor de red funcion:
-> difusion de audio y video a internet,ayudas para crear pagina web;
-> Streaming video y audio a h264,aac,asf multimedia streaming,y puede ser visto en tiempop real, realplayer, 
     windows media player flash player,IE o navegador firefox,sobre pc remoto,macintosh,telefonos moviles ect.;
-> en modo usuario h264,no necesita conocer la direccion IP. soporta servicio de dominio dyndns.org, changeip.org, dynddns.us servicio de dominio dinámico.
