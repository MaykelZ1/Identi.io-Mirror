Bloquear Facebook Mikrotik RouterOS

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 9 años










 

Queridos amigos bienvenidos, en esta ocacion les traigo un post de como bloquear Facebook con Mikrotik. No se si lo abran utilizado antes pero es una asombrosa herramienta para aquellos que administran redes, etc etc.




01 - Ingresamos a nuetro microtik por la herramienta Winbox.








02 - Ingresamos a IP - Firewall.




 

03 - Una ves ingresado, con el signo + generamos una nueva regla.
 




04 - En la solapa general, en chain seleccionamos la opcion forward.



 

05 - En la solapa Advanced, Src Address List ( Deberiamos tener hecha una Address List con el rango de IP a restringir el acceso a Facebook). En la opcion Content ( la palabra a bloquear el acceso, en nuestro caso seria Facebook).

 




06 - En la solapa Action, seleccionar la accion Drop ( que significa cortar).
 




07 - Una ves realizado esto tendremos el acceso bloqueado a facebook. Ahora les comento como realizar una Address Lists para seleccionar un segmento a bloquear. En la misma pestaña de Firewall nos dirigimos a la solapa de Address List, seleccionamos el Signo + para realizar una nueva entrada e ingresamos los datos como se ven a continuacion.
