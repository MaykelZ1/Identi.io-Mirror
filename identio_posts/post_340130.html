Mac OS X Tema para Ubuntu 14.04 - Paquete Macbuntu

  • Categoría: Linux
  • Publicado hace más de 11 años



Instalar Mac OS X Theme for Ubuntu 14.04 le ayudará a hacer que Ubuntu 14.04 se parece a Mac OS X. Esta Macbuntu Transformation Pack Incluye GTK Temas, temas para Ubuntu Unity, temas de iconos para Ubuntu 14.04, Logos, Boot Splash sceen, pantalla de conexión webkit LightDM tema (por WattOS). Este pack es desarrollado por noobslab con la ayuda de bluedxca93 de gnome-look.org.

Esta Transformation Pack Mac OS X se compone de cuatro temas GTK Mac, tres temas de iconos de Mac, pantalla de arranque Mac y Mac como LightDM tema. Los comandos y las instrucciones para cambiar un Ubuntu 14.04 se parece a Mac OS X se dan a continuación.




COMENCEMOS CON EL TOTORIAL PARA INSTALAR EL TEMA DE MAC!

Para cambiar el tema GTK, el tema iconos, cursores, Fuentes necesitamos Tweak Tool Unidad. Instale Tweak Tool Unity en Ubuntu 14.04 con los comandos que figuran a continuación.
sudo apt-get install unity-tweak-tool



También puede utilizar la herramienta Tweak Ubuntu para cambiar los temas.
sudo add-apt-repository ppa:tualatrix/ppa
sudo apt-get update
sudo apt-get install ubuntu-tweak
Instalar Mac OS X tema para Ubuntu 14.04

Instalar el tema GTK, Icon tema, Ratón tema puntero y fuentes de Mac OS X y aplicarlos a ubuntu 14.04. Ejecutar los comandos en la terminal (pulse Ctrl + Alt + T).
sudo add-apt-repository ppa:noobslab/themes
sudo apt-get update
sudo apt-get install mac-ithemes-v3
sudo apt-get install mac-icons-v3

Ahora Tweak Tool Unidad Abierta y la sección seleccione Temas. En la sección de temas GTK seleccione Mbuntu tema. En la ficha Iconos seleccione Mbuntu-osx y en la pestaña cursores seleccione Mac-cursores.









La apariencia del escritorio de la Unidad es ahora casi igual que Mac. Usted tiene iconos de Mac, y punteros de ratón de Mac.

Lo siguiente es añadir un logotipo Dock de Mac, Mac y Nombre en la barra superior.
Instalar Docky, Mac OS X como Muelle en Ubuntu 14.04

Docky es un mac os x peso muy ligero como muelle para ubuntu. Tiene mac como efectos de activación para los iconos. Para instalar docky en ubuntu 14.04 ejecutar las órdenes dadas a continuación.
sudo add-apt-repository ppa:docky-core/ppa

sudo apt-get update

sudo apt-get install docky



Ocultar Ubuntu Unidad Lanzador

Puede ocultar la unidad lanzador de Apariencia -> Comportamiento y se apagará. Ahora su escritorio tiene una bonita mac os x fondo de pantalla con mac os x muelle.



Substituya el texto "Ubuntu Escritorio 'con' Mac OS X 'en el panel superior




Introduzca los siguientes comandos para cambiar el texto 'Escritorio de Ubuntu "en el panel:

 
cd && wget -O Mac.po http://drive.noobslab.com/data/Mac-14.04/change-name-on-panel/mac.po

cd /usr/share/locale/en/LC_MESSAGES; sudo msgfmt -o unity.mo ~/Mac.po;rm ~/Mac.po;cd
Si desea volver al texto 'Ubuntu Escritorio', introduzca los siguientes comandos en la terminal:

cd && wget -O Ubuntu.po http://drive.noobslab.com/data/Mac-14.04/change-name-on-panel/ubuntu.po

cd /usr/share/locale/en/LC_MESSAGES; sudo msgfmt -o unity.mo ~/Ubuntu.po;rm ~/Ubuntu.po;cd


Introduzca los siguientes comandos para reemplazar el logotipo de Apple en el lanzador

wget -O launcher_bfb.png http://drive.noobslab.com/data/Mac-14.04/launcher-logo/apple/launcher_bfb.png

sudo mv launcher_bfb.png /usr/share/unity/icons/


Si desea logo Ubuntu espalda en lugar del logotipo de la manzana introducir los siguientes comandos en la terminal:


wget -O launcher_bfb.png http://drive.noobslab.com/data/Mac-14.04/launcher-logo/ubuntu/launcher_bfb.png
sudo mv launcher_bfb.png /usr/share/unity/icons/



8: Reemplazar Overlay barras de desplazamiento con normal



Ingrese siguiente comando en la terminal para desactivar la barra de desplazamiento de superposición:

gsettings set com.canonical.desktop.interface scrollbar-mode normal

Si quieres volver a la antigua barra de información, introduzca siguiente comando:

gsettings reset com.canonical.desktop.interface scrollbar-mode

9: Saque de puntos blanco y Ubuntu Logo de bloqueo de pantalla:
Ubuntu 14.04 tiene nueva pantalla de bloqueo, que el tema ya se indica para LightDM pero este paso es necesario si usted está usando la pantalla de bloqueo de unidad.



Introduzca los siguientes comandos en el terminal para eliminar los puntos blancos y el logo de Ubuntu:

sudo xhost +SI:localuser:lightdm
sudo su lightdm -s /bin/bash
gsettings set com.canonical.unity-greeter draw-grid false;exit
sudo mv /usr/share/unity-greeter/logo.png /usr/share/unity-greeter/logo.png.backup


Si desea volver a continuación, utilice los siguientes comandos.

sudo xhost +SI:localuser:lightdm
sudo su lightdm -s /bin/bash

gsettings set com.canonical.unity-greeter draw-grid true;exit

sudo mv /usr/share/unity-greeter/logo.png.backup /usr/share/unity-greeter/logo.png




10: Logo de Apple en Lanzador



Introduzca los siguientes comandos para instalar Apple Logo:


wget -O launcher_bfb.png http://drive.noobslab.com/data/Mac-14.04/launcher-logo/apple/launcher_bfb.png

sudo mv launcher_bfb.png /usr/share/unity/icons/

Si desea una copia de Ubuntu logo ingrese los siguientes comandos en la terminal:

wget -O launcher_bfb.png http://drive.noobslab.com/data/Mac-14.04/launcher-logo/ubuntu/launcher_bfb.png
sudo mv launcher_bfb.png /usr/share/unity/icons/



11: Auto-ocultar la Unidad Lanzador:
Puede ocultar la unidad lanzador de Apariencia  -> Comportamiento  y apagar .



12: Tweak Tool Unidad para cambiar temas e iconos:

Usted puede utilizar estas herramientas populares para cambiar temas e iconos en Ubuntu, también puedes cambiar otros ajustes de estas herramientas de retoque. También puede instalar Ubuntu Tweak . Ingrese siguiente comando para instalar las herramientas de retoque.

sudo apt-get install unity-tweak-tool

13: Instalar iconos monocromáticos de LibreOffice:Iconos humanos son por defecto en LibreOffice qué tipo de aspecto extraño con esta transformación pack, para hacer LibreOffice más elegante, con iconos monocromáticos siga estas instrucciones. Ingrese siguientes comandos para instalar los iconos monocromáticos de LibreOffice.

sudo apt-get install libreoffice-style-sifr


Después de la instalación vaya al menú LibreOffice seleccione "Herramientas"> "Opciones"> "LibreOffice"> "Ver" y seleccione "Sifr" en "Tamaño de icono y el estilo". Ver siguientes capturas de pantalla





14: Fuentes (Opcional) Mac:


La última vez que recibí la solicitud acerca de las fuentes de Mac, así que esta vez yo los estoy incluyendo con este paquete.Así que aquí están las fuentes para usted.



wget -O mac-fonts.zip http://drive.noobslab.com/data/Mac-14.04/macfonts.zip
sudo unzip mac-fonts.zip -d /usr/share/fonts; rm mac-fonts.zip
sudo fc-cache -f -v
