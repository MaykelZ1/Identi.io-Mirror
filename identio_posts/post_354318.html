Requisitos mínimos para Windows

Sistema Operativo: Windows XP Service Pack 3
Procesador: Core 2 Duo 2GHz o similar
Memoria: 3 GB RAM
Tarjeta gráfica: 512 MB NVidia o ATI
DirectX: Version 9.0
Disco Duro: 3 GB de espacio disponible
Tarjeta de Sonido: Direct X 9.0c
No recomendado para gráficas Intel
