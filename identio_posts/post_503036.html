Glary Utilities Pro 5,92,0,114 DC 23,02,2018 + Por

  • Categoría: Software / Programas
  • Publicado hace más de 7 años


Glary Utilities Pro - es un potente conjunto de ajustes del sistema y utilidades para la puesta a punto y la mejora de la productividad global y proteger su PC. Este software le permite limpiar archivos basura comunes del sistema, entradas de registro no válidas y los rastros de Internet (Hay un enchufe, se puede lograr la plena compatibilidad de las aplicaciones con más de 45 programas diferentes.

Además, Glary Utilities Pro optimiza el proceso de uso de la memoria del sistema, busca, arregla o elimina accesos directos inactivos, edita la lista de programas que se cargan con el sistema operativo y elimina las aplicaciones especificadas.

Principales características de Glary Utilities Pro:
• Configurando el sistema operativo.
• Corrección de errores en el registro del sistema.
• Eliminar archivos innecesarios.
• Desinstalar programas.
• Eliminación de rastros de trabajo en la computadora.
• Gestión de procesos.
• Bloqueo de malware.
• Optimización de la memoria operativa.
• Administre la lista de aplicaciones que se descargan cuando se inicia el sistema.
• Interfaz multilingüe

Un conjunto de utilidades incluidas en Glary Utilities PRO:
• Disk Cleaner: elimina los datos no deseados de sus discos y restaura el espacio en disco
• Registry Cleaner: limpie el registro, escanee y limpie el registro para mejorar el rendimiento del sistema.
• Fixer de accesos directos: elimina los errores de acceso directo en los menús y en el escritorio
• Desinstalar Manager: desinstala por completo los programas que ya no necesita.
• Administrador de inicio: administra los programas de ejecución automática
• Memory Optimizer: comprueba y optimiza la memoria en segundo plano.
• Administrador de menú contextual: administra el menú contextual, administra los elementos del menú contextual para archivos y carpetas ...
• Borrar pistas: borra todos los rastros de estar en Internet: eventos, cookies, historial de Internet ...
• Trituradora de archivos: borra archivos para siempre sin recuperación.
• File Undelete: forma rápida y efectiva de recuperar archivos borrados accidentalmente
• File Encrypter y Decrypter: protege sus archivos del acceso y uso no autorizados.
• Disk Analyzer: muestra el espacio en disco ocupado por archivos y carpetas.
• Buscar archivos duplicados: busque y elimine carpetas duplicadas y vacías
• File Splitter and Joiner: divide los archivos grandes en archivos más pequeños para administrarlos y luego los recopila.
• Administrador de procesos: supervisa los programas que se ejecutan en su computadora y bloquea el spyware y los troyanos.
• Asistente de Internet Explorer: administra los complementos de Internet Explorer
• Herramientas estándar de Windows: proporciona acceso directo a configuraciones útiles para las funciones de las ventanas predeterminadas

¿Qué hay de nuevo en la versión 5.92.0.114:
* Optimized Disk Cleaner: soporte agregado para 'Ashampoo Burning Studio 19' y 'Kantaris Media Player'
* Optimized Tracks Eraser: soporte agregado para 'PerfectDisk 12' y 'Office 2016'
* Optimized Startup Manager: optimizó el algoritmo de coincidencia de adquisición de elementos de inicio y aumentó la precisión en un 30%
* Optimized Duplicate Cleaner: menú optimizado "Archivo-> Lista de carpetas excluidas" para adaptarse a múltiples idiomas.
* Búsqueda rápida optimizada: arreglado.
* Configuración optimizada: resolvió el problema de Avira falsos positivos.
* Mejoras menores de GUI
* Corrección de errores menores
