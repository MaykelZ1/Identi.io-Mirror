Primeros pasos con la Consola GNU/Linux

  • Categoría: Linux
  • Publicado hace más de 11 años
En el Mundo GNU/Linux, la consola es una herramienta que se usa frecuentemente. Si buscamos información sobre algún problema o tuneo con nuestra distro, lo más probable es que la solución involucre usar la consola. Eso no quiere decir que la solución para nuestro problema, requiera usar siempre la consola.

Por ejemplo, la consola para Opensuse con KDE se llama Konsole, la podemos buscar en nuestros programas, o presionando Alt + F2  y escribiendo en el cuadro de texto "Konsole".


 

Ya hace algo de tiempo que en GNU/Linux la consola fue quedando en un segundo plano, como debe ser, si me lo preguntan. Y es que ¿a quién le gusta escribir comandos en la pantallita negra, para copiar un archivo, actualizar el sistema, etc? A muy pocos creo yo.

 

Pero como nunca está de más saber usarla, por aquello que sea necesario, quiero dejar una lista de comandos muy útiles:


 
1. "cd" - Es usado para navegar por los directorios.
Ejemplo : cd /home/opensuse -> Ésto nos lleva al directorio "/home/opensuse"

 
 
2. "ls" - Nos muestra una lista con el contenido de un directorio
Ejemplo : ls /home/opensuse -> Lista el contenido del directorio "/home/opensuse"

 
 
3. "mkdir" - Crea un directorio
Ejemplo : mkdir /home/nueva -> Creará una carpeta con el nombre "nueva" "/home/nueva"

 
 
4. "cp" - Copia de archivos
Ejemplo : cp /home/opensuse/1.txt /home -> copia el archivo 1.txt "/home/opensuse/1.txt" en el directorio "/home"

 
 
5. "mv" - Se utiliza para mover archivos
Ejemplo : mv /home/opensuse/1.txt /home -> Mueve el archivo 1.txt "/home/opensuse/1.txt" al directorio "/home"

 
 
6. "rm" - Se utiliza para eliminar archivos
Ejemplo : rm /home/opensuse/1.txt -> Elimina el archivo 1.txt que se encuentra en "/home/opensuse/1.txt"

 
 
7. "clear" - Se utiliza para limpiar la pantalla

 
 
8. "pwd" - Se utiliza para saber en que path estamos trabajando

 
 
9. "chmod" - Se utiliza para cambiar los permisos de un archivo
Ejemplo : chmod 755 /home/opensuse/1.txt -> Este cambio da todos los permisos menos el de edición al archivo 1.txt

 Para más información de chmod y los permisos de un archivo dejo éste link
 
 
10. "man" - Entrega informacion sobre el commando ingresado (man=manual)
Ejemplo : man pwd -> Nos da la informacion en linea del comando 'pwd' 

 
 
11- "ps" - Lista los procesos que están actualmente corriendo.

 
 
12. "find" - Se utiliza para localizar un archivo/directorio.
Ejemplo: find /home/suse -type f -name 1.txt -> Busca el archivo 1.txt en el directorio /home/opensuse

 
 
13. "exit" - Se utiliza para salir de algun programa o del shell.

 
 
14. "su" - Da momentáneamente privilegios de ROOT

 
 
15. "shutdown" - Apaga el sistema
Ejemplo : shutdown -h now -> Apaga el sistema en ese momento y NO lo vuelve a reinicia

 
 
16. "halt" - Apaga el sistema de la misma manera que el mandato anterior

 
 
17. "reboot" - Reinicia el sistema

 
 
18. "startx" - Inicia el entorno gráfico

 
 
19. "?" Representa un solo caracter.
Ejemplo : ls ?.txt (lista todos los archivos de un solo caracter con extensión txt)

 
 
20. "*" Representa todo.
Ejemplo : ls *.rpm (lista todos los archivos con extensión rpm)

 
 
21. "kill" - Mata el proceso
Ejemplo : kill -9 123 -> Mata el proceso con número de PID 123 sin darle tiempo a terminar

 
 
22. "hostname" - Devuelve el nombre de la maquina




Bonus:
(gentileza de @rojored, )

23. "eject" - Con éste comando se conseguirá la expulsión de la unidad de CD, siempre y cuando esta no esté en uso
