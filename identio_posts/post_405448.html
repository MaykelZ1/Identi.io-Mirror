10 Script Imprescindibles para Greasemonkey













Greasemonkey es una extensión para el navegador Mozilla Firefox que permite, por medio de pequeñas porciones de código creadas por usuarios, modificar el comportamiento de páginas web específicas. Con esta extensión es posible mejorar la experiencia de lectura de un sitio web, hacerlo más usable, añadir nuevas funciones a las páginas web, corregir errores, mejorar servicios de búsquedas y muchas otras cosas más.




Colección de 10 Script para Greasemonkey y Tampermonkey que no debes dejar pasar:


(1) Anti-Adblock Killer
(2) AdsBypasser
(3) AntiAdware
(4) Direct Google
(5) Direct Google Images
(6) Select-click-copy Enabler
(7) Google Picasa Get ImageUrl    mod
(8)  Just Image
(9) R@p1dL33ch @d5 & T1m3r byp@55!!   mod
(10) 5h0w L1nk5


(otros aportes:identi.li)

Instalar Tampermonkey Chrome
https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=es








(1) Anti-Adblock Killer

Anti-Adblock Killer es un script creado para eludir las protecciones utilizadas en algunos sitios web que obligan al usuario desactivar AdBlockers.




https://greasyfork.org/es/scripts/735-anti-adblock-killer-reek






(2) AdsBypasser

Similar a safebrowse, nos permite eludir el conteo de algunos redirectores.

Lite edition removes image-hosting site support from Full edition. If you prefer to use other userscripts to deal with image-hosting sites, you can use the Lite edition.



https://greasyfork.org/es/scripts/4881-adsbypasser






(3) AntiAdware
Como muestra la imagen algunos sitios de descarga como el de flash u otros nos meten diversos programas extras para monetizar el producto, con este script podremos saltar dicha descarga.



https://greasyfork.org/es/scripts/4294-antiadware


(4) Direct Google

Este script nos dejara la URL directa al sitio web y nos dejara la URL "En caché" a mano.




https://greasyfork.org/es/scripts/568-direct-google







(5) Direct Google Images

Este script nos dejara la URL directa a la imagen sacando la redirección de google.





https://greasyfork.org/es/scripts/6674-direct-google-images






(6) Select-click-copy Enabler

En algunos sitios web deshabilitan la copia de texto u imagen, con este script podrás saltar tal restricción.



https://greasyfork.org/en/scripts/560-select-click-copy-enabler







(7) Google Picasa Get ImageUrl    mod

Como muestra la imagen con este script podremos obtener la URL directa o con
[ img ] [ / img ] desde picasa web.



https://greasyfork.org/es/scripts/15431-google-picasa-get-imageurl






(8)  Just Image

Este script les valdrá en algunos host de imagen, lo que hace es redirigirlos directamonte a la imagen.

http://userscripts-mirror.org/scripts/show/430370






(9) R@p1dL33ch @d5 & T1m3r byp@55!!   mod

Con este script podremos generar directamente enlaces evitando esos molestos pops.



https://greasyfork.org/es/scripts/15430-r-p1dl33ch-d5-t1m3r-byp-55
