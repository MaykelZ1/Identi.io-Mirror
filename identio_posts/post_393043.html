VMware Workstation Pro 12

  • Categoría: Reciclaje
  • Publicado hace más de 10 años


La introducción de Vmware Workstation 12 Pro

 VMware Workstation 12 Pro continúa la tradición de VMware de ofrecer características de vanguardia y el rendimiento que los profesionales técnicos dependen de todos los días cuando se trabaja con máquinas virtuales. Con soporte para la última versión de Windows y Linux, los últimos procesadores y hardware, y la capacidad de conectarse a VMware vSphere y vCloud Aire, es la herramienta perfecta para aumentar la productividad, ahorrar tiempo y conquistar la nube.
Construido para Microsoft Windows 10 y Más

Vmare Workstation 12 Pro es la mejor manera de evaluar y probar Windows 10 con tus existente de Windows o Linux PC. En unos pocos pasos simples que usted puede instalar fácilmente en Windows 10 en la maquina virtuales a partir de una imagen ISO del disco o utilizando el Asistente Virtual Machines Workstation. Con la estación de trabajo 12 Pro puede empezar a tomar ventaja de Windows 10 últimas características como Cortana el asistente digital personal, entintado capacidades en el nuevo navegador web de borde, o incluso empezar a construir aplicaciones universal para Windows 10 dispositivos. Usted puede incluso pedir Cortana lanzar VMware Workstation derecha desde Windows 10.
Para las organizaciones y usuarios técnicos que se ejecutan las últimas distribuciones de Linux, la estación de trabajo 12 Pro es compatible con Ubuntu 15.04, Red Hat Enterprise Linux 7.1, Fedora 22, y muchos más. Usted puede incluso crear hipervisores anidados para ejecutar Hyper-V o VMware ESXi y vSphere para construir el laboratorio vSphere definitiva para la formación, la demostración demostraciones y pruebas. Estación de Trabajo 12 Pro ha añadido soporte para VMware fotones para que pueda empezar a construir y experimentar con la aplicación en contenedores que se ejecuta en un entorno virtualizado.
Mostrar potentes gráficos 3D

 Vmware Workstation Pro ahora soporta DirectX 10 y OpenGL 3.3 que ofrece una experiencia más fluida y sensible cuando se ejecutan aplicaciones 3D. Con estas capacidades añadidas que ahora tienen acceso a todas las aplicaciones que requieren X10 directa como Mapa Poder herramienta de Microsoft para Excel, lo que le permite visualizar datos en una nueva dimensión. Además, la estación de trabajo hace que correr muy exigentes aplicaciones 3D como AutoCAD o SOLIDWORKS un broche de presión en una máquina virtual.
Listo para pantallas de alta resolución

 VMware Workstation Pro ha sido optimizado para soportar alta resolución 4K UHD (3840 x 2160) pantallas para ordenadores de sobremesa y + (3200x1800) muestra QHD utilizados en ordenadores portátiles y tablets x86 que le proporciona una experiencia nítida y detallada. Estación de Trabajo 12 Pro ahora soporta múltiples monitores en el host con diferentes configuraciones de DPI para que pueda ejecutar su nueva pantalla 4K UHD junto con su pantalla 1080P HD existente.
Crear máquinas virtuales potentes

 Con VMware Workstation 12 Pro, puede crear máquinas virtuales con hasta 16 CPU virtuales, discos virtuales 8 TB y 64 GB de memoria para ejecutar las aplicaciones de escritorio y de servidor más exigentes en un entorno virtualizado. Usted puede dar a las aplicaciones de uso intensivo de gráficos un impulso adicional mediante la asignación de hasta 2 GB de memoria de vídeo a una máquina virtual.

Requisitos del sistema

VMware Workstation Pro se ejecuta en PC x86 estándar con procesadores Intel y AMD.

VMware recomienda lo siguiente:

    Procesador Intel Core? Solo x86 de 64 bits o equivalente, procesador de doble núcleo AMD Athlon? 64 FX o equivalente.
    Velocidad de 1,3 GHz o superior.
    2 GB de RAM como mínimo; se recomiendan 4 GB de RAM o más.

Instalación de Workstation Pro:

1,2 GB de espacio disponible en disco para la aplicación. Espacio en disco adicional suficiente para cada máquina virtual. Consulte el espacio en disco recomendado por el proveedor para cada sistema operativo invitado.

Para la compatibilidad con Windows DirectX 10 en una máquina virtual:

    3 GB de RAM (PC host).
    Intel de doble núcleo a 2,2 GHz o superior o AMD Athlon 4200+ o superior.
    NVIDIA GeForce 8800GT y superior o ATI Radeon HD 2600 y superior.

Workstation 12 Pro necesita un procesador de 64 bits y un sistema operativo host de 64 bits.
