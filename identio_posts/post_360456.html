OblyTile v0.9.8










En windows 8 y 8.1 existen las live tiles que no son más que esos pequeños espacios en nuestro menú inicio. Estando aquí se pueden organizar de todas maneras y tamaños, y como es bien sabido en Win 8.1 permite interactuar de manera distinta con ellas.


Una de las principales desventajas de estas versiones es que al crear el acceso directo de un programa en el menú lo hacia de manera clásica, ya que le asigna un color que le de su gusto, el icono por defecto y más.


Por esta razón existe OblyTile, este es un programa que permite crear todo tipo de tiles de programas, enlaces a páginas web y carpetas para modificarlos a su gusto para tenerlos en nuestro menú de Windows.





Funciones de OblyTile


Para comenzar a utilizarlo, se descarga el programa del enlace que se encuentra al final del post. No requiere ningún tipo de instalación por lo que sólo basta con ejecutarlo en modo administrador para iniciarlo.






Para comenzar a crear un tile se escribe el nombre que llevara en el apartado Tile Name, ya que se escribió el nombre en los puntos suspensivos delante de Program Path se selecciona si será un acceso directo a un programa, un enlace web, o una carpeta.





Para el icono, se descarga uno que sea de su agrado de internet y lo coloca en  Tile Square Image, esta imagen se puede eliminar al finalizar, no afecta en nada.

En el apartado de colores, se selecciona el color que se desea para el tile y para la letra.






Ya que se termino de configurar el Tile se da clic en Create Tile para lo cual al finalizar Windows te lanzará una notificación de que se creó satisfactoriamente.






Y de esta manera es como queda creada. 






Una gran ayuda para quienes desean tener accesos directos de sus programas, páginas o carpetas en el menú inicio.



-Windows 8, 8.1 en todas sus versiones 32 & 64 bits
