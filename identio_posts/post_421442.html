Ubuntu After Install

  • Categoría: Linux
  • Publicado hace más de 9 años
Hola gente de Identi y sobre todo a los usuarios Linux como yo.
Este es mi primer post, así que sepan disculpar posibles errores en el mismo y estoy abierto a sugerencias y consejos.
 Ubuntu After Install es un software pensado para automatizar la tarea de instalación de soft y así evitar el tedioso trabajo de ingresar los repositorios de dicho soft a mano.
Los que instalan cada 6 meses una nueva versión de Ubuntu, saben de lo que hablo.
Este programa está en desarollo y cada vez son más los programas que instala en forma automática.
Contiene listas para instalar, 31 de las aplicaciones más populares y útiles disponibles para Ubuntu, muchas de las cuales no están en el centro de software y requieren que agreguemos PPAs o que descarguemos paquetes .DEB.
 
Ubuntu After Install lo hace todo por nosotros, para ahorrarnos el largo y tedioso proceso de seleccionar e instalar cada aplicación por separado. El programa instala la última versión de todas las aplicaciones, así como los repositorios requeridos para mantener el software actualizado.
 Instalación
Ubuntu After Install es compatible con Ubuntu 12.04 y versiones posteriores.
 
Opción 1:
 
Puedes añadir el repositorio desde la terminal e instalar Ubuntu After Install, escribiendo:
 
sudo add-apt-repository ppa:thefanclub/ubuntu-after-install
sudo apt-get update
sudo apt-get install ubuntu-after-install
 
Opción 2:
 
Si no te gusta la terminal, puedes simplemente descargar el paquete .DEB y al hacer doble clic sobre el, se abrirá el centro de software para que lo instales.

La página del proyecto desde donde pueden descargar el .deb
thefanclub.co.za/how-to/ubuntu-after-install ( añadir www. al principio)
