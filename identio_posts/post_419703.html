Tutorial Virtualbox (2da parte)

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 9 años
Que tal amigos, pues aquí les traigo la segunda entrega de este tutorial. En la primera vimos como instalar Virtualbox y sus aditamentos. Hoy ya pasaremos a lo bueno, es decir instalar nuestra maquina virtual con el sistema operativo que deseemos probar. En el primer tutorial tienen los programas (ultima version). Hagamos un breve raconto de los pasos seguidos en el post anterior:


1) Instalamos Virtualbox, usando el correspondiente instalador según tengamos en la compu Windows, linux o Mac.


2) Para ello vamos al menu archivo ---> preferencias ---> extensiones, damos clic en el icono del triangulo invertido naranja y seleccionamos el extension pack y lo instalamos. Una vez instalado nos aparecerá este mensaje, cerramos. Ya tenemos instalado el extension pack.






Ahora procederemos a preparar nuestra maquina virtual, para luego instalar el sistema operativo huesped (el cual previamente deben haber descargado, se puede usar tanto una imagen iso como el propio dvd de instalación. Les recomiendo la imagen iso, asi no gastan en dvd. Pero si ya lo tienen en dvd tambien se puede usar.




Abrimos Virtualbox una vez instalado.




1) En la parte superior izquierda hacemos clic en "nueva"





Esto iniciara el asistente para instalar el sistema operativo huésped.




2) En la siguiente pantalla en nombre ponemos el nombre del sistema operativo a instalar, en tipo corresponderá decir si es Linux, Windows Mac o Solaris. Y en version bueno la version del sistema operativo que estamos instalando.

Ejemplo. Vamos a instalar Windows 7 de 64 bits, pondremos

como sigue en la captura siguiente.


En nombre en este ejemplo pondremos Windows 7

En tipo seria Windows

Version: windows 7 64 bits





Por supuesto todo depende de que vayamos a instalar. damos clic en siguiente.


3) Ahora seleccionaremos la cantidad de memoria ram que le vamos a asignar a nuestra maquina virtual. Mucha gente deja todos los valores por defecto que pone la instalacion, pero personalmente he experimentado algunos problemas, lo ideal es siempre elegir lo maximo (siempre dentro de las rayitas verdes, si se pasan a las anaranjadas les va a tirar error luego y se les colgara la maquina... si sabre yo de eso jaja). Pongan asi como muestra la imagen.




Damos clic en siguiente.


4) En la siguiente pantallla damos clic en crear.




5) Luego en la siguiente pantalla dejamos como esta todo y damos clic en crear








Aqui les recomiendo no dejar lo que viene por defecto, porque despues al actualizarse el sistema operativo se les acaba el espacio y es un dolor de cabeza para ampliar el tamaño del disco duro. Les recomiendo poner mas de 50 gb o lo que ustedes consideren. De todas maneras se ira administrando dinamicamente es decir por mas que ponga 80 gb si en total tienen consumidos 30 ese es el tamaño real. Me paso instalando windows, le deje los 25 gb que dice el programa y en cuanto puse windows update chau mi espacio en disco jaja. Personalmente les pongo unos 70 gb y tengo espacio para estar holgado y tranquilo.


6) Una vez que dimos en crear ya tendremos preparada nuestra maquina para instalar nuestro sistema operativo huesped. En el próximo tutorial veremos como instalar el sistema operativo que queramos probar. Y aun nos faltaría instalar las guest addition para poder compartir archivos entre anfitrión y huesped y por ultimo las configuraciones como para poder tener disponibles puertos usb y habilitar para usar con el celu, entre otros.



Hasta el próximo tutorial!.
