Norton Internet Security 2014 Full

  • Categoría: Software / Programas
  • Publicado hace más de 11 años

Norton Internet Security 2014 es una suite de seguridad robusta y fiable para tu PC. Incluye un motor potente, firewall, función Identity Safe y una usabilidad al más alto nivel. Esta nueva versión no revoluciona el producto, pero optimiza varias características. 

Herramientas avanzadas de defensa

Norton Internet Security 2014 ofrece protección completa y entiempo real contra virus, gusanos, troyanos, spyware y rootkit, a la vez que actúa como filtro para el correo electrónico, programas de mensajería instantánea y para la navegación por Internet.

Norton es capaz también de analizar tu muro de Facebook, que cada vez más se está convirtiendo en una herramienta de difusión de malware.

Gracias a la tecnología SONAR, el módulo de defensa proactiva puede identificar con más precisión posibles amenazas desconocidas. Norton Internet Security 2014 puede bloquear además páginas web y descargas peligrosas, e incluye opciones para configurar a fondo el Modo Silencioso.

Integra también Norton Power Eraser, una herramienta avanzada a usar cuando el motor falla, con el escaneo clásico, al eliminar un malware que supuestamente ha atacado el sistema.

Norton Internet Security 2014 incluye una evolución del móduloInsight, que analiza archivos y procesos en ejecución, clasifica los seguros y los excluye de la fase de escaneo, optimizando los recursos disponibles para tu sistema.

A diferencia de Norton Antivirus, Internet Security ofrece algunas opciones más, comenzando con un firewall de dos vías que te permite filtrar las conexiones entrantes y salientes. Este firewall cuenta con diversas opciones de configuración y está organizado de tal manera que incluso los usuarios menos experimentados podrán usarlo sin problemas. También encontrarás aquí los controles parentales (en Norton Family) para proteger la navegación por Internet de los niños.
