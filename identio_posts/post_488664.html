Obtén Windows Fall Creators Update vía MCT

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 8 años


Aquí les enseñare como obtener Windows 10 Fall Creators Update que el día 17 de octubre salio de forma oficial, existen varias maneras de descargar la ISO oficial de Windows pero en este caso usaremos una herramienta ofrecida por microsoft llamada Media Creation Tool (MCT) que permite descargar la ISO oficial en el idioma, arquitectura (x86 ó x64) que prefieran, también permite hacer una AiO con ambas arquitecturas para que tengan todo en una sola ISO, esta herramienta también permite hacer una actualización de su sistema, claro que solo sirve si tienen una versión anterior a Fall Creators Update (Build RTM 16299.15).



Empecemos...

1- Descarguen la herramienta oficial buscando en Google: Media Creation Tool ó Descargar imagen de disco de Windows 10 (archivo ISO).



2- Ejecutan la herramienta esperan unos segundos y aceptan los términos de licencia.






3- Una vez aquí como les comente pueden directamente darle a Actualizar este equipo ahora y les dará la opción de conservar archivos o hacer una actualización limpia, pero en este caso le daremos a Crear medios de instalación.




4- Aquí desmarcan la opción Usa las opciones recomendadas para este equipo, y seleccionan el idioma y arquitectura, sobre la edición solo les saldrá Windows 10 eso es porque la ISO a descargar incluyen todas las ediciones (Home, Home N, Home Single Language, Education, Education N, Pro, Pro N) no incluye la edición Enterprise.

*En arquitectura la opción Ambos se refiere a lo que comente, es para hacer una AiO con todas las ediciones y las dos arquitecturas.








5- Una vez aquí les da la opción de insertarlo directamente en un USB, mínimo 8gb como lo indica, pero como aquí les enseño como obtener la ISO, seleccionaremos Archivo ISO...




...y seleccionamos donde se va a guardar la ISO, nombrándolo de manera que lo identifiquen.




6- Empezará a descargar y a esperar...



...hasta que termine.




7- Listo, para montarlo en un USB usar el programa llamado Rufus.





NOTA. Si usaron la opción "Ambos" en arquitectura, cuando van a bootear la imagen, les saldrá de esta manera:
