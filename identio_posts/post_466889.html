Windows 7 SP1 Colección (MSDN)(SVF)(Retail) EN-ES

  • Categoría: Reciclaje
  • Publicado hace más de 8 años






Colección de Microsoft Windows 7 SP1, son isos originales provenientes de MSDN de Microsoft eso significa que son completos sin ningún tipo de modificación, como es para colección cada iso esta en un archivo SmartVersion (.svf) para que ocupe menos espacio; La colección contiene solo el idioma Español Internacional(ES) y Inglés(EN), también esta el MUI(Language Pack).





Que es un archivo .svf?

-SmartVersion es una herramienta para almacenar multiples versiones de sus archivos dentro de un archivo SmartVersion (SVF). SmartVersion usa la compresión inteligente y algoritmos de comparación de archivo. Sólo los cambios entre las versiones de producto diferentes son almacenados en los archivos SVF. Esto elimina datos redundantes en sus archivos SVF y hace seguro los parches que usted crea para las nuevas versiones de sus productos son cuanto menos posible.


Como extraigo la .iso de un archivo .svf?

-Para extraer es necesario un programa de nombre SVF eXtractor (Incluído en la descarga).




NOTA1. Subido a una carpeta en MEGA.NZ

NOTA2. SOLO la iso de windows 7 sp1 ultimate contiene las demás versiones las cuales son: Home Premium, Home Basic, Professional, Starter. Por defecto esta bloqueado, para desbloquear solo hay que eliminar el archivo ei.cfg ubidado en "sources/ei.cfg" dentro de la iso. Esto es solo para poder seleccionar durante la instalación las versiones antes mencionadas.

NOTA3. La extracción de las isos es escalonado, ejemplo: si se quiere extraer la iso <es_professional_n_x64> se necesita la iso <es_ultimate_n_x64> y para tener esa iso necesito <es_ultimate_x64> y para tener esa necesito <en_ultimate_x64>.


NOTA4. Como pueden ver todo se extrae con las isos ultimate en inglés x86-x64 siendo las unicas que estan en formato .iso los demas en formato comprimido .svf.

NOTA5. Se recomienda tener todo en una misma carpeta incluido el programa "svfx.exe" para extraer mas cómodamente.

NOTA6. Mas detalles de como extraer en el archivo leeme.txt dentro de la descarga.
