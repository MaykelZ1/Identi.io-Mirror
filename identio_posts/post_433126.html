Windows Codename "Neptune" Build 5.111

  • Categoría: Reciclaje
  • Publicado hace más de 9 años
Windows Neptune fue una versión experimental de Microsoft Windows que fue desarrollada durante 1999 hasta el 2000. El objetivo era obtener una vista preliminar de ideas para un hogar de usuarios del sistema operativo de las mismas fuentes, como la basada en Windows NT 2000, que fue desarrollado para usuarios de negocios, en sustitución de los antiguos basado en MS-DOS de Windows 9x. Sin embargo Windows Neptune se combinó con Windows Odyssey para unirse al proyecto "Whistler" el cual fue dado a conocer a finales del 2001 como Windows XP. Unos meses antes fue lanzado Windows Millenium Edition.¡Un saludo!
