IsoBuster Pro 4.0.0.00







IsoBuster Pro es un software de recuperación de datos simple pero potente para recuperar datos de discos ópticos, discos duros, discos de estado sólido, y unidades USB. IsoBuster tiene la capacidad de acceder y recuperar los datos borrados o perdidos de los sistemas de archivos dañados y discos dañados físicamente incluyendo CD, DVD y discos Blu-ray, USB, unidades de disco duro y SSD, y mucho más. Es compatible con todos los formatos comunes (sistemas de archivos), incluso se puede rescatar archivos perdidos de un mal o de la papelera discos / unidades, guardar documentos importantes, fotos, mp3, vídeos, ficheros de archivo, imágenes de disco (como ISO, BIN y NRG), etcétera

IsoBuster es una aplicación de recuperación de datos altamente fiable, diseñada para restaurar los datos dañados de diferentes tipos de dispositivos. Originalmente diseñado para recuperar archivos de discos ópticos, IsoBuster amplió sus capacidades. Como tal, ahora es compatible con todos los medios de almacenamiento como dispositivos flash, tarjetas de memoria, unidades de disquetes, discos Zip, discos duros, y más. IsoBuster cuenta con una interfaz sencilla y fácil de usar. Es que todas las pistas y sesiones existentes en los medios de almacenamiento al igual que el explorador muestra, lo que hace que es posible visualizar los datos recuperables con mayor claridad.


Características principales:


Habilidades para trabajar con archivos de CD comprimido directos
Comparar los datos o archivos de imagen con su suma de comprobación
motor de extracción compleja y altamente eficaz
Creación de archivos de imágenes, archivos de control y cuesheet
Posibilidades para restaurar los datos de discos dañados
Los datos de proceso de varios tipos de imágenes virtuales
Varios mecanismos para recuperar archivos perdidos o eliminados
Muestran y permiten extraer las diferentes grabaciones
Soporta HFS y HFS +, el sistema de archivos Apple Mac
Soporta varios tipos de dispositivos (medios de almacenamiento)
Acceder a archivos y rescate, y más suprimirse.


¿Qué hay de nuevo en IsoBuster Pro 4.0 :


Los cambios en la interfaz gráfica de usuario que soporta más reciente del sistema operativo
Nuevas consultas de búsqueda predefinidas añadido
Nueva unidad / detección automática de eliminación
Nueva fácil historial de navegación (pan rallado)
muy potente funcionalidad nueva búsqueda
Otras correcciones de errores y mejoras.
