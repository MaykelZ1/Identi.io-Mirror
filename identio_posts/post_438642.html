¿Qué es HTML y para qué sirve?

  • Categoría: Apuntes y Educación
  • Publicado hace más de 9 años
HTML



Definición de HTML (HyperText Markup Language): 
Significa lenguaje de marcas de hipertexto, haciendo referencia a el lenguaje de programación estándar ´para el diseño de paginas web.Es un lenguaje, ya que contiene distintos códigos y sintaxis como cualquier otro lenguaje de programación.

¿Como funciona?

HTML consiste de una serie de cortos códigos escritos en un archivo de texto, llamadas etiquetas. El texto se guarda como un archivo HTML y visto a través de un navegador web (IE, Mozilla, Chrome, etc). Este navegador lee el archivo y traduce el texto en una forma visible. Se puede utilizar cualquier cosa, desde un editor de texto rudimentario (bloc de notas, WordPad, etc) a un potente editor gráfico para crear paginas HTML.


¿Qué son las etiquetas?

Las etiquetas son los que separan el texto normal del código HTML. Casi en su totalidad las etiquetas se identifican de la siguiente manera: <html> , <ul>, <div> entre otras. Contamos con una diversidad de etiquetas para imágenes, vídeos, textos, tablas y muchas mas.



Se puede identificar que las etiquetas utilizadas en la imagen son: html, body, center, h1 y font. Para cerrar una etiqueta se utiliza el siguiente código: 
</ejemplo>

¿ Y en que nos beneficia?

Una pagina HTML nos brinda una variedad de ventajas tanto para dar a conocer una empresa, generar ingresos o simplemente mostrar un producto e información al mundo.
