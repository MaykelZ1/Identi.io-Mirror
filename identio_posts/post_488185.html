TeraCopy Pro 3,26 (14/Oct/2017)




TeraCopy para Microsoft Windows


Una de las quejas más comunes sobre las versiones más recientes de Windows es la velocidad de copia lenta, especialmente cuando se transfieren muchos archivos a través de la red. Si desea acelerar su copia o si transfiere regularmente grandes cantidades de datos y tiene que detener el proceso para realizar alguna otra tarea de disco intensivo, este programa puede ser justo lo que necesita.

TeraCopy está diseñado para copiar y mover archivos a la máxima velocidad posible. Se salta archivos malos durante el proceso de copia y, a continuación, los muestra al final de la transferencia para que pueda ver cuáles necesitan atención. TeraCopy puede comprobar automáticamente los archivos copiados para detectar errores calculando sus valores de suma de comprobación CRC. También proporciona mucha más información sobre los archivos copiados que su contraparte de Windows.

TeraCopy se integra con el menú del botón derecho del Explorador de Windows y se puede configurar como controlador de copia predeterminado.


Copia archivos más rápido

TeraCopy utiliza buffers dinámicamente ajustados para reducir los tiempos de búsqueda. La copia asincrónica acelera la transferencia de archivos entre dos discos duros físicos.

Detener y reanudar las actividades de transferencia de archivos

Detenga el proceso de copia en cualquier momento para liberar los recursos del sistema y continuar con un solo clic.

Error de recuperación

En caso de un error de copia, TeraCopy intentará varias veces recuperarse y, en el peor de los casos, simplemente saltará el archivo, sin terminar la transferencia completa.

Lista de archivos interactivos

TeraCopy muestra las transferencias fallidas de archivos y le permite corregir el problema y volver a copiar sólo los archivos problemáticos.

Integración de Shell

TeraCopy puede reemplazar completamente las funciones de copia y movimiento de Explorer, que a su vez le permiten trabajar con archivos como de costumbre.

Soporte Unicode completo.

Compatibilidad con Windows 8 x 64.



TeraCopy Pro *

La versión de pago del programa incluye las siguientes características:
Copiar / mover a sus carpetas favoritas.
Guarde los informes como archivos HTML y CSV.
Seleccione archivos con la misma extensión / carpeta.
Elimine los archivos seleccionados de la cola de copias.
¡Más funciones próximamente!



Arregle problemas fácilmente

En caso de un error de copia, TeraCopy intentará recuperarse. Si no puede recuperarse en el peor de los casos, simplemente saltará el archivo problemático, sin terminar la transferencia completa. También puede esperar la reconexión de red o dispositivo.

Los archivos innecesarios pueden omitirse para reducir el tiempo de transferencia. Durante el proceso, TeraCopy muestra errores detectados y le permite solucionar el problema recopiando sólo los archivos problemáticos.




Verificación de archivos

TeraCopy puede verificar los archivos después de haberlos copiado para garantizar que sean idénticos. Esto se hace comparando los hashes de los archivos de origen y de destino.

Algoritmos admitidos: CRC32, MD5, SHA-1, SHA-256, SHA-512, Panamá, Tiger, RipeMD, Whirlpool y xxHash.





Confirmar drag'n'drop

TeraCopy puede mostrar opcionalmente un cuadro de diálogo de confirmación en cada operación de arrastrar y soltar. Puede evitar que mueva las carpetas accidentalmente de una ubicación a otra.





Flujo de trabajo más rápido

TeraCopy puede conservar el historial de los directorios utilizados recientemente, así como mostrar las carpetas que están abiertas en el Explorador y otros administradores de archivos.
