Mame UI (x32 & x64) + 2734 ROMS




Hola amigos. Para los que echan de menos esos antiguos juegos Arcade, les traigo el MAME UI con 2734 ROMS (clónicos incluídos) para que se diviertan.

Si bién el  Acceso Directo da al MAME32, también está dentro de la carpeta instalada el MAME64. Las ROMS deben quedar dentro de la carpeta MAME32/ROMS (deben dejarlas con el formato ZIP).


INSTRUCCIONES DE INSTALACIÓN:
Para los Novatos, seguir los siguientes pasos:
1) Ejecutar MameUI32_Setup.exe.
2) Las ROMS debens ser descomrimidas dentro de la carpeta de instalación.
3) Entrar a la carpeta de instalación y correr MamemUI32.exe.
4) Sólo la 1ª vez, ir a File, Audit all Games para que se genere el índice en forma correcta y no tengan problema para accesar a los Juegos.
5) Doble Click sobre el juego.
6) Cuando aparezca un texto con la palabra OK al final, sólo digitar OK.
7) Después da una información básica sobre Video, Audio, etc., a lo cual pulsan la Tecla Espaciador.


INSTRUCCIONES DE USO
Como está basado en las antiguas máquinas de juego que funcionaban en base a Fichas o Monedas:
1) Pulsar la tecla 5 para insertar fichas.
2) Pulsar la tecla 1 para jugar.
3) Pulsar la tecla P para pausar y quitar pausa.
4) Pulsar la tecla TAB para ingresar a opciones del juego.
5) Pulsa la tecla ESC para salir del juego.


LISTA DE JUEGOS
