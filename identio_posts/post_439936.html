Comencemos!

1. ¿Qué es una lista y sangrado?
     Una lista es el conjunto de textos ordenados por números, letras o símbolos. Un ejemplo simple es el de este post que esta enumerado y organizado por los números 1, 2, 3 y 4.


2. Listas en HTML.
     Para las listas se utilizan diferentes etiquetas las cuales veremos a continuación:

         No enumeradas: Para las listas no enumerada utilizamos la etiqueta <ul>. Vamos a utilizar la
                                       etiqueta <li> para que aparezcan de la siguiente manera.




         Enumeradas: Para las listas enumeradas utilizamos la etiqueta <ol>. Vamos a utilizar la
                                  etiqueta <li> para que aparezcan de la siguiente manera.




3. Sangrado en HTML.
      La sangría es una especie de margen que se establece a ambos lados del texto. Para que un texto
       aparezca sangrado, se inserta con la etiqueta < blockquote >.
