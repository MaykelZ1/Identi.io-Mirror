Windows 10 Actualizado marzo 2017
MSDN 1703 15063 ESPAÑOL 32 & 64 BITS
+ WINDOWS SPY DESTROY


Esta nueva versión de Windows busca corregir muchas de las críticas y pesadillas que surgieron
debido al gran cambio de Windows 8. Windows 10 también es un intento más profundo de unificar la experiencia de computadoras y dispositivos móviles, al igual que el gran ecosistema
de Microsoft.
las novedades integradas en las isos son las siguientes
- microsoft update 15063 marzo 2017
- update build 1703 español  32 & 64 bits

- ISO windows ES 10 enterprise 1703 64 bits actualizado marzo 2017
- ISO windows ES-MX 10 enterprise 1703 32 bits actualizado marzo 2017

Requisitos de instalacion

Básicamente, si tu PC puede usar Windows 8.1, entonces no debería haber problemas.
Si no estás seguro, no te preocupes: Windows revisa el sistema para ver si puede instalar
la versión preliminar.

Requisitos de instalacion

IDIOMA: ESPAÑOL Es-Mx
TAMAÑO: 3,97 GB windows 64 bits enterprise / 2,91 GB windows 32 bits enterprise
MEDICINA INCLUIDA: si
INSTRUCCIONES: SI
SERVIDORES DE DESCARGA: MEGA Y ZIPPYSHARE
CONTRASEÑA: dsfansub

INSTALACION
- Queme el archivo ISO en un dvd o prepare una unidad booteable USB con la misma.
- se recomienda instalar desde cero sin conexion a internet, una vez finalizada la instalacion
es posible actualizar el sistema para asi despues activar el windows
- una vez actualizado el windows e instalado el office 2016 se recomienda usar el programa
destroy 10 windows spy para desinstalar one drive e  inhabilitar y eliminar otros programas
intrusivos de windows para asi usarlo sin problemas.

- una vez hecho ello proceder a la activacion del sistema sin problemas.
estas actualizaciones se descargan cuando actualizas el windows 10 por primera vez manualmente.
