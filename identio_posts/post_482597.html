Titulo: Megabasterd
Peso: 30mb
Version: Portable.
Servidor: ZippShare




*Compatible con todo sistema operativo, Linux, MAC y Windows en cualquiera de sus versiones.
*Podrás hacer múltiples descargas, se recomiendan 3 para una conexión de 1 mb de bajada.
*Podrás crear una cola de descargas, para que tus archivos se descarguen uno tras otro.
*Solo funciona para descargas y subidas al servidor de Mega.
*Finalmente simplemente deberás instalar JAVA para poder ejecutar el software sin problemas.
*Disfrutaras de subir y descargar archivos, sobre todo descargar sin limitaciones.





Paso1. Ejecutar como administrador y aparecera  la siguiente ventana






Paso 2. Copian el link de mega y luego ir a la ventana File y New Download.







Paso 3. Los Link seran detectados automaticamente y le dan clic al boton azul. y a descargar.






Tambien lo puedes usar para subir archivos a MEGA solo dan de alta su cuenta en la configuracion.
