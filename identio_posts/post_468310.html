Cisco Network Magic

  • Categoría: Software / Programas
  • Publicado hace más de 8 años
Cisco Network Magic Pro Edition 5.5.9195




Con Network Magic Pro proporciona herramientas de hágalo usted mismo "para ayudarle a configurar, administrar y proteger la red como un profesional de TI.
Características:
- Fácil de utilizar-No conocimientos necesarios de TI
- Realizar tareas de ubicación centralizada
- Conectarse a la red
- Añadir un dispositivo
- Administrador de conexión wi-fi
- Apoyo NAS
- Compartir carpetas e impresoras
- Nuevas Mejoras en la seguridad (compatible con el cifrado de WPA)
- Monitoreo y Control - Control Parental Nueva
- Informes de red - sitios Web visitados, la aplicación de software utilizada, y añadió dispositivos o eliminado
- Administrar y solucionar problemas
- La prueba de velocidad de Internet
- Salud y Alertas de Seguridad - proactivamente supervisa la actividad de red y la salud
- Reparación de conexión a Internet
- Crear una red sin fisuras a través de Windows, XP, Vista y Mac

Requisitos del sistema:
* Sistema operativo: Microsoft Windows Vista (todas las ediciones, incluidas las ediciones 64-bit), Windows XP (todas las versiones con SP1 o posterior, incluyendo edición de 64 bits), Windows 2000 Pro (SP3 o posterior)
* Procesador: Intel Pentium III o equivalente
* Memoria: 256 megabytes (MB) de RAM para Windows 2000 y XP; 512 megabytes (MB) de RAM para Windows Vista
* Adaptador de red: por cable (Ethernet) o inalámbrica, con conexión a Internet (router, gateway)
* Espacio en disco: 100 megabytes (MB)
* Navegador Web: Microsoft Internet Explorer versión 5.5 o posterior
