 Todo sobre EPub
Es un nuevo estandar de libros digitales que va a co-existir por un tiempo con PDF.

Ventajas

  Su contenido adapta automaticamente al tamaño de la pantalla, o mejor decir que ePub es un formato re-fluente.

 El PDF se ve bien solo en dispositivos a los cuales fue diseñado ? si preparas un PDF pare verlo en pantalla de un monitor de 17 pulgadas no se ve nada bien en la pantalla de un telefono celular, y al reves ? un documento PDF diseñado para iPhone no se vería bien en pantalla de una netbook. El EPub resuelve este problema.

Todos los dispositivos lo aceptan, sean computadoras, móviles, e-readers o tablets.


Programas Para Leer EPub Desde Una Computadora (Independientes del Sistema Operativo)


FBReader


Es un programa versátil para leer archivos Epub. El programa funciona con los siguientes sistemas operativos:

Windows
MacOS X
Linux
BlackBerry
Android


EPUBReader


Si utilizas Firefox como navegador web, esta es la manera más fácil de mostrar archivos EPUB. Después de instalar este plugin, su navegador es capaz de abrir, mostrar y guardar archivos Epub. Esta solución funciona para todos sistemas operativos en los que se puede instalar Firefox (Windows, MacOS X, Linux).

Calibre


Es una herramienta potente para administrar sus libros electronicos. También permite convertir libros electronicos entre muchos formatos. El único requisito para utilizar Calibre es que Python está instalado en su ordenador.



Lectores Epub para Mac OS, iPhone y iPad


BookReader


Es un programa para abrir Ebooks con Mac. La "Light Version" está disponible gratuitamente para su descarga.

Bluefire Reader  


El lector Bluefire es quizás el mejor programa para leer archivos ePub con iPad y iPhone. El programa es gratuito y también es compatible con libros en formato PDF.

iBooks


Producto exclusivo de Apple para el iphone, el ipad o el ipod touch . Permite organizar tus libros en una biblioteca virtual de madera con un efecto muy conseguido. Una vez dentro las posibilidades de edición y visualización son muy completas. Tiene también un efecto de ?pasar página? muy real.



lectores Epub para Android





Aldiko


 Un lector para los usuarios de móviles Android. Des de aquí también puedes organizar y leer tus libros. Ampliar la letra, poner notas y cambiar los formatos del texto y de visualización del libro.



Conversor de EPub y Otros Formatos



Calibre




Creador y Editor de EPub


Sigil


 Sigil es una aplicación libre, gratuita y multiplataforma (Linux, Windows y Mac) para la edición y gestión de libros electrónicos.
