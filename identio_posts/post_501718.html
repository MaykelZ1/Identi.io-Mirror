> > > > INFORMACION < < < <

--Nintendo Entertainment System (conocida también como NES o Nintendo NES) fue una videoconsola de ocho bits perteneciente a la tercera generación en la industria de los videojuegos. Fue lanzada por Nintendo en Norteamérica, Europa y Australia entre 1985 y 1987.

 Nestopia es un emulador de Nintendo Entertainment System (NES) de código abierto, escrito en C++ . Está diseñado para emular el hardware con la mayor precisión posible. Originalmente sólo para Windows pero luego fue adaptado a Mac OS X y Linux.

Según muchos usuarios la emulación es perfecta, con un resultado incluso mejor que el original. Esto se consigue por medio de filtros de imagen y sonido con los que pueden darle más vida a los viejos juegos de Nintendo.

Nestopia dispone de las mismas opciones del resto de emuladores, pero añade otras. Por ejemplo, puede grabar automáticamente, grabar y reproducir vídeos, configurar distintos mandos y dispositivos, jugar en red, introducir trucos.

Nestopia emula la CPU del NES a velocidad exacta, lo que garantiza la plena compatibilidad con el software que se esté utilizando se basan en la frecuencia del procesador. Otras característica incluyen:

Soporte para 201 mappers diferentes.
Emulacion de Famicom Disk System (FDS)
Soporte para Zapper Light Gun.
Soporte de parches IPS .
Onda precisa de emulación de vídeo--

- Emulador de Nintendo (NES) [Nestopia Ultima Version][Español]





> > > > INSTRUCCIONES DE INSTALACIÓN < < < <

1. Primero tenemos que descargar el enlace subido a MEGA o GOOGLE DRIVE.

2. Una ves descargado el archivo .RAR, le darán Click Derecho y descomprimir.

3. Cuando se haya descomprimido los archivos, les quedara un Block de texto y una carpeta llamada "Emulador de Nintendo (NES)".



4. Ahora le daran Doble Click a la carpeta y una ves dentro encontraran una gran variedad de archivos, el que vamos a utilizar es el archivo .exe llamado "nestopia" y simplemente lo ejecutan.



5. Una ves ejecutado, se les abrirá el tablero del emulador de Nintendo (NES) Nestopia.



6. Simplemente Comenzamos con la configuración de este emulador, para eso lo primero nos dirigimos a ->Options ->Vídeo...



7. Se nos abrirá una pestaña nueva, aquí vamos a ser la mayoría de las configuraciones de este emulador, dejaran tal cual como les aparece en la imagen exceptuando en la opción ->Device que colocaran su tarjeta gráfica y en la opción ->Mode, colocaran su resolucion de pantalla.



8. Y lo mas importante en la Opción ->Filter, colocaran el filtro o como se conocen Shader que mas les acomode a su Equipo en mi caso "Hqx" y le darán clik a la opción de "Setting".





9. Continuamos con la configuración, para esto nos vamos a la pestaña ->Options ->Sound...



10. Se nos abrirá una pestaña nueva, y al igual que el caso anterior simplemente dejan tal cual como esta en la imagen.



11. Ahora seguimos configurando, para esto nos vamos a la pestaña ->Options ->Imput...



12. Se nos abrirá una pestaña nueva, aquí podrán configurar el teclado o el joystick que tengan para emular los botones del control de la Consola Nintendo (NES).



13. Seguimos configurando, para esto nos vamos a la pestaña ->View ->Show, y seleccionan los 3 parámetros marcados.



14. Ya casi terminando de configurar, nos vamos a la pestaña ->Machine ->Región, y selecciona el parámetro Auto.



15. Por ultimo vamos a ejecutar un juego para ver si el emulador quedo bien configurado y sin problemas, para esto se dirigen a la pestaña ->File ->Open...



16. Se nos abrirá una nueva pestaña y simplemente buscan la carpeta donde tienen sus juegos de Nintendo (NES).
