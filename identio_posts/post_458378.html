(Usuarios de Windows XP y Vista) Fin de Firefox

  • Categoría: Off-Topic
  • Publicado hace más de 8 años
Lamentablemente hay una mala noticia.

La versión 52 de Firefox es la última actualización completa para Windows XP y Windows Vista. Se lanzarán actualizaciones de seguridad, pero no incluirán nuevas funciones.

¿Por qué Firefox paraliza el soporte a los usuarios de Windows XP y Vista?

Firefox es uno de los únicos navegadores que ofrecen soporte para Windows XP y Vista.
El propio Microsoft suprimió su soporte a Windows XP en 2014 y hará lo mismo con Windows Vista
en 2017. Los sistemas operativos incompatibles pueden ser peligrosos, ya que no reciben
actualizaciones de seguridad y contienen exploits ya conocidos, lo que dificulta el mantenimiento
de Firefox en esas versiones.

Las actualizaciones de seguridad de Firefox para los usuarios de Windows XP y Vista continuarán
hasta septiembre de 2017, pero no incluirán nuevas funciones. A medidados de 2017, se
anunciará la fecha definitiva de fin, basada en el número de usuarios que sigan utilizando
Windows XP y Vista.

¿El cambio a un navegador diferente me seguirá manteniendo protegido?

Lamentamos decir que no. La mayoría de los navegadores (como Google Chrome y Microsoft
Internet Explorer) ya han finalizado la compatibilidad con Windows XP y Windows Vista.

Si no comtemplas la actualización de tu sistema operativo, puedes continuar con las
actualizaciones de seguridad de Firefox hasta septiembre de 2017.

¿Puedo aún navegar con seguridad en Firefox?

Si en marzo de 2017 estás utilizando Firefox con Windows XP o Windows Vista, se actualizará
automáticamente a la versión de Firefox Extended Support Release. Seguirás recibiendo
importantes actualizaciones de seguridad de Firefox, pero no recibirás otras funciones o
actualizaciones. No necesitarás hacer nada más.

¿Cómo puedo obtener las funciones más recientes de Firefox?

Si quieres mantener tu Firefox actualizado con todas las últimas funciones y actualizaciones,
tendrás que actualizar el sistema operativo de tu ordenador.

 - Actualizar a una versión de Windows que Microsoft todavía admita, como Windows 7, 8 o 10.
   Consulta aquí para obtener más información. Las versiones de Windows que no son compatibles
   con Microsoft no son fiables ni seguras de usar, lo que también dificulta el mantenimiento de
   Firefox en esas versiones.
 - (Avanzado): Cambia a un sistema operativo basado en Linux. Consulta los sitios web de
   soporte para la versión de Linux que te interesa.

Soy una empresa usuaria de Firefox, ¿cuándo terminará el soporte para Windows XP y Vista?

Según la planificación, las empresas que utilizan Firefox deben considerar septiembre de 2017
como la fecha de finalización del soporte para Windows XP y Vista.
¿Cuándo se dejará de dar soporte para Firefox Nightly y Aurora en Windows XP y Vista?

El soporte en Windows XP y Vista para los usuarios de Firefox Nightly y Aurora ya ha finalizado.
Deberías pensar en actualizar a la versión más reciente de Firefox, que puedes descargar aquí o
actualizar el sistema operativo a una versión de Windows que siga estando administrada por
Microsoft, como Windows 7, 8 o 10.

Fuente:
