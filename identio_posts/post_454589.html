Formatear y Recuperar Usb Con Diskpart Windows

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 8 años
Diskpart es una “herramienta de gestión de discos”
Si piensas que tu unidad de almacenamiento está fallando en su funcionamiento y piensas que podría ser un problema de software, puedes usar el comando Diskpart de Windows para solucionarlo.

¿Que es Diskpart?


Es una utilidad de línea de comandos que permite administrar discos. Algunos comandos sólo pueden ser accedidos a través de Diskpart y no mediante el “Administrador de discos” de Windows. Esta es la razón de que sea una herramienta de gran alcance para ver ciertos discos que no aparecen en el “Administrador de discos”, convertir un disco básico en un disco dinámico, usar el comando “clean”, borrar toda la información del disco, administrar las particiones (crear, borrar, editar), asignar, modificar y borrar una letra del lector y formatear una partición con muchas posibilidades.

Puede ser una herramienta muy útil en algunos casos donde el “Administrador de discos” no puede realizar determinadas operaciones tan simples como formatear. Existen muchos comandos para administrar tus discos. Aquí mostrare los comandos para poder listar, seleccionar, limpiar, y formatear un disco. Esta herramienta está integrada a la mayoría de los modelos de Windows: XP, Vista, 7, 8, 8.1 y 10.


IMPORTANTE: Realizaremos un formateo lento que borrara toda la información de tu USB.

Comenzamos:

Ejecutar CMD como Administrador (En la barra de búsqueda según tu versión de Windows)
escribir:
CD/ (Enter)
diskpart (Enter)

escribir:
list disk (Enter)

Nos muestra una lista de los discos o unidades instaladas, posteriormente seleccionar el del USB

escribir:

select disk # (Enter)

IMPORTANTE: Reemplazar y verificar que # es el número de nuestro USB a reparar generalmente es 1

aparecerá:

el disco # es ahora el disco seleccionado

escribir:

clean (Enter)

aparecerá:

diskpart ha limpiado el disco satisfactoriamente

escribir:

create partition primary (Enter)
aparecerá:

diskpart ha creado satisfactoriamente la partición especificada

escirbir:

select partition #

IMPORTANTE: Reemplazar y verificar que # es el número de nuestro USB a reparar generalmente es 1

aparecera:

la particion # es ahora la seleccionada

escribir:

active (Enter)

aparecerá:
diskpart marca la particion actual como activa

escribir:

format fs=FAT32 (Enter)
IMPORTANTE: Reemplazar FAT32 por NTFS según desees el tipo de formateo, usualmente utilizaremos NTFS por la gran capacidad de las memorias actuales.

Esperar hasta que aparezca:

100 por ciento completado diskpat formateo el volumen correcto

escribir:

assign (Enter)

aparecerá:

diskpart asigno correctamente una letra de unidad de montaje

escribir:

exit (Enter)

escribir:

exit (Enter)


Con ello saldremos hasta nuestro escritorio de Windows nuevamente y estaremos realizando un formateo más a fondo a nuestra memoria USB con lo cual la mayoría de las veces podremos reutilizarla nuevamente si está fallando en su funcionamiento y piensas que podría ser un problema de software
.
