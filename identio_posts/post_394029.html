  • Categoría: Software / Programas

Duplicate File Remover es una potente herramienta para localizar archivos duplicados en tu ordenador (deduplicación). Se examina el equipo y listas de archivos duplicados en base a byte a byte de comparación, lo que garantiza el 100% de precisión. Puede encontrar duplicados de cualquier archivo: texto, binario, música, vídeo o imágenes. Para los archivos multimedia (MP3, OGG / Vorbis, WMA), el contenido de las siguientes etiquetas también pueden ser analizados: "Artista", "Álbum", "Título" y "Comentario". La propiedad Resumen de archivos también se puede buscar: título, tema, categoría, y el autor. Los archivos duplicados se pueden eliminar, vinculado duro, mover o copiar. La lista de archivos duplicados puede ser ordenada, exportada a un informe HTML, y actuar en consecuencia.
