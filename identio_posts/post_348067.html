  • Categoría: Software / Programas
  • Publicado hace más de 10 años


Bienvenidos a mi post.

Hoy les traigo este programa que sirve para crear hasta cuatro unidades virtuales de reproducción de CD o DVD. Podrás utilizar el contenido de estos elementos desde el disco rígido, sin que puedan impedírtelo los sistemas de seguridad.

DAEMON Tools Lite es un programa capaz de crear hasta cuatro unidades virtuales de reproducción de CD o DVD. Podrás utilizar el contenido de estos elementos desde el disco rígido, sin que puedan impedírtelo los sistemas de seguridad. Podrás leer y manipular el contenido de CD y DVD con sistemas anticopia. Soporta los siguientes formatos de imagen: BWT, CDI, B5T, CCD, CUE, ISO, MDS, NRG y PDI . 
Con el uso de este programa ya no desperdiciarás CD o DVD. 

Peso: 12.3 MB
Servidor: MEGA
Sistema: Windows XP
                  Windows Vista
