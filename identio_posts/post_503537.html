> > > > INFORMACION < < < <

-- DeSmuME es un emulador de Nintendo DS para Microsoft Windows, Mac OS, Linux y Amiga OS, aunque también se instala en la Wii. Soporta la mayoría de los ROMs de Nintendo DS, tanto homebrew como comerciales.

Es, junto a No$GBA, uno de los emuladores de NDS más usados y valorados. Actualmente se a implementado Loader ds icon + Bios + Firmware and Cheat´s; en fin esperamos que estas versiones con estos complementos sean de tu agrado te saluda Inmortalgames.

La consola tiene su propio sistema de arranque firmware. Al encender el equipo, una advertencia de salud y seguridad se muestra en primer lugar, al ser tocado el aviso, se carga el menú principal, similar a la consola Wii.

El menú principal presenta al jugador siete opciones: jugar a un juego de DS, el uso del PictoChat, iniciar la descarga DS, jugar un juego de Game Boy Advance, cambiar el brillo de la pantalla, activar la alarma y acceder a la configuración de la consola.--

> > > > INSTRUCCIONES DE CONFIGURACIÓN < < < <

1. Primero tenemos que descargar el enlace subido a MEGA o GOOGLE DRIVE.

2. Una ves descargado el archivo .RAR, le darán Click Derecho y descomprimir.

3. Cuando se haya descomprimido los archivos, les quedara un Block de texto y una carpeta llamada "Emulador de Nintendo DS (NDS)".



4. Ahora le darán Doble Click a la carpeta y una ves dentro encontraran una gran variedad de archivos, el que vamos a utilizar es el archivo .exe llamado "DeSmuME-x64" o "DeSmuME" si tienen sistema de 32bis y simplemente lo ejecutan.



5. Una ves ejecutado, se les abrirá el tablero del emulador de Nintendo DS (NDS).



6. Simplemente Comenzamos con la configuración de este emulador, para eso lo primero nos dirigimos a la pestaña ->Config ->Emulation Settings.



7. Se nos abrirá una pestaña nueva, si es su primera ves configurando este emulador dejaran tal cual esta en la imagen, si no es así modifiquen como mejor les de resultados.



8. Seguimos con la configuración, para esto nos dirigimos a la pestaña ->Config ->3D Settings.



9. Como siempre se nos abrirá una pestaña nueva y dejaran como siempre tal cual esta en la imagen.



10. Continuamos con la configuración, para esto nos dirigimos a la pestaña ->Config ->Sound Settings.



11. De nuevo se nos abrirá una nueva pestaña y de nuevo dejaran tal cual esta en la imagen.



12. Seguimos configurando, para esto nos dirigimos a la pestaña ->Config ->Firmware Settings.



13. Como es de costumbre se nos abrirá una nueva pestaña, colocaremos los datos de Usuario y el idioma.



14. Seguimos con la configuración, para esto nos dirigimos a la pestaña ->Config ->Control Config.



15. Se nos abrirá una nueva pestaña, aquí simplemente podrán configurar su joystick o teclado, para emular los botones de la consola Portátil Nintendo DS.



16. Continuamos con la configuración, para esto nos dirigimos a la pestaña ->View ->Magnification Filter, varios parámetros para marcar y seleccionaran el Filter que mas se adapte a su equipo, en mi caso seleccione la opción "5xBRZ".



17. Ya para terminar la configuración del emulador, nos dirigimos a la pestaña ->View ->LCDs Layout, en este punto marcaran la opción en que se mostrara la pantalla con los juegos, en mi caso seleccione la opción "Horizontal".



18. Punto Opcional: si quieren que el emulador les muestre los FPS cuando ejecutan un juegos, se dirigirán a la pestaña ->View ->HUD, seleccionan la opcion "Display FPS".



19. Por ultimo vamos a ejecutar un juego para ver si el emulador quedo bien configurado y sin problemas, para esto nos dirigimos a la pestaña ->File ->Open ROM...



20. Como es de costumbre se nos abrirá una nueva pestaña y buscaran la ruta en donde tienen la carpeta de sus juegos de Nintendo DS (NDS).
