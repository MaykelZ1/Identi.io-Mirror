  • Categoría: Reciclaje

Bienvenidos a mi post

El dia de hoy les traigo el sistema operativo Yosemite para VMware


Imagenes






Informacion importante1. Una vez instalado VMware instalar el unlocker
2. Copiar la carpeta del sistema operativo a la carpeta de maquinas virtuales
3. Esperar el primer inicio y configurar a tu gusto el sistema
4. El sistema va un poco lento y no puedes cambiar la resolucion para eso pasa a la 2a parte del tutorial

Paso ImportantisimoUna vez iniciado el sistema por primera vez apagar la maquina virtual y seguir el paso del tutorial 2 incluido en la descarga
 Seguir paso a paso el tutorial del video ya puedes cambiar la resolucion de la pantalla y se nota la mejora de fluided del sistema
