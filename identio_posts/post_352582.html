Cambia fácil la extención de uno o varios archivos

  • Categoría: Software / Programas
  • Publicado hace más de 10 años


File Extension Changer hace que sea muy fácil cambiar la extensión (s) de uno o varios archivos. Se integra en el menú contextual de los archivos y carpetas sin el uso de archivos DLL proporcionando así la facilidad de uso sin cargar el sistema cuando no esté en uso. Soporta Drag n función Drop ayudar a cambiar las extensiones de varios archivos simultáneamente. También cuenta con una versión completamente portátil del programa de apoyo a la integración contexto menú.


 características:

     Al integrar en el menú contextual del archivo, el Extension Changer hace que sea muy fácil cambiar la extensión de un archivo. * Mediante la integración en el menú contextual de la carpeta, las extensiones de varios archivos se puede cambiar de una vez.
    
En el modo normal también, los archivos y las carpetas se pueden agregar mediante arrastrar y soltar y el otorgamiento de diversos archivos, incluso de diferentes carpetas se pueden cambiar en cuestión de segundos.

Características del Modo avanzado incluyen cambio condicional de extensiones en el que el usuario puede proporcionar algunas condiciones sobre la base de que las extensiones se cambian. También esta lista se puede guardar como un archivo para su uso posterior.
     Nueva ruta de los archivos se puede proporcionar para mover todos los archivos con otro nombre a la nueva carpeta.
     Archivo por lotes puede ser creado para deshacer los cambios realizados. Por otra parte, lote o archivo REDO se pueden crear para el proceso de cambio de nombre por el que todo el cambio de nombre puede ser hecho de nuevo.
     Tener registro de lo que se hace con los archivos de registro que se pueden crear para cada sesión de registro de las actividades de cambio de nombre.
     Entradas del menú contextual se pueden agregar o quitar fácilmente usando Registry Manager incluido con File Extension Changer.
     Versión portátil también disponibles. No quiero otro programa en el menú de inicio o simplemente quiere poner File Extension Changer en tu memoria USB. Ahora usted puede. Integración Shell está todavía alcanzable mediante el Administrador del Registro.


 NOTA - Si experimenta algún problema al utilizar File Extension Changer, por favor consulte las siguientes notas que espero resolver muchos de los problemas comunes relacionados con el programa. No dude en enviarme un correo electrónico si usted está en duda.

     Para los usuarios de Windows Vista, File Extension Changer no funcionará, ya que requiere el archivo "comdlg32.ocx". Pero usted puede hacer que funcione mediante la descarga y el registro de la misma. He aquí un buen artículo sobre cómo hacerlo. Después de hacer esto, el programa va a funcionar tan bien como en Windows XP.
     Recuerde que cambiar la extensión de un archivo no va a cambiar o convertir su contenido de acuerdo con la extensión. Por ejemplo, cambiar la extensión de un archivo "WAV" a archivo "MP3" sólo cambiar el nombre del archivo, pero el contenido del archivo se encuentra todavía en el formato WAV. Del mismo modo cambiar la extensión de un archivo 'BMP' a 'JPG' no va a cambiar el formato del archivo. Puede parecer a trabajar, pero en realidad no lo es. Así que, si usted necesita esa funcionalidad, descargar un conversor de audio / vídeo / imagen correspondiente. Eso te ayudará a convertir el formato de un archivo y, obviamente, cambia su extensión.
     La estructura de directorios del archivo zip se debe mantener el correcto funcionamiento del programa. De lo contrario es posible encontrar ciertos problemas durante el cambio de nombre en relación con BackUp, inicie sesión y Rehacer creación del archivo.
     El usuario debe tener en cuenta que File Extension Changer no utilizan ningún tipo de DLL (Dynamic Link Library) archivos, incluso para la integración shell. En su lugar utiliza registro para eso. Ningún componente de File Extension Changer está presente en el tiempo de ejecución cuando no se está ejecutando no se carga el sistema cuando no se está ejecutando.
     Entradas del menú contextual pueden ser añadidos o eliminados incluso después de la instalación desde Inicio-> Programas-> Extensión de archivo Cambiador> Configuración del menú contextual o en versión portátil mediante el Administrador de registro incluida con File Extension Changer.
     El paso anterior requiere que el usuario tenga privilegios administrativos. Si no es así, inicie sesión en una cuenta que tenga privilegios de administrador o póngase en contacto con su administrador.
     Copia de seguridad, rehacer y archivos de registro se guardan en el archivo de la carpeta de Extension Changer para el modo DragNDrop y modo de directorio. Para el modo Shell, el archivo de copia de seguridad se guarda en la carpeta en la que está presente el archivo cuya extensión se cambia.
     Copia de seguridad, rehacer y archivo de registro (para el modo DragNDrop y modo de directorio) sólo se puede crear cuando File Extension Changer está presente en un medio que no esté protegido contra escritura. De lo contrario, el programa mostrará un error. Mientras se ejecuta el programa desde un medio de sólo lectura, deshabilitar la copia de seguridad, vuelva a realizar y registrar la creación del archivo por archivo Extension Changer para que funcione correctamente.
     Es muy importante que los archivos de copia de seguridad se ejecutan en orden inverso (es decir, el archivo de copia de seguridad más reciente se debe ejecutar primero y luego su anterior y así sucesivamente) mirando a su fecha y hora del sello en su nombre de archivo. Archivo de registro también se puede referir a la misma para File Extension Changer v3.2.1. De no hacerlo, no dará lugar a los resultados esperados. También cambiar de nombre o la extensión manualmente, con algún otro software o con File Extension Changer (sin modo de copia de seguridad) después de que el archivo de copia de seguridad se creó para un cambio anterior para el mismo archivo hará que el archivo indetectable desde el mismo proceso de copia de seguridad.
     NO seleccione 'Extensión Change' desde el menú contextual cuando se selecciona más de un archivo, ya que abrirá muchos casos de extensión de archivo del cambiador. En cambio, para múltiples archivos, ejecutar el programa desde el grupo 'Programas' en 'Menú Inicio' y luego añadir los archivos que desee ser renombrados en el cuadro de lista desplegable.
     Proveer de extensión inadecuado en un archivo puede quedar inservible. En ese caso, cambie la extensión del archivo a la original, ya sea utilizando el archivo de copia de seguridad «lote» (si es creado) o manualmente.
     Si la ruta del archivo se cambia mediante la opción 'Cambiar ruta', el archivo renombrado se moverá a la carpeta.
     Si se marca 'Cambiar ruta', el archivo por lotes de copia de seguridad no podrá deshacer el camino. Sin embargo, el nombre de archivo se puede revertir de nuevo a la original.
     Archivo por lotes REDO no funcionará si se selecciona la opción 'Cambiar ruta' durante su período de sesiones.
     El uso de archivos Extension Changer desde el menú contextual de un archivo para los archivos de los LNK 'de extensión, (es decir, archivos de vínculos o accesos directos) en el menú de shell dirigirá hacia el archivo de destino real y no el acceso directo en sí.
     Usando Archivo Extension Changer desde el menú contextual de una carpeta mostrará los archivos presentes en la carpeta. Los archivos presentes en las subcarpetas no se muestran ni afectados.
     En el modo de arrastrar y soltar, el usuario debe agregar sólo archivo (s). Los archivos pueden estar presentes en diferentes carpetas. Pero la adición de carpeta (s) no hará nada. Si desea cambiar el nombre de los contenidos de una carpeta, utilice 'Agregar directorio "o haga clic derecho en una carpeta y seleccione' Extensión Cambio (s) '.
     La integración en el menú contextual de los archivos y carpetas es opcional y se puede cambiar volviendo a instalar la aplicación. Recuerde que debe desinstalar la aplicación por primera vez.
     Nuevo nombre + extensión no debe ser igual a cualquier otro archivo ya está presente en la carpeta de destino o no debe convertirse mismo después del cambio. (Por ejemplo - si las extensiones de los siguientes dos archivos ABC.INI y ABC.INF presente en la misma carpeta (o va a ser desplazado a misma carpeta de destino después de renombrar) se deben cambiar a TXT, a continuación, nuevo nombre de archivo + extensión para ambos archivos se convertirán mismo).
     Update Checker se retira de la extensión de archivo Versión cambiador 3.3.1 ya que muchos programas anti-virus están dando un falso positivo. Si está utilizando una versión anterior del archivo Extensión Changer, te puedo asegurar que no contiene ningún tipo de virus / spyware / malware etc. Simplemente descarga un archivo de texto en su carpeta temporal para comprobar la última versión del programa. Si usted todavía está en duda, simplemente borrarla. No afectará File Extension Changer de ninguna manera.
Instrucciones
1 - Selecionamos el (los) archivos y damos click derecho y en el menú contextual selecionamos "Change Extension".

2 - En File Extension Changer (Original File Name) podremos ver el nombre y la extención original del archivo.
En la opción New File Name podremos cambiar tanto el nombre como la extención del archivo (como pueden ver lo he cambiado de txt a rar), ahora solo damos click en Change and Exit.





 SO: Windows 98 / NT / 2000 / ME / XP. Funciona en Windows Vista / Windows 7 por un pequeño tweak (Ver notas más arriba para más detalles)/Windows 8 funciona solo desde el menú contextual.
