Microsoft presenta el nuevo Windows 10

  • Categoría: Reciclaje
  • Publicado hace más de 11 años
Finalmente y contra todo pronóstico, la nueva versión del sistema operativo más famoso de Microsoft se llamará Windows 10 y unificará móviles, tablets y PC.



Ni Windows TH, ni Windows 9, el nombre de la nueva versión del sistema operativo de Microsoft será Windows 10, saltándose así un número desde su predecesor Windows 8.


Esta versión será la encargada de traer la unificación entre dispositivos móviles, tablets, portátiles y ordenadores de sobremesa bajo un único sistema y ecosistema con una fuerte integración entre todos los dispositivos.





También traerá de vuelta a la vida el, demandado por muchos, menú de inicio "clásico", aunque adaptado a los tiempos que corren con un toque de Modern UI, igualmente, las aplicaciones ModernUI podrán manejarse como ventanas convencionales dentro del escritorio, pudiendo ejecutarlas en modo ventana o de manera completa.


Igualmente, la interfaz se adaptará según el tipo de dispositivo que estemos usando, teniendo el escritorio convencional en un PC de sobremesa, y ModernUI en un tablet por ejemplo. De esta manera, la Charm Bar lateral de Windows 8, solo se mostrará a los usuarios de pantallas táctiles, e incluso el menú de inicio cambiará dependiendo de si tenemos pantalla táctil o no, de tal manera que si conectamos un teclado y ratón, el sistema cambiará directamente a la interfaz de sobremesa.






Además, la compañía lanzará mañana mismo su programa de previews, por lo que cualquiera podrá instalar las últimas mejoras del sistema operativo antes de que lleguen en versión final, algo muy parecido a lo que ya hace Microsoft con su programa de Windows Phone Preview for Developers.


Windows 10 llegará al mercado a mediados de 2015,, tras la conferencia BUILD de Microsoft y se lanzará para móviles, PCs y tablets, sustituyendo también a Windows Phone 8.1 y utilizando un único núcleo para todos los dispositivos.
