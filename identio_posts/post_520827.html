Windows MiniOS XP 2018 + Server

  • Categoría: Sistemas Operativos
  • Publicado hace más de 6 años


La apariencia consta de dos temas de colores planos, uno “Blue” por defecto y una versión “Dark” u oscura que puedes seleccionar alternativamente, estos temas son un fork echo por mí. Los iconos serán iguales a los de Windows10. También incluí un Pack de Extras que el usuario puede usar opcionalmente, este se puede encontrar en Inicio, Todos los Programas, MiniOS_Extras, ahí verán diferentes aplicaciones que pueden usar, como un Dock o Barra, Tema Aero, Notas Rápidas, WinFlip, Miniaturas en la barra de Tareas, entre otras más. No incluyo más temas ni wallpapers para no crecer el ISO más de lo necesario.


CAMBIOS

Se eliminaron componentes, servicios y aplicaciones, por estar obsoletos o por ser innecesarios para la mayoría de los usuarios, ayudando a si a mejorar el uso de espacio en disco, la velocidad, el consumo de recursos del sistema, se optimizaron los servicios, entre otros cambios. Aparte de lo que fue eliminado siempre busqué que este sistema mantuviera la misma compatibilidad de hardware que el XP SP3 original.

NOTA: Para instalar desde USB recomiendo el Rufus que se encuentra en el ISO, aunque se pueden usar otros softwares para ello, de tener algún problema entonces hacerlo del modo clásico, quemar un CD, XP es obsoleto, algunos BIOS no permiten su booteo, esto no es un problema del MiniOS, sino del XP en general.
———————————————————————–
NOTA: Si se va a instalar en una máquina virtual recomiendo usar VirtualBox, VMWare por defecto no usa las configuraciones aplicadas en ningún Windows Desatendido si se configura al inicio. De usar VMWare no poner el ISO cuando creamos la máquina, lo hacemos después de creada para que instale correctamente.
                                   
Este Windows no es solo para equipos de pocos recursos, puedes usarlo en un buen ordenador también, es cierto que es pequeño, tiene la misma compatibilidad de hardware que el XP original.
Quiero hacerles conocer que el MiniOS es usado en entornos empresariales también, no hay problema alguno en ello, es perfectamente compatible con todo el hardware que XP SP3 original, así que si quieres usarlo en un entorno más allá de lo personal lo puedes hacer sin riesgo alguno.
