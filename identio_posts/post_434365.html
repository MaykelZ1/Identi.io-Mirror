(PsVita) Henkaku Psvita CFW (Tutorial)




¿Que es Henkaku?

Henkaku es el primer CFW nativo de Psvita, para cargar copias de seguridad, Homebrew y emuladores, claro que existian otros tipos de CFW como el Rejuvenate el cual trabaja de manera nativa al igual que henkaku pero era muy complicado tanto de instalar como de usar, o el T-N TotalNoob el cual trabaja mediante exploits del emulador de PSP pudiendo cargar tanto isos de PSP como toda la Scene de PSP con una gran compatibilidad. El henkaku es facil de instalar, facil de usar y con un gran futuro sin embargo tiene sus desventajas que mencionaremos mas adelante. asi pues continuemos.

Requisitos

    Psvita Fat/Slim ambos modelos son compatibles
    Firmware 3.60
    (Aclarando este apartado, el henkaku solo funciona en este Firmware si tienes uno inferior o superior el exploit no se inciara, si el firmware que tiene tu PSVita es inferior debes actualizar de forma segura a 3.60, si es superior nada que hacer, tener paciencia mientras hallan un nuevo exploit).
    Acceso a internet tanto en la Vita como en la PC.

NOTA: Las instrucciones en las imagenes

ACLARACIONES Y DUDAS

DESVENTAJAS

    Necesitas una conexion a internet para el inicio del henkaku, es decir, si quieres llevar tu consola contigo debes iniciar henkaku y mantener la consola suspendida, no apagada, la suspension mantiene el CFW y tus aplicaciones funcionaran pero al apagarla las Aplicaciones instaladas no funcionaran.
    El CFW aun esta en desarrollo y se sabe que tendra una gran acogida, pero si tienes otro CFW como el T-N Total Noob piensa bien si instalas o no el Henkaku puesto que este aun no tiene compatibilidad con la carga de Isos de PSP, por lo tanto si pierdes el T-N no tendras acceso al homebrew de PSP.
    El espacio en la memoria, como es sabido, el homebrew para henkaku pesa un poco mas que el de PSP al ser nativo tiene mejor desempeño pero los emuladores ocupan mas espacio y pues ya sabemos lo costosas que son las memorias para PsVita, ojala en algun futuro se fabriquen Adaptadores para SDs

FAQ.

    ¿Con Henkaku puedo jugar Backups de PsVita?

Si es posible, sin embargo el proceso de instalacion de los backups es un poco complicado, pronto habra un tutorial para la instalacion de los backups.

    ¿Henkaku es un CFW permanente?

Desafortunadamente no, al reiniciar la consola el CFW se pierde, para activarlo de nuevo basta con abrir el explorador de internet y visitar Henkaku.xyz dar install y listo podremos usar de nuevo las aplicaciones o juegos que hayamos instalado.

    ¿Riesgo de Baneo?

Como en todo uso de software no licenciado existe el riesgo de baneo, sin embargo no ha aparecido el primer caso, se recomienda que al usar el CFW tengas la consola sin conexion a internet y no sincronizes trofeos, esto para prevenir.
