Light Image Resizer v5.1.3.0








Light Image Resizer


Pruebe el software de conversión de imagen y el convertidor de imagen ObviousIdea. Fácil pero potente, cambia el tamaño y simplemente convierte imágenes, una a una o en modo por lotes. La edición masiva se puede usar para aplicar perfiles a un conjunto de imágenes

El programa está disponible como una extensión de shell de Windows. Simplemente haga clic con el botón derecho para cambiar el tamaño de las imágenes fácilmente como copias o continuar trabajando en las fotos originales. Luego, comparta sus fotos redimensionadas en línea con amigos.

Cree imágenes amigables para el correo electrónico o úselos para publicar en Internet

Cambiar el tamaño y convertir

Use el Light Image Resizer para cambiar el tamaño de las imágenes. El convertidor de imágenes por lotes puede convertir fácilmente sus imágenes en diferentes formatos. Seleccione su resolución de salida, cambie el tamaño del original o cree una copia, mueva y / o cambie el nombre de los archivos o comprima, elija un destino específico para sus imágenes procesadas. Puede trabajar en fotos individuales o editar grandes cantidades de imágenes simplemente haciendo un cambio de tamaño de lote con solo un clic.

Configuración extra

Agregue su propia marca de agua para proteger su trabajo o agregue un copyright a las fotos con formato de texto html y soporte de transparencia. Personalice las imágenes convirtiéndolas en sepia o agregue un borde. Use la vista previa en vivo para ver cómo se verá el resultado final antes de procesar la imagen. El Resizer Photo Resizer / Picture Resizer de ObviousIdea funciona en archivos, subcarpetas y carpetas, y es compatible con formatos RAW. Con soporte multi-core integrado, puede convertir imágenes aún más rápido. Inicie la aplicación fácilmente desde el explorador de shell de Windows con un clic derecho en la imagen o carpeta.

Salida y exportación

Las fotos se pueden agregar directamente como archivos adjuntos, individualmente o en un archivo ZIP, para enviarlas por correo electrónico utilizando la función de publicación, sin exceder el tamaño máximo permitido por los servidores de correo electrónico. Cree archivos PDF a partir de sus imágenes, como jpg a PDF; haz esto para imágenes individuales o múltiples. Edite, cree, exporte, realice una copia de seguridad de nuevos perfiles con su configuración favorita. Los perfiles más populares ya están predefinidos, como imágenes de fondo de resolución FULL HD, Ipod, NTSC, 640 x 480. Acceso a los perfiles en 1 clic con la integración de shell.

Marco digital

Transfiera fotos y optimice el espacio en marcos digitales con el Asistente de Marco de Imagen Digital. Puede usar las opciones de cambio de tamaño de la imagen para cambiar el tamaño de la resolución compatible con el marco y agregar aún más fotos para su visualización.

REQUISITOS DEL SISTEMA

Mínimo: Intel Pentium III / AMD Athlon XP, 512 MB de RAM (Windows XP)
Recomendado: Núcleo Intel 2 Duo o equivalente, 2 GB de RAM (Windows 7 o por encima)
Mejor Presentación: Core i5 / i7 o equivalente, 4 GB de RAM (Windows 7 o por encima)

Sistemas operativos compatibles

Nuestro convertidor de imagen y la imagen Resizer es compatible con Windows XP (SP3), Windows Vista, Windows 7, Windows 8, Windows 10
