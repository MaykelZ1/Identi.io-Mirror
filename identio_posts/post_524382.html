Hoy quiero tocar un tema típico del diseño web, los menús.

Crear un menú básico es sencillo, ¿Qué es un menú básico? Bueno, por ejemplo el de Identi cuando tu cuenta no esta abierta. Ahora vamos a ver como se hace el menú de Identi como práctica.

CSS 1: Lo primero que tendríamos que hacer, es un reseteo de los estilos por defecto del navegador. Si no entienden muy bien que esto, pueden dejar en los comentarios y hago otro post exclusivamente sobre eso:



Lo que hace la imagen de arriba, es básicamente quitarle todos los márgenes y paddings que tengan todos los elementos, y quitarle el subrayado a los enlaces.

HTML 1: Vamos a hacer lo primero de todo, agregar las clases necesarias para crear el menú. Acá tengo que dar una critica, y es que el menú de Identi, o en realidad toda la web de Identi, esta muy mal diseñada para lo que es la web hoy en día, así que no voy a usar el HTML original de la web porque no tendría sentido, si no que uso el mío.

Vamos a agregar las siguientes clases:

- .main-menu: Clase utilizada para el contenedor principal del menú
- .main-menu-items: Clase utilizada para el contenedor principal de la lista
- .main-menu__item: Clase utilizada para cada ítem del menú



CSS 2: Empecemos con el menú.

1- Agregar fondo al menú: Vamos a agregar un fondo negro a todo el menú





2- Alinear todo el contenido: Vamos a hacer que todo el menú solo mida el 90% de todo el ancho de la pantalla, y vamos a alinearlo al centro del menú. Para este menú, vamos a usar flexbox. Puedes buscar en Google para saber que es, no usen float: left; Eso ya pasó, usen Flexbox y CSS Grid.





3- Vemos que cada ítem del menú se ve uno mas horrible que el otro, vamos a arreglar esto cambiando el color, separándolos un poco arriba y abajo y cambiando la fuente.





4- Vamos a agregar los efectos del menú, que básicamente es uno solo. Tenemos que hacer que el menú al pasar el mouse por cada ítem, este cambie de color y agregar el subrayado del enlace.





4- Si se fijan en el menú de Identi cuando no tienen su cuenta abierta, hay un botón para acceder a su cuenta, pero este esta alineado diferente. Para hacer eso, vamos a crear una clase para mover ese ítem.





5- El menú básicamente ya esta terminado, así que vamos a tocar un poco el HTML para que el menú sea el de Identi, ya que también hay que agregar la clase align-right al ítem para que se mueva.
