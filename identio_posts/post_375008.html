Cómo mostrar el icono de actualización de Windows

  • Categoría: Sistemas Operativos
  • Publicado hace más de 10 años





Cómo mostrar el icono de actualización de Windows 10, si no está visible.



Microsoft está promocionando actualmente la actualización hacia Windows 10 en Windows 7 o Windows 8.1 anunciándolo como una actualización gratuita que pronto será liberado. Windows 10, el cual será lanzado para PC y dispositivos tablet en menos de dos meses, se ofrece como una actualización gratuita para la mayoría de los sistemas Windows.






La notificación de actualización sólo se muestra en el área de la bandeja del sistema del sistema operativo si algunas actualizaciones ya se instalaron en el ordenador. Mientras este es el único requisito, la funcionalidad parece estar rota en algunos sistemas no mostrando el icono en ellos, incluso si cumplen todos los requisitos previos.












Puesto que Windows 10 se proporciona como una actualización gratuita, sólo hay una razón para reservar la actualización usando la notificación "Obtenga Windows 10": la pre-carga de los archivos de instalación. 





Si desea actualizar a Windows 10 el día en que sea liberado por Microsoft, se beneficiará de los archivos de instalación en la pre-carga ya que los servidores pueden sobrecargarse el día anunciado debido a que los usuarios intentaran actualizarse al nuevo sistema operativo. 





Reparar el problema del mensaje de actualización a Windows 10.







Lo primero que pueden querer hacer es asegurarse de que todos los cambios necesarios se hallan instalado en el sistema. Una vez que haya hecho eso, es posible que desee ejecutar un método tras otro para ver si el problema se soluciona.




Mientras que usted puede hacerlo de forma manual, es posible que pueda ejecutar un pequeño script por lotes para Windows, que lo hará de forma semiautomática.



El script esta disponible y todo lo que tiene que hacer es descargarlo a su sistema, extraer el archivo y ejecutar el archivo win10fix_full.bat con derechos de administrador. Esto lo hará haciendo un clic derecho sobre él y seleccionando la opción "Ejecutar como administrador".




Nota: Los usuarios cautelosos pueden querer comprobar el código en un editor de texto antes de ejecutar el script. Allí encontrará la lista de los comandos que utiliza el archivo por lotes que determina si todas las actualizaciones requeridas están instaladas y los comandos que se utilizaran para intentar solucionar el problema del icono. 




Lo primero que debe hacer es el paso (1), que es ejecutar una comprobación del estado de actualización. Esto comprobará si todos los cambios necesarios se han instalado en el sistema.




Si ese es el caso, entonces debe realizar el paso (2), (3) y después el (4), una tras otro para mostrar el icono de actualización en el área de la bandeja del sistema.
