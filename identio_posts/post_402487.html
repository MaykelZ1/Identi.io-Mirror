DnsConfig - Configura tus servidores dns - Mega

  • Categoría: Reciclaje
  • Publicado hace más de 9 años







DNS/CONFIG es una pequeña herramienta que permite configurar rápidamente las direcciones TCP/IPv4 y TCP/IPv6 de los servidores DNS en nuestra PC, estas direcciones son por lo general asignadas por el proveedor de servicio de Internet (ISP) y permiten la comunicación de nuestro equipo a un servidor que contiene una enorme base de datos con los nombres de dominio y su respectiva dirección IP haciendo posible la navegación en Internet como la conocemos y el acceso a las distintas paginas web. DNS/CONFIG es una herramienta que sirve para modificar dichas direcciones por otras preestablecidas en el programa o introducir de manera manual una dirección de preferencia personal.

¿Qué es un servidor DNS?

Básicamente los servidores DNS son equipos encargados de guardar en una base de datos la relación que existe entre el nombre de un dominio (Pagina Web) o servicio (Correo electrónico) y sus direcciones IP.

A nosotros los seres humanos se nos da mas fácil recordar nombres que números pero las computadoras, servidores y equipos conectados a la red se comunican entre si mediante una cadena numérica asignada conocida como IP, cuando escribimos el nombre de un dominio en el navegador, nuestra PC envía ese nombre a un servidor DNS y este le devuelve la dirección IP asociada a ese dominio haciendo posible la comunicación entre nuestro equipo y el servidor o host remoto para adquirir la pagina web principal o el servicio solicitado y mostrarlo en pantalla.

Si el servidor DNS no tiene en su base de datos un registro asociado a la solicitud enviada por nuestra PC (Si la pagina que quieres ver no esta registrada en el servidor DNS), puede responder al PC con la dirección de un servidor DNS que pueda tener información acerca de ese host remoto, o bien puede consultar al otro servidor DNS. Este proceso puede repetirse hasta que el equipo cliente (nuestra PC) reciba las direcciones IP o hasta que se establezca que el nombre consultado no pertenece a ningún host (que la pagina web no existe).



Los servidores DNS también realizan su función de manera inversa, si en el navegador escribimos directamente la dirección IP, este devolverá el nombre de dominio asociado a dicha dirección y nuestro navegador nos mostrará la página web asociada a esa IP.

Tenemos que tomar en cuenta que no existe una base de datos única con todas las direcciones IP y que los servidores DNS realizan consultas entre si para verificar quien tienen la información solicitada y estos están estructurados jerárquicamente con un estrato superior compuesto por 13 servidores distribuidos por todo el globo conocidos como «Los Servidores Raíz».

¿Como afecta el rendimiento del servidor DNS a mi PC?

Cuando los servidores DNS están saturados (muchas solicitudes) o estos están Down (Por fallas eléctricas, mantenimiento,etc.) nuestro navegador de Internet se mantiene esperando la respuesta para la conexión y luego de un tiempo de espera nos puede decir que la pagina web que buscamos acceder no esta disponible cuando en realidad se trata de un error en la comunicación con un servidor DNS, pero por lo general, lo que experimentamos es una lentitud al navegar en Internet debido al retraso en la respuesta del servidor.

¿Cuáles pueden ser los beneficios de usar otro servidor DNS?

Velocidad: Un servidor DNS que tarde menos en devolver la dirección IP del dominio o servicio solicitado permite obtener una mayor velocidad de navegación ya que reduce el tiempo de carga de las paginas haciendo nuestra experiencia al navegar la Web mucho mas fluida.

Seguridad: Existen servicios públicos gratuitos que contienen filtros que bloquean sitios en la web que están clasificados como peligrosos porque tienen virus o malware y otros pueden también bloquear sitios que contienen material pornográfico o contenido para adultos, perfectos para los niños, instituciones religiosas o cualquier entidad publica o privada que desea bloquear ese tipo de contenido sin tener que instalar programas adicionales.

Privacidad: Puedes burlar la vigilancia, censura o el control al acceso a servicios y paginas web que según varias fuentes en Internet, se implementa en mas de 40 países. También puedes evitar el registro de datos personales por parte de las empresas que brindan estos servicios y el uso que le das a los mismos usando direcciones como las DNS de Censurfridns.dk (Los servidores DNS guardan los registros de las consultas que realizan los usuarios).
