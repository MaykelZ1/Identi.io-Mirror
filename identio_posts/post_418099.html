CD con MS-DOS y Sistemas Operativos de 16/32 bits

 
 


 

Les traigo un CD con Arranque MS-DOS 6.22, que contiene los instaladores de:
 
1) MS-DOS 6.22,
 
2) DR Dos 7.03
 
3) OpenDOS 7.01
 
4) Windows 3.11
 
5) Windows 95
 
6) Windows 98
 
7) Windows 98SE
 
8) Windows ME
 
 
 
Además, trae Parches y Actualizaciones para Windows 95/ME y Office 97. 

NOTA: El PC arranca con MS-DOS, por lo que quedan en una consola como con el comando CMD. Para los que no saben cómo realizar el Formateo del PC, echen un vistazo AQUÍ. 
 

Espero les sea de utilidad. Saludos y bendiciones
