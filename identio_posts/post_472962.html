Windows 7 MiniOS|32 + 64 + Boot|UB|







MiniOS; Mini de pequeño y OS de Operate System (Sistema Operativo) ya que esta es la idea, hacer un Mini Sistema Operativo que fuera pequeño, veloz, atractivo y que tenga a la vez la misma compatibilidad de hardware que el original. Con el tiempo ha evolucionado, nunca quise forzar a nadie a que use algo que no quiere, por ello no incluyo nada adicional más allá de los extras, cada cual es libre de elegir que softwares usar. Con el tiempo, necesidades, ideas, y cooperación tenemos la actual versión. Ahora tengo esta version sobre Windows7 Ultimate SP1.


Apariencia

La apariencia consta de dos temas de colores planos, uno “Dark” u oscura por defecto y una versión “Blue” que puedes seleccionar alternativamente. Los iconos son iguales a los de Windows10.


Cambios

Se eliminaron componentes, servicios y aplicaciones, por estar obsoletos o por ser innecesarios para la mayoría de los usuarios, ayudando a si a mejorar el uso de espacio en disco, la velocidad, el consumo de recursos del sistema, se optimizaron los servicios, entre otros cambios. Aparte de lo que fue eliminado siempre busqué que este sistema mantuviera la misma compatibilidad de hardware que el Windows7 SP1 original.


NOTA: Para instalar desde USB recomiendo el Rufus, aunque pueden usar cualquier otro software que usaran para hacerlo.


NOTA: Si se va a instalar en una máquina virtual recomiendo usar VirtualBox, de usar VMWare no poner el ISO cuando creamos la máquina, lo hacemos después para que instale correctamente.
