Como usar el Joystick de PS3 en PC (Tutorial)

  • Categoría: Hazlo tu mismo
  • Publicado hace más de 9 años
Hola comunidad Identi! Hoy les vengo a traer un tutorial bastante lindo para que sepan usar sus Joysticks de play 3 en la compu


VALIDO TANTO PARA JOYSTICK DE PS3 ORIGINALES COMO TRUCHOS/REPLICAS - 100% FUNCIONAL


1) Deberan ingresar en el siguiente sitio web: Link (No hay Copyright para este programa) 


2) Descargan el programa Download latest version here. [1.2.0.160] tal y como se muestra en la imagen de abajo. 




3) Descargan la actualización Download latest update here. [1.2.2.175] tal y como se muestra en la imagen de abajo. 




4) Extraen el primer archivo descargar osea latest version here. [1.2.0.160] en donde ustedes quieran. 


5) Luego de realizar el paso anterior extraen la actualización osea latest update here. [1.2.2.175] en el mismo lugar y le dan a reemplazar a todo lo que les pida. 



6) Ahora deberan descargar unos drivers que seran los del mando de Xbox 360 para Windows, desde el siguiente enlace: Link 


7) Una vez descargado dicho archivo proceden a instalarlo. 



8) Ahora conectan por cable USB el Joystick a la Compu. 



9) Una vez hecho lo anterior proceden a ingresar a la carpeta del ScpServer y ejecutar como administrador el ScpDriver.exe. 


10) En el programa abajo hay 3 opciones, deben destildar la opcion de Bluetooth Driver y luego darle a instalar. 



11) Ahora le dan a Install y durante la instalación les aparecera una opcion de si permitimos instalar un dispositivo a lo que le decimos que si (Esto no le puede salir a todos). 


12) Una vez finalizado les debería quedar algo como esto. 


13) Cierran el programa y ahora ejecutan el ScpServer.exe como Administrador. 


14) Ahora una vez ejecutado el programa les deberia aparecer 1 Pad conectado o la cantidad de Joysticks que ustedes conectaron, en mi caso 1 porque tengo conectado 1 solo  


15) Lo tildan a ese PAD y luego aprietan el Botón de PS del Joystick para que se ajuste la configuracion y quede como el mando 1. Veran que ahora al lado aparecera como USB CHARGING eso significa que ya esta listo para usar. 




Espero que les sirva como a mi
