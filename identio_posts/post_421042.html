Navegando por T! me di cuenta que nadie podia poner en pantalla completa el MUGEN. Y cuando preguntaban como se hacia todos contestaban (ALT+ENTER)... Pero para que eso funcione sebemos hacer lo siguiente:

1ero) Entrar en la carpeta "data".
2ndo) Ejecutar el archivo "MUGEN.cfg" con el block de notas.
3ero) Una vez hecho esto buscamos lo siguiente: [Video Win].
4rto) Justo debajo de ese titulo encontramos:
";Enter the width and height of the resolution you want to set here.
;The optimal resolution is 320x240, but if your video card has problems
;You may want to try 640x480.
Width = 320
Height = 240"

Lo Unico que debemos modificar son los datos que dicen:
Width = 320
Height = 240

Reemplazandolos por:

Width = 640
Height = 480

Y por ultimo donde dice:

" ;Set this parameter to 0 to disable screen stretching, and set it to 1 if
;you want to scale it up to fit the current resolution.
Stretch = 0 "

Tienen que poner: Stretch = 1
