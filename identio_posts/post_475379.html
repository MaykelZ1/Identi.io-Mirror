JRiver Media Center 23.0.17 Multilenguaje








JRiver Media Center: la aplicación de gestión de medios todo en uno convierte cualquier PC con Windows en un centro de entretenimiento para el control completo de sus medios digitales: audio, video, imágenes y televisión. Conecta PC a estéreo, TV, dispositivos de control remoto, cámaras digitales, escáneres y reproductores de MP3 portátiles. Reproduce todos los medios, rasga, quema y organiza toda tu música, imágenes y video. MC codifica y reproduce todos los formatos de medios populares. Las potentes utilidades incluyen Media Server para transmitir música e imágenes a PCs remotas. Media Center es más que un jugador de clase mundial. También es una red de medios de comunicación, una casa entera de música, película, televisión y solución fotográfica. El líder en sonido de calidad audiophile. La base de datos de medios más potente disponible; Una herramienta esencial para grandes colecciones.


Características:



Audio 



* Soporte nativo de DirectShow DSP filtra 
* Audiophile - WASAPI salida exclusiva utilizando DAC USB de alta gama 
* Audiophile - Añadida la capacidad para reproducir archivos de la memoria en lugar de disco 
* Audiophile - Apoyo a la gama alta USB DAC de con WASAPI 
* auriculares DSP hace Escuchando más natural y cómodo. 
* SoundcardSwitch comando para el lanzador, permite la selección de la tarjeta de sonido 
* Vista del álbum 3D ajusta el tamaño del texto para mostrar más pistas cuando sea necesario 
* Vista del álbum 3D permite jugar, agregar, barajar y añadir como próximo a jugar



Imágenes



* Previsualización de imagen permite el zoom desde una miniatura para rotar, eliminar, comparar, editar y más 
* Marcos de imagen en miniatura (opcional) 
* Photoshop PSD soporte de archivos 
* Copia de comando único / enviar a un editor como Photoshop y pila nueva imagen con original


Video



* Etiquetado de vídeo utilizando archivos de sidecar 
* Captura de capítulo para MKV y MP4 
* Capítulos añadidos a la pantalla en pantalla 
* Panel de información muestra metadatos


Televisión


* Añadido nueva columna “visto” a los metadatos 
* EPG puede recuperar automáticamente los datos XMLTV 
* correlación automática de los nombres de los canales de XML y los nombres de los canales de MC 
* Panel de información muestra los metadatos, incluyendo arte programa de TV 
* arte programa de TV desde YADB


Vista del teatro


* Ver Hulu y YouTube desde la vista de teatro 
* Vistas anidadas personalizables en vista de teatro 
* Panel de información de archivo personalizable para mostrar metadatos 
* Mejoras de velocidad y calidad 
* Fondos personalizados 
* Soporte de pantalla táctil 
* Reloj en pantalla


Remotos y servidores


* Remitentes y servidores 
* Remoto de retransmisión utilizando transceptores IR para controlar un decodificador o receptor 
* Asistente de configuración remota 
* Media Center Aprendizaje remoto y voladura 
* Media Center puede conectarse a otro Media Center para actuar como un control remoto TRemote) 
* Nuevo plug-in de control remoto HID - soporta accesos directos con teclado, mouse y gamepad 
* Control remoto con interfaz de servicio Web basada en REST en Library Server 
* Servidor de biblioteca soporta lectura y escritura para permitir el acceso compartido a bases de datos multiusuario
