WinXPup 4.0 [Actualizaciones abril 2014 XP]

Características de winXPup4.00

 
 Actualizaciones hasta Abril del 2014
 Requiere Windows XP SP3 Español para ejecutarse. No requiere versiones previas de winXPup, es totalmente independiente!
 
 Seis (6) modos de actualización:

1. Total: Incluye IE8 y WMP11 con todas sus actualizaciones.
2. Corporativo: Actualiza el sistema con el IE6 original.
3. Corporativo PRO: Incluye IE7 con todas sus actualizaciones.
 
4. Integrate Corporativo: Actualiza un disco de instalación (IE6).
5. Integrate Total: Actualiza un disco de instalación (IE8 & WMP11).
6. Integrate CorpPRO: Actualiza un disco de instalación (IE7 & WMP11).

 Detecta las versiónes de: 
 
 1. Internet Explorer (IE8, IE7 e IE6). *
 2. Windows Media Player. WMP11 se ejecuta en todas las versiones de Windows (Para Modo de actualización Total). *
 3. Herramienta de eliminación de software malintencionado (Actualizada). *
 
 4. Sistema de actualizaciones opcionales: Windows Search 4, DirectX 9 y  .Net Framework totalmente desatendido (1.1, 2, 3, 3.5 y 4.0). 
 5. Instalación exclusiva de actualizaciones opcionales. Puede usar winXPup para hacer instalaciones desatendidas de los diferentes .Net, DirectX y Windows Search
 
 * Omite la reinstalación de los mismos si no es requerida.
 
 
 Instalación desatendida con argumentos del ejecutable:
 
 
 1. '-ai1' :: Modo Total
 
 2. '-ai2' :: Modo Corporativo
 3. '-ai3' :: Modo Corporativo PRO
 4. '-ai4' :: Modo Integrate

 Ejemplo: wnp_iii9_esn.exe -ai3 (Modo Corporativo PRO)   
 1. Reinicio del sistema automatizado para continuar con las actualizaciones.
 2. Si el sistema se reinicia mientras winXPup se está ejecutando, la instalación de winXPup, luego de reiniciar el PC, continua en el punto previo antes del reinicio, eliminando la necesidad de volver a ejecutar winXPup. Ha sido probado tanto cerrando la sesión, así como con apagar y  reiniciar; al entrar de nuevo al sistema las actualizaciones continúan donde iban previamente.
 3. Generación de archivo .log de procesos realizados 
 
 
 Integración Automatizada (Compatible con Windows 7):
 Seleccione el CD/DVD ó la imagen ISO de windows XP para Integrar actualizaciones; winXPup copiará el CD/DVD a su disco duro, lo actualizará y finalmente generará una nueva IMAGEN ISO BOOTEABLE que winXPup puede quemar en CD/DVD tanta veces como desee el usuario.
 
 
 winXPup también ejecuta la herramienta de detección de software malintencionado a discreción, sirviendo de está manera como sistema de detección. Está función se ampliará en versiones sucesivas. (Solo se ejecuta en Windows XP)
 
 
Validación del disco de instalación a Actualizar. winXPup detectará la versión indicada para actualizar su disco de instalación y escogerá el modo correcto (Total ó Corporativo) de integración. Detecta versiones desatendidas de Windows XP con sus diferentes caracteristicas.
Nota: Si winXPup detecta un modo incompatible de Integración no se realizará el proceso.
