Detalles
Sistema de automatización radial version Profesional con llave física de uso
Administre su emisora de radio en forma sencilla y confiable

Sincronice sus comerciales emitiendo publicidad de manera puntual y ordenada

Emita su radio por internet directamente desde la misma pc

Atienda llamadas telefónicas de oyentes o corresponsales

Grabe la cinta testigo durante 1 año y acceda a la misma de manera ordenada

Grabe llamadas telefónicas de oyentes o corresponsales

Ideal para:
-Automatización de emisoras de radio

-Generación de radios para internet

-Musicalización de locales comerciales, centros de compras, empresas: Permite insertar informacion en formato audio dentro de la programacionmusical
-Iglesias, emite de manera ordenada (intro de predica-predica-cierre de predica) permitiendo utilizar diferentes voces pregrabadas respetando el tema de cada predica y nombre del pastor.
 
Nueva interfase de usuario.
Nuevo motor de reproducción con tecnología ShotPlay.
Realice búsqueda personalizada con mayor velocidad.

Personalice sus mezclas como un DJ profesional.
Compatible con Microsoft Windows 7 y Microsoft Windows Server 2008, aproveche las ventajas de los
nuevos sistemas operativos.
Control independientemente del nivel de audio de cada ventana.
SQL Server, FTP Server, Web Server, Remote Control Server.
Integre placas económicas más rápidas y de última tecnología.
NO necesita conexion a internet para funcionar 
Soporte tecnico por internet con costo adicional de u$ 100 + impuestos por año. 



 





Caracteristicas
Audio
Ultra baja latencia con Tecnología Hardata Shot-Play.
Soporte de múltiples marcas.
Nuevo algoritmo de mezcla.
Procesador de audio multibanda integrado.
Soporte de placas ASIO.
Soporte de nuevas calidades de audio (nuevos formatos, bits por muestra: 8, 16, 32, 24, 32 bits y punto flotante; sampling rate: 8 a 96 KHz).

Usabilidad
En este nuevo modelo se agregaron las solicitudes de usuarios avanzados con las siguientes mejoras:

Nueva interfase de usuario, más clara y simple de usar.
Nueva ventana de exploración avanzada de datos.
Busquedas avanzadas.
Nuevo editor de audio integrado.
Nueva ventana de Emisión.
Nueva ventana de Voice Over.
Nueva ventana de Instant Play con botonera.

Musicalizador
Formatos y subformatos.
Separación de artista.
Repetición por prioridades.
Filtrado por año del album.

Gestión de archivos
Múltiples Storages para poder segmentar el almacenamiento de los archivos.
Soporte para contar con múltiples versiones de un mismo archivo.
Base de datos de auditoría, separada de la base de datos principal.
Separación de las consultas de auditoría de las consultas real time.
Metadata extendida.
Soporte de imágenes asociados a los materiales de audio.
Soporte de Álbum.
Soporte de múltiples artistas.

Noticias
Captura de cables de agencias noticiosas.
Nueva ventana de emisión de noticias. 
Nueva ventana Prompter de noticias.

Web
Servicios Web integrados.
SDK para desarrollar aplicaciones Open Source sobre los servicios Web.
Gestión de usuarios, seguridad, reportes, auditoría y materiales vía Web.

Plataforma
Hardata Hdx Server 3.
Administración de usuarios y grupos de usuarios.
Permisos configurables para emisoras/ventanas/materiales.

Sistema Operativo y Base de datos
Windows XP, Vista, 7, Server 2008.
Soporte de SQL Server 2003, Server 2008."
Acceso Web (Administración de contenidos + Reportes)
Incorpora acceso web integrado, capaz de administrar contenido y generar reportes remotos de los contenidos emitidos.El acceso Web permite previsualizar contenido, importar y exportar contenido al servidor, y generar reportes de emisión. 

Logger (Grabación continua)
El sistema puede realizar grabaciones continuas de una o varias señales. Puede ser utilizada como copia legal. 
 
Administracion de llamadas
Atiende y graba las llamadas de los oyentes, mientras graba y administra las llamadas de los corresponsales a través de un sistema de comandos DTMF. 
Búsqueda y edición de noticias
Ideal para producción y emisión de noticieros, captura automáticamente noticias de los cables de agencias de noticias usando herramientas de búsqueda y edición, y soporta la creación de archivos de texto con audio embebido.
 
MusicalizadorPermite crear fácilmente programación musical  estableciendo criterios de selección tales como artista, género, ritmo, prioridad, año, etc.
Hora y estado del clima
Con actualización meteorológica desde internet, emita la información de hora y clima con solo presionar un botón, o inclúyala en la programación.
Soporte RDS
Muestra el nombre de la emisora, título e intérprete del tema musical y mensajes personalizados en la radio.
 
Asistencia para vivoNovedoso, sencillo y práctico asistente para vivo que permite disparar audios en forma instantánea.
One-Touch herramientasAvanzadas herramientas hacen de la administración de contenidos para emisión, una tarea fácil y eficiente con opciones de Play Over, Voice Over, Fade, Fade Over y Crossfade. 

Mezcla Automática

Mejor que un DJ en vivo. Detecta automáticamente el punto óptimo de mezcla entre los archivos de audio (música, artística) resultando en una transición suave y con niveles óptimos de sonido.
