Importante - AOMEI PE Builder Free

  • Categoría: Software / Programas
  • Publicado hace más de 11 años

AOMEI PE Builder es una herramienta gratuita que le ayuda en la creación de un entorno de arranque basado en Windows PE sin necesidad de instalar AIK / WAIK. Integra un conjunto de herramientas que le facilita y permiten recuperar el sistema cuando existen problemas de arranque. Basado en el entorno de Windows PE tradicional y muy similar al escritorio de Windows de toda la vida, incluye explorador de archivos y su menú de inicio clásico. AOMEI PE Builder le permite seleccionar entre diferentes formas de crear un dispositivo de arranque (CD/DVD,USB) o creando una imagen ISO.

Explicado con otras palabras es un sistema de arranque que le permite ejecutar un Windows reducido para poder recuperar el sistema o archivos en un Windows nativo, lo puede arrancar desde un CD/DVD o usando un Pendrive.

AOMEI PE Builder Free incluye por defecto AOMEI Partition Assistant Standard que le permite gestionar particiones y AOMEI Backupper Standard con la cual puede crear imágenes ISO del disco duro o copiar archivos y carpetas fácilmente, ambas aplicaciones vienen incluidas en la creación de AOMEI PE Builder.




También incluye pero bajo selección:

7-Zip: Completo compresor y descompresor de archivos.
Everything: Un sencillo buscador de archivos y carpetas en Windows.
IrfanView: Visor de imágenes muy popular.
Notepad++: Editor de notas más completo que el típico bloc de notas.
SumatraPDF: Visor de archivos PDF.
Q-Dir: Gestor de archivos
Recuva: Te permite recuperar archivos borrados en Windows.
PENetwork: Pequeña utilidad para instalar y gestionar diferentes parámetros de red.
Filezilla: Gestor FTP muy completo.
QTWeb: Navegador web ligero y seguro.
OSFMount: Permite montar imágenes
BOOTICE: Restaura el MBR y la partición Boot Record.
NTPWEdit: Editor de contraseñas, puede cambiar o eliminar contraseñas de las cuentas de usuarios.







Otra opción muy interesante es la posibilidad de añadir archivos o aplicaciones portables que después puede ejecutar en el entorno de Windows PE. También puede crear unidades virtuales para organizar archivos o programas para localizarlos fácilmente.
