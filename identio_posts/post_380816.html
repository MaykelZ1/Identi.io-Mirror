WinToUSB 2,2 [Instala windows desde una memoria]

  • Categoría: Software / Programas
  • Publicado hace más de 10 años

WinToUSB es un software libre que permite que usted instale y que funcione el sistema operativo de Windows en una impulsión dura del USB o un USB Flash Drive, usando una imagen de la ISO o la impulsión de CD/DVD como la fuente de la instalación. En la más nueva versión usted puede reproducir el OS actual de Windows (Windows 7 o más adelante) a una impulsión del USB como Windows para ir espacio de trabajo. WinToUSB también apoya crear la impulsión bootable del USB de WinPE, puede ayudarle a transferir el contenido de WinPE al USB para conducir y para hacer la impulsión bootable. Las características dominantes de WinToUSB incluyen: Interfaz fácil de utilizar del mago que proporciona las instrucciones paso a paso para crear Windows para ir impulsión del USB. Instale Windows/WinPE de una imagen de la ISO o de la impulsión de CD/DVD.Reproduzca el OS actual de Windows (Windows 7 o más adelante) como un Windows para ir impulsión del USB. Ayuda para Windows 7/8/8,1/10 Preview/2008 técnico R2/2012/2012 R2 y WinPE 2/3/4/5. Utilice cualquier edición de la inspección previo técnica 10 de Windows 8/8.1/Windows para crear Windows para ir impulsión del USB. Ayuda para disposiciones del disco de MBR y de GPT. Notas importantes: Windows 7/2008 R2 no tiene ayuda incorporada 3,0 del USB, así que Windows 7/2008 R2 tendrá que ser pateado de un puerto 2,0 del USB. Las impulsiones del flash del USB son muy lentas. Toma un de largo plazo para instalar y funcionar Windows de un USB Flash Drive, recomiende altamente el usar de un disco duro del USB.Windows a ir las impulsiones se puede patear en diversas computadoras, así que usted puede llevarlo a dondequiera y utilizarlo en cualquier computadora. Windows 7/2008 R2 no es totalmente portables. Usted puede tener problemas de la activación y del conductor cuando booting en diversas computadoras. Usted necesita ser administrador en la computadora que usted está instalando el WinToUSB encendido.
