WinSetupFromUSB 1.6 | Pendrive con Multiboot



WinSetupFromUSB 1.6 es un Software que te permitirá múltiples Instaladores de Windows (2000, XP, 2003, Vista, 7, 8, 8.1, 10, 2008 Server, 2012) y Linux (varias distribuciones) en un mismo Pendrive.

INSTRUCCIONES

Primero: Una vez que descompriman, ejecuten el programa de acuerdo a su versión de Windows (de 32 ó 64 bits):






Segundo: Como consejo, utilicen un Pendrive de 8GB o más. Ahora, la primera vez que usen el WinSetupFromUSB deberán formatear el Pendrive, por lo que tendrán que marcar la casilla "Auto format it with FBinst" (las opciones "align" y "copy BPB" deben quedar seleccionadas). Si pretenden tener sólo Windows Vista ó superior, pueden usar el formato NTFS; si van a incluir Windows 2000/XP/2003, el formato debe ser FAT32.


Tercero: Este paso sólo les servirá si pretenden instalar  Windows 2000/XP/2003. Debemos seleccionar la casilla que está bajo el título "Windows 2000/XP/2003".




Después de clickear sobre el botón "...", seleccionaremos la carpeta donde está el Instalador (normalmente la carpeta I386):




Cuarto: Este paso sólo les servirá si pretenden instalar  Windows Vista ó superior. Debemos seleccionar la casilla que está bajo el título "Windows Vista/7/8/server 2008/1212 based ISO".



Después de clickear sobre el botón "..." y seleccionaremos la ISO correspondiente a la Windows a instalar.

Luego, marcaremos la casilla "Advanced Options" y aparecerá una ventana de opciones en la que deberemos marcar la casilla "Custom menu names for Vista/7/8/Server Source":


Quinto: Clickeamos sobre el botón "GO" y se iniciará el trabajo. Para el caso de haber seleccionado "Auto format it with FBinst" aparecerán las siguientes ventanas:



En ambas ventanas hay que clickear sobre el botón "Yes" (recuerden que el formatear el Pendrive implicará perder TODO lo que en él había).


Sexto: Para el caso el haber seleccionado la casilla "Windows Vista/7/8/server 2008/1212 based ISO" aparecerá don ventanas similares a la siguiente:



En la primera ventana hay que ingresar el nombre de la carpeta donde va a quedar el Instalador (máximo 7 caracteres) y en la segunda ventana va un nombre interno (máximo 35 caracteres). Ninguno de dichos  nombres aparecerá visualmente en el Menú de Arranque del Pendrive.





Séptimo: Se iniciará el trabajo en el Pendrive:
