Configurar Red wifi 3dsp 2310U Geo Eco-pc en Ubuntu







Las personas que tengan una de las mini laptop Geo Eco-pc y quieran instalar Linux Ubuntu notarán que el adaptador de red inalámbrico no es reconocido por el sistema, pero tranquilos que aquí está la solución: 

Ir a: aplicaciones/ accesorios/ terminal 

$ sudo su 

$ apt-get install aptitude 

$ aptitude install build-essential linux-headers-`uname -r` 

$ apt-get install git 

$ git clone http://github.com/franciscosouza/3dsp.git 

$ cd 3dsp 

$ sudo bash Install_3DSPUSB.sh 

listo! ahora para que la aplicación se ejecute sin pedir clave: 

sudo sh -c "echo $USER ALL=NOPASSWD: /usr/sbin/3dsp-wifi-radar >> /etc/sudoers" 

Y ahora para que arranque al iniciar sesión, Sistema -> Preferencias -> Aplicaciones al inicio, pulsar Añadir y : 






Para ejecutar el programa: 

Aplicaciones/Accesorios/3DSP uWb 
Aplicaciones/Accesorios/3DSP Wifi Radar (en esta aplicación seleccionan la conexión de su preferencia y conecta) 


Recomendación: 

si el Firefox se configura para trabajar sin conexión de forma automática... instala esto: 

https://addons.mozilla.org/en-US/firefox/addon/disable-automatic-offline-mana/
