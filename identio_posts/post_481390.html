[USB multiboot] EasyBoot [PORTABLE] [MEGA]

  • Categoría: Software / Programas
  • Publicado hace más de 8 años


Hola a todos, en este nuevo post les traigo esta increible utilidad, EasyBoot 6.6.0.800 portable multilingual, incluido el español como siempre, es super util para hacer un USB multiboot personalizado con tus fondos y menus, perfecto para dar ese toque profesional a tu USB de reparaciones.







EasyBoot es una utilidad con la que podrás crear un disco de arranque para Windows añadiendo las funciones que más te interesan: realizar particiones del disco o restaurar el sistema, entre otras, con una interfaz visual personalizable.

EasyBoot te permite personalizar el entorno gráfico con un fondo propio, textos, barras y botones de función, como por ejemplo el acceso directo al modo y fuente de arranque.

Además, EasyBoot te brinda soporte para menús de arranque en varios idiomas. Una vez configurado, el programa genera un fichero que basta con grabar para obtener un CD o DVD listo y funcional.






- Windows 7, Windows 8, Windows 8.1, Windows 10 o versiones superiores
- Un procesador Intel Pentium 4 o mejor
- 512MB de RAM
- Pendrive
