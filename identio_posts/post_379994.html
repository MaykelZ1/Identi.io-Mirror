DREAMWEAVER CC 2015

Encuentra las fotos perfectas
Encuentra las imágenes perfectas para tu sitio en la tienda (marketplace) de Adobe Stock. Explora una biblioteca de fotos e ilustraciones cada vez más amplia y descárgalas directamente en tu proyecto desde Creative Cloud.

Previsualiza en cualquier dispositivo
Prueba e inspecciona sitios web en varios dispositivos a la vez. Solo tienes que escribir una URL corta en el navegador de tu dispositivo móvil para previsualizar tus sitios.

Crea sitios web adaptativos
Desarrolla sitios web que se adapten dinámicamente a diferentes tamaños de pantalla con Bootstrap integrado. Revitaliza tus sitios existentes, empieza desde cero, utiliza plantillas o incorpora componentes de Bootstrap. Previsualiza tus diseños en tiempo real cambiando el tamaño de la Vista en vivo en Dreamweaver. Define consultas multimedia visualmente y más.
Exportación de lotes desde PSD
Ahorra tiempo extrayendo rápidamente imágenes optimizadas para la Web de documentos de Adobe Photoshop en diferentes resoluciones y formatos.

Codificación más rápida
Los nuevos temas del editor de código, la validación de código en tiempo real con Linting y la función de autocompletar compatible con Emmet hacen que la escritura de código en Dreamweaver resulte más rápida y fácil que nunca.

Y mucho más
También incluye un nuevo panel de DOM para navegar y editar elementos de páginas HTML con facilidad, previsualizaciones en tiempo real de colores e imágenes en la vista de código, nuevos fragmentos de código adaptativo para acelerar el desarrollo, inserción automática de archivos dinámicos al guardar y más.




REQUISITOS DEL SISTEMA



Windows


Procesador Intel® Pentium® 4 o AMD Athlon® de 64 bits.

Microsoft® Windows® 7, Windows 8 o Windows 8.1.

2 GB de RAM.

1.1 GB de espacio disponible en el disco duro para la instalación; se requiere espacio libre adicional durante la instalación (no se puede instalar en dispositivos de almacenamiento flash extraíbles).

Resolución de 1280 x 1024 con tarjeta de vídeo de 16 bits.

Se requiere disponer de una conexión a Internet y registrarse para la activación del software y la validación de los abonos obligatorias, y para el acceso a los servicios online*.




Mac OS


Procesador Intel multinúcleo.

Mac OS X v10.9, 10.10

2 GB de RAM.

1.1 GB de espacio disponible en el disco duro para la instalación; se requiere espacio libre adicional durante la instalación (no se puede instalar en un volumen que utilice un sistema de distinción entre mayúsculas y minúsculas en archivos, ni en dispositivos de almacenamiento flash extraíbles).

Resolución de 1280 x 1024 con tarjeta de vídeo de 16 bits.

Se requiere disponer de una conexión a Internet y registrarse para la activación del software y la validación de los abonos obligatorias, y para el acceso a los servicios online*.
