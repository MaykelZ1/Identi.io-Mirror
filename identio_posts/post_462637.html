Windows 10 LTSB es una versión del sistema operativo de Microsoft especialmente pensada para aquellos usuarios que no necesitan aplicaciones universales, ni a Cortana ni la tienda de aplicaciones de Microsoft. Un Windows 10 como debería haber sido desde los inicios, muy similar a su predecesor Windows 7.

Esta rama del sistema operativo también se caracteriza por tener garantizados 10 años de actualizaciones y, además, solo recibe actualizaciones y parches de seguridad y corrección de errores, nada más, por lo que en ningún momento recibe nuevas funciones y características adicionales a diferencia de los usuarios de las versiones de Windows 10 convencionales, reduciendo también así la tolerancia ante fallos.

Este Windows está basado en la rama Windows 10 Enterprise, por lo que cuenta con todas las opciones de este sistema operativo para empresas, por ejemplo, permitiendo a los administradores tener un mayor control sobre la telemetría y funcionar en entornos centralizados, así como poder hacer uso de funciones como Windows to Go.

Como hemos dicho, Microsoft no quiere que los usuarios utilicen este sistema operativo, y una muestra de ello es la dificultad para encontrar una copia de este sistema. Windows 10 LTSB solo se encuentra disponible como una parte de Windows 10 Enterprise, la cual solo puede comprarse como volúmenes de licencias para empresas o pagando 7 dólares al mes.
Como podemos ver, Windows 10 LTSB es todo lo que han pedido los usuarios. Y tecnicos inside lo trae con mas de 230 optimizaciones y mejoras graficas entre muchas otras cosas asi que si eres un gamer este es tu windows y si no lo eres tambien ya que algo mas veloz y optimizado es imposible.
