Recupera tu unidad - Diskpart - CMD

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 9 años







En esta ocacion quiero compartir con ustedes un comando muy util como lo es el Diskpart, es muy util a la hora de recuperar una unidad dañada. Tambien para verificar si nuestro sistema esta reconociendo nuestra unidad de disco local o extraible, cabe aclarar que es al momento de formatear perderemos todos los datos alojados en la unidad.




- Oprimir las reclas Windows + R

- Se abrira la ventada de ejecutrar, en dicha ventana escribiremos el comando diskpart. En el SO. XP el comando es diskpart.exe




- Cuando se abra la ventana de cmd para ver los discos que se encuentran en nuestro sistemas, deberemos escribir list disk.



- Deberemos buscar la unidad a formatear por el tamaño, una vez encontrada para seleccionarla deberemos escribir selec disk x ( La x es el numero del disco a formatear).
importante: Una mala manipulación en este paso puede resultar en la pérdida de datos que no deseas eliminar.





- El comando Detail Disk nos permite tener mas detalle de la unidad.




- Una vez que estés seguro de que has seleccionado la unidad correcta, escribe clean. En caso de que la unidad tenga más de una partición escribe clean all:


- Cuando termines el procedimiento anterior escribe create partition primary:

- Escribe después select partition 1:

- Luego ingresa el comando active:

- Por último, escribe format fs=fat32 quick o escribe format fs=ntfs quick. Dependiendo del formato que se le quiera asignar a la unidad.

Bueno esto es todo por ahora, espero que les sea de utilidad...
