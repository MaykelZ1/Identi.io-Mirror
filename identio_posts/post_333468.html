Solución: JDownloader no reconoce links (W8/Java7)

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 11 años
Solucionado: JDownloader no reconoce links usando Windows 8 y Java 7








Algunos usuarios de Windows 8 se encuentran con este problema debido a incompatiblidades entre la versión de JAVA y una actualización de JDownloader la cual no se soluciona reinstalando ninguno de los 2 software mencionados.


La solución es ir al directorio donde tenemos instalado jDownloader y buscar el ejecutable. Normalmente es C:Program FilesJDownloader



Una vez allí, sobre el archivo clickeamos con el botón derecho y le damos a Propiedades. Nos desplazamos a la pestaña Compatiblidad y activamos la opción: Modo de compatiblidad con Windows XP SP3. 










Una vez realizado esto, forzaremos la compatiblidad hacia atrás de la aplicación y de forma automática debería actualizarnos los plug-ins. 



Una vez actualizado se reiniciará y volverá a iniciar con el problema resuelto y las bbdd de plug-ins actualizada.
