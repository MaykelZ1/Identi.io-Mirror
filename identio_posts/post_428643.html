Android 6.0 RC2 para pc

  • Categoría: Sistemas Operativos
  • Publicado hace más de 9 años
Ya puedes descargar e instalar Android 6.0 RC2 para PC
Windows es el sistema operativo para ordenadores que domina a nivel mundial. Es un hecho indiscutible, aunque poco a poco comienzan a abrirse grietas bajo sus pies. Lo que no ha Ubuntu promete conseguirlo Android, cuya nueva versión para PC ya puedes descargar e instalar.

Se trata de una muy buena adaptación para ordenadores del sistema operativo líder en dispositivos móviles. La han desarrollado los miembros del equipo Android x86, y ya va por la versión Android 6.0 RC2 Marshmallow. Es totalmente gratis, aunque para instalarla deberás crear un USB booteable.

Android para PC es un sistema operativo especialmente recomendable si tienes un ordenador viejo. Consume pocos recursos, lo que lo convierte en una alternativa perfecta a Windows para ordenadores con menos de 2 GB de RAM. No es el único sistema operativo ligero. También están Remix OS o CloudReady, aunque todos ellos tienen algo en contra: su dependencia de Internet.

La segunda versión de este firmware viene con varias mejoras con respecto a la primera. Principalmente los cambios se han producido en cuanto a estabilidad y seguridad, aunque pueden calificarse como menores. Puedes descargar la imagen .iso tanto para ordenadores de 64 bits como para PC de 32.

Hay que aclarar que esta versión para ordenadores creada por Android x86 no es el mismo sistema operativo que equipan los Chromebook. Estos portátiles vienen con un firmware desarrollado por el fabricante, a diferencia de Android 6.0 RC2, producido por un equipo de voluntarios. No obstante, las críticas recibidas son bastante buenas.

La principal desventaja con la que ha contado siempre Linux es la experiencia de usuario. Aunque tiene muchos puntos positivos, sus potenciales usuarios notan la diferencia notable entre este SO de código abierto y Windows.

En el caso de los Chromebook y las adaptaciones de Android para ordenador, los consumidores vienen de usar móviles Android, por lo que están completamente familiarizados con el sistema.

http://www.android-x86.org/download
