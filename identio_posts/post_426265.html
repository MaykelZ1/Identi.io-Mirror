Vokoscreen grabar escritorio en Fedora

  • Categoría: Linux
  • Publicado hace más de 9 años



En windows tenemos a Camtasia como el sowftare más común para grabar el escritorio, quizas luego le siga VLC, que por  cierto, también se puede utilizar en las distro GNU/Linux, sin embargo, también existen programas específicos para linux que se utilizan para dicha actividad.
Vokoscreen es una aplicación gráfica para grabar nuestro escritorio, con una variedad de opciones o extras que harán que nuestro resultado final sea el deseado. Usa en el background a ffmpeg, por lo que la potencia de esta aplicación no es inferior si la comparamos con otras similares, de hecho Vokoscreen nos brinda opciones que ciertamente encontraremos muy útiles.

Este Software fue desarrollado por Volker Kohaupt. Y su sitio web en GitHub es: https://github.com/vkohaupt/vokoscreen



Requiere la instalación del repositorio de rpmfusion, por lo tanto se instala primero desde la terminal:

$ sudo dnf install --nogpgcheck http://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm

$ sudo dnf install --nogpgcheck http://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm

Ahora instalar Vokoscreen:

$ sudo dnf install vokoscreen


La interfaz de Vokoscreen es realmente simple, al igual que su uso:

Imagen 1

Como pueden apreciar (ver imagen 1) así es como se verá Vokoscreen al iniciarlo. Tenemos los siguientes botones de acción:
Iniciar: iniciar grabación
Detener: detener grabación (en caso de haber alguna en curso)
Pausar: pausar/renaudar grabación (en caso de haber alguna en curso)
Reproduci: ver el vídeo que recién grabamos
Enviar: enviar por email algún vídeo que hayamos grabado

Esas son las opciones de control o acción, luego tenemos las opciones para personalizar lo que deseamos captar en realidad.
Pestaña  Monitor/Display/Ventana
Imagen 2:

En esta pestaña se puede configurar cómo se realizará la grabación:

Pantalla: se grabará la pantalla completa
Ventana: se grabará solo una ventana o aplicación
Área: se grabará solo una porción de la pantalla, un área determinada
Ampliación: marcar esta opción para que se muestre una lupa que agrandará la zona donde se encuentre el puntero del ratón. Válido destacar que tenemos tres tamaños para la lupa, para hacerla más grande o menos grande, seleccionar el tamaño deseado entre los disponibles debemos hacer clic en el pequeño botón con tres puntos [...] que aparece a la derecha de la palabra Lupa.
ShowKey: Muestra en pantalla las teclas que se van presionando.

Pestaña micrófono a utilizar (Audio de entrada)
Imagen 3

En esta pestaña se configura la entrada de audio que se utilizará para la grabación, si es el interno del equiop utilizar la primera opción, en caso contrario utilizarlo como lo muestro en la imagen 3.

Pestaña Vídeo de salida
Imagen 4

En la pestaña se configuran los fotogramas por segundo, el codec a utilizar para el video de salida, tienen dos opciones avi o mkv.
También la configuración del audio, además de poder seleccionar si se incluye el puntero del mouse en la grabación


Pestaña de configuración:

Imagen 5

En la pestaña configuración (Imagen 5), se asigna la ruta en la que se van a guardar los archivos, seleccionar el reproductor de vídeo que se utilizará para ver el resultado una vez finalizada la grabación, así como las opciones de minimizar el ícono de vokoscreen al iniciar la grabación.

Pestaña cámara Web.
Imagen 6

Además puedes anexar una grabación de la cámara en el video (imagen 6),, puedes selecionar la ubicación de la camara.
