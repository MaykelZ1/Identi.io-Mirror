Google Chrome es un navegador de código abierto gratuito. Tiene un diseño simple que hace que sea más fácil trabajar en Internet. Entre las características únicas de Chrome se encuentran las pestañas aisladas que evitan que el navegador se bloquee y ofrecen una protección más confiable contra las amenazas de Internet. En Chrome, se implementa un nuevo motor javascript V8, que permite que el navegador funcione con nuevas aplicaciones web.
Google Chrome incluye muchas funciones, administrando qué webmasters pueden mejorar el nivel de conveniencia para los usuarios finales. Google Chrome tiene incorporado el software Gears que permite a los webmasters aprovechar la API, como el almacenamiento sin conexión. Además, gracias a Google Chrome, sus aplicaciones web no serán diferentes de los programas de "escritorio", ya que el navegador se puede iniciar en un modo con una interfaz de usuario mínima, cuando solo se muestra un encabezado.
Velocidad: el navegador se inicia rápidamente y carga rápidamente páginas web
• Google Chrome se puede abrir rápidamente desde el escritorio.
• Google Chrome descarga instantáneamente páginas web.
• Google Chrome maneja páginas web interactivas, aplicaciones web y código de JavaScript con una velocidad sin precedentes.

Simplicidad: el navegador está diseñado según los requisitos de eficiencia y conveniencia
• Una ventana se usa para buscar y navegar.
• Las pestañas se pueden colocar en cualquier orden, fácil y rápidamente.
• Los sitios favoritos siempre están a mano: sus miniaturas se muestran en la página de acceso rápido.

Estilo: gracias a los temas, el navegador será aún más brillante y atractivo
• Diseña tu navegador utilizando paletas de colores, patrones, imágenes y gráficos de autor. Visita la galería de temas de Google Chrome.
