Error Windows 10 Driver Sonido Realtek

  • Categoría: Sistemas Operativos
  • Publicado hace más de 10 años
Hola amigos de IDENTI!!! La mejor Comunidad de la Web...



Después de un largo tiempo fuera de IDENTI, he decidido regresar para presentarles la solución al problema de Drivers en Windows 10 Build 10041. Como aún no ha salido la versión oficial, muchos fabricantes no publican en su web los drivers para este sistema, pero, pueden ser descargados desde Windows Update.

El problema está, cuando algunos drivers se descargan desde WinUpd y al momento de ser instalados presentan un error 0000b54 (algo asi); dicho error se produce debido a que el tiempo predeterminado por Windows para instalar aplicaciones se agota.

Para solucionar dicho error deben hacer lo siguiente:

1. Ingresar como Administrador en la Aplicación gpedit.msc



2. Directiva de Equipo Local / Configuración del Equipo / Plantillas Administrativas / Sistema / Instalación de Dispositivos ... Luego dan doble Clic sobre "Configurar tiempo de espera de instalación de dispositivos"



3. Finalmente, Habilitan la opción y cambian el valor del tiempo de espera a 1800 segundos (con media hora creo que es tiempo más que suficiente jajajaa)



Luego, intenten nuevamente instalar los controladores desde Windows Update, esperan a que se instalen y Listo!!!... problema resuelto...

Saludos a todos
