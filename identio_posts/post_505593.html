ES File Explorer Pro v1,1,3 [Mod Alien]

  • Categoría: Moviles / Celulares
  • Publicado hace más de 7 años




Descripción general: Esta es la versión Pro de ES File Explorer (Administrador de archivos), la aplicación imprescindible para todos los usuarios de Android. ¡Puede disfrutar de todos los beneficios a continuación una vez que tenga ES File Explorer Pro!

Administrador de archivos: administre sus archivos tal como lo hace en su computadora de escritorio o portátil usando Selección múltiple, Cortar / Copiar / Pegar, Mover, Crear, Eliminar, Cambiar nombre, Buscar, Compartir, Enviar, Ocultar, Crear acceso directo y Marcador. Todas las operaciones pueden realizarse en archivos locales (en su dispositivo Android) o incluso de forma remota (desde su computadora a través de una red) 
?Administrador de aplicaciones: categorice, desinstale, haga copias de seguridad y cree accesos directos a sus aplicaciones 
?Remote File Manager: cuando esta característica está habilitado, puede administrar archivos en su teléfono desde su computadora 
?Código de seguridad incorporado y compatibilidad con RAR: le permite comprimir y descomprimir archivos ZIP, descomprimir archivos RAR y crear archivos ZIP cifrados (AES de 256 bits)
? Reproductores y reproductores integrados para varios tipos de archivos: Incluye fotos, música y videos; admite aplicaciones de terceros como Quick Office para una mejor productividad 
?Muestra miniaturas: para APK e imágenes 
?Text viewers and editors 
?Accede a tu PC doméstica: a través de tu teléfono inteligente a través de WiFi con SMB 
?Funciones como cliente FTP y WebDAV: administra archivos en Servidores FTP, FTPS, SFTP y WebDAV del mismo modo que gestiona archivos en su tarjeta SD 
?Buscador de archivos Bluetooth: puede copiar y pegar archivos entre dispositivos preparados para Bluetooth. ES File Explorer (Administrador de archivos) admite OBEX FTP para explorar dispositivos y transferir archivos entre dispositivos Bluetooth
?Haga tareas con un solo clic, aumente la memoria y acelere su dispositivo: incluye un widget simple que permanece en su pantalla de inicio para permitirle conocer su situación actual de RAM y eliminar automáticamente las tareas, con una lista de ignorar para ignorar las aplicaciones que desea sigue corriendo. El módulo Administrador de tareas es obligatorio para esta función. 
?Cache Cleaner y Auto-start Manager: elimine los archivos basura que ocupan un valioso espacio de almacenamiento. El módulo Administrador de tareas es obligatorio para esta función. 
?Root Explorer: el último conjunto de herramientas de administración de archivos para usuarios root. Proporciona acceso a todo el sistema de archivos y todos los directorios de datos, y permite al usuario cambiar los permisos.
?Soporta múltiples idiomas: inglés, ruso, japonés, coreano, francés, español, alemán, italiano, checo, húngaro, ucraniano, tamil, catalán, turco, lituano, portugués y más ...
