Kingdoms Of Amalur Reckoning




Kingdoms of Amalur: Reckoning para PC es el primer juego basado en el nuevo universo de Amalur, un mundo de fantasía con 10.000 años de historia ideado por el escritor R.A. Salvatore, el diseñador de juego Ken Rolston y el dibujante y guionista de cómics Todd MacFarlane. Se trata de un juego de rol de mundo abierto en el que exploraremos este mundo de fantasía, con libertad para movernos por sus diferentes regiones, tratar con sus razas y llevar a cabo todo tipo de aventuras.

REQUISITOS PARA WINDOWS

MÍNIMO

SO: Windows XP SP3 / Windows Vista SP2 / Windows 7 SP1
Procesador: Intel Core 2 Duo a 2.2 GHz o AMD Athlon 64 X2 5000+ a 2.6 GHz
Memoria: 1 GB de RAM para Windows XP / 2 GB de RAM para Windows Vista y Windows 7
Disco Duro: 10.5 GB de espacio libre
Gráficos: nVidia GeForce 8800 con 512 MB de VRAM o superior / ATI Radeon HD3850 con 512 MB de VRAM (con soporte para Pixel Shader 3.0), resolución mínima soportada: 1280x720
Sonido: Dispositivo compatible con DirectX 9.0c
Versión de DirectX : 9.0c
