TeraCopy Pro 3.2 Multilingual + Portable










Una utilidad popular diseñada para copiar archivos de forma más rápida y confiable, proporcionando al usuario muchas características. TeraCopy utiliza buffers dinámicamente ajustados para reducir los tiempos de búsqueda. Puede reanudar las transferencias de archivos rotas y omitir archivos dañados durante el proceso de copia.


Solucionar problemas fácilmente 


En caso de un error de copia TeraCopy intentará recuperarse. Si no puede recuperarse en el peor de los casos, simplemente saltará el archivo problemático, sin terminar la transferencia completa. También puede esperar a que se vuelva a conectar la red o el dispositivo. 
Los archivos innecesarios se pueden omitir para reducir el tiempo de transferencia. Durante el proceso, TeraCopy muestra errores detectados y le permite solucionar el problema recopiando sólo los archivos problemáticos.
Verificación de archivos 
TeraCopy puede verificar los archivos después de haberlos copiado para asegurarse de que son idénticos. Esto se hace comparando hashes de archivos de origen y destino. 
Algoritmos soportados: CRC32, MD5, SHA-1, SHA-256, SHA-512, Panamá, Tiger, RipeMD, Whirlpool y xxHash.

Confirm drag'n'drop 

TeraCopy puede mostrar opcionalmente un diálogo de confirmación en cada operación de arrastrar y soltar. Puede impedirle mover las carpetas accidentalmente de una ubicación a otra.

Flujo de trabajo más rápido 

TeraCopy puede mantener un historial de directorios usados ??recientemente, así como mostrar carpetas que están abiertas en el Explorador y otros administradores de archivos.

Otras características


Integración de Shell 

TeraCopy puede reemplazar completamente las funciones de copia y movimiento de Explorer, permitiéndole trabajar con archivos como de costumbre.

Exportar informes Pro 

Guardar listas de archivos con toda la información relacionada como archivos HTML y CSV.

Editar listas de archivos Pro 

Se pueden eliminar archivos y carpetas innecesarias para reducir el tiempo de transferencia.

Preservar las marcas de tiempo de la fecha 

TeraCopy mantiene la hora y la fecha originales de los archivos al copiar.

Confiado por expertos forenses 

TeraCopy está incluido en DEFT (Digital Evidence & Forensics Toolkit).

Copiar archivos bloqueados 

Copie archivos utilizando un servicio de Windows elevado y copia de sombra de volumen, si es necesario.
