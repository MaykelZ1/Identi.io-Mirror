Adobe Flash Player - plug-in diseñado para todos los navegadores populares - Internet Explorer, AOL, Firefox, Mozilla, Netscape y Opera, con el que puede reproducir archivos de vídeos en Flash (SWF), ver videos en línea y escuchar audiofaylyi en línea que está disponible de forma gratuita descargar diakov.net. Sólo con este complemento en el navegador puede mostrar correctamente las páginas web y elementos web interactivos, desarrollados por la tecnología Flash. Esta tecnología es ampliamente utilizada para la creación de banners publicitarios, animaciones, juegos, y jugar en las páginas web de vídeo y audio.

Adobe Integrated Runtime (AIR) - es un entorno independiente de la plataforma de Adobe para ejecutar una aplicación que te permite usar HTML / CSS, Ajax, Adobe Flash y Adobe Flex para migrar las aplicaciones web para el escritorio. Adobe AIR incluye la posibilidad de crear un paquete de instalación que puede ser fácilmente instalado en el ordenador del usuario. Los paquetes son compatibles con todas las plataformas soportadas por este sistema, pero que existe en los sistemas de gestión de paquetes OS modernos que sólo admite YUM para Linux. 

Adobe Shockwave Player - componente para la reproducción de películas Flash, que proporciona la mejor calidad de visualización de contenido interactivo páginas web de entretenimiento o de negocios de la naturaleza en flash. 

Adobe Shockwave - una tecnología que le permite disfrutar de todos los placeres de gráficos 3D interactivos en Internet, es para mostrar los datos y creado Adobe Shockwave Player. Reproductor de Shockwave Player ha sido reconocida como la herramienta estándar para la reproducción de material multimedia publicados en Internet.
