Windows 7 portable

  • Categoría: Sistemas Operativos
  • Publicado hace más de 9 años





Información:
Windows 7 Portable es una versión  para iniciar Windows 7 sin necesidad de instalarlo en nuestro equipo.

La mejor utilidad de esta versión, es que en caso de fallo de nuestro sistema operativo o que no pueda arrancar, lo podamos usar para acceder a nuestros archivos y respaldar o intentar arreglarlo.

Lo podemos cargar desde una unidad extraible como un USB o un disco duro externo.

INSTRUCCIONES:
Formatear usb en NTFS.
Descomprimir el .rar y copiar todos los archivos a un usb y listo.
En caso de no funcionar, descomprimir el archivo .rar, crear un iso con los archivos descomprimidos usando PowerIso, despues crear una usb booteable con rufus.

Nota: solo sirve para USB, no para CD.

Peso: 272mb

Servidor: Mega.
