Microsoft Toolkit 2.5.3

  • Categoría: Software / Programas
  • Publicado hace más de 10 años




Microsoft Toolkit es un conjunto de herramientas y funciones para la gesstión de licencias, la implementación y la activación de Microsoft Office y Windows.
Todas las funciones se ejecutan en segundo plano y la interfaz gráfica está deshabilitado para evitar la ejecución de múltiples funciones, ya que podrían entrar en conflicto o causar daños si se ejecutan simultáneamente.


 
Los Funciones de personalización de Microsoft Office de instalación (Personalizar Tab Setup), AutoKMS desinstalador (si está instalado AutoKMS), AutoRearm desinstalador (si está instalado AutoRearm), Oficina Uninstaller y Product Key Checker funcionan incluso si Microsoft Office o Windows no está instalado/compatibles.


Requisitos:
-Microsoft .NET Framework 4.0/4.5
-Microsoft Office 2010 en adelante
-Windows Vista o posterior


Cambios
-AutoKMS habilita el servicio de registro de sucesos de Windows si esta desabilitado
-Se cambio el driver del adaptador TAP para soporte de Windows 10
-Se cambió la ubicación del folder WinDiver para reducir los errores de borrado
-Soporte para la versión preliminar de Windows 10 (Technical Preview)
-Reelaborada la detección de la versión de Windows
-Actualizada la librería TaskScheduler
