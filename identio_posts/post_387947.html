requisitos del sistema windows

  
 Requisitos Mínimos
Windows XP Service Pack 3, Windows Vista, 7 y 8 (32bit/64bit)
Tarjeta Gráfica DirectX 9 con 512Mb Video RAM:
- AMD Radeon HD 2600 XT
- nVidia 8600
CPU Dual Core:
- AMD Athlon64 X2 2.1 Ghz (4050+)
- Intel Core2 Duo 1.86 Ghz (E6300)
1GB de Memoria (2GB en Vista)
Hardware Recomendado
Windows Vista, Windows 7 o Windows 8
Tarjeta Gráfica DirectX 11 con 1GB Video RAM:
- AMD Radeon HD 4870
- nVidia GTX 480
CPU Quad Core :
- AMD Phenom II X2 565
- Intel Core i5-750
4GB de Memoria
