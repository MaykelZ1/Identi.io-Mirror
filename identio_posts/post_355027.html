Después de instalar XUbuntu pero automatizado

  • Categoría: Linux
  • Publicado hace más de 10 años





La mayor parte de nosotros, una vez instalado una nueva versión de nuestra distribución favorita o simplemente una nueva distribución solemos instalar nuestras aplicaciones favoritas o las utilizamos habitualmente, y además personalizamos nuestra instalación.Sin embargo, lo que no puedo evitar de ninguna de las maneras es instalar, una y otra vez las mismas aplicaciones, las que suelo utilizar mas a menudo, y esto es algo, que cada vez me resulta mas tedioso.
¿Como evitar esto en Xubuntu??


Postinstalación de XUbuntu

¿Porque XUbuntu?

Al igual que indica el autor de este script, en mi caso yo utilizo XUbuntu para equipos de pocos recursos. Básicamente equipos donde tenía instalado un XP. Es una forma de aprovechar un viejo ordenador, que de otra forma tendríamos que desecharlo puesto que es incapaz de mover otros sistemas operativos.

¿Que nos ofrece este script?

Como he comentado en la introducción, se trata de evitar tener que actualizar, personalizar, configurar e instalar una y otra vez las mismas aplicaciones, de forma, que con este script dejarás XUbuntu en perfectas condiciones para que cualquier usuario pueda trabajar sin necesidad de preocuparse de si se trata de Windows o GNU/Linux.
¿Qué es lo que hace este script?

En primer lugar actualiza tu equipo
Insltala las tipografías mas habituales de Windows
Instala Chrome.
Instala Firefox
Instala xubuntu-restricted-extras
VLC, el mejor reproductor de vídeo hasta la actualidad. Reproduce cualquier tipo de archivo.
My wheather-indicator. Para tener el tiempo en el panel.
Youtube indicator. Para descargar videos de youtube.
Touchpad-indicator para activar o desctivar el touch pad de tu portatil
Openshot, excelente editor de video.
Codecs de compresión, para poder descomprimir cualquier tipos de archivos.
Gimp: el mejor editor gráfico del mundo GNU y sus plugins mas conocidos.
Brasero: Para grabar CD y DVD.
dropbox: El sistema de almacenamiento en la nube.
calibre: El gestor de libros electrónicos mas usado de la actualidad.
Libdvdread4 : librerias necesarias para ver DVD en linux.
Libreoffice: Una suite ofimática para tus documentos.
Preload: para acelerar la apertura de aplicaciones.
Gedit: el fantástico editor de texto de gnome

Además,

Quita todos los paquetes descargados durante las instalaciones.
Desactiva el informe de errores.

¿Como ejecutar el script?

Es realmente muy sencillo, dado que tan solo tenemos que crear un archivo (postinstalacion por ejemplo) y copiar y pegar el contenido anterior en el mismo. A continuación le damos permisos para ejecutarlo,


chmod 777 postintalacion


Por último ejecutamos el script


sudo ./postinstalacion



Conclusiones
 
Se trata de una opción muy interesante, pero todavía me parece mas interesante personalizarla y configurarla con tus propias aplicaciones, de forma que con el paso del tiempo y el uso de XUbuntu o Ubuntu, irás añadiendo mas aplicaciones (y quitando otras) para dejar un script que seguro te ahorrará mucho tiempo, y te evitará la tediosa labor de instalar las aplicaciones una y otra vez.
