Java, ¿Qué es Java? - Es un lenguaje de programación con el que tenemos la posibilidad de hacer cualquier clase de avance y está enfocado a contemplar las pretensiones tecnológicas de compañías. Uno de los conceptos más atrayentes de la tecnología Java es que es un lenguaje sin dependencia de la interfaz. Esto significa que si hacemos un programa en Java va a poder

trabajar en algún PC del mercado


Independientemente del S.O. en el que hayamos ido a usar el avance. Esta es la más grande virtud de Java en relación a sus máximos competidores en lenguajes de programación y simplifica el trabajo del programador, logrando centrarse en un exclusivo avance sin dependencia de la interfaz dónde vaya a usarse dicho avance. Esta aptitud del lenguaje Java se consigue por la Máquina Virtual de Java. La máquina virtual de Java hace de puente entre el S.O. y el software de Java y facilita que el S.O. comprenda toda clase de avance.
