Tor Browser Bundle 4-0-5 Portable

  • Categoría: Software / Programas
  • Publicado hace más de 10 años


Tor es un conjunto de herramientas para una amplia gama de organizaciones y personas que quieren mejorar su seguridad y la seguridad en Internet. Usando Tor puede ayudarle a anonimizar la navegación web y la edición, la mensajería instantánea, IRC, SSH, y otras aplicaciones que utilizan el protocolo TCP. Tor también proporciona una plataforma en la que los desarrolladores de software pueden crear nuevas aplicaciones con una función de anonimato, seguridad y características de privacidad. El Tor Browser Bundle le permite usar Tor en Windows, Mac OS X o Linux sin necesidad de instalar ningún software. Puede ejecutarse desde una unidad flash USB, viene con un navegador web pre-configurado para proteger su anonimato, y es autónomo.

Navegar de forma anónima proporciona un poderoso escudo contra la vigilancia de la red y análisis de tráfico. La protección de su privacidad en línea sólo puede lograrse mediante herramientas de software adicionales que pueden ocultar su ubicación y puede evitar que su información y navegación hábitos en línea de ser revelado a nadie.

Si alguna vez estás en necesidad de una herramienta de este tipo, Tor Browser Bundle es una sugerencia que puede agregar a la lista. La aplicación se basa en una red de servidores virtuales con el fin de hacer que sus sesiones de navegación anónima. Tor pone a disposición un conjunto de reglas y algoritmos que las rutas de las comunicaciones a través de varios servidores en todo el mundo. De este modo, todo el tráfico se asignó al azar a modo de seguimiento de la fuente se convierte en una tarea casi imposible.

El paquete de descarga es un archivo de extracción automática que incluye la aplicación de Tor en sí, que se puede ejecutar desde la línea de comandos, una cruz GUI controlador de la plataforma llamada Vidalia y una versión a medida de Firefox equipado con un conjunto de complementos que garanticen el anonimato en línea.



Aconsejamos al usuario medio para considerar el uso de la aplicación Vidalia, ya que la interfaz gráfica te pone a prueba menor que el comando versión puntual de Tor. La primera vez que ejecute Vidalia, se conecta automáticamente a la red Tor y si el proceso se realiza correctamente, un estado se mostrará en la ventana principal.

La interfaz de Vidalia encierra una colección de accesos directos que le permiten detener el servicio, para empezar a configurar los relés, para desencadenar una visión global de la red y más importante de todo, se divierte un botón para el uso de una nueva identidad en cualquier momento que desee. Si usted está interesado en el uso de ancho de banda, Vidalia cuenta con una utilidad independiente sólo para mostrar un gráfico de ancho de banda que mide la tasa de recibir / enviar los paquetes.

La versión portable de Firefox incluido en el paquete de descarga está personalizado con algunas extensiones que hacen que la experiencia de navegación anónima fácil. Estos incluyen: Torbutton - le permite crear una nueva identidad cada vez que inicie una sesión de navegación, NoScript - bloquea los scripts de dominios no confiables y HTTPS Everywhere - le permite cifrar la comunicación con una gran cantidad de sitios populares.

Con todos los componentes que encajona, Tor Browser Bundle es una de las herramientas de navegación anónima más dignos de confianza que hay. Se puede configurar desde el símbolo del sistema, ejecute desde el controlador de interfaz gráfica de usuario dedicado - Vidalia y viene con un navegador web pre-configurado. Cualquiera de estos métodos a elegir el resultado es una experiencia de navegación por Internet segura y controlada.
