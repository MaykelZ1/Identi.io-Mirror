 Aomei Tech:   Del disco dinámico Manager de Aomei favorable la edición se diseña especialmente para los usuarios de la pequeña empresa y trae una serie de funciones: Gestión de volúmenes dinámicos, Convertidor de discos y Asistente Paso a Paso para darle un control total sobre sus PCs que ejecutan Windows 7/Vista/XP/2000 (32-bits y 64 bits) Sistemas Operativos. Le permite cambiar el tamaño / mover / reducir / ampliar el volumen dinámico para maximizar el uso del espacio en disco, ampliar el volumen del sistema dinámico para resolver el problema de poco espacio en disco en la unidad C, convertir discos dinámicos a básicos y vice versa sin perder datos, arreglar el problema de partición GPT protección mediante la conversión de disco GPT a MBR, convertir el disco MBR a GPT para crear volúmenes superiores a 2 TB, añadir la unidad de RAID 0 y retire la unidad de RAID 0 para ajustar (ampliar y reducir) volumen RAID 0, mover las rebanadas de volumen de un disco a otro, etcétera Las características adicionales: Todos los cambios se pueden previsualizar y corregido antes de aplicar el disco físico. Soporta todos los dispositivos de almacenamiento que pueden ser reconocidos por Windows, como IDE, SAS, SSD, SATA, SCSI, discos externos USB y disco alambre del fuego. Es compatible con el sistema de archivos incluyendo FAT, FAT32 y NTFS. Apoyo estilos disco incluyendo MBR (Master Boot Record) y GPT (Tabla de particiones GUID). Soporta hasta 5 TB volúmenes dinámicos y 256 unidades de disco duro dentro de un mismo sistema. Todas las operaciones siempre como objetivo el hacer que su computadora trabaje eficientemente.
