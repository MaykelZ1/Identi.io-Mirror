Desarrollo web con Sass, CoffeeScript y Emmet(V2B)

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 10 años


Aprende a desarrollar sitios web con herramientas rápidas y productivas

Aprende a desarrollar sitios web de forma profesional con Sass, CoffeeScript y Emmet herramientas que te facilitarán el trabajo diario con CSS, javascript y HTML. Emmet escribirá el código HTML por ti, CoffeeScript revisará la sintaxis y Sass te ayudará a estructurar fácilmente tus reglas de estilo. Descubre cómo los tres juntos te permitirán construir archivos en formatos estándar de una manera ultra rápida y ágil.
Hoy en día el desarrollo web ha madurado lo suficiente para crear herramientas que nos permiten llegar a los límites de las tecnologías estándar. Por mucho tiempo hemos creado archivos en texto plano y escrito instrucciones en lenguajes de programación que no facilitan el desarrollo de sitios web complejos.
Antes creábamos un archivo CSS y nos preocupábamos más por el orden de las instrucciones y no por el estilo del sitio web. Ahora tenemos Sass, un lenguaje para poder estructurar correcta y fácilmente nuestra vista además de ofrecer caracteristicas que en otros años hubiéramos soñado
Durante mucho tiempo nos peleamos con los browsers y nuestros archivos javascript porque tal vez nos faltó un punto y coma, pero ahora tenemos CoffeeScript, un lenguaje de programación que nos revisa la sintaxis y nos permite escribir de forma más sencilla nuestras funciones.
Cada que tenemos que crear un sitio web y empezar con el layout siempre tenemos que crear etiquetas HTML y muchas veces esas mismas etiquetas las repetíamos una y otra vez; ahora tenemos Emmet, un proyecto que nos va a permitir escribir en una pequeña sintaxis etiquetas HTML, Lorems y templates en un momento.
Las tres herramientas juntas nos darán nuevas cotas de efectividad, agilidad y rapidez en nuestro trabajo profesional como desarrolladores web.
