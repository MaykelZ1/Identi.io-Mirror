Nueva Build de Windows10 - 14271

  • Categoría: Reciclaje
  • Publicado hace más de 9 años

Nuevas Build 14271 para PC y 14267.1004 para Windows 10 Mobile por el anillo rápido

 
 

 
Por desgracia el MWC 2016 no está siendo todo lo emocionante y apasionante que nos gustaría con ver nuevos terminales con Windows 10 Mobile, sin embargo las noticias continúan y parece ser que Microsoft se está poniendo las pilas y de que manera con la llegada de nuevas Builds, pues nos ha llegado una nueva Build 14271 para PC y 14267.1004 para Windows 10 Mobile a través del anillo rápido en la que se encargan de seguir mejorando la estabilidad del sistema operativo, así como corregir los errores pertinentes que os comentaremos a continuación.



Mejoras Build 14271 para PC

 

 
·         Corregido un problema por el que los bordes de las ventanas de aplicaciones pasaban del color de énfasis a color negro negro después de cada actualización a una nueva versión.


·         Actualizado los iconos de control de la música que se muestran en la barra de tareas en aplicaciones de música como Groove para un aspecto más fresco y limpio a resoluciones más altas.


·    Corregido un problema por el que la barra de tareas a veces no se ocultaba automáticamente y mostraba de forma inesperada en la parte superior de las ventanas de pantalla completa como la que muestra por encima de una presentación de PowerPoint en el modo de presentación de diapositivas.


·         Arreglado un problema donde el ?Ocultar notificaciones durante la presentación de? preferencia configuración se pierde después de actualizar a una nueva Build.


·         La cabecera de toda aplicación en el Centro de actividades se puede hacer clic en lugar de sólo el nombre de la aplicación o ?x?.


·         Hemos hecho que sea más fácil descartar todos los avisos de una aplicación específica en el Centro de Acción mediante el aumento de la zona objetivo para abarcar toda la cabecera de aplicación.


·         Corregido un problema por el que las notificaciones de interacción no mostrarían un botón para introducir texto en algunos casos.


·         Cambio rápido de usuario debería funcionar ahora con foto contraseña.


·         Corregido un problema por el que algunas aplicaciones Win32 de escritorio no aparecían en Inicio.


·         Arreglado un problema donde el acento de color no cambia automáticamente cuando se elige la opción de presentación de diapositivas para el fondo de escritorio.

 
 

 Errores encontrados Build 14271 para PC
 

 
    Algunos equipos tienen pantallazo azul cuando se reanuda desde la hibernación. Al desactivar la hibernación es una solución en algunos casos hasta que se solucione.

    Si usted tiene Kaspersky Anti-Virus, Seguridad en Internet, o el total de Kaspersky Security Suite instalado en su PC hay un error de controlador conocido que impide que estos programas funcionen como se espera en construye a partir de la Subdivisión de Fomento. Nos hemos asociado con Kaspersky para solucionar este problema en futuras versiones, pero en este momento no hay soluciones alternativas conocidas. Si bien este problema está presente, se recomienda utilizar Windows Defender u otro producto antivirus de terceros de su elección para permanecer protegidos.


    Siempre encendido, ?siempre muestran todos los iconos en el área de notificación?, este ajuste altera la estructura del área de notificación (?bandeja del sistema?).


    Al navegar en Microsoft Edge, los gráficos muestran el uso de la biblioteca D3.js no se mostrarán correctamente ? por ejemplo, render como negro, colocado de forma incorrecta. sitios afectados son conocidos Cortana, Bing.com, y el PowerBI.com.

   Mejoras Build 14267.1004 Windows 10 Mobile
 

    Solucionado el problema reportado por los Insiders_ en la Build 14267 donde el Lumia 550 no cargaba o no se conectaba a través de USB.

    Solucionado el problema que impedía notificaciones de correo de voz.

    Introducido el correo de voz visual para dispositivos de dual SIM como el Lumia 950 XL.

    Publicado una actualización de la aplicación Contactos que lleva de nuevo la función de la historia.

 
 

 Errores encontrados Build 14267.1004 Windows 10 Mobile
 

 
    La aplicación Comentarios de Windows anclada en Inicio, después de actualizar a la Build 10586.107 ya no se mostrará como anclada.


    Algunos dispositivos tienen problemas de conexión a pantallas inalámbricas. Se está investigando.


    Al utilizar Continuum en el Lumia 950 XL, la experiencia con el trackpad no está funcionando correctamente.


    Soporte para Alcatel Onetouch Fierce XL

 
 

¿Te ha llegado la actualización? ¿Has notado alguna mejora más?
