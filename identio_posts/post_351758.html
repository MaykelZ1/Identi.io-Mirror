PES 2015 + Voces Latino










HOLA GENTE, HOY LES TRAIGO PRO EVOLUTION SOCCER 2015, SE TRATA.... YA... TODOS CONOCEN AL PES... COMENCEMOS 

Nombre: PES 2015
Fecha de lanzamiento: 13 de noviembre de 2014
Plataforma: PC
También disponible para: X360  XOne  PS3  PS4
Género: Deportes, Fútbol
Duración estimada: Incalculable
Desarrollador: Konami
Distribuidor: Konami
Idioma: Textos: Español
Voces: Español 

Requisitos mínimos:

Sistema operativo: Windows 7 SP1 / Vista SP2
Procesador: Intel Core 2 Duo @ 1.8GHz (AMD Athlon II X2 240 o equivalente procesador)
Memoria: 1GB RAM
Tarjeta gráfica: Nvidia GeForce 7800/ATI Radeon X1300/Intel HD Graphics 2000. Compatible DirectX 9.0c con 512MB.
Disco duro: 8GB de espacio libre.

Requisitos recomendados:

Sistema operativo: Windows 7 SP1 / Vista SP2
Procesador: Intel Core i3 530 (AMD Phenom II X4 925 o equivalente procesador)
Memoria: 2GB RAM
Tarjeta gráfica: Nvidia GeForce 7950/ATI Radeon HD 2400/Intel HD Graphics 3000. Compatible con DirectX 9.0c con 1024MB
Disco duro: 8GB de espacio libre.



AL DESCOMPRIMIR LES VAN A SALIR 3 CARPETAS CON.. EL JUEGO BASE, OTRA CON EL IDIOMA LATINO Y OTRA CON EL PARCHE. LO QUE TIENEN QUE HACER ES:
1- INSTALAR EL JUEGO
2- CRACKEARLO
3- COPIAR LA CARPETA "Data" DEL IDIOMA Y PEGARLA EN LA RUTA DE INSTALACION
4- COPIAR EL CONTENIDO DE LA CARPETA "Parche Online" Y PEGARLO EN LA RUTA DE INSTALACION (PARA ESTO, NECESITAN TENER SU CUENTA STEAM ABIERTA, SI NO TIENEN CUENTA STEAM NO HAGAN ESTE PASO, SINO, NO SE LES VA ABRIR EL JUEGO)
5- JUGAR
