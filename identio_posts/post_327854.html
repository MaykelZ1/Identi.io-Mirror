Mass Effect 2[MG][Repack][DLC][Es][PC]










SINOPSIS

Segunda entrega del gran RPG galáctico creado por BioWare, que nos llevará de nuevo a visitar los rincones más recónditos del universo. Acción, exploración, rol y un gran argumento son sus señas más significativas.


INFORMACIÓN

Desarrollador: BioWare
Editor: EA Games
Género: Acción-Primera Persona(RPG)(Ciencia Ficción)
Plataforma: PC
Fecha de Lanzamiento: 28 de Enero del 2008
Idioma: Multi Lenguaje
Tamaño: 8.7GB(Dos Discos cada uno 4.6GB y 4.1GB)(Gracias a Brantekk)


REQUERIMIENTOS

Requisitos mínimos:
Sistema Operativo = Windows XP SP3 / Windows Vista SP1 / Windows 7
MicroProcesador = 1.8GHz Intel Core 2 Duo o superior AMD CPU
Memoria = 1 GB RAM para Windows XP / 2 GB RAM para Windows Vista y Windows 7
15 GB
Tarjeta Gráfica = 256 MB (con soporte Pixel Shader 3.0). Placas soportadas: NVIDIA GeForce 6800 o superior; ATI Radeon X1600 Pro o superior. NVIDIA GeForce 7300, 8100, 8200, 8300, 8400, y 9300; ATI Radeon HD3200, y HD4350 están por debajo de los requisitos mínimos.
DVD ROM = 1x Velocidad
Tarjeta de Sonido = DirectX 9.0c compatible
DirectX = DirectX 9.0c de agosto de 2008 (incluido)
Periféricos = Teclado y Ratón

Requisitos recomendados:
Sistema Operativo = Windows XP SP3 / Windows Vista SP1 / Windows 7
MicroProcesador = 2.6+ GHz Cure 2 Duo Intel o superior AMD CPU
Memoria = 2 GB RAM
Tarjeta Gráfica = ATI Radeon HD 2900 XT, NVIDIA GeForce 8800 GT o superior recomendadas
Tarjeta de Sonido = DirectX 9.0c compatible
DirectX = DirectX 9.0c de agosto de 2008 (incluido)
Periféricos = Teclado y Ratón


INSTRUCCIONES

1. Descomprimir
2. Montar o Grabar
3. Instalar(Autocrackiable)
4. Desbloquear el Firework
5. Jugar.
