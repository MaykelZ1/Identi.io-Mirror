Modificando el menú de IDENTI (HTML y CSS básico)

  • Categoría: Tutoriales / Cursos
  • Publicado hace más de 6 años

Atención: Este post, es una continuación del post "HTML y CSS Básico: Como crear un menú" donde aprendían a crear un menú básico clonando el menú de Identi.

En este post, vamos a tomar ese menú y modificarlo.

¿Qué vamos a hacer?

- Insertar logo
- Mover los items de acceso y registro de usuarios
- Cambios de diseño

Clases nuevas:

- main-menu-logo: Contenedor principal de logo
- main-menu__item--register: Clase para modificar estilos de un item
- main-menu__item--access: Clase para modificar estilos de un item

Vamos a tener que editar un poco el código CSS y HTML, empecemos por lo primero.

Insertar logo Identi y mover items de acceso y registro de usuarios: Para insertar un logo, bastaría con insertar una imagen en vez de texto (Con CSS redimensionamos la imagen)
Para mover los dos items a la izquierda, simplemente encerramos los items que necesitamos que estén del otro lado en un div, y gracias a la clase "align-right" que habíamos creado en el otro post lo podemos alinear fácilmente.



Bien, ahora varias cosas que hay que tocar en CSS, vamos parte a parte.

- Logo: Para el logo, vamos a hacer algo fácil. Simplemente vamos a cambiarle el ancho por 40px y darle un margen a la derecha de 15px.



- Rediseñar los items: Para esto, solo hay que eliminar y agregar un par de lineas.

- Cambiar padding
- Cambiar fuente
- Agregar una transición
- Cambiar efectos mover

- Fuente: Usaremos la fuente Open Sans, de Google Fonts



- Padding y transición: Haremos un padding mas grande, para que el menú no sea tan fino.



- Efectos mover: Vamos a cambiar el efecto de cambiar el fondo, por agregar una linea por debajo del ítem. Esto lo logramos con box-shadow



Crear el botón de crear cuenta y acceder: Para esta parte, hay que tener el código del post "Como crear botones", porque utilizaremos el código creado en ese post para crear los botones. Hay que agregar las clases necesarias para crear el botón, lo cual nos quedaría así:



Las clases "--register" y "--access", son las clases que vamos a utilizar para rediseñar algunas cositas. Vamos ahora por esa parte:

--register y --access: Estas dos clases vamos a utilizarlas para poder reescribir propiedades de css sin tener que estar tocando otra cosa.



"!important" es necesario ya que no queremos (O por lo menos en mi caso) que el botón de acceder y crear cuenta tenga el efecto de la linea por debajo.

Ahora que el item "registrate" ya no existe más, porque ahora es "crear cuenta" y es un botón, hay que eliminar el código que habíamos hecho para que sea de color verde y hay que cambiar el :nth-child(4) del item "imagenTi" por :nth-child(5)
