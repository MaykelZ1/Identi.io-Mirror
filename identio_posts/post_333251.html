Qué Hacer después de instalar Linux Mint 17







Este es mi primer post, así que... sean pacientes!

Soy un nuevo usuario de Linux y me he decantado por Linux Mint 17, después de pasar por Xubuntu y Elementary OS. En mi opinión, esta versión de Linux, derivada de Ubuntu, es la que me ha dado, por mucho, la experiencia "out the box" que tanto espera uno al instalar un SO.

En fin, ya instalado en el sistema, llega la primordial pregunta... ¿y ahora, qué?

Pues bien, ya habiendo Googleado un poco puedo compartirles la siguiente información que puede quitarles varios dolores de cabeza encima.


1. Consigue más fondos de pantalla.

Cada lanzamiento de Linux Mint viene con un nuevo grupo de fondos de pantalla, pero también puedes conseguir los de versiones anteriores con este comando:

$ sudo apt-get install mint-backgrounds-*


2. Añade un navegador adicional.

Linux Mint 17 trae Firefox instalado como navegador por defecto. Puedes conseguir más navegadores como Chromium y Google Chrome. Chromium se encuentra disponible en los repositorios:

$ sudo apt-get install chromium-browser


- Pasar Chromiun al español:

$ sudo apt-get install chromium-browser-l10n

- Instalar la última versión de Firefox:

$ sudo add-apt-repository ppa:mozillateam/firefox-next
$ sudo apt-get update
$ sudo apt-get install firefox
$ sudo apt-get install firefox-locale-es


3. Instalar Flash Player.

El paquete del Adobe Flash Player está instalado por defecto en Mint, así que debería poder reproducir juegos y vídeos en Flash dentro de Firefox. Para Chromium, sin embargo, el Flash Player basado en Pepper no está incluido y tiene que ser instalado a mano. Utiliza el siguiente comando para descargar el paquete y colocarlo dentro de Chromium.

$ sudo apt-get install pepperflashplugin-nonfree

El comando de arriba debería funcionar, pero en caso de que falle se puede rehacer el proceso con el siguiente comando:
$ sudo dpkg-reconfigure pepperflashplugin-nonfree


4. Instalar los codecs multimedia.

El paquete Ubuntu Restricted Extras instalará lo esencial para reproducir formatos como mp3. También instalará las fuentes de Microsoft:
$ sudo apt-get install ubuntu-restricted-extras


Para activar la reproducción de DVDs encriptados, es necesario instalar este paquete (Linux Mint suele traerlo):

$ sudo apt-get install libdvdread4

$ sudo /usr/share/doc/libdvdread4/install-css.sh


5. Instalar fuentes propietarias.

En los repositorios de Ubuntu podemos encontrar algunas fuentes típicas de Windows. Las instalamos desde una terminal con el siguiente comando:

Code:
$ sudo apt-get install msttcorefonts gsfonts-x11 


6. Instalar Skype.

La popular aplicación de mensajería y videollamadas ya se encuentra disponible en los repositorios de los socios de Canonical, así que con ejecutar este comando bastará:
$ sudo apt-get install skype


7. Instalar la última versión de LibreOffice.

Esta suite de ofimática ya viene incluida en Linux Mint, pero soy partidario de tener la última versión estable, por el tema de la compatibilidad con la de M$.

para esto, deberán abrir un terminal y usar los siguientes comandos:

$ sudo add-apt-repository ppa:libreoffice/ppa
$ sudo aptitude update && sudo aptitude install libreoffice
$ sudo aptitude install libreoffice-l10n-es


8. Instalar KolourPaint.

Para aquellos que editan imágenes a nivel básico y extrañan Paint de M$, esta es la mejor solución. Potente y versátil, KolourPaint no tiene nada que envidiarle a su colega propietario.

Para instalarlo, deberán abrir un terminal y usar el siguiente comando

$ sudo apt-get install kolourpaint4
$ sudo apt-get install oxygen-icon-theme
$ sudo apt-get install kdelibs-bin kdelibs5-data kdelibs5-plugins


9. Instalar Temas de apariencia e íconos de M$.

La idea es hacer la transición entre Win y Linux lo menos traumante que se pueda. Mucho ayuda el encontrar un sistema que se "parece" (visualmente hablando) a lo que veníamos usando.

Hay que agradecer a NoosLab, por colocar en repositorios estos temas e íconos. Mucho googlear me permitió encontrar los comandos de instalación desde terminal.

$ sudo add-apt-repository ppa:noobslab/icons
$ sudo add-apt-repository ppa:noobslab/themes
$ sudo apt-get update
$ sudo apt-get install win-themes
$ sudo apt-get install win-icons


Un truco; en Mate, basta con configurar los íconos y las ventanas, barras de avance y demás desde el apartado de "apariencia" del Centro de Control.

En Cinnamon, hay mayor compatibilidad con los temas GTK2 y GTK3, por lo que la opción "obtener mas temas en linea" es altamente recomendada.

10. Actualizar el sistema.

Casi para terminar, deben asegurarse de que el sistema está al día. Para ello, lo más fácil y directo es abrir un terminal y escribir estos comandos:
$ sudo apt-get update

$ sudo apt-get upgrade


11. Retirar lo que ya no se use.

Cuando hayan terminado, usen este comando para eliminar lo que ya no haga falta:
$ sudo apt-get autoremove


Esto es todo. Espero les haya servido. Algunas cosas las tomé de arteescritorio.com, otras, son de cosecha íntegramente mía.
