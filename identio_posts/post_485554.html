Consigue Windows de manera oficial y más

  • Categoría: Reciclaje
  • Publicado hace más de 8 años











Amigos de identi aquí les dejo información de donde pueden descargar isos oficiales de Windows, tanto versiones finales como versiones PreRelease (insider), como ya se dejo de dar soporte a Windows Vista y anteriores aquí solo encontrarán desde Windows 7 hasta 10 y sus respectivas diferentes ediciones, también encontrarán herramientas para desarrolladores, Office desde 2007 a 2016, les diré como obtener la iso de las builds insider, haré tambien otro post explicando las diferentes formas de integrar actualizaciones, software a windows para que preparen un SO a su comodidad, son muchas cosas por eso será en otro post.



Gracias al usuario @adguard por hacer una web donde ha hecho una recopilación de windows, office, y software para desarrolladores, todas originales aunque no proviene desde la MSDN, los enlaces se generan por petición a servidores de microsoft por lo tanto los archivos son oficiales, aqui podrán descargar:

-Windows 7 SP1
-Windows 8.1 With Update 3
-Windows 10 y sus diferentes build's
-Office 2007
-Office 2010
-Office 2013
-Office 2016
-Windows 10 Insider (no aparecen todos, porque microsoft publico de manera abierta solo hasta la build 16278, las más actualizadas se consiguen mediante descargas UUP para luego generar la iso)
-Software de implementación y desarrollo.
-Tambien versiones por Volumen (VL) de Windows y Office (SQL).
-También enlaces directos a descargas por filtración ESD.
-Paquetes de idioma para Windows final y insider en formato .cab

Generalmente lo actualiza agregando más software, como comento el recopila software para ponerlo de forma gratuita al publico. La página web es esta:



TechBench



Aquí podrás consultar los archivos de la MSDN, no hay descarga, es para estar seguro si la iso que bajaste es de la MSDN y información de la misma:


MSDN Info



Y desde aquí pueden descargar los archivos UUP con enlaces de servidores a microsoft:


UUP Files

Ahora les diré como obtener las isos de Windows Insider para que ustedes mismos los obtengan o para compartir, sería bueno que lo aprendido aquí le sirviera a algunos usuarios de identi para que también publique las isos de Windows Insider tanto en español es-es como es-mx porque en internet se encuentra mucho en inglés y otros idiomas pero en español poco, aparte porque ya están saliendo las RS4.


Bueno como mencioné algo más arriba y en diferentes post donde publico las builds de windows ¡nsider, estas se obtienen mediante los archivos UUP (Unified Update Plataform) las cuales si eres insider, seguro ya las tienes porque se descargan vía windows update cuando se actualiza a una nueva build. Estas se encuentran en C:WindowsSoftwareDistributionDownload en una de esas carpetas con el nombre en hash, la carpeta que contenga archivos parecidos a estos será la que contenga el sistema o la nueva build descargada:
(perdonen por no poner imágenes precisas)




Una vez localizada la carpeta, tocará extraer la imagen del sistema de esa nueva build (La iso generada a partir de esos archivos resultará en la misma edición instalada, osea si tienes windows 10 Enterprise insider preview, la iso resultante será la misma edición no contendrá otras ediciones, pero claro la misma edición pero nueva build) si desean descargar todas las ediciones habría que examinar los archivos .xml de en esos archivos para encontrar el enlace oficial para descargar esa edición, lo cual es un problema, pero si solo desean la edición que tiene entonces no es un problema.

Si en cambio desean obtener un AIO con todas las ediciones lo cual también es valido por microsoft (osea que microsoft también entrega aio's de manera oficial vía ESD) tendrán que buscar en esos archivos .xml ; pero si se fijaron más arriba el problema ya nos lo resulve el amigo @adguard el cual nos entrega esos archivos UUP para descargar y en múltiples idiomas, solo sería cuestión de descargar todos los archivos dándole en Select Edition/All Edition.

Una vez tengamos los archivos UUP procederemos hacer la iso para eso usaremos un Script llamado Decrypt, (tambien lo pueden conseguir en la página de @adguard):

Primer Caso: Si descargaron los UUP desde la pagina de @adguard solo bastaría con copiarlos/moverlos todos a la carpeta UUPs.
Segundo Caso: Si en cambio deciden hacerlo con los UUP que descargaron vía Windows Update, solo habría que decirle al script en que carpeta se encuentran esos archivos.
