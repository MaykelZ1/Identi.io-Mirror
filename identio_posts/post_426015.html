VisualC++ AIO

  • Categoría: Software / Programas
  • Publicado hace más de 9 años
Que tal seguidores de Identi, en esta ocasion les traigo un AIO generado por mi mismo en el cual contiene las versiones mas actualizadas de los complementos de Visual C++ y visual Studio. Pero, para que sirve?



El Microsoft Visual C++ Redistributable es una serie de archivos que se deben instalar en el sistema para que se puedan usar ciertos programas programados con Visual C++.
 Imaginate que el programa es alguien que entra a una biblioteca y empieza a pedir libros - el Visual C++ Redistributable son los libros, por asi decirlo.
Por funcionarte un programa sin el C++ Redistributable te puede funcionar,
sin embargo no es recomendable que ejecutes el programa sin tener lo que se requiere.




Este AIO contiene los paquetes de visual c++ desde el 2008 hasta el 2015 x32 y x64. El archivo se llama Visual C++.exe y pesa 74.3 Mb.
Espero les sirva!
