* Descripción *

Windows 8.1, es la actualización del sistema operativo de Windows 8.
Esta actualización incluye importantes mejoras, no sólo a nivel de
rendimiento sino también de diseño entre unas de ellas las mas vistas
es el botón de inicio que vuelve a aparecer, ya no se volverá a dar
clic secundario es un espacio en el Inicio para entrar a Todos los
Programas sino que ahora aparecerá un botón en la parte inferior de
la ventana que permitira la busqueda facil de los programas, entre
otras cosas. Esta version incluye todas las actualizaciones desde la
version de Windows 8.1 Pro Update 1 hasta la, actualmente
Windows 8.1 Pro Update 3, esta actualizacion se creo con el proposito
de corregir algunos problemas, mejorar la seguridad y otras cosas mas.
