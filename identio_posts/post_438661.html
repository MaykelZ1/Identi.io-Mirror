Vamos a empezar!


1. ¿Como esta compuesto un documento HTML?
  Los documentos HTML se diseñan de la siguiente manera y orden:

  <html>   con esta etiqueta indicamos el tipo de documento.
  <head>   es la cabeza del documento.
  <title>    indicaremos el titulo del documento.
  </title>   cerramos la etiqueta title.
  </head>  cerramos la etiqueta head.
  <body>   es el cuerpo del documento html donde se mostrara el contenido.
  </body>  cerramos la etiqueta body.
  </html>  cerramos la etiqueta html.


2. Herramientas a utilizar.
    Las herramientas necesarias en este caso para empezar a realizar documentos html son las siguientes:

    Bloc de notas.

Navegador web (mozilla, IE, etc).



3. Nuestro primer ejercicio:
   Vamos a abrir el bloc de notas y escribiremos el siguiente codigo

<html>
<head>
 <title>Primer Ejercicio</title>
</head>
<body>
  Realizaste el primer ejercicio, Felicitaciones!!!!
</body>
</html>

Una vez copiado el codigo guardamos el archivo con cualquier nombre pero con una extensión .HTML:

ejemplo: ejercicio1.html

CUIDADO! en la parte de "Codificación" vamos a poner UTF-8 (para no tener problemas con las ñ o cualquier acento dentro del texto) y guardamos.
