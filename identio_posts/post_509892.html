ILLUSTRATOR CC 2018
ULTIMA VERSION
Illustrator está por todas partes
Millones de diseñadores y artistas utilizan Illustrator CC para crear de todo, desde iconos web y empaquetados de productos hasta ilustraciones de libros y vallas publicitarias.



Obras emblemáticas de cualquier tamaño
Consigue todas las herramientas de dibujo que necesitas para convertir sencillos colores y formas en logotipos, iconos y gráficos sofisticados. Las ilustraciones de Illustrator son vectoriales, por lo que puedes redimensionarlas para pantallas de móviles y para vallas publicitarias, pero siempre quedarán nítidas y atractivas.




Tipografías impresionantes que lo dicen todo
Añade el nombre de una empresa a un logotipo, diseña un folleto o crea el boceto de un sitio web con las mejores herramientas de texto que existen. Añade efectos, gestiona los estilos y edita caracteres individuales para crear diseños tipográficos que expresen tus mensajes a la perfección.




Llama la atención en todas partes
Crea dibujos a mano alzada o calca y vuelve a colorear fotos importadas para convertirlas en obras de arte. Utiliza tus ilustraciones en cualquier parte: trabajos impresos, presentaciones, sitios web, blogs y redes sociales.



REQUISITOS

macOS
Procesador Intel multinúcleo con compatibilidad de 64 bits.
Mac OS versión 10.13 (High Sierra), Mac OS versión 10.12 (Sierra), o Mac OS X versión 10.11 (El Capitan).
2 GB de RAM (se recomiendan 8 GB).
2 GB de espacio disponible en el disco duro para la instalación; se necesita espacio libre adicional durante la instalación (no se puede instalar en un volumen que utilice un sistema de archivos con distinción entre mayúsculas y minúsculas, ni en dispositivos de almacenamiento extraíbles basados en memoria flash).
Resolución de pantalla de 1024 × 768 (se recomienda 1280 × 800).
Opcional: para utilizar las funciones de rendimiento de GPU: el Mac debe tener un mínimo de 1 GB de VRAM (se recomiendan 2 GB) y el equipo debe ser compatible con OpenGL versión 4.0 o superior.
Para comprobar los valores de VRAM, seleccione Mac > Acerca de este Mac (información de gráficos).
Para comprobar que el equipo es compatible con la versión necesaria de OpenGL (4.0 o posterior), consulte este documento de Soporte de Apple.
