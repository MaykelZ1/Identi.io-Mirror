> > > > INSTRUCCIONES DE INSTALACIÓN < < < <

1. Primero tenemos que descargar el enlace subido a MEGA o GOOGLE DRIVE.

2. Una ves descargado el archivo .RAR, le darán Click Derecho y descomprimir.

3. Cuando se haya descomprimido los archivos, les quedara un Block de texto y una carpeta llamada "Emulador de Super Nintendo (SNES)".



4. Ahora le daran Doble Click a la carpeta y una ves dentro encontraran una gran variedad de archivos, el que vamos a utilizar es el archivo .exe llamado "Snes9X" y simplemente lo ejecutan.



5. Una ves ejecutado, se les abrirá el tablero del emulador de Super Nintendo (SNES) Snes9X.



6. Simplemente Comenzamos con la configuración de este emulador, para eso lo primero nos dirigimos a ->Vídeo ->Display Configuration...



7. Se nos abrirá una pestaña nueva, si no saben como configurar y para que sirve cada opción, dejaran tal cual como les aparece en la imagen. Exceptuando en la opción ->Fullscreen Display Setting que colocaran su resolucion de pantalla.



8. Y lo mas importante en la Opción ->Use Shader, marcaran esa casilla y en la opcion ->OpenGL Shader File, buscaran los Shader con los 3 puntitos "..."







9. Continuamos con la configuración, para esto nos vamos a la pestaña ->Sound ->Settings...



10. Se nos abrirá una pestaña nueva, y al igual que el caso anterior simplemente dejan tal cual como esta en la imagen.



11. Ahora seguimos configurando, para esto nos vamos a la pestaña ->Imput ->Imput Configuration...



12. Se nos abrirá una pestaña nueva, aquí podrán configurar el teclado o el joystick que tengan para emular los botones del control de la Consola Super Nintendo (SNES).



13. Por ultimo vamos a ejecutar un juego para ver si el emulador quedo bien configurado y sin problemas, para esto se dirigen a la pestaña ->File ->Load Game...



14. Se nos abrirá una nueva pestaña y simplemente buscan la carpeta donde tienen sus juegos de Super Nintendo (SNES).
