OGWhastapp Update sin caducidad

  • Categoría: Moviles / Celulares
  • Publicado hace más de 10 años




Amigos de IDENTI,

Quiero traerles sin mas la ultima version de famoso OGWhatsapp, que no es mas que la versión de Whatsapp modificada con otro nombre para poder correr 2 Whatsapp con 2 numeros distintos, hay muchos videos, pero no vi en identi ningun paso a paso con imagenes, y ya que muchos tubieron problemas a partir de marzo del 2015 en adelante, les traigo esta version que ya no tiene problemas de caducidad.

 Bueno el metodo siempre es el mismo(5 pasos):



1 ) Si tenemos instalado Whatsapp hacemos un backup de las conversaciones si es que queremos guardarlas





2) Instalar OGWhastApp.apk (copiar a la SD o memoria del dispositivo y correr la instalacion solo pulsar sobre el archivo)



O copian el archivo bajado de internet al dispocitivo via USB, o como mas les guste


(Se que el ejemplo dice GoogleCamera.apk es lo mismo deberia decir OGWhatsapp.apk)

Buscan en el administrador de archivos el Archivo que bajan de este post, y al pulsar sobre el saldra algo así



Esto es solo un ejemplo(se que no dice OGWhatsapp), pero es el mismo caso pulsan Instalar y les advertirá de la seguridad

 

Van a Ajustes o settings, Seguridad, y activan los Origenes Desconocidos para poder instalar el OGWhatsapp

Luego lo abrimos




3) Registramos el numero que tendra OGWhatsapp (supuestamente una segunda linea)



Para registrarlo se puede seguir la guia oficial de Whatsapp - > registrarlo 




4) Vamos al administrador de Archivos y cambiamos de nombre a la Carpeta de instalacion tal como lo muestra la siguiente imagen 




Cambiar de nombre la carpeta que crea de WHATSAPP a OGWHATSAPP





5) Instalar nuevamente el Whatsapp Original de Playstore, registrarlo nuevamente con una linea que lo venias usando, ahi puedes restaurar el backup.



Listo ya contamos con Whatsapp para un numero y OGWhatsapp para otro.



Espero les sea de utilidad.
