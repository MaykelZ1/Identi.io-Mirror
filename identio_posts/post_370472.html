version 6



Deep Freeze Standard 6.61
 
 
 
Informacion Del Programa:
 
 
 
Plataformas: Win95/98/NT/ME/2000/XP/Vista/7.
 
Idioma:Español.
 
Deep Freeze requiere Windows 2000, Windows XP (32 64 bits) o Windows Vista (32 64 bits) o
 
Windows 7 (32 64 bits) y un 10% de espacio libre en el disco duro. Los requerimientos de
 
hardware coinciden con los que se recomiendan para el sistema operativo host.
 
 
 
Deep Freeze, es una aplicación disponible para Microsoft Windows, Mac OS XP, y SUSE Linux sistemas operativos que permite a los administradores de sistemas para proteger
 
el sistema operativo principal y los archivos de configuración en una estación de trabajo o servidor mediante la restauración de un equipo de nuevo a su original configuración cada vez
 
que se reinicie el equipo.
 
 
 
Operación
 
 
 
Deep Freeze es un piloto a nivel de kernel que protege la integridad del disco duro mediante la reorientación de la información que se escriben en el disco duro
 
o una partición, dejando intactos los datos originales. Esto redirige la información ya no se hace referencia una vez que se reinicie el equipo, restaurando así el sistema a su estado
 
original a nivel de sector de disco. Esto permite a los usuarios hacer ?virtual? cambios en el sistema, dándoles la apariencia de que puede modificar los archivos de núcleo
 
o incluso eliminarlas, e incluso hacer que el sistema inutilizable a sí mismos, pero al reiniciar el configuró originalmente ?congelado? estado de la operación sistema se restaura.
 
 
 
Para realizar cambios, un administrador del sistema debe ?descongelar? la partición protegida mediante la desactivación de Deep Freeze, hacer los cambios
 
necesarios y, a continuación «congelación» de nuevo por volver a habilitar Deep Freeze. Estos cambios forman parte de la partición protegida y se mantendrán después de reiniciar.
 
?Congelación? y ?descongelar? se puede hacer a nivel de estación de trabajo o de forma remota a través de la Deep Freeze Enterprise Console.
 
 
 
Deep Freeze también puede proteger a un equipo de malware perjudicial, ya que elimina automáticamente los archivos descargados cuando se reinicia el equipo.
 
Limitaciones y la seguridad
 
 
 
Deep Freeze sólo protege las estaciones de trabajo en un ?recién arrancado? del Estado. Es decir, Deep Freeze evita su manipulación permanente con unidades de disco duro
 
protegida / particiones en los reinicios, pero la actividad del usuario entre reinicios no está limitado por el programa. Por ejemplo, Deep Freeze no impide la instalación de aplicaciones,
 
un usuario podría instalar una versión modificada de un navegador Web (pero aparentemente inocuos para el usuario sin saberlo) diseñado para enviar en secreto
 
las contraseñas de los usuarios a un servidor conectado a Internet. Como solución alternativa, Deep Freeze puede ser configurado para reiniciar después de cerrar la sesión
 
de usuario, apagado después de un periodo de inactividad seleccionado, o el reinicio / apagado a una hora programada en un intento por garantizar que dichas
 
instalaciones no se mantienen (como reiniciar el sistema vuelve al sistema a su estado original, sin modificar el estado).
 
 
 
Deep Freeze no puede proteger el sistema operativo y el disco duro en el que se instala si el equipo se inicia desde otro medio (como un disco duro externo, un dispositivo USB,
 
discos ópticos, o un servidor de red), o cuando el volumen total de unidades en el sistema tiene más de 2 TB [2]. En tales casos, un usuario tendría acceso real al contenido
 
de la (supuestamente) el sistema congelado. En un equipo basado en Windows, este escenario puede evitarse mediante la configuración del CMOS en la estación de trabajo sólo para
 
arrancar el disco duro que debe protegerse y para entonces la contraseña que protege el CMOS. Esta es una precaución normal
 
para la mayoría de los ordenadores de acceso público.
 
 
 
Otras características de Deep Freeze incluyen:
 
 
 
* Consola de Administración
 
* Protección de uno o varios discos / particiones.
 
* Creación de hasta 15 Password permanentes
 
* Passwords de una sola vez (One time password ? OTP)
 
* Más de 100 GB de espacio descongelado (Thawed: areas del disco que no se protegen).
 
* Posibilidad de elegir particiones del disco descongeladas.
 
* Modo silencioso
 
* Reinicios o apagados programados.
 
* Tiempos de mantenimiento programados.
 
* Compatible con actualizaciones de Windows y Servidores SUS.
 
* Ejecución de procesos Batch en tiempos de mantenimiento.
 
* Protección de password mediante cifrado.
 
* Compatible con redes LAN y WAN
 
* CMOS protegida.
 
* MBR (Master Boot Record) protegido.

VERSION 7




Deep Freeze Enterprise es una novedosa herramienta que protege instantáneamente las configuraciones de un sistema operativo. No importa qué cambios le hagas luego, simplemente reinicia el ordenador y volverás a tenerlo en su estado original.
 
Olvídate de sistemas corruptos, ataques de virus, pérdidas de datos, y otros problemas graves, siempre estarás seguro de que tu sistema funcionará tal y como lo dejaste la última vez.
 
Entre sus múltiples posibilidades, Deep Freeze Standard permite la protección por password, protege el CMOS, el Master Boot Record, soporta múltiples discos duros y particiones, soporta SCSI, ATA, SATA, IDE y es compatible con FAT, FAT32 y NTFS.
 
Deep Freeze no protege contra los reinicios con disquetes de arranque o unidades de CD-ROM. Es recomendable configurar el CMOS para evitar el inicio del equipo con disquetes de arranque o unidades de CD-ROM y el CMOS debe ser protegido con una contraseña.
 
Deep Freeze Enterprise es ideal para grandes entornos informáticos.
 
 
 
 
 
CARACTERISTICAS
 
El Deep Freeze de Faronics es ampliamente considerado como el líder la industria del software del tipo ?reinicie y restaure?. El principal objetivo de Deep Freeze es que, mientras esté instalado, la computadora estará ?congelada?. Cualquier cambio que se le realice a una computadora ?congelada? será revertido al reiniciar la PC, o sea que el disco duro automáticamente revertirá a su imagen creada durante el último estado descongelado (Thawed).
 
Cuando una computadora está ?descongelada? (Thawed), cualquier cambio que se le realice se mantendrá, a menos que sea manualmente revertido. Los usuarios también pueden crear una ?partición descongelada?. En ella, los datos pueden ser guardados y no se eliminarán al resetear la máquina.
 
Esencialmente, Deep Freeze restaura el Disco duro de una computadora al estado inicial del sistema operativo (determinado por el administrador) con cada reseteo.

Consola de Administración

USO Y FUNCIONAMIENTO
 
Funcionamiento
 
La protección funciona desde que el equipo inicia y antes de que empiece a cargar el sistema operativo, razón por la cual protege el equipo de cualquier virus, incluso los que atacan el MBR y las particiones NTFS, incluso si el equipo se ha iniciado en Modo Seguro.
 
Uso
 
Faronics diseñó Deep Freeze enfocado en cuatro mercados específicos:
 
Educación ? incluyendo Escuelas, Universidades, etc.
 
Gobierno
 
Bibliotecas ? incluyendo las bibliotecas públicas y privadas, así como también otros computadores de acceso público.
 
Salud ? útil para Hospitales, Centros de Enfermería, etc.
 
Otro uso muy conocido de este software es en Cibercafés, donde cada vez que un usuario deja una pc ésta se resetea y vuelve a estar en el estado inicial, para que otro usuario pueda utilizarla en su óptimo estado de rendimiento.
 
REQUERIMIENTOS DEL SISTEMA
 
Windows
 
Es totalmente compatible con Windows XP, Vista y Windows 7. Deep Freeze para Windows es compatible con múltiples formatos de disco duro, así como con entornos de arranque múltiple. Programa períodos de Mantenimiento Descongelado para la ejecución de actualizaciones.
 
Server
 
Es compatible con Windows Server 2008 R2. Utilice el Control de la Línea de Comandos para gestionar la implementación de Deep Freeze de forma remota, y User Server Seed para la comunicación e instalación.

BlueLife KeyFreeze v1.3





BlueLife KeyFreeze es un programa gratuito para Windows, que nos permite bloquear el teclado y el ratón, con un solo clic, de tal manera que solo podrá utilizarse hasta que el propio usuario lo desbloquee.
 
Esta característica puede ser de gran utilidad para evitar que los niños pequeños jueguen con el teclado y el ratón, mientras el usuario esta ausente del ordenador. En este caso no es necesario apagar el PC o bloquear la pantalla, basta solo con ejecutar la aplicación para evitar que se puedan utilizar estos dos elementos.
 
La interfaz del programa es simple e intuitiva; solo consiste en un botón en color rojo que se debe oprimir para bloquear el teclado y el ratón, mientras que para desbloquearlos solo basta con pulsar CTR+ALT+DEL.
