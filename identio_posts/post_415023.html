Programar el apagado de tu PC sin programas

  • Categoría: Hazlo tu mismo
  • Publicado hace más de 9 años
Bienvenidos y Gracias por pasar 



   

Muchas veces en mi caso he querido programar el apagado del PC pero los programas que he probado son un verdadero cacho, así que comencé a usar el CMD de Windows 7 y mediante comandos programar el apagado.

Lo incomodo de esto es en la noche (cuando el sueño no me deja razonar)....


Bueno al grano lo primero que haremos sera abrir el bloc de notas

y escribimos el siguiente código:



@echo off
echo Indique tiempo de apagado del equipo  
echo. 
set /p tiempo= Minutos: 
cls
echo el equipo se apagara en %tiempo% minutos
set /a tiempo=tiempo*60
shutdown /s /t %tiempo%
echo.
echo desea cancelar?
echo        1.Si   2.No
set /p cancelar=   
if %cancelar%==1 shutdown /a
exit



para los entendidos, notaran que es un código bastante básico.


Finalmente guardamos el archivo como un archivo  bat, el archivo quedaría con un nombre así "apagado.bat" lo ejecutamos y listo, ya tenemos un archivo propio y seguro para apagar la pc.


eso es todo, espero les sirva
