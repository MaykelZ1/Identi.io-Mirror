VMware Fusion 7 Pro FUll

  • Categoría: Mac
  • Publicado hace más de 11 años
Con el lanzamiento de OS X Yosemite a la vuelta de la esquina (ayer se lanzó la developer preview 7) todo el software para Mac se apresura en ser compatible con el nuevo sistema operativo. Las máquinas virtuales no iban a ser menos, ya que todo su negocio se basa en ser compatibles con los sistemas operativos y si hace unas semanas fue Parallels hoy tenemos la actualización de VMware Fusion 7 con compatibilidad para Yosemite y Windows 8.1.


Esta nueva versión del conocido software de máquinas virtuales es compatible al cien por cien con Yosemite, permitiéndonos ejecutar cualquier aplicación Windows en nuestro Mac sin necesidad de instalar el sistema operativo de Microsoft. Como añadido, VMware Fusion 7 es capaz de virtualizar Windows 8.1 de manera casi nativa, es decir, que las aplicaciones virtualizadas son capaces de utilizar casi toda la potencia de la máquina en la que se está ejecutando. Y si tenemos un Mac con retina display lo veremos todo mucho mejor, ya que esta nueva versión optimiza Windows para las pantallas retina.


Junto al lanzamiento de VMware Fusion 7 también se ha lanzado la versión pro, la cual tiene las mismas características pero añade una mejor optimización para las GPU, redes virtuales y compatibilidad con más máquinas virtuales a la vez.
