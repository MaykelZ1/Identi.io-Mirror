Multisystem
Instalar varias distros en un pendrive y elegir con cual queremos iniciar



Los usuarios GNU/Linux, no sé si todos, pero muchos de nosotros sufrimos de "versionitis", extraño mal del cuál no se conoce una cura definitiva, y que se manifiesta haciendo que cada tanto nos mudemos de distro, o necesitemos probar ésta o aquella nueva versión.
Hace un par de años, cansado de formatear unos viejos pendrives de 2 GB para instalar y bootear una versión diferente de linux, me topé con este programa, que me permitió tener todas las distros que quisiera, así como alguna que otra ISO con utilitarios , tipo Hiren's boot cd y las versiones de windows que quisiera y poder elegir al momento de bootear con cuál quiero iniciar el equipo. La única limitación es el tamaño del pendrive, obviamente a mayor capacidad, más versiones podremos agrupar en un sólo pen.
Multisystem es una herramienta que nos permite crear un medio de instalación usb para varias distros, optando por la que queremos iniciar. Además en aquellas distros que lo permitan se puede activar la persistencia lo que hace que se mantengan los datos y modificaciones que hagamos.

Para descargar e instalar Multisystem tenemos dos caminos
Script de instalación, el cuál les dejo en la descarga, y al que deberemos descomprimir y en la carpeta donde lo hagamos, abriremos una terminal y ejecutamos:


sudo ./install-depot-multisystem.sh

O podemos en derivadas Debian ( Ubuntu, Mint, Elementary, Deepin, etc)

Agregamos el repositorio

sudo apt-add-repository 'deb http://liveusb.info/multisystem/depot all main'

Importamos la clave de autenticación

wget -q -O - http://liveusb.info/multisystem/depot/multisystem.asc
sudo apt-key add -

Recargamos

sudo apt-get update

Instalamos

sudo apt-get install multisystem

Este último paso, sólo si usan DEBIAN
Los usuarios Debian deben agregar su $USER al grupo adm

sudo /usr/sbin/usermod -a -G adm "$SUDO_USER"

Una vez instalado, lo iniciamos

Ahora con Hiren's Boot CD también agregado, vamos a probarlo
Multisystem tiene incorporada la posibilidad de probar nuestro pendrive, sin necesidad de reiniciar la máquina y bootear desde él.
Para ello incluye Qemu y VirtualBox y podemos optar por uno o otro



Vamos a iniciar Hiren's Boot CD con Qemu y utilizar el mini windows XP
Booteando







MultiSystem hará de tu pendrive USB una verdadera navaja suiza ya que te permitirá crear de manera rápida un Live USB Multisistemas. Dicho en pocas palabras: tendrás un USB de arranque en donde la única limitante es la misma capacidad del propio USB.

Multisystem tiene otras opciones como la de actualizar o cambiar el gestor de arranque, formatear el pendrive, añadir el modo de persistencia o descargar imagenes iso. A mi la integración con máquina virtual no me funcionó. Otra opción es la de lanzar el usb desde un cd (muy útil en equipos viejos) o desde el lanzador de Windows

También posee una lista de distros, con la posibilidad de cliquear en cada nombre y llevarte a la descarga de la ISO
