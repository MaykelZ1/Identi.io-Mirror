Ghost Windows Xp universal










En menos de 10 minutos (dependiendo de tu hardware) tendras instaldo el Windows Xp Service Pack 3 + Drivers + software basico
Es una imagen Universal , un clon de Xp que se puede instalar en cualquier Pc o Lap top sin importar el Hardware



Que tal amigos , es mi primer post, el "backup" no es mio, lo hizo Tuxsis, pero el autor (que tiene un blog) de pronto desapareció  y los links se cayeron hace mucho tiempo y ya nunca respondio preguntas ni los resubio.

En esta publicación les quiero compartir algo que no he encontrado casí en la web.
Se trata de un Windows Xp universal para clonar de forma desatendida.
Le llamé: Recovery Ghost Tuxsis V1

Es decir, NO ES UN DISCO COMUN, ES UNA IMAGEN QUE SE PEGA JUNTO CON EL PROGRAMA ONE KEY EN UNA PARTICION  para restaurar Windows Xp en cualquier computadora (Sin importar el hardware). Ya no necesitarás perder tiempo reinstalando todo el software y configuraciones. En menos de 10 minutos tendrás tu sistema listo para usar con el software y configuraciones necesarios
Si te dedicas a armar y vender equipo de cómputo; dar mantenimiento a computadoras...etc.
Podrás clonar con este disco en cualquier computadora en menos de 10 minutos, con programas básicos y configuraciones especiales para mejor rendimiento.



¿Qué contiene esta imagen universal de Ghost?
*Windows Xp sp3 actualizado 2011
*100% actualizable,
*Versión Lite. (reducida para mejor rendimiento)


Detalles de soporte:
*SATA: Sí.
* Soporte RAID: Sí.
* Soporte SCSI: Sí.
* Auto controladores: Sí (Skydriver XP V3 
* Internet Explorer 8 Final: Sí
* Windows Media Player 11
* Revisiones: Sí (Actualización 15/03/2011)
* Actualización en línea: Sí.


Aplicaciones:
Ccleaner 3.6
Winrar 3.9
Mozilla 3.6
Foxit Reader
Microsoft Office 2003
Dvd Shrink
Microsoft silverlight
MozBackup
Usb disk security
Flash Player

Detalles de la imagen ISO de ghost universal Xp.

*Archivos comprimidos y cortados con HJ SPLIT
*Sistema NTFS
*Tamaño de la imagen 699 MB.
********************************************************************************************************************************
instalacion:

Pegar la imagen Backup viejas .gho y el programa ONE Key.exe en una particion que no sea donde esta el Siste Operativo (por ejemplo disco local D) y ejecutan el one key y dan click en restaurar y luego en yes es todo
- See more at: http://www.identi.li/index.php?topic=138785#sthash.dsQCX52J.dpuf

Recomendaciones Previas   !!!!!!! LEE ANTES DE INSTALAR!!!!!!!

*Realizar un respaldo de tu sistema operativo o minimamente de tus archivos importantes.
*No instalar si se requiere de un controlador sata para el disco duro, ya que generara un error y se quedara en un bug eterno al cargar la pantalla de inicio de windows despues de toda la instalacion.
*Particionar el disco duro o tener otra unidad en la cual copiar la imagen .GHO osea el archivo y reinstalar el sistema con onekey para que sea mas facil el proceso(automatico, con solo unos clics) y sobre todo mas rapido.
*Si despues de restaurar la imagen del sistema tienen el error, de que falta archivo NTLDR se soluciona  facil.Tienen que iniciar o bootear su computadora con hirens cd y con el programa boot ice que viene dentro de este disco.
BootIce viene en el Hirens 14 y 13 esta herramienta creo que tambien esta en versiones mas antiguas y actuales como lo menciono.
Para solucionar el problema bootean con el hirens, ahora en el menu elegimos la opcion de MiniWindows xp y esperan a que cargue el programa buscan el programa BootIce en los programas del hirens, en el menu particion/boot/mbr y es ahi donde encuentran al programa BootIce
cuando cargue el programa tienen que seleccionar la unidad donde se restauro la imagen "DESTINATION DISK" si solo tienen un disco aparecera por default pero si tienen mas discos o particionen asegurense de elegir la particion o disco donde se restauro la imagen una vez seleccionada la particion o disco duro correcto le daremos un clic en el boton PROCESS MBR despues aparecera una nueva ventana que se llama MASTER BOOT RECORD, mucho cuidado con esto que es donde esta la solucion y deben de poner la opci?
