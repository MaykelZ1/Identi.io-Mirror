GiliSoft USB Lock 5.1.0 [Full]

  • Categoría: Reciclaje
  • Publicado hace más de 10 años

 USB Lock es una herramienta de prevención de fuga de datos que impide fugas y copia de sus datos en unidades USB, discos externos, CD / DVD u otros dispositivos portátiles. Una vez instalado, USB Lock le permite bloquear todas las unidades y dispositivos que no pertenecen a usted. Con bloqueo de USB, puede compartir su PC con cualquier persona sin temor a theft.This de datos es una herramienta fácil de Software Secure USB y poderosa suite de Endpoint DLP que le ayuda a bloquear el puerto USB, grabadora de DVD / CD de sólo lectura, bloquear algunos sitios web, prohibir a algunos programas y desactivar varios dispositivos.

Características y beneficios:

Memorias USB Block
Desactivar la lectura desde discos USB o desactivar la escritura en discos USB. Bloqueo USB no permite ningún tipo de unidad USB para acceder a su equipo a menos que usted lo autorice. Por defecto, todos los tipos de unidades USB se bloquean incluyendo duros externos, FireWire, Enhanced mini-USB, Host Controller Interface (HCI), HP-IL, COM, LPT, IrDA, USB on-the-go, U3, EHCI, RAID Controller, adaptador Host, Cable serie (uso con transferencia de datos), Serial ATA, ACCESS.bus y cualquier dispositivo de almacenamiento que está conectado al puerto USB que muestra una unidad en el sistema.

CD Lock, Bloque Medios y discos Blu-ray
Desactivar pueden leer discos DVD / CD o hacer grabadora de DVD / CD de sólo lectura. La aplicación también bloquea cualquier disco que utiliza el cubo del disco, bahía, combo o unidad de CD / DVD y asigna una letra de unidad, por ejemplo; CD-R, CD-RW, CD-RAM, DVD-R, DVD-RW, DVD-RAM, HD-R, HD-RW, HD-RAM, Blu Ray-R, Blu Ray-RW, Blu Ray-RAM , disquete A, B disquete y unidades Zip.

Bloqueo Sitio Web
Bloquear el acceso a algunos sitios web. Esta utilidad le permite bloquear sitios web no deseados de la pantalla en Internet Explorer. Si un sitio web está bloqueado el usuario es enviado a una página en blanco o una "página bloqueada" y el contenido de la página original no se cargan en el PC. Detener la carga de banners y anuncios, deje a sus hijos de pasar horas en salas de chat o eliminar sitios web no deseados desde su punto de vista. Evite que sus hijos tengan acceso a ciertos contenidos sitios web tales como los sitios para adultos y juegos de azar.

Programas Lock
Bloquear correr cualquier programa, incluyendo Internet Explorer, Outlook, AOL, AIM, y más. Usted puede incluso bloquear el panel de control con un solo clic de botón. Otras características incluyen la capacidad de elegir su propio mensaje para mostrar si alguien intenta ejecutar uno de sus programas bloqueados, y también incluye la protección de contraseña de modo que sólo se puede desbloquear el programa cuando esté listo para usarlo. Permite que todo el equipo se mantenga activo y funcionando, y solo cerraduras e impide el acceso a los programas que usted especifique. Fácil de usar interfaz que muestra el estado actual de todos los programas bloqueados, y te permite hacer clic con facilidad y desbloquear como desee

dispositivos de bloqueo
El programa puede ser utilizado para restringir leída o escribe el acceso a los dispositivos de medios extraíbles como CD, DVD, disquete, flash y unidades USB. También se puede utilizar para desactivar la impresora, Moderna, com puertos LPT, infrarrojos, bluetooth, 1394.

Notificación de Alarma
Si el usuario ingresa una contraseña incorrecta más de 5 veces, enviará notificaciones de alarma a su e-mail.

Copy Protection
El programa utiliza un nivel avanzado de la tecnología de prevención de fuga de datos que no permite la duplicación de sus archivos importantes y material de derecho de autor a cualquier unidad USB u otros dispositivos de almacenamiento tales sin su permiso. El programa funciona mediante el bloqueo de todos los tipos de dispositivos de almacenamiento no autorizados, como unidades USB, discos externos, CDs, DVDs, etc., de esta manera se evita el plagio, la piratería, la distribución y la copia ilegal de sus datos



Prevención de Fuga de Datos
USB Lock es un software de prevención de fuga de datos. Evita que tus datos de conseguir trascendido a unidades USB y otros dispositivos de almacenamiento, ya que permite controlar qué dispositivos pueden acceder a su ordenador mientras que bloquea todos los otros dispositivos no autorizados que no le pertenecen a usted. Con bloqueo de USB instalado en su computadora, usted puede sentirse seguro de que sus datos se mantendrán en su PC a salvo y seguro.
