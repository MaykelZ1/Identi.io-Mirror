Smoke Patch  release for pes 2018




Smoke Patch X15 (10.1.5) release for PES 2018, please read the patch information before downloading or installing smoke patch.

This version have a reworked system files to expand the patch, it is important to start a NEW MASTER LEAGUE and BAL after installing this version, future versions will be based on this system files so no need to start new after this version.


X15 General Features:
- includes konami PD2
- optional offline patch 1.03
- all teams have correct names, logos and managers
- all teams have correct stadium names
- correct players names for all teams
- removed fake and doubled players
- correct team squads
- added new leagues
- added new teams and nationals
- added new players
- add real kits for all teams and referees
- updated boots
- updated gloves
- updated balls
- a lot of updates and enhancements


Main changes in this version:
- updated boots
- updated balls
- updated gloves
- updated logos
- added full Liga Aguila instead of PLA league
- added 55 new teams
- added 8 new national teams
- aligned all teams ID with konami
- all teams have correct stadium names
- all teams have correct rivalries
- other fixes and updates


Updated graphics:
this version have a major graphics updates, including players boots and gloves, league balls and emblems, and a lot of other enhancements.


Added teams attributes:
in this version we worked on the edit file details, now all the teams have correct stadium names, and all revelries are correctly assigned.

Installation:

- Extract patch files 
download all parts and be sure all are completed, extract only part one the rest will be extracted automatically, don't move any of the installation files or folders.

- Run SetupsmokeX15.exe
run the installer and choose PES 2018 installation folder, the patch must be installed in the same game folder, be sure you don't get errors and that the installer finished successfully.


NOTES:
- new ML and BAL modes are required after installing patch X15
- no previous versions required, only the game and this version.
- do not use old edit00000 files (using older or other edit files will cause bugs)
- this patch is made to focus on offline modes, online feature will not find opponents due the difference between the patch system files and the original game.
