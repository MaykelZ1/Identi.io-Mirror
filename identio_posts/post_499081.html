.NET Framework 3.5 Windows 10 Standalone Offline

  • Categoría: Software / Programas
  • Publicado hace más de 7 años

Cuando usamos Windows 10 en algún momento queremos utilizar algun programa o juego que nos requiere .NET Framework 3.5 y nos rompemos la cabeza queriendolo instalar por que Windows Update se tarda mucho en descargarlo e instalarlo, y recurrimos a querer descargarlo, pero sorpresa Microsoft no permite la descarga standalone (individual y offline) de esta utilería y nos llega a frustrar esta situación, no obstante buscamos y buscamos por internet, y solo hay scripts para instalarlo desde USB o un medio de instalación de Windows 10.
Funciona con instalaciones de Windows 10 de 32 y 64 bits, si tienen algún problema avísenme
Este artículo pone fin a todos esos problemas con una descarga única y un video tutorial de como se instala.
