Liberar Memoria RAM Automáticamente
[Windows 7, 8, 8.1, 10 y XP]












Hola a todos, esta vez les traigo un software pequeño, invisible y súper práctico.


Clean Memory es un software capas de refrescar o actualizar el uso de memoria ram de las distintas aplicaciones que tengamos en funcionamiento en nuestra pc, utilizando la función propia del sistema.

Clean Memory se ejecuta de forma automática (cada 15 minutos de forma predeterminada) liberando en cuanto sea posible la memoria ram reservada para procesos en segundo plano y sobre todo para los navegadores.



Como funciona?


No es ningún secreto que muchos programas usan y abusan de nuestra memoria ram, un ejemplo claro es Mozilla Firefox (Google Chrome no se escapa), por cada pestaña que tengas abierta en el navegador consume mas o menos de 40mb hasta 250mb de ram extra, y si tenes muchas extenciones más aun.  


Los programas utilizan la memoria ram, que es muchísimo mas rápido para leer y escribir datos que un disco duro, para realizar sus cálculos y procesos internos (por eso no se puede tener PC sin memoria ram), cada clic que hacemos en el navegador implica un calculo interno, es decir, RAM.


Lo que sucede es que los datos de los cálculos realizados se van acumulando en la memoria (por si el programa los vuelve a requerir), pero normalmente el programa no los utiliza, sino que cada vez realiza nuevos cálculos cuando es necesario... y se van acumulando...y solo se borran al cerrar el programa, lo que hace Clean Memory es borrar todos esos datos que ya se utilizaron de la memoria ram mientras el programa sigue ejecutándose, OJO "pero no borra ningún dato que esta en uso", solamente se deshace de los datos que no sirven, evitando así la sobrecarga de nuestra memoria ram, sobre todo si es de 1gbmb o menor.



*Funciona en todas las máquinas




El proceso es sencillo
1-) Instalan
2-) Listo..!



Pasos para configurar los intervalos de ejecución:
(Si quieren que la memoria ram se libere cada 1 minuto o cada 2 horas)
1- Van a Panel de Control > Programar tareas
2- Seleccionan y hacen doble clic en Clean System Memory
3- Pestaña Desencadenadores > doble clic a Al iniciar el sistema
4- Y cambian el valor de Repetir cada 15 minutos por el tiempo que deseen, desde 1 minuto hasta varias horas.
