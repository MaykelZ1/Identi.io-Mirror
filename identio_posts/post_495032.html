poner letras a canciones - Lyrics Finder

  • Categoría: Software / Programas
  • Publicado hace más de 7 años
NOTA: ESTO NO ES IMPORTANTE PARA EL POST PERO ME GUSTA PONER UN POQUITO DE LO QUE PASO CUANDO ENCONTRÉ EL PROGRAMA


bueno, un mal dia se me perdio mi celular, por lo que recurri a mi viejo ipod para seguir escuchando musica, pero de momentos queria cantarlas o ver como eran, cosa que en mi android podia hacerlo por medio de una aplicacion que me descargaba automaticamente las canciones que yo tubiera, pero en ipod, nop, note que habia algo para visualizar letras pero, aunque uno consiga las canciones legitimamente, estas no siempre (casi nunca) tienen incorporadas las letras en los metadatos (eso lo explico mas adelante), asi que desde itunes podia ponerlas pero tenia que hacerlo uno por uno, buscar y pegar, y como mi libreria cuenta con mas de 3500 canciones, asi que... no que flojera, me puse a buscar si habia algo que pudiera usar en mi ipod, pero no, sin embargo encontre un programita que haciaalgo parecido a lo que necesitaba.

asi que decidi compartir esto con ustedes


Lyrics Finder es nuestra herramienta!
(disponible para windows mac y linux)


desarrollada por MEDIAHUMAN, este es un programa gratuito asi que honor a quien honor merece






Requisitos del sistema:Windows 10, Windows 8, Windows 7, Windows Vista, Windows 2003, Windows XP para sistemas de PC (32 bit/64 bit)macOS 10.13 (High Sierra), macOS 10.12 (Sierra), OS X 10.11 (El Capitan), OS X 10.10(Yosemite), OS X 10.9 (Mavericks), OS X 10.8 (Mountain Lion), Mac OS X 10.7 (Lion), Mac OS X 10.6 (Snow Leopard)Alrededor de 40 Mb de espacio libre en el disco duro




como se usa

arrastra una cancion o una carpeta completa a la zona en blanco del programa




el programa hace todo en automatico, si las canciones ya tenia la letra, esta no se reemplazara 


la interfaz es facil de entender, del lado izquierdo, si el cuadro tiene un punto verde, es que encontro la letra, si no aparece es por el el programa, bueno, no la encontró, pero te da la opcion de que la busques manualmente presionando el icono del plantea, te abrira el navegador, o una pestaña nueva con los datos a buscar, que hay que tener en cuenta eso, si el artista o el titulo estan erroneos es puede que sea el motivo por el cual no las encontro, (en este ejemplo, bueno ni haciendo una busqueda manual pude encontrarlas, pero solo me paso con este grupo, con los demás no tuve ningun problema)

el icono de en medio es para volver a buscar, y el de el lapiz para escribir manualmente




el programa trabaja de una manera muy rapida, me sorpendio que tan rapido fue, duraba segundos en terminar el proceso

esto es muy util si usas un reproductor que pueda mostrar las letras o lyrics (como algunos prefieren llamarle) que esten guardadas en los metadatos, 

que son los metadatos? para resumir de forma simple, es esa informacion que tienen los archivos para describir algunos detalles, como por ejemplo la fecha de creacion, y de ultima modificacion, siendo de los mas comun de ver, en el caso de los archivos de audio, pueden contener los datos de compositor, album, año, grupo, genero, portada y el que nos interesa en este caso, la letra


en mi caso yo uso un ipod touch, su reproductor permite ver las letras guardadas de esta forma, el que uso en android tambien hace eso,  pero prefería usar una aplicacion para esto xD
