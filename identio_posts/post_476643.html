ESET Endpoint Security
6.5.2107.1
[x86 - x64]

Protección antimalware
Elimina todo tipo de amenazas tales como virus, rootkits, gusanos y software espía. Detecta las amenazas especialmente dirigidas para Mac OS y Windows para proteger los entornos multiplataforma.

Protección contra páginas fraudulentas
Te protege contra los intentos de robo de contraseñas, información bancaria y otros datos por páginas web fraudulentas que se hacen pasar por legítimas.

Bloqueo de exploits
Protege los sistemas más vulnerados como navegadores web, lectores de documentos PDF, Java, clientes de correo electrónico y componentes de MS Office.

Control de acceso a la información
Bloquea los dispositivos no autorizados (CDs/DVDs y dispositivos con puerto USB) en tu sistema. Permite el filtro de páginas web con más de 140 categorías o reglas personalizables. Permite aplicar políticas para permitir que los dispositivos extraíbles accedan al sistema y así reducir el riesgo de infección por malware y fugas de información.

Antispam
Filtra el correo no deseado y analiza el correo entrante en busca de amenazas. Es totalmente compatible con Microsoft Outlook (POP3, IMAP, MAPI).

Sistema de prevención de intrusiones (HIPS)
Monitoriza los puntos de entrada al sistema y ciertas actividades del mismo y bloquea los intentos de intrusión sospechosos.

Análisis avanzado de memoria
Mejora la detección de APTs (Amenazas persistentes avanzadas) que utilizan múltiples capas de cifrado para llevar a cabo su actividad.

Herramientas integradas de diagnóstico y recuperación
Proporciona snaphsots de diagnóstico profundo de las características clave del sistema, desde procesos a claves de registro. Permite a tu empresa utilizar un medio de recuperación de arranque en caso de fallo del sistema.

Protección completa para equipos Windows

Requisitos del sistema:
Sistemas operativos: Microsoft Windows® 10, 8.1, 8, 7, Vista, XP, 2000
Compatible con el nuevo ESET Remote Administrator 6  
Nota: No es compatible con las versiones anteriores de ESET Remote Administrator.
