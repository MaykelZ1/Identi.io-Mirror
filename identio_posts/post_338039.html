*Requisitos del sistema:
- Requisitos mínimos del sistema para Nero 2014 Platinum
- Windows® XP SP3 (32 bits), Windows Vista® SP2 o posterior (32/64 bits), Windows® 7 SP1 Home Premium, Professional o Ultimate (32/64 bits), Windows® 8 (32/64 bits), Windows® 8.1 (32/64 bits)
- Procesador Intel® o AMD de 2 GHz.
- 1 GB de RAM
- 5 GB de espacio en el disco duro para una instalación típica de todos los componentes (incluidas las plantillas, el contenido y el espacio temporal en disco)
- Tarjeta gráfica compatible con Microsoft® DirectX® 9.0.
- Unidad de DVD para instalación y reproducción.
- Unidad de CD, DVD o Blu-ray grabable o regrabable para grabación
- Windows Media® Player 9 o posterior.
- Los componentes de terceros, como Microsoft Windows® Installer 4.5, Microsoft .NET® 4, Microsoft® DirectX® o Adobe Flash se suministran con el producto o se descargan de forma automática si no se incluyen en el paquete.
- Se precisa conexión a Internet para determinados servicios. Los costes de conexión a Internet son responsabilidad del usuario.
- La reproducción de Blu-ray Disc, Blu-ray 3D Disc y la edición de vídeos HD requieren Windows Vista® SP2 (32/64 bits), Windows® 7 SP1 Home Premium, Professional o Ultimate (32/64 bits), Windows® 8, (32/64 bits), Windows® 8.1 (32/64 bits)
- La edición Ultra HD (4K) requiere sistemas operativos de 64 bits: Windows® 7 SP1 Home Premium, Professional o Ultimate, Windows® 8, Windows® 8.1
- Para consultar otros requisitos, consulte www.nero.com.
Nota:
- En los sistemas operativos de 64 bits, las aplicaciones se ejecutan en el modo de simulación de 32 bits.
- El sistema operativo debe instalar y reconocer correctamente los dispositivos para que el producto funcione correctamente.
- Se recomienda encarecidamente llevar a cabo la instalación de los controladores de dispositivos más recientes certificados por WHQL
- La configuración del servidor de acceso multimedia requiere un PC conectado a una red local.
